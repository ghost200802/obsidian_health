# 朋友圈素材池模板

## 定义

朋友圈素材池不是文案库，而是“原料库”。它既存放客户原话、真实场景、常见卡点和你的专业判断，也存放能建立你个人品牌印象的内容，供[[朋友圈内容运营工作流]]和[[朋友圈内容运营]]反复调用。

## 使用原则

- 一条素材只记一个问题，不要一条里塞满多个主题
- 优先记录客户原话，而不是先写成漂亮文案
- 先记场景和判断，再决定是否值得发朋友圈
- 如果连“这条是说给谁听的”都说不清，先不要入池
- 素材池至少保持两类内容：**专业内容** 和 **个人品牌内容**
- 个人品牌内容不等于随手发生活，而是能让人更理解你的工作方式、价值观和审美取向
- 所有内容必须去标识化，不能保留真实姓名、单位、体检图或聊天截图

## 素材来源优先级

1. [[客户复盘与随访]]中的高频卡点
2. [[1对1营养咨询]]中的客户原话
3. [[减脂案例方法库]]中的去标识化案例模式
4. 体检报告解读、复查和指标变化中的高频问题
5. 线下活动、读书会、深度对话后的真实反馈
6. 你的阅读、工作观察、生活方式和价值观表达
7. 官方指南和已有 wiki 页面中的专业判断边界

## 字段说明

- **日期**：记录素材进入池子的日期
- **来源**：复盘、咨询、案例、体检解读、指南、个人观察、生活片段
- **关系基础**：客户、老客户、读书会、老同学、弱关系、沉睡联系人等
- **人群**：减脂需求、体检指标异常、慢病风险、泛健康关注者等
- **内容轨道**：专业内容 / 个人品牌内容
- **场景/原话**：尽量写接近客户真实表达的话
- **触发点**：加班、熬夜、外卖、应酬、周末松掉等
- **我的判断**：你对问题本质的判断
- **证据/支撑**：案例经验、已有 wiki 页面或官方指南
- **可写标题**：把这条素材转成朋友圈标题的第一版
- **内容类型**：唤醒 / 专业 / 证据 / 品牌 / 转化
- **承接动作**：私聊、体检解读、1对1咨询、长期管理、仅唤醒
- **优先级**：高 / 中 / 低
- **状态**：未写 / 已排期 / 已发布 / 已沉淀

## 空白模板

```md
# 朋友圈素材池

## 使用说明
- 每次咨询或复盘后补 1-3 条即可
- 优先记录“客户怎么说”与“我怎么判断”
- 同时保持“专业内容”和“个人品牌内容”都在持续积累
- 优先记录那些能明确指向某类人群和某个可服务问题的素材
- 每周从这里挑 3-4 条，进入[[朋友圈每周选题板模板]]
```
