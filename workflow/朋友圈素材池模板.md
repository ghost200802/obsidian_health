# 朋友圈素材池模板

## 定义

朋友圈素材池不是文案库，而是“原料库”。它优先存放客户原话、真实场景、常见卡点和你的专业判断，供[[朋友圈内容运营工作流]]和[[朋友圈内容运营]]反复调用。

## 使用原则

- 一条素材只记一个问题，不要一条里塞满多个主题
- 优先记录客户原话，而不是先写成漂亮文案
- 先记场景和判断，再决定是否值得发朋友圈
- 如果连“这条是说给谁听的”都说不清，先不要入池
- 生活感和 plog 只有在能回到某个服务问题时，才值得进素材池
- 所有内容必须去标识化，不能保留真实姓名、单位、体检图或聊天截图

## 素材来源优先级

1. [[客户复盘与随访]]中的高频卡点
2. [[1对1营养咨询]]中的客户原话
3. [[减脂案例方法库]]中的去标识化案例模式
4. 线下活动、读书会、深度对话后的真实反馈
5. 官方指南和已有 wiki 页面中的专业判断边界

## 字段说明

- **日期**：记录素材进入池子的日期
- **来源**：复盘、咨询、案例、指南、个人观察
- **关系基础**：客户、老客户、读书会、老同学、弱关系、沉睡联系人等
- **人群**：高压职场、情绪性进食、高敏感、慢病合并减脂等
- **场景/原话**：尽量写接近客户真实表达的话
- **触发点**：加班、熬夜、外卖、应酬、周末松掉等
- **我的判断**：你对问题本质的判断
- **证据/支撑**：案例经验、已有 wiki 页面或官方指南
- **可写标题**：把这条素材转成朋友圈标题的第一版
- **内容类型**：唤醒 / 专业 / 证据 / 转化
- **承接动作**：私聊、体检解读、1对1咨询、长期管理、仅唤醒
- **优先级**：高 / 中 / 低
- **状态**：未写 / 已排期 / 已发布 / 已沉淀

## 空白模板

```md
# 朋友圈素材池

## 使用说明
- 每次咨询或复盘后补 1-3 条即可
- 优先记录“客户怎么说”与“我怎么判断”
- 优先记录那些能明确指向某类人群和某个可服务问题的素材
```
