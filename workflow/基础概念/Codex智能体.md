# Codex

## 定义

Codex 是 OpenAI 推出的 Agent 工具，能够自主执行代码编写、文件操作等复杂任务。在 [[LLM-Wiki知识管理]] 方法论中，Codex 作为 Agent 执行引擎，负责自动完成 [[Ingest工作流]] 中的知识编译操作。

## 在 LLM Wiki 中的角色

- **知识编译执行者**：按照 AGENTS.md 中定义的规则，自动将原始资料编译为结构化知识
- **文件操作**：在 [[Wiki层]] 创建和更新知识页面
- **链接维护**：使用 Obsidian `[[]]` 双链语法建立知识网络
