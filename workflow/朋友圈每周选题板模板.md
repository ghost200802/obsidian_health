# 朋友圈每周选题板模板

## 定义

每周选题板的作用，是把[[朋友圈素材池模板]]里的原料排成一周 3-4 条可发布内容。它解决的是“这周到底发什么、先发哪条、每条承担什么作用”。

## 使用方式

- 每周固定一次排版，最好放在周日或周一早上
- 排题前先看一眼封面图和个性签名，确认它们和本周主轴没有跑偏
- 先定本周主轴，再从素材池挑 3-4 条，不要反过来
- 同一周尽量围绕一个人群和一个核心问题展开
- 每条内容只承担一个角色：唤醒、专业、证据或转化

## 本周计划模板

```md
# 朋友圈每周选题板

## 本周主轴
- 本周想反复出现的问题：
- 本周主要服务的人群：
- 本周希望别人记住我的一句话：
- 本周门面是否一致：封面 / 个签 / 最近几条朋友圈
- 本周轻转化目标：引导私聊 / 引导预约 / 维持存在感

## 发布节奏

| 日期 | 角色 | 面向谁 | 素材来源 | 标题角度 | 开头场景 | 核心判断 | 结尾动作 | 承接动作 | 状态 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 周一 | 唤醒 |  |  |  |  |  |  |  | 未写 |
| 周三 | 专业 |  |  |  |  |  |  |  | 未写 |
| 周五 | 证据 |  |  |  |  |  |  |  | 未写 |
| 周日 | 转化 |  |  |  |  |  |  |  | 可选 |

## 复盘
- 哪条引发了私聊：
- 哪条只有点赞没有咨询：
- 这周有没有让更多人知道我现在在做什么：
- 哪个表述最容易让人记住：
- 哪个问题值得回到[[朋友圈素材池模板]]继续积累：
```

## 示例

### 本周主轴

- 本周想反复出现的问题：高压工作日为什么更容易晚上失控
- 本周主要服务的人群：[[高压职场人群减脂]]
- 本周希望别人记住我的一句话：很多人减脂失败，不是不懂吃，而是工作日的节律撑不住
- 本周门面是否一致：封面和个签都在表达“高压工作也能执行的减脂支持”
- 本周轻转化目标：引导高压职场人私聊自己的卡点

### 发布节奏

| 日期 | 角色 | 面向谁 | 素材来源 | 标题角度 | 开头场景 | 核心判断 | 结尾动作 | 承接动作 | 状态 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 周一 | 唤醒 | 高压上班族 | 素材池第 1 条 | 下班后失控，往往不是你嘴馋 | “很多人回家第一件事不是吃饭，是回血。” | 这是疲惫和压力后的补偿性进食 | 邀请读者对照自己最容易失控的时段 | 私聊 | 未写 |
| 周三 | 专业 | 外卖减脂人群 | 素材池第 2 条 | 外卖减脂为什么总觉得饿 | “点轻食的人，很多时候不是吃多，而是吃不住。” | 结构和饱腹设计错误，比单纯热量更常见 | 给 1 个外卖结构建议 | 1对1咨询 | 未写 |
| 周五 | 证据 | 平台期人群 | 素材池第 3 条 | 体重没掉，不等于你这周白做了 | “有位客户体重没动，但夜间加餐明显少了。” | 执行质量改善通常早于体重变化 | 提醒先看趋势，不要急着推翻方案 | 体检解读 | 未写 |
| 周日 | 转化 | 高压工作减脂人群 | 个人观察 | 我通常怎么帮高压工作的人做减脂 | “我不会先给很满的食谱，而是先找工作日最容易掉线的点。” | [[1对1营养咨询]]更重要的是找障碍，不是发模板 | 提醒适合人群可私聊沟通 | 私聊 | 可选 |

## 排题检查

- 这一周的 3-4 条，是否围绕同一个主轴
- 是否至少有 1 条来自真实客户原话
- 是否至少有 1 条体现你的专业判断，而不是常识复述
- 是否至少有 1 条能让人感受到你做过真实案例
- 是否至少有 1 条让人更清楚你能解决什么问题
- 是否只有 1 条承担轻转化，不至于周周都像广告
