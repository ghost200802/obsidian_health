# 朋友圈每周选题板模板

## 定义

每周选题板的作用，是把[[朋友圈素材池模板]]里的原料排成一周 3-4 条可发布内容。它解决的是“这周到底发什么、先发哪条、每条承担什么作用”。

## 使用方式

- 每周固定一次排版，最好放在周日或周一早上
- 先定本周主轴，再从素材池挑 3-4 条，不要反过来
- 同一周尽量围绕【减脂需求】或【体检指标异常】中的一个主轴展开
- 每条内容只承担一个角色：唤醒、专业、证据、品牌或转化

## 本周计划模板

```md
# 朋友圈每周选题板

## 本周主轴
- 本周想反复出现的问题：
- 本周主要服务的人群：减脂需求 / 体检指标异常
- 本周希望别人记住我的一句话：
- 本周是否需要插入 1 条个人品牌内容：
- 本周轻转化目标：引导私聊 / 引导预约 / 维持存在感

## 发布节奏

| 日期 | 角色 | 面向谁 | 素材来源 | 标题角度 | 开头场景 | 核心判断 | 结尾动作 | 承接动作 | 状态 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 周一 | 唤醒 |  |  |  |  |  |  |  | 未写 |
| 周三 | 专业 |  |  |  |  |  |  |  | 未写 |
| 周五 | 证据 |  |  |  |  |  |  |  | 未写 |
| 周日 | 品牌/转化 |  |  |  |  |  |  |  | 可选 |

## 复盘
- 哪条引发了私聊：
- 哪条只有点赞没有咨询：
- 哪条个人品牌内容更容易让人记住我：
- 哪个表述最容易让人记住：
- 哪个问题值得回到[[朋友圈素材池模板]]继续积累：
```

## 示例

### 本周主轴

- 本周想反复出现的问题：体检单一堆箭头，到底先从哪里改
- 本周主要服务的人群：体检指标异常
- 本周希望别人记住我的一句话：很多异常指标，不是要你同时做十件事，而是先抓最关键的生活动作
- 本周是否需要插入 1 条个人品牌内容：需要
- 本周轻转化目标：引导体检解读和私聊咨询

### 发布节奏

| 日期 | 角色 | 面向谁 | 素材来源 | 标题角度 | 开头场景 | 核心判断 | 结尾动作 | 承接动作 | 状态 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 周一 | 唤醒 | 体检指标异常 | 素材池第 1 条 | 体检单一堆箭头，很多人第一反应就错了 | “看到甘油三酯、尿酸、转氨酶都高，很多人会直接慌。” | 大多数人不是缺更多知识，而是缺一个正确起手式 | 邀请读者回想自己体检后第一步做了什么 | 私聊 | 未写 |
| 周三 | 专业 | 体检指标异常 | 素材池第 2 条 | 指标异常，第一步先改什么 | “不是每个箭头都要单独救火。” | 先抓能联动改善体重、饮食结构和节律的动作，通常更有效 | 给 1 个起步建议 | 体检解读 | 未写 |
| 周五 | 证据 | 减脂需求 | 素材池第 3 条 | 体重和指标，有时是一起往回走的 | “有位客户体重先稳住后，甘油三酯和尿酸也开始松动。” | 结果改善往往来自持续执行，不是单点猛改 | 提醒先看趋势，不要急着换方案 | 1对1咨询 | 未写 |
| 周日 | 品牌 | 泛健康关注者 | 个人观察 | 为什么我做营养管理，总先看‘做不做得下去’ | “我不会先把方案写得很满，而是先判断这个人能不能长期执行。” | 这代表你的工作方法和价值观 | 留一句工作理念或轻互动 | 仅唤醒 | 可选 |

## 排题检查

- 这一周的 3-4 条，是否围绕同一个主轴
- 是否至少有 1 条来自真实客户原话
- 是否至少有 1 条体现你的专业判断，而不是常识复述
- 是否至少有 1 条能让人感受到你做过真实案例
- 是否至少有 1 条在强化你的个人品牌印象
- 是否至少有 1 条让人更清楚你能解决什么问题
- 是否只有 1 条承担轻转化，不至于周周都像广告
