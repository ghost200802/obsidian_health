# 体重管理指导原则（2024年版）

> 2024年发布的体重管理指导原则，涵盖体重分类标准、评估方法、干预治疗及特殊人群管理。

## 文献概要

本指导原则于2024年发布，全文30页，约809KB。文件系统规定了体重分类标准、体重评估的6个维度、干预治疗策略、健康体重维持方法，以及学龄前儿童、学龄儿童青少年、孕产妇、老年人群和职业人群等特殊人群的体重管理要求。是当前中国[[体重管理]]领域最新的权威指导文件。

## 一、体重分类标准详述

### 1.1 中国成人BMI分类标准

指南明确了基于[[BMI与体重分类|BMI分类标准]]的体重分类体系：

| 分类 | BMI（kg/m²） |
|------|-------------|
| 体重过低 | < 18.5 |
| 正常体重 | 18.5 ~ 23.9 |
| 超重 | 24.0 ~ 27.9 |
| 肥胖 | ≥ 28.0 |

> 切点说明：中国成人BMI分类采用18.5、24.0、28.0三个切点，与WHO标准（18.5、25.0、30.0）不同，更符合中国人群的体脂分布特征和健康风险。

### 1.2 中心型肥胖腰围标准

| 性别 | 中心型肥胖切点 |
|------|---------------|
| 男性 | 腰围 ≥ 90 cm |
| 女性 | 腰围 ≥ 85 cm |

> 中心型肥胖（腹型肥胖）是心血管代谢疾病的重要独立危险因素，腰围测量是评估内脏脂肪积聚的简易指标。

### 1.3 7岁以下儿童年龄别BMI标准

- 采用**标准差法**进行分类
- 以WHO儿童生长标准或中国儿童生长参照标准为依据
- BMI ≥ 同年龄同性别第95百分位（或+2 SD）为肥胖
- BMI在同年龄同性别第85至95百分位之间（或+1 SD至+2 SD）为超重
- BMI < 同年龄同性别第3百分位（或-2 SD）为体重过低

### 1.4 7-18岁儿童青少年BMI标准

- 采用**性别年龄别BMI**标准
- 参照"学龄儿童青少年营养不良筛查标准"（WS/T 456-2014）和"学龄儿童青少年超重与肥胖筛查标准"（WS/T 586-2018）
- 不同性别、不同年龄段的BMI切点不同，需查专用表格

### 1.5 80岁以上高龄老人BMI适宜范围

- 高龄老人BMI适宜范围：**22.0 ~ 26.9 kg/m²**
- 不建议高龄老人过度减重，适度超重可能对生存有利
- 需结合身体功能状态和合并症综合判断

### 1.6 孕期体重增长范围

根据孕前BMI不同，孕期体重增长推荐范围不同：

| 孕前BMI | 孕期体重增长推荐范围 |
|---------|---------------------|
| 体重过低（< 18.5） | 12.5 ~ 18.0 kg |
| 正常体重（18.5 ~ 23.9） | 11.5 ~ 16.0 kg |
| 超重（24.0 ~ 27.9） | 7.0 ~ 11.5 kg |
| 肥胖（≥ 28.0） | 5.0 ~ 9.0 kg |

> 参见[[妊娠期营养]]获取更多孕期营养管理信息。

## 二、体重评估6个维度详述

体重评估是制定个性化体重管理方案的基础，指南从6个维度进行全面评估。

### 2.1 个人史/既往史

| 评估项目 | 具体内容 |
|---------|---------|
| 家族史 | 肥胖、糖尿病、心血管疾病、高血压等代谢性疾病家族史 |
| 减重史 | 过往减重尝试、减重方法、减重效果、体重反弹情况 |
| 肥胖相关疾病史 | 2型糖尿病、高血压、血脂异常、睡眠呼吸暂停、多囊卵巢综合征、痛风等 |
| 特殊用药史 | 可能导致体重增加的药物（糖皮质激素、抗精神病药、抗抑郁药等） |

### 2.2 生活方式风险评估

| 评估领域 | 评估内容 |
|---------|---------|
| 饮食习惯 | 能量摄入水平、膳食结构、进餐频率、在外就餐情况、含糖饮料摄入 |
| 运动情况 | 运动类型、频率、强度、时长、久坐时间 |
| 睡眠状况 | 睡眠时长、睡眠质量、是否存在睡眠呼吸暂停 |
| 心理状态 | 压力水平、情绪状态、是否存在情绪性进食 |
| 吸烟饮酒 | 吸烟史、饮酒频率和量 |

### 2.3 体格检查/人体成分/能量消耗

| 指标 | 说明 |
|------|------|
| BMI | 身体质量指数 = 体重(kg) / 身高(m)² |
| 腰围 | 脐水平周径，评估中心型肥胖 |
| 臀围 | 臀部最宽处周径，计算腰臀比 |
| 体脂率 | 通过[[身体成分|人体成分分析]]或皮褶厚度测量 |
| 内脏脂肪面积 | 通过生物电阻抗或CT/MRI评估 |
| 基础代谢率 | 静息能量消耗，可通过间接测热法或公式估算 |

> 参见[[BMI与体重分类]]和[[身体成分]]了解更多测量方法。

### 2.4 医学检查评估

| 检查类别 | 具体项目 |
|---------|---------|
| 血常规 | 白细胞、红细胞、血红蛋白、血小板等 |
| 尿常规 | 尿蛋白、尿糖、尿酮体等 |
| 血糖相关 | 空腹血糖、餐后2h血糖、[[HbA1c]]（糖化血红蛋白） |
| 血脂谱 | 总胆固醇、甘油三酯、LDL-C、HDL-C |
| 肝功能 | ALT、AST、GGT等 |
| 肾功能 | 肌酐、尿素氮、eGFR |
| 甲状腺功能 | TSH、FT3、FT4（排除甲状腺因素导致的体重异常） |
| 影像学检查 | 腹部B超（脂肪肝评估）等 |

### 2.5 运动能力评估

| 评估维度 | 说明 |
|---------|------|
| 力量 | 肌肉力量测试（握力等） |
| 速度 | 有氧运动能力 |
| 耐力 | 心肺耐力（如6分钟步行试验） |
| 柔韧性 | 关节活动度 |
| 协调性 | 平衡能力测试 |
| 关节活动范围 | 评估运动安全性，排除运动禁忌 |

### 2.6 特定人群评估要点

- **儿童青少年**：重点关注生长发育曲线、性发育状态、骨龄
- **孕产妇**：关注孕前体重、孕期增重、胎儿发育
- **老年人**：关注肌少症筛查（SARC-F）、跌倒风险、认知功能
- **疾病患者**：关注原发病控制情况、用药与体重的关系

## 三、干预与治疗策略详述

### 3.1 生活方式干预

生活方式干预是体重管理的首选和基础措施，包括三大核心要素：

#### 3.1.1 膳食干预

指南推荐了4种主要膳食干预方法：

| 方法 | 具体方案 | 适用人群 |
|------|---------|---------|
| **限制能量饮食** | 每日减少500-1000千卡（或减少30%能量）；女性1000-1200千卡/天，男性1200-1600千卡/天 | BMI ≥ 24的超重及肥胖人群 |
| **均衡膳食模式** | 在目标能量范围内均衡分配宏量营养素；脂肪供能比20%-30%，蛋白质15%-20%，碳水50%-65% | 适合大多数减重人群 |
| **高蛋白/低碳水/生酮饮食** | 蛋白质供能比20%-30%或更高；碳水供能比<40%或极低；脂肪供能比相应增加 | 需在专业人员指导下短期使用 |
| **间歇性能量限制** | 5:2间歇性断食（5天正常饮食，2天极低能量600-1000千卡）；或时间限制进食（如16:8） | 依从性较好的超重/肥胖人群 |
| **代餐** | 使用营养代餐品替代1-2餐/天，保证蛋白质和微量营养素供给 | 需要严格控制能量摄入者 |

> 参见[[限能量膳食]]、[[减重膳食补充剂]]和[[减重运动处方]]了解更多。

#### 3.1.2 运动干预

| 推荐要素 | 具体要求 |
|---------|---------|
| 运动量 | **≥ 250分钟/周** 中等至较大强度有氧运动 |
| 运动类型 | 有氧运动（快走、慢跑、游泳、骑车等）+ 抗阻运动（力量训练） |
| 抗阻运动频率 | 2-3次/周，涉及主要肌群 |
| 日常活动 | 减少久坐时间，增加日常身体活动 |

> 运动干预应遵循[[减重运动处方]]原则，循序渐进，个体化制定。

#### 3.1.3 精神心理支持

指南推荐以下精神心理支持技术：

| 技术 | 说明 |
|------|------|
| **动机访谈** | 以患者为中心的咨询技术，增强行为改变动机 |
| **自我监控** | 定期记录体重、饮食、运动，提高自我觉察 |
| **刺激控制** | 识别并管理诱发过度进食的环境刺激 |
| **情绪管理** | 学习应对负面情绪的健康策略，减少情绪性进食 |
| **认知重建** | 识别并修正关于食物、体重、体像的不合理认知 |
| **问题解决** | 培养应对体重管理过程中障碍的技能 |

> 参见[[减重行为干预]]和[[正念减重]]了解更多心理干预方法。

### 3.2 医学治疗

#### 3.2.1 中医治疗

| 方法 | 说明 |
|------|------|
| **中药** | 辨证论治，常用方剂包括防风通圣散、大柴胡汤、二陈汤等 |
| **针灸** | 体针选取天枢、中脘、足三里、丰隆等穴位 |
| **穴位埋线** | 将可吸收线植入穴位，产生持续刺激效果 |
| **耳针** | 耳穴贴压，选取饥点、神门、内分泌等 |
| **振腹疗法** | 通过腹部振颤手法调节脏腑功能 |
| **太极拳** | 传统功法运动，调节身心平衡 |

> 参见[[针灸减重]]了解更多中医减重方法。

#### 3.2.2 药物治疗指征

- BMI ≥ 28.0（肥胖），经生活方式干预3-6个月效果不佳
- BMI ≥ 24.0（超重）且合并以下至少一种并发症：糖耐量异常/糖尿病、高血压、血脂异常、睡眠呼吸暂停等
- 需在医生指导下使用获批的减重药物，并定期评估安全性和有效性

#### 3.2.3 手术治疗指征

- BMI ≥ 32.5，经生活方式干预和药物治疗效果不佳
- BMI ≥ 27.5 且合并2型糖尿病或严重心血管代谢并发症
- 手术方式包括：袖状胃切除术、胃旁路术等

> 参见[[减重代谢手术]]和[[减重手术营养管理]]了解更多。

### 3.3 减重目标

| BMI范围 | 减重目标 |
|---------|---------|
| 24 ≤ BMI < 32.5 | 每周减0.5-1 kg，6个月内减重5%-15% |
| BMI ≥ 32.5 | 分阶段目标：先减5%，再减10%，最终目标15%或以上 |

> 减重速度不宜过快，以减少胆囊疾病、营养不良等风险。初始减重3-6个月后进入体重维持阶段。

### 3.4 全流程体重管理

指南提出"院内诊疗 + 院外管理"的全流程体重管理新模式：

| 环节 | 内容 |
|------|------|
| **院内诊疗** | 完成体重评估、制定个体化方案、医学检查、多学科会诊 |
| **院外管理** | 自我监测、定期随访、远程指导、社区支持 |
| **技术手段** | 5G通信、云存储、人工智能（AI）、可穿戴设备、移动健康APP |

> 参见[[全流程体重管理]]和[[全生活方式管理]]了解更多。

## 四、健康体重维持

减重成功后进入体重维持阶段，这是体重管理的长期挑战。

### 4.1 核心原则

- **吃动平衡是关键**：维持能量摄入与消耗的平衡
- 体重维持需要持续的生活方式管理，而非短期的严格限制

### 4.2 自我监测

| 方法 | 频率 |
|------|------|
| 定期称重 | 每周至少1次，固定时间（如晨起空腹） |
| 记录饮食 | 持续记录或定期回顾饮食日记 |
| 记录运动 | 追踪运动类型、时长和频率 |

### 4.3 合理膳食

| 要素 | 建议 |
|------|------|
| 三餐分配 | 早餐30%、午餐40%、晚餐30% |
| 饮水 | 充足饮水，每天1500-1700 ml |
| 食物选择 | 以[[平衡膳食|均衡膳食]]为基础，避免高能量密度食物 |

### 4.4 适量运动

| 运动类型 | 推荐量 |
|---------|--------|
| 中等强度有氧运动 | **≥ 150分钟/周** |
| 抗阻运动 | 2-3次/周 |
| 减少久坐 | 每久坐1小时起身活动5-10分钟 |

> 参见[[减重维持]]了解更多体重维持策略。

## 五、特殊人群管理（6类）

### 5.1 学龄前儿童

| 管理要点 | 具体措施 |
|---------|---------|
| 膳食管理 | 培养健康饮食行为，规律进餐，避免过度喂养；限制含糖饮料和高能量零食 |
| 身体活动 | **180分钟/天** 身体活动（包括中等到较大强度至少60分钟） |
| 睡眠管理 | 保证充足睡眠，3-4岁10-13小时，5-6岁10-12小时 |
| 行为习惯 | 减少屏幕时间，培养自主进食能力 |

> 参见[[儿童青少年肥胖营养]]了解更多。

### 5.2 学龄儿童青少年

| 管理要点 | 具体措施 |
|---------|---------|
| 饮食调整 | 保证充足营养供给以支持生长发育，控制超重肥胖流行 |
| 身体活动 | 每天至少**60分钟**中等到较大强度运动 |
| 屏幕时间 | 电子产品使用时间**< 2小时/天** |
| 心理关注 | 关注体重相关的心理问题，避免体重歧视 |

### 5.3 孕产期女性

| 管理要点 | 具体措施 |
|---------|---------|
| 孕前管理 | 孕前BMI维持在18.5-23.9的理想范围 |
| 孕期管理 | 根据孕前BMI分阶段控制能量摄入，监测体重增长曲线 |
| 产后管理 | 产后**1年内**逐步恢复至孕前体重，鼓励母乳喂养 |

> 参见[[妊娠期营养]]了解更多。

### 5.4 老年人群

| 管理要点 | 具体措施 |
|---------|---------|
| 健康饮食 | 适量优质蛋白质摄入，预防肌少症；充足蔬果和全谷物 |
| 规律活动 | 适合年龄的运动，平衡训练预防跌倒 |
| 睡眠管理 | 保证约**7小时**睡眠 |
| 心理支持 | 关注心理健康，预防老年抑郁 |

> 参见[[老年少肌性肥胖]]了解更多。

### 5.5 职业人群

| 管理要点 | 具体措施 |
|---------|---------|
| 健康食堂 | 推广健康食堂，提供低能量、营养均衡的工作餐 |
| 工间操 | 推行工间操制度，增加工作期间身体活动 |
| 减少久坐 | 优化工作环境，设置站立式办公，减少久坐时间 |

### 5.6 体重过低人群

| 管理要点 | 具体措施 |
|---------|---------|
| 膳食策略 | **高能量、高蛋白**饮食，增加健康脂肪摄入 |
| 运动策略 | 以**增肌抗阻运动**为主，促进肌肉量增长 |
| 疾病排查 | 排除导致体重过低的疾病因素（甲亢、消化道疾病等） |
