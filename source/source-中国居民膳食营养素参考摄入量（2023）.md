# 中国居民膳食营养素参考摄入量（2023）

> 中国营养学会 编著 | 655页 | 63.6MB

## 文献概要

本书是中国营养学会发布的最新版[[膳食营养素参考摄入量]]（Dietary Reference Intakes），是**中国营养学领域最核心的参考标准**。全书655页，以"总论→各营养素DRIs→应用"三篇结构，系统制定了中国居民各类营养素的[[推荐摄入量]]（推荐摄入量）、[[适宜摄入量]]（适宜摄入量）、[[可耐受最高摄入量]]（可耐受最高摄入量）、[[宏量营养素可接受范围]]（宏量营养素可接受范围）、PI（预防非传染性慢性病的建议摄入量）和SPL（特定建议值）等关键参考数值。所有临床营养、公共营养、[[食谱编制]]等工作均以本书数据为根本依据。

## 核心内容

### 2023版 vs 2013版主要变化

| 变化类别 | 2013版 | 2023版 | 意义 |
|---------|--------|--------|------|
| **PAL身体活动水平** | 1.50/1.75/2.00 | **1.40/1.70/2.00** | 更准确反映国人实际活动水平 |
| **年龄分组** | 14岁~/80岁~ | 新增30岁~；14岁~拆为12岁~/15岁~；80岁~改为75岁~ | 更精细的年龄分层 |
| **能量EER** | 较高 | 整体降低约100 kcal/d | 符合国人能量平衡需要 |
| **孕中期能量** | +300 kcal/d | **+250 kcal/d** | 降低50 kcal |
| **孕晚期能量** | +450 kcal/d | **+400 kcal/d** | 降低50 kcal |
| **乳母能量** | +500 kcal/d | **+400 kcal/d** | 降低100 kcal |
| **老年人蛋白质RNI** | 0.98 g/(kg·d) | **1.17 g/(kg·d)**（≥65岁） | 预防肌少症 |
| **老年人蛋白质AMDR** | 10-20%E | **15-20%E**（≥65岁） | 提高下限 |
| **成人钙RNI(≥50岁)** | 1000 mg/d | **800 mg/d** | 下降 |
| **孕期/哺乳期钙** | 额外增加 | **不额外增加** | 与同龄女性一致 |
| **18-49岁女性铁RNI** | 20 mg/d | **18 mg/d** | 下降 |
| **孕中期铁增加** | +4 mg | **+7 mg** | 升高 |
| **孕晚期铁增加** | +9 mg | **+11 mg** | 升高 |
| **碳水化合物** | 无EAR | **新增EAR=120g/d** | 首次制定 |
| **添加糖限制** | 无 | **<50g/d，最好<25g/d** | 新增建议 |
| **3-17岁EPA+DHA** | 仅妇幼人群 | **已制定AI** | 新增儿童青少年 |
| **SPL新增** | 无 | 甜菜碱、辅酶Q10、菊粉、谷物β-葡聚糖 | 拓展DRIs范围 |
