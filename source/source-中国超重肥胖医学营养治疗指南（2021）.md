# 中国超重肥胖医学营养治疗指南（2021）

> 中国超重肥胖医学营养治疗指南，系统梳理13种医学减重膳食方法，2021年发布。

## 文献概要

本指南是针对中国超重和肥胖人群的医学营养治疗（[[医学营养治疗]]）权威指导文件，全文共55页，约2.7MB。指南基于国内外循证证据，详细介绍了13种医学减重膳食方法，包括[[限能量膳食]]、[[高蛋白膳食]]、[[低碳水化合物饮食]]、[[间歇性断食]]、低GI饮食、DASH饮食、地中海饮食、[[代餐]]等，并对每种方法的适用人群、实施方式、减重效果和安全性进行了系统评价。

## 核心内容

### 超重肥胖的医学营养治疗原则

- 综合评估患者身体状况，制定个体化营养治疗方案
- 能量负平衡是减重的核心原理
- 膳食结构调整与行为干预相结合
- 长期坚持与定期随访

![[raw/中国营养学会/中国超重肥胖医学营养治疗指南（2021）.pdf#page=5]]

### 13种医学减重膳食方法

#### 一、限能量膳食（CRD）
- 在目标能量摄入基础上每日减少500~750 kcal
- 三大营养素供能比例：碳水化合物40%~55%、脂肪20%~30%、蛋白质15%~20%
- 适用于大多数超重/肥胖患者
- 详见 [[限能量膳食]]

#### 二、高蛋白膳食
- 蛋白质供能比20%~30%，或蛋白质摄入1.2~1.5 g/kg/d
- 增加饱腹感、减少瘦体重丢失
- 肾功能不全者慎用
- 详见 [[高蛋白膳食]]

#### 三、低碳水化合物饮食
- 碳水化合物供能比<40%，甚至<20%（极低碳水）
- 短期减重效果显著，长期安全性有待更多研究
- 详见 [[低碳水化合物饮食]]

#### 四、间歇性能量限制（IER）
- 包括隔日禁食（ADF）、5:2断食、限时进食（TRE）等
- 详见 [[间歇性断食]]

#### 五、低血糖指数（GI）饮食
- 选择GI值较低的食物，控制血糖波动
- 有助于改善胰岛素敏感性

#### 六、DASH饮食
- 富含水果、蔬菜、低脂乳制品、全谷物、禽肉、鱼类和坚果
- 限制饱和脂肪、含糖饮料和 sweets
- 原为降压设计，兼具减重效果

#### 七、地中海饮食
- 以植物性食物为主，橄榄油为主要脂肪来源
- 大量证据支持其心血管保护作用

#### 八、代餐
- 使用营养代餐粉/棒替代1~2餐
- 严格保证蛋白质、维生素、矿物质摄入
- 详见 [[代餐]]

#### 九~十三、其他方法
- 极低能量膳食（VLCD）、素食减重膳食、个性化营养干预、生物节律饮食、中医食疗等

### 各膳食方法减重效果比较

| 膳食模式 | 6个月预期减重 | 适用人群 | 注意事项 |
|----------|-------------|----------|----------|
| 限能量膳食（CRD） | 5%~10%体重 | 广泛适用 | 基础方案 |
| 高蛋白膳食 | 8%~12%体重 | 无肾病者 | 关注肾功能 |
| 低碳水化合物饮食 | 6%~10%体重 | 胰岛素抵抗者 | 关注血脂 |
| 间歇性能量限制 | 5%~8%体重 | 难以持续限能量者 | 禁食日依从性 |
| 低GI饮食 | 4%~8%体重 | 糖代谢异常者 | 需GI食物知识 |
| DASH饮食 | 3%~6%体重 | 高血压合并肥胖 | 兼顾降压 |
| 地中海饮食 | 3%~5%体重 | 心血管高危人群 | 长期获益 |

## 指南制定方法

本指南由中华医学会、中国营养学会等5个学会联合制定，基于GRADE分级系统进行证据评价和推荐分级。

- **证据等级**：A（高）→ B（中）→ C（低）→ D（极低）
- **推荐强度**：强推荐、弱推荐
- 按**PICO格式**（人群、干预、对照、结局）评价临床问题
- 采用**德尔斐法则**形成专家共识
- 共检索近**20年文献**，纳入**560篇参考文献**
- 2021年正式发布，是当前中国超重/肥胖医学营养治疗领域的权威指导文件

## 全生活方式管理（核心原则）

全生活方式管理（Comprehensive Lifestyle Management）是本指南的核心原则，强调多要素协同干预：

- **三要素结合**：[[医学营养治疗|饮食管理]] + [[身体活动|体育锻炼]] + [[减重行为干预|行为干预]]
- 每周不少于**150 min**运动
- 保持**6个月内不少于14次随诊**并持续**1年**
- 可使超重/肥胖者平均**减重8 kg**
- 全生活方式管理**优于单纯饮食或运动干预**
- 膳食、运动和行为三者缺一不可，协同增效

## 13种膳食方法详细推荐

| 膳食方法 | 推荐等级 | 核心推荐意见 | 关键证据要点 |
|---------|---------|------------|------------|
| 限能量膳食(CRD) | A（强推荐）| 基础减重方案，每日减少500-1000kcal | 3-6个月减重5%-10% |
| 高蛋白膳食(HPD) | B（弱推荐）| 蛋白质供能比20%-30%，增加饱腹感 | 减重依从性更好，对肾功能无不良影响 |
| 低碳水化合物饮食(LCDs) | A（强推荐，短期）| 碳水供能比≤40%，短期控制体重 | T2DM患者短期改善血糖控制 |
| 生酮饮食 | B（强推荐）| 碳水供能比≤20%，需临床营养师指导 | 长期安全性未证实 |
| 间歇性能量限制(IER) | A（强推荐）| 隔日禁食、5:2等模式 | 减重效果与持续限能量相当 |
| 低GI饮食 | B（弱推荐）| 选择低GI食物，控制血糖波动 | 增加饱腹感，改善胰岛素抵抗 |
| DASH饮食 | B（强推荐）| 富含蔬果低脂乳，限钠 | 降低体重、BMI、体脂含量 |
| 地中海饮食 | A（强推荐）| 植物性食物为主，橄榄油 | 降低体重和心血管风险 |
| 代餐食品 | B（弱推荐）| 替代1-2餐，配合维生素矿物质补充 | 1年内减重效果更好 |
| TRF时间限制进食 | D（弱推荐）| 限定进食时间窗口 | 长期证据不足 |
| 益生菌 | C（弱推荐）| 特定菌株协助减重 | 鼠李糖乳杆菌等有效 |
| 营养教育 | C（强推荐）| 线上线下结合 | 改善饮食结构和依从性 |
| 行为辅导 | A（强推荐）| 自我监测、目标设定、问题解决 | 所有减重者都应给予 |

## 运动处方

运动是[[减重]]全生活方式管理的关键组成部分：

- **适度减重**：每周至少**150 min**中等强度运动
- **减重≥5%**：每周**300 min**中-高强度运动
- **有氧运动 + 抗阻训练联合**（证据等级A）：两种运动方式协同效果最佳
- **HIIT（高强度间歇训练）**：具有时效优势，节省约**40%**运动时间
- **累积性运动**：效果优于一次性持续运动，可分多次完成

## 药物治疗

药物治疗是生活方式干预无效后的辅助手段：

- **适应证**：BMI≥28经3个月生活方式干预无效；或BMI≥24合并并发症
- **奥利司他**：配合低脂饮食，主要不良反应为胃肠道不适
- **T2DM减重药物**：
  - [[二甲双胍]]（证据等级A）
  - GLP-1受体激动剂（证据等级A）
  - SGLT-2抑制剂（证据等级A）

## 减重手术后营养管理

减重手术（代谢手术）患者的围手术期和术后营养管理至关重要：

- **术前**：限饮食预减重，减少肝脏体积和内脏脂肪，降低手术风险
- **术后蛋白质**：60-80g/d
- **术后维生素D**：筛查+预防性口服3000U/d
- **术后维生素B12**：350-1000μg/d常规补充
- **术后铁**：定期监测，缺铁时补充铁剂+维生素C
- **ERAS理念**应用于围手术期（加速康复外科）
- 术后需长期随访监测营养状况

## 特殊人群

### PCOS（多囊卵巢综合征）
- 6个月减重5%-10%
- 各种饮食模式均可，CRD为基础

### T2DM（2型糖尿病）
- 减重目标5%-10%
- [[限能量膳食]]为基石，多种膳食模式可选

### MAFLD（代谢相关脂肪性肝病）
- 禁酒
- 可选择[[地中海饮食]]或间歇性能量限制（IER）

### 痛风
- 推荐[[地中海饮食]]或[[DASH饮食]]

### 围孕期
- 综合生活方式干预改善孕期增重

### 儿童青少年
- CRD为基础方案
- 每天至少60min中高强度运动
- 以家庭为中心的干预模式

### 老年少肌性肥胖
- 蛋白质1.0-1.5g/(kg·d)
- 不推荐极低能量饮食
- 重视抗阻运动保留肌肉量

## 微量营养素补充

- [[限能量膳食]]尤其极低能量饮食时应补充**复合维生素与微量元素**（C，强推荐）
- 有缺乏风险者补充**钙和/或维生素D**（B，强推荐）
- **个体化饮食方案**防止铁缺乏（C，弱推荐）

## 减重维持

减重后的体重维持是长期管理的关键环节：

- **3-6个月**限能量高蛋白膳食更有助于维持（证据等级A）
- **IER**在维持期作用无显著劣势（证据等级C）
- 长期（≥1年）综合减重维持计划
- **自我监测**：定期称重
- **目标设定**与**问题解决**：持续行为管理

## 精准营养

精准营养是个体化营养干预的前沿方向：

- **3个层级**：群体→个体化→基因营养
- **FTO基因**检测可辅助制定精准方案
- **肠道菌群P/B比值**可作为个性化营养的生物标志物
- 肠道菌群分析有助于预测个体对不同膳食模式的反应
