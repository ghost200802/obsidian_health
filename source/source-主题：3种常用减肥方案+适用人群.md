# 主题：3种常用减肥方案+适用人群

## 文件信息
- 原始文件：`Apple Notes/1 个人/公共营养师/科普/主题：3种常用减肥方案+适用人群.md`
- 文件类型：Markdown 笔记
- 上级分类：公共营养师 / 科普

## 内容定位
这份笔记目前主要展开的是 CRD（限能量平衡膳食）在减脂中的应用逻辑、食物结构和三餐搭配思路，适合补充[[限能量膳食]]和[[减重]]的实践层表达。

## 核心摘要
笔记把 CRD 描述为“在保证营养的基础上吃适当份量”的主流减重方法，强调其适用范围广、食物结构均衡、可迁移到食堂和外卖场景。内容重点不是精算热量，而是围绕主食、蛋白质、蔬菜、水果、奶类和豆制品建立可执行的三餐框架。

## 关键观点
- CRD 的核心不是极端限制，而是在能量受控前提下维持平衡膳食。
- 三餐结构比单个食物更重要，早餐和午餐的蛋白质与蔬菜配置会影响全天饱腹感。
- 主食并不是减脂的敌人，关键在于总量、搭配和全天分布。
- 外卖或食堂场景也能套用主食 + 蛋白质 + 蔬菜的搭配公式。
- 坚果、奶类、豆制品等食物可作为补充，但同样需要份量意识。

## 保留价值
这份资料适合作为[[限能量膳食]]的场景化说明，让“均衡、可持续、能落地”这几个原则更容易被客户理解和执行。
