# 中国居民健康体重管理之减重行动20条

> 中国营养学会肥胖防控分会联合西安交通大学全球健康研究院，组织跨学科专家编写，发表于《中国糖尿病杂志》2023年12月。基于科学循证，系统提出20条减重行动建议。

## 文献概要

本共识由中国营养学会肥胖防控分会组织编写，采用专家综览和意见集中法，检索汇总2020年以来高级别证据（系统综述、Meta分析、指南、共识），经多轮专家讨论（2022年12月至2023年8月）形成。共识旨在向公众及卫生工作者推荐基于高级别科学证据的减重原则和实用方法，涵盖膳食、运动等生活方式管理和临床治疗措施。

## 背景

- 过去30年中国居民超重/肥胖快速增长：成年人超重/肥胖率超过**50%**，学龄儿童青少年接近**20%**
- 约5~6亿成年人和5000万儿童青少年超重/肥胖
- 超重/肥胖增加[[高血压]]、[[糖尿病]]、心脑血管疾病、特定癌症等多种慢性病患病风险
- 全球疾病负担研究：**44%**的T2DM、**23%**的冠心病、**7%~41%**特定癌症负担可归因于超重/肥胖
- 预测到2030年：成年人肥胖超重率可达**70%**，儿童可达**30%**；归因于超重/肥胖的医疗费用约**4180亿元**，占全国医疗费用总额**22%**

## 共识20条详细内容

### 要点一：生活方式干预是科学减重的首选和基础措施（第1~2条）

**第1条：超重/肥胖是热量摄入多于消耗所致，减重需要控制饮食，增加身体活动。**
- 超重/肥胖定义：热量摄入多于消耗，多余热量以脂肪形式积聚体内
- 诊断标准：BMI ≥ 24.0 为超重，BMI ≥ 28.0 为肥胖
- 中心性肥胖：腰围男性 ≥ 90.0 cm、女性 ≥ 85.0 cm
- 中心性肥胖的健康危害比全身性肥胖更大
- 儿童青少年有年龄-性别BMI参考界值（附表2）
- 关联：[[BMI与体重分类]]、[[肥胖]]

**第2条：不健康生活方式是超重/肥胖的主要危险因素，全生活方式管理可帮助有效减重。**
- 主要危险因素：高脂高糖等高热量和低纤维膳食、饮食不规律、缺乏身体活动
- 其他致病因素：遗传、心理、经济、文化等
- [[全生活方式管理]]：同时实施饮食、运动和行为三要素的多维度干预
- 全生活方式管理优于单纯饮食或运动干预，带来多重健康效应
- 超重/肥胖者可通过加强体重监测、设定改善目标、坚持健康生活方式、获取社会支持等措施有效减重
- 关联：[[减重行为干预]]、[[身体活动]]

### 要点二：合理膳食是科学减重的基础（第3~9条）

**第3条：在控制总热量的基础上，坚持谷类为主的平衡膳食模式有利于减重。**
- 控制总热量的平衡膳食模式是减重基础
- 地中海饮食：以蔬菜水果、鱼类、五谷杂粮、豆类和橄榄油为主，与体重改善、[[心血管疾病]]和代谢性疾病风险下降相关
- 其他健康膳食模式：得舒饮食（DASH）、北欧饮食、[[中国江南饮食]]（东方健康饮食）
- 中国江南饮食特点：食材种类丰富，蔬菜水果摄入量大，水产品和禽类白肉多于红肉，油盐摄入较少，口味清淡，以蒸煮为主
- 选择适合自己的科学膳食模式并长期坚持是关键
- 关联：[[地中海饮食]]、[[中国江南饮食]]、[[膳食指南]]

**第4条：膳食中适当增加奶类及其制品有助于减重。**
- 牛奶对超重/肥胖、T2DM和CVD可能具有潜在改善作用
- 奶及奶制品可提供优质[[蛋白质]]，是膳食[[钙]]的最佳来源
- 《[[中国居民膳食指南（2022）]]》推荐：每人每天摄入300~500 g液态奶或等量奶制品
- 乳糖不耐受或牛奶蛋白过敏者除外

**第5条：控制总热量、改变三大供能营养素配比和间歇性断食有助于短期减重，但长期效果缺少证据支持。**
- 控制总热量、改变三大供能营养素配比（低碳水、低脂肪、高蛋白等）有短期减重效果，长期效果不明显，不推荐长期使用
- 极低和低热量饮食短期减重2%~15%，可控制血糖和改善脂质代谢，但长期效果尚无证据
- [[间歇性断食]]（轻断食）：正常热量与热量限制交替进行的膳食模式，有一定减重作用，但长期效果尚无证据
- 关联：[[低碳水化合物饮食]]、[[间歇性断食]]、[[生酮饮食]]

**第6条：低碳水化合物饮食有利于中短期体重控制，但其长期安全性和有效性证据不足。**
- 低碳水化合物饮食：碳水供能比从55%~65%减少到≤40%，[[脂肪]]供能占比≥30%
- 极低碳水化合物饮食（碳水供能比<10%）：即[[生酮饮食]]，脂肪和蛋白质与碳水供能比3:1~4:1
- 生酮饮食减重机制：缺乏碳水导致胰岛素分泌降低，抑制脂肪生成并促进脂肪分解；高蛋白产生饱腹感，影响食欲控制激素（瘦素、食欲刺激素）
- 短期应用减重效果显著，长期安全性和获益尚不明确
- 需在专业人员科学指导下进行
- 关联：[[低碳水化合物饮食]]、[[生酮饮食]]、[[胰岛素抵抗]]

**第7条：夜间饮食和不吃早餐增加肥胖风险，不利于减重。**
- 进食时间在导致肥胖和影响减重效果中的作用日益受到重视
- 不吃早餐和[[夜间饮食]]与体重增加相关，甚至导致肥胖
- 不吃早餐会增强饥饿感，增加后续进食量，加重[[胰岛素抵抗]]
- 吃早餐有利于维持血糖稳定，避免因禁食时间过长导致胃饥饿素浓度增大引起食欲亢进
- 夜间摄入食物量和营养特征可能对代谢和昼夜节律产生负面影响

**第8条：降低进食速度有利于恢复和保持健康体重。**
- 进食速度过快与肥胖相关，特别是腹型和内脏型肥胖
- 降低进食速度有利于减重：减少热量摄入，降低饥饿素分泌，减轻饥饿感
- 升高神经肽Y和GLP-1饱食激素分泌，增加饱腹感
- 建议成人每餐进食时间 > 20 min
- 关联：[[饮食行为]]、[[减重行为干预]]

**第9条：尚无明确科学证据证实天然膳食补充剂的减重效果，需谨慎使用。**
- [[减重膳食补充剂]]包括：肉碱、白芸豆提取物、壳聚糖、羟基柠檬酸、辣椒素、吡啶甲酸铬、共轭亚油酸、葡甘露聚糖、铬、瓜尔胶、丙酮酸盐、咖啡因等
- 减重效果缺乏足够证据，多数研究存在方法学缺陷、样本量不足、缺乏长期随访
- 部分膳食补充剂可能存在肠道不良反应，甚至引起肝肾功能损害
- 建议谨慎使用

### 要点三：运动干预是科学减重的核心（第10~11条）

**第10条：有氧、抗阻、高强度间歇运动均可有效减重。**
- [[有氧运动]]：运动强度较低，如快走、长跑、广场舞、长距离骑行等，长期规律有利于减脂
- [[抗阻运动]]：可预防/减少通过限制热量减重过程中的肌肉量下降
- [[高强度间歇运动]]（HIIT）：可增加运动后过量氧耗，增强减脂效率
- 关联：[[减重运动处方]]、[[身体活动]]

**第11条：维持中等强度运动 > 250 min/周可帮助减重，较高强度运动减重效果更明显。**
- 150~250 min/周中等强度运动 → 适度减重
- > 250 min/周 → 明显减重并维持效果
- 按储备心率百分比法：中等强度运动达40%~60%，高强度运动达60%~80%
- 靶心率 = 安静心率 + (最大心率 - 安静心率) × 运动强度百分比；最大心率 = 220 - 年龄
- 有基础疾病（T2DM、高血压、心脑血管疾病）患者需专业评估确认安全后，才可从事较高强度运动
- 关联：[[减重运动处方]]

### 要点四：良好的睡眠和健康、积极的心态，有助于减重（第12~14条）

**第12条：每天睡眠时间 < 6 h 增加肥胖风险，> 7 h 有利于减重，特别是有利于预防腹型肥胖。**
- 每天睡眠时间 < 6 h 与肥胖、T2DM、CVD、代谢性疾病发生风险增加相关
- 长期睡眠障碍 → 慢性压力 → "下丘脑-垂体-肾上腺"轴功能长时间活跃 → 糖皮质激素分泌增加 → [[胰岛素抵抗]]和体重增加
- 睡眠障碍还可通过增加焦虑和抑郁情绪，减少身体活动，影响热量消耗
- 关联：[[胰岛素抵抗]]

**第13条：长期持续的心理压力可增加肥胖风险，保持良好的心理和情绪有助于保持健康体重。**
- 长期心理压力干扰认知功能（执行功能和自我调控能力）
- 慢性压力诱导暴饮暴食和摄入高热量食物，减少体力活动和睡眠时间
- 通过"下丘脑-垂体-肾上腺轴"促进高热量食物摄入，影响肠道微生态和肽类激素分泌（瘦素、食欲刺激素、神经肽Y）
- 关联：[[减重行为干预]]

**第14条：正念及照镜子有利于恢复健康体重。**
- [[正念减重]]：改善饮食行为失调、强化自我激励和控制，提高减肥计划有效性
- 与其他心理方法联合应用效果更好
- 照镜子：可改变认知，影响膳食行为，促进减重，长期效果需进一步研究
- 关联：[[减重行为干预]]

### 要点五：药物和手术治疗是临床减重的重要手段（第15~17条）

**第15条：[[针灸减重]]辅以生活方式管理，有助于减重。**
- 针灸可调节食欲、肠动力和新陈代谢，减轻压力、调节情绪
- 针灸通过改变中枢神经递质，调节饥饿和饱腹感，增加脑啡肽、β-内啡肽和血清素水平，降低食欲
- 耳针、针刺、穴位埋线等可能有帮助，但长期效果尚需更多证据

**第16条：超重/肥胖合并"三高"并发症的患者，经3~6个月生活方式干预仍无法有效减重，可考虑药物治疗。**
- 超重/肥胖合并"三高"（高血压、高血糖、高血脂），改变生活方式无效 → 可使用减重药物
- **奥利司他**：中国目前唯一批准的非处方减重药物，肠道胰脂肪酶抑制剂，减少膳食脂肪吸收
  - 常见不良反应：皮肤油性斑点、胃肠排气增多、大便紧急感、油性大便
  - 可导致脂溶性[[维生素D]]吸收减少，用药期间应注意补充
- **T2DM合并肥胖患者**：可在医生指导下选择二甲双胍、GLP-1受体激动剂、SGLT-2抑制剂等
  - 作用：抑制食欲、延缓胃排空速度、增加体内葡萄糖排出
  - 常见不良反应：胃肠道事件
- 药物治疗后应定期进行有效性和安全性评估
- 关联：[[减重]]、[[肥胖]]

**第17条：[[减重代谢手术]]治疗肥胖有效，适用于重度肥胖且生活方式干预无效患者。**
- 重度肥胖：BMI ≥ 37.5 kg/m² 或 ≥ 32.5 kg/m² 伴有并发症
- 可考虑强化全生活方式干预 + 药物治疗 + 减重代谢手术
- 最常见手术类型：胃袖状切除术和Roux-en-Y胃旁路手术
- 术后10年体重减轻可达27%~69%
- T2DM合并肥胖患者术后10年T2DM缓解率约25%~62%
- 手术可能并发症：[[维生素]]缺乏、胃食管反流、倾倒综合征
- 术前全面临床内科评估，术后长期随访和营养管理
- 关联：[[减重手术营养管理]]、[[肥胖]]

### 要点六：肥胖的全生命周期防治，关注儿童肥胖问题（第18~20条）

**第18条：超重/肥胖儿童体重干预目标是维持体重不增或增长速度减缓，最终达到健康体重，而非必须降低绝对体重。**
- 干预目标：保证身高稳定增长的同时，体重不增或增长速度减缓
- 首推[[生活方式干预]]，父母和家庭参与至关重要
- 实施应针对整个家庭，促进以家庭为中心的生活方式改变
- 确定生活方式干预无效后可采用药物治疗辅助，需在专业医生指导下使用
- 关联：[[儿童青少年肥胖营养]]

**第19条：超重/肥胖儿童减重应注重膳食结构调整，减少外出就餐，控制进食速度、零食及含糖饮料量。**
- 坚持控制食物总量、调整饮食结构和行为的原则
- 减少快餐食品、零食，避免含糖饮料摄入
- 多吃水果和蔬菜，保证每日至少饮用液态奶300 ml或等量奶制品
- 减少外出就餐，控制进食速度至20~30 min/餐
- 关联：[[饮食行为]]、[[儿童青少年肥胖营养]]

**第20条：超重/肥胖儿童至少应进行150 min/周中高强度有氧运动和抗阻训练，减少久坐，电子设备使用 < 2 h/d。**
- 每日进行中高强度、全身性有氧运动30~60 min（如慢跑、滑冰、游泳等），每周 ≥ 4 d，时长 ≥ 150 min
- 每周进行3~4次抗阻训练（如引体向上、仰卧起坐、高抬腿运动等），建议隔日进行
- 运动需长期坚持，同时减少久坐行为
- 关联：[[儿童青少年肥胖营养]]、[[减重运动处方]]

## 保留价值

- **权威性高**：中国营养学会肥胖防控分会组织跨学科15名资深专家编写
- **时效性强**：基于2020年以来最新证据
- **覆盖全面**：从生活方式到临床治疗，从成人到儿童全生命周期
- **实用性强**：20条具体可操作的行动建议，面向公众和卫生工作者
- **与已有知识互补**：与[[source-中国超重肥胖医学营养治疗指南（2021）]]、[[source-体重管理指导原则（2024年版）]]形成减重领域三重证据体系

## 关键数值速查

| 指标 | 标准/建议值 |
|------|-----------|
| BMI超重 | ≥ 24.0 kg/m² |
| BMI肥胖 | ≥ 28.0 kg/m² |
| 男性腰围（中心性肥胖） | ≥ 90.0 cm |
| 女性腰围（中心性肥胖） | ≥ 85.0 cm |
| 中等强度运动（适度减重） | 150~250 min/周 |
| 中等强度运动（明显减重） | > 250 min/周 |
| 每餐进食时间建议 | > 20 min |
| 睡眠时间建议 | > 7 h/天 |
| 奶类推荐摄入量 | 300~500 g/d |
| 儿童电子设备使用 | < 2 h/d |
| 儿童有氧运动 | ≥ 150 min/周 |
| 减重代谢手术适应证 | BMI ≥ 37.5 或 ≥ 32.5 伴并发症 |
| 奥利司他使用前提 | 3~6个月生活方式干预无效 |
