# 成人高脂血症食养指南（2023年版）问答

## 元信息

- **发布机构**：国家卫生健康委办公厅
- **发布日期**：2023-01-29
- **原文链接**：[国家卫生健康委办公厅关于印发成人高血压食养指南等4项食养指南的通知](https://www.nhc.gov.cn/sps/c100088/202301/f01895a06c5349ef999f25da833c166d.shtml)
- **对应正文**：[[source-成人高脂血症食养指南（2023年版）]]

## 核心摘要

问答稿把高脂血症食养的高频问题压缩成更好懂的说明，尤其适合解释为什么要重点盯住脂肪供能、胆固醇和食物标签。它强调每种证型都给了 3 天示例食谱，并提醒使用者把食药物质也放回全天总膳食去核算。对[[血脂异常]]页来说，这份问答能补上“怎么解释给患者听”的那一层。

## 关键观点

1. **重点不只在“少油”**：问答稿强调脂肪供能一般不超过总能量 20%-25%
2. **实操工具明确**：正文提供不同证型 3 天食谱和常见食物交换表
3. **会看会选很重要**：适合配合食品标签阅读和食物替换做长期管理
4. **适合作为患者教育版本**：比正文更适合回答“高脂血症是不是只看胆固醇”
