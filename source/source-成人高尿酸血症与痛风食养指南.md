# 成人高尿酸血症与痛风食养指南（2024年版）

> 国家卫生健康委办公厅发布的针对成人[[高尿酸血症]]与[[痛风]]的专项食养指南。涵盖疾病特点与分型、7条食养原则、食物嘌呤含量分类表、食物选择指导、不同疾病阶段/地区/季节食谱示例、中医辨证分型食养方及食物交换表。

## 文献概要

本指南根据《健康中国行动（2019—2030年）》和《国民营养计划（2017—2030年）》相关要求制定。主要面向未合并肾功能不全等相关合并症的成人[[高尿酸血症]]、[[痛风]]人群以及基层卫生工作者（包括营养指导人员）。

### 核心背景数据
- 我国成人居民[[高尿酸血症]]患病率：**14%**
- [[痛风]]患病率：**0.86%～2.20%**
- 男性高于女性，城市高于农村，沿海高于内陆
- 痛风患病率呈逐年上升趋势，发病年龄趋于年轻化
- 高尿酸血症和痛风是[[慢性肾脏病]]、[[高血压]]、[[心血管疾病]]及[[糖尿病]]等疾病的**独立危险因素**

## 疾病特点与分型

### 西医分期
| 分期 | 特征 |
|------|------|
| 无症状高尿酸血症期 | 无症状高尿酸血症及无症状单钠尿酸盐晶体沉积 |
| 急性痛风性关节炎期 | 关节炎突然发作，关节红肿热痛，疼痛剧烈 |
| 痛风间歇期 | 两次急性痛风性关节炎发作之间的阶段 |
| 慢性痛风性关节炎期 | 关节持续疼痛，[[血尿酸]]水平持续波动，可伴痛风石 |

### 中医辨证分型

| 证型 | 常见分期 | 主要表现 | 舌脉特征 |
|------|---------|---------|---------|
| 湿浊证 | 无症状HUA期、痛风间歇期 | 肢体困乏沉重，形体肥胖，嗜食肥甘，口中粘腻不渴，大便黏 | 舌淡胖或有齿痕，苔白腻，脉滑 |
| 湿热证 | 急性痛风性关节炎期 | 关节红肿灼热疼痛、疼痛剧烈，发作频繁，或伴发热，烦躁不安，口苦口臭 | 舌质红，苔黄腻或黄厚，脉弦滑或滑数 |
| 痰瘀证 | 痛风间歇期、慢性痛风性关节炎期 | 关节肿痛反复发作，关节局部硬结或皮色暗红，或关节刺痛，屈伸不灵活、关节变形 | 舌质紫暗，苔白腻，脉弦或弦滑 |
| 脾肾亏虚证 | 慢性痛风性关节炎期 | 关节疼痛反复发作、活动不灵活、僵硬或变形，腰膝酸软，乏力明显，肢体困倦沉重，腹胀 | 舌淡胖或有齿痕，舌苔白腻，脉沉缓或沉细 |

## 食养原则7条

### 原则1：食物多样，限制嘌呤
- 每天食物品种不少于**12种**，每周不少于**25种**
- 合理调整膳食中[[碳水化合物]]、[[蛋白质]]和[[脂肪]]提供的能量比例
- 动物内脏（肝、肾、心）嘌呤含量普遍高于普通肉类，应**尽量避免**
- 鸡蛋蛋白、牛奶嘌呤含量较低，可放心食用
- 大豆嘌呤含量略高于瘦肉和鱼类，但植物性食物中嘌呤人体利用率低
- 豆腐、豆干等豆制品加工后嘌呤含量有所降低，可**适量食用**
- 宜选择低[[血糖生成指数]]的碳水化合物类食物
- 每天全谷物食物不低于主食量的**30%**
- [[膳食纤维]]摄入量达到**25～30g**
- 有痛风发作病史者应尽量避免食用既往诱发痛风发作的食物

### 原则2：蔬奶充足，限制果糖
- 维生素、植物化学物等营养成分可促进肾脏尿酸排泄
- 新鲜蔬菜（如菊苣、鲜百合）、水果（如富含[[维生素C]]的樱桃、草莓等）、植物性饮料（如无糖咖啡、茶）和草本植物（如薏苡仁、益智仁）等含有丰富的植物化学物（生物碱类、酚酸、黄酮类、芪类及苷类）
- 每天多食新鲜蔬菜，推荐每天摄入**不少于500g**，深色蔬菜应占一半以上
- 乳蛋白是优质蛋白的重要来源，可以促进尿酸排泄，鼓励每天摄入**300mL以上**或相当量的奶及奶制品
- 限制果糖含量较高的食品：含糖饮料、鲜榨果汁、果葡糖浆、果脯蜜饯等
- 水果中的维生素C、黄酮、多酚、钾、膳食纤维等可改变果糖对尿酸的影响，水果摄入量与痛风无显著相关性
- 建议每天水果摄入量**200～350g**

### 原则3：足量饮水，限制饮酒
- 每天建议**2000～3000mL**，尽量维持每天尿量大于2000mL
- 优先选用白水，也可饮用柠檬水、淡茶、无糖咖啡及苏打水
- 避免过量饮用浓茶、浓咖啡等，避免饮用生冷饮品
- 酒精代谢影响嘌呤释放并促使尿酸生成增加，酒精还导致血清乳酸升高，减少尿酸排泄
- 黄酒嘌呤含量较高，其次是啤酒；白酒嘌呤含量低但酒精度高，易使乳酸堆积
- 急性痛风发作、药物控制不佳或慢性痛风性关节炎患者应**不饮酒**

### 原则4：科学烹饪，少食生冷
- 推荐每天食盐摄入量不超过**5g**，每天烹调油不超过**25～30g**
- 减少油炸、煎制、卤制等烹饪方式
- **提倡肉类汆煮后食用，尽量不喝汤**
- 腊制、腌制或熏制的肉类嘌呤、盐分含量高，不宜食用
- 经常食用生冷食品容易损伤脾胃功能，可导致尿酸盐结晶析出增加
- 痛风患者应**少吃生冷食品**

### 原则5：吃动平衡，健康体重
- 超重肥胖会增加痛风风险，减轻体重可显著降低[[血尿酸]]水平
- 每天可减少250～500kcal能量摄入，通过运动消耗250～500kcal能量
- 避免过度节食和减重速度过快，以每周减低**0.5～1.0kg**为宜
- 18～64岁成年人BMI适宜范围**18.5～23.9kg/m²**，65岁及以上老年人**20.0～26.9kg/m²**
- 运动强度以低、中强度有氧运动为主，每周4～5次，每次30～60分钟
- 选择对关节冲击力小的运动：慢跑、走路、骑自行车、[[太极拳]]、八段锦、游泳等
- 运动期间或运动后应及时补充水分
- **痛风性关节炎期应减少或避免运动**
- 养成规律作息、劳逸结合的习惯

### 原则6：辨证辨体，因人施膳
- 脾胃失调、湿浊内生是痛风反复不愈的症结所在
- 痛风食养关键在于**调理脾胃**
- 推荐食用：白扁豆、玉米须、麦芽、山药、芡实、大枣、橘皮、山楂、五指毛桃、茯苓等
- 各证型具体食养方案：
  - **湿浊证**：避免寒冷诱发，严格限制嘌呤和蛋白质总摄入量，低盐膳食，避免甜、油腻、酸、涩食物；可食用薏苡仁、橘皮、茯苓代茶饮
  - **湿热证**：严格限制嘌呤、蛋白质摄入，尤其限盐和烟酒，推荐食用山竹、西瓜、荸荠等，可用赤小豆、木瓜、薏苡仁等煮汤
  - **痰瘀证**：多饮水，严格限制高脂、高胆固醇食物和高盐膳食，可食用木耳、山楂、桃仁等
  - **脾肾亏虚证**：宜食温热性食物，可食生姜、黄芪、茯苓、核桃、荔枝等

### 原则7：因地因时，择膳相宜
- 沿海地区：有长期食用海产品和高嘌呤煲汤习惯
- 内陆地区：常吃畜禽肉和淡水鱼虾，蔬果摄入相对较低
- 高原地区：常吃青稞、酥油、牛羊肉，蔬菜水果及饮水量较少
- **春季养肝**：膳食清淡爽口，多吃含丰富维生素、[[膳食纤维]]的蔬菜水果
- **夏季祛暑**：食物清淡、营养丰富、易消化，推荐丝瓜、冬瓜，少吃海鲜、动物内脏、畜肉，少吃生冷
- **秋季养阴润肺**：少吃鱼虾螃蟹，推荐莲子、莲藕、荸荠、百合等甘凉生津润燥之品
- **冬季温补**：控制火锅、烤串、肉汤等摄入
- 痛风性关节炎遇冷容易诱发，秋冬、夏秋季节交替要注意保暖

## 食谱示例概况（附录3）

### 无症状高尿酸血症期与痛风间歇期
- 能量范围：1600～1800kcal
- 适合正常体重成人、轻体力活动人群
- 坚持食物多样、能量适宜、主食定量、蛋白适量、蔬菜丰富
- 覆盖不同地区（华南、西南、华东、西北、东北、华北）和四季

### 急性痛风性关节炎期及慢性痛风性关节炎期
- 能量范围：1400～1600kcal
- 更严格的嘌呤限制
- 蛋白质来源以鸡蛋、低脂牛奶为主

### 食谱营养素参考范围
- 蛋白质：**10%～20%**总能量
- 碳水化合物：**50%～60%**总能量
- 脂肪：**20%～30%**总能量
- 全天植物油25g，盐4g
