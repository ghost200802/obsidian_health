# 我们都期待「事事有回应」，但大多数人都误解了它的含义——

**许多朋友因为等不到回应而产生深切的对自我的怀疑，却从来没有意识到，其实这是一种非常基础的需求的满足——“被回应”的需求。**
因为得不到回应而产生的痛苦，从来不是我们的问题，而是因为我们最基本的**“被回应需求”**<span style="font-family:.PingFangSC-Regular;">没有得到满足。</span>

- <span style="font-family:.PingFangSC-Semibold;color:#94110ff;"><b>我们有刻在基因里的被回应需求，这是作为人类、也包括其他高等生物的本能需求。</b></span>

**当基础的认知闭合需求得到满足，我们对于关系里不确定性的容忍程度也会上升，**哪怕是高压力状态下，也不会因为等不到对方的回应而抓狂和和向对方施压，整体的关系也会因此变得松弛。

- <span style="font-family:.PingFangSC-Semibold;color:#94110ff;"><b>回应为我们提供掌控力和安全感，构建起向外探索的“内心安全基地”。</b></span>

<p style="text-align:justify;margin:0">
</p>
- <span style="font-family:.PingFangSC-Semibold;color:#94110ff;"><b>总是能给予我们回应的人，会让我们感到对方可信任、有责任感、温暖。</b></span>
**一个可以被预测到、一定会尽ta所能给我们回应的人，就是**<span style="color:#94110ff;"><b>这个世界对我们而言温柔的存在</b></span><span style="font-family:.PingFangSC-Semibold;"><b>。</b></span>


<span style="font-family:.PingFangSC-Semibold;color:#94110ff;"><b>事事有回应表示：</b></span>
<span style="font-family:.PingFangSC-Semibold;color:#94110ff;"><b>我认为你对我而言是重要的（认知层面），</b></span>
<span style="font-family:.PingFangSC-Semibold;color:#94110ff;"><b>我希望感受到你传递的情绪（情感层面），</b></span>
<span style="font-family:.PingFangSC-Semibold;color:#94110ff;"><b>我相信我的回应对你、对这段关系都是有必要的（信念）。</b></span>


**回应的正确打开方式是：**
1. **我们大胆直接表达需求，对方准确get到。对于回应方来说**，事事有回应的前提是识别对方传递的需求。**最忌按照自己所想办事，而根本不顾及对方到底想要什么。**一个简单的识别方式就是：如果对方在叙述完事情后，后续说的是具体的困境，这就意味着对方可能需要你做些什么，但如果对方谈论的是自己的感受，或许更需要你提供一些情感上的支持。当然，你如果觉得不确定的话，也可以再问问对方，“我能为你做些什么吗？”
2. **约定好关键事情、时刻，尽量做到及时回复。找个机会聊聊你们认为特别需要对方在的时刻或场景，让对方心里对此有个数。**也推荐观看爱情三部曲《爱在黎明破晓前》《爱在黄昏日落时》《爱在午夜降临前》等片子，**看看电影里的人都是如何回应以及“完美错过”关键时刻回应的。**
3. **使用正确的回应方式。**
4. **等回应方多点耐心，回应方无法及时回应时记得解释下**
