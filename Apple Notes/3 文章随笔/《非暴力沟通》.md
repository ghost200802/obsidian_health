# 《非暴力沟通》

暴力沟通的四大来源
1、道德评判
2、比较
3、回避责任 （把“我不得不”改成“我选择”）
4、强人所难

**不带评论的观察是人类智力的最高形式**

区分观察（陈述事实）和评论（总结性判断）

绝大部分痛苦来自“无法准确表达自己的需求”
绝大部分愤怒来自“自己的需求未被满足”

<span style="font-family:.PingFangSC-Semibold;color:#ff260ff;"><b>非暴力沟通的</b></span><span style="color:#ff260ff;"><b>4个步骤</b></span>
1⃣️ 陈述观察（对方的行为）
2⃣️ 表达感受（自己的感受）
3⃣️ 询问原因（对方的需求）
4⃣️ 提出需求（务必精准，切忌模糊）

<span style="font-family:.PingFangSC-Semibold;color:#ff260ff;"><b>表达感激</b></span><span style="color:#ff260ff;"><b>3个步骤：</b></span>
1⃣️ 陈述对方做了什么行为使我们的生活改善
2⃣️ 陈述对方满了我们哪些需求
3⃣️ 我们现在心情如何？

如何给人提意见，可使用BIC工具：
B：Behavior，陈述对方行为
I：Impact, 陈述影响（短期局部看到的负面效果，叫影响）
C：Consequence, 长期的后果，要和这个人自身的利益挂钩，
