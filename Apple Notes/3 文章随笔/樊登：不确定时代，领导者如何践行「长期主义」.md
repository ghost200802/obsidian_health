# 樊登：不确定时代，领导者如何践行「长期主义」

### <span style="color:#ff260ff;"><b>1）幂次曲线是什么？</b></span>

<span style="font-family:.PingFangSC-Semibold;color:#432ffff;"><b>我们过往的大量记忆都是你想象的。</b></span>
**当人们讲自己过往的故事、干货、经验时，讲出来的内容和过去实际发生的客观情况，两者之间的差异是非常大的。**
公司里领导者的职位高，只是因为你愿意承担责任且恰好是公司的创始人，并不是领导者比下属聪明

<span style="font-family:.PingFangSC-Semibold;color:#432ffff;"><b>幂次曲线不是一开始就增长的，在爆炸性增长前还有一段几乎是横线的积蓄期。</b></span>

### <span style="color:#ff260ff;"><b>2）如何熬过幂次曲线的横线？</b></span>
1. <span style="font-family:.PingFangSC-Regular;color:#432ffff;">意义：</span><span style="font-family:.PingFangSC-Regular;">事业的意义来自于他人。当你能够看到这件事对社会的影响时，你的动力会强劲且持久。</span>
2. <span style="font-family:.PingFangSC-Regular;color:#432ffff;">乐趣：</span><span style="font-family:.PingFangSC-Regular;">一个人获得乐趣是能力，而不是一个状态。创业一定是你自己走，然后小心摸索，看看怎样能够走出一条完全不同的道路。所以乐趣这件事情是能够带领我们走出很多坑的东西，因为你乐此不疲。</span>
3. <span style="font-family:.PingFangSC-Regular;color:#432ffff;">反脆弱</span><span style="color:#432ffff;"><b>：</b></span><span style="font-family:.PingFangSC-Regular;">反脆弱生意模型的最大特点就是</span>——<span style="font-family:.PingFangSC-Semibold;"><b>成本是有底的，但收益的上限是不确定的，收益多大都有可能，这个就是反脆弱的生意。</b></span>


<p style="text-align:justify;margin:0">意义和乐趣代表着你愿意坚持下去，<b>反脆弱代表着你有能力坚持下去。</b>
