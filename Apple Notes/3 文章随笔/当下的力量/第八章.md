# 第八章

大部分人追求肉体上的欢愉或者各种形式的心理满足，因为他们相信这些事情会使他们幸福或者会将他们从恐惧或匮乏的感觉中解放出来。幸福可以被视为一种源于肉体享乐的富有活力的感觉，或者是一种源于某种心理满足的更为安全、更为圆满的自我感觉。这就是从不满足或匮乏感中去寻找拯救。

真正的拯救是成就满足，是和平，是生命的圆满。<span style="color:#432ffff;">它就是做你自己，在你的体内感受没有对立面的美善。</span><span style="font-family:.PingFangSC-Regular;">在这种状态中，你本体的喜悦不依赖于任何外界的事物；真正的拯救是了解到，你是那个滋生万物的无时间、无形式的至一生命不可分割的一部分。</span>

你的大脑说，在你被解放或变得圆满之前，你需要时间来寻找、挑选、达到、获得、成为或理解某事。你视时间为达到救赎的手段，事实上时间是救赎的最大障碍。

你认为，此刻由于你还不够圆满或不够好，所以你不能从你的现实情况中到达那里，实际上，此时此地是你到达那里的唯一途径。所以拯救的方法不止一个：<span style="color:#432ffff;">你可以利用任何一个条件，但不需要任何特殊的条件</span><span style="font-family:.PingFangSC-Regular;">。</span>

大部分的爱情关系似乎都会变得爱恨交织。爱给予你的痛苦就像给予你的欢乐一样多。通常，<span style="color:#432ffff;">情侣们对这种周期性变化都很上瘾。</span><span style="font-family:.PingFangSC-Regular;">这样戏剧化的方式使得他们活得更带劲儿！</span>

<span style="font-family:.PingFangSC-Regular;">浪漫的爱情关系 VS 真爱</span>
<span style="font-family:.PingFangSC-Regular;color:#432ffff;">真爱是没有对立面的，因为它是超越大脑而产生的。爱是难以持久不变的</span><span style="font-family:.PingFangSC-Regular;">，就像有意识的人一样，非常稀少。</span>

我们都倾向于把爱情存在的障碍归因于爱情关系的消极面；同样，你也可能把障碍的产生归因于你的伴侣，而不是你自己。

在积极的一面，你与你的伴侣相爱。一开始，这是一种深深的满足状态。你感到充满活力。你的存在突然变得有意义起来，因为有人想念你，需要你，使你变得特殊，对对方来说也是一样。当你们在一起的时候，你感觉很圆满。在这种强烈的感觉中，有需要和依赖的成分。

<span style="font-family:.PingFangSC-Regular;"><u>沉溺上瘾和追寻圆满</u></span>
人类从生理和心理2个方面都在追求圆满。

<span style="font-family:.PingFangSC-Regular;color:#432ffff;">只要你认同你的思维，你的自我感就是源于外在。</span><span style="font-family:.PingFangSC-Regular;">也就是说，你从那些与你的自我感丝毫没有任何关系的事情中</span>——<span style="font-family:.PingFangSC-Regular;">你的社会角色、财产、外表、成功与失败、信仰等</span>——<span style="font-family:.PingFangSC-Regular;">寻找你的自我感。</span>

如果在爱情关系中，你既体验到了爱，又体验到了爱的对立面——攻击和情感暴力等——那么<span style="color:#432ffff;">你很可能就是将自我依恋和沉溺上瘾与爱混淆在一起了。</span><span style="font-family:.PingFangSC-Regular;">你不可能在这一刻爱你的伴侣，而在下一刻马上攻击他或她。</span>

真爱是没有对立面的。如果你的“爱”有对立面，那么这就不是真爱，而是小我对更深层、更完整的自我感的需求，一种其他人暂时能满足的需求。

<span style="font-family:.PingFangSC-Regular;color:#432ffff;">关系本身不会造成痛苦和不快乐，它们只是将已经在你内在的痛苦和不快乐引发出来。</span>

如果他们将注意力集中在当下，<span style="color:#432ffff;">他们要面对的第一件事可能就是自己的痛苦</span><span style="font-family:.PingFangSC-Regular;">，这是他们所恐惧的。</span>

<span style="font-family:.PingFangSC-Regular;"><u>从上瘾到开悟的爱情关系</u></span>

<span style="font-family:.PingFangSC-Regular;">我们能将上瘾的爱情关系转化成真正的爱情关系吗？？？</span>
<span style="font-family:.PingFangSC-Regular;color:#94110ff;">请将你的注意力更多地集中在当下时刻，</span><span style="font-family:.PingFangSC-Regular;">这会使你更多地感受临在并加强你的存在感。</span>

<span style="font-family:.PingFangSC-Regular;color:#432ffff;">爱情最伟大的催化剂就是完全接受你伴侣的一切，而不是去批判或以任何方式改变他或她。</span>

事实上，当你通过接受事实而让批判停止时，你就从你的思维中解放出来了。这样，你就立即超越了小我。所有的思维游戏和沉溺依赖都将消失。<span style="color:#94110ff;">（如果对方做不到呢？）</span>

爱不需要依赖一个人、一个外在的形式才能存在。在你临在的定静中，你可以感觉到你的无形式和无时间的本质，就是赋予你肉体生命的那个未显化状态。这样，你就能感觉到在其他人和其他生物内的相同深度的生命。你的观察超越了形式和分离的屏障。这就是合一的体现。这就是爱。

对真爱感受的强烈程度却因人而异。<span style="color:#432ffff;"><b>如果某人对你的爱的反应比其他人对你的爱的反应更清晰、更深，并且你对这个人也有着同样的感觉，这时，我们可以说你和他或她是在爱情关系中。</b></span><span style="font-family:.PingFangSC-Regular;">你和那个人的联结，和你与公交车上坐在你的旁边的人，或鸟、树、花的联结都是一样的，只是</span><span style="font-family:.PingFangSC-Regular;color:#432ffff;">你对这种联结感受的程度不一样</span><span style="font-family:.PingFangSC-Regular;">。</span>

每一个危机不仅代表着危险，也蕴藏着机会。数百万人现在过着单身生活，或者成为单身父亲或母亲，无法再建立爱情关系或者不愿意重蹈覆辙。其他人则从一段关系走到另外一段关系，在欢乐——痛苦的周期里挣扎，不断地从异性那里追求满足和不可捉摸的目标。还有一些人则为了孩子或安全感、习惯性、害怕孤单或其他共同利益的约定等而妥协，有的是因为无意识地对刺激的情绪障碍和痛苦上瘾，从而继续生活在这种充满痛苦、失调的爱情关系中。

如果爱情关系强化了自我思维模式并激活了痛苦之身，你为什么不接受这个事实而要去逃避呢？为什么不去与关系共存，而要去避免亲密关系或继续幻想理想伴侣的出现，来解决你的问题或给你满足感？<span style="color:#432ffff;">承认和接受这些事实，也会使你从中获得一定程度的解放。</span>

如果你对你伴侣的所有无意识行为都做出反应，你自己就会变得无意识，但是如果你能记得去觉知你的反应，你就不会迷失。

爱情关系不是用来使你快乐或满足的，如果你仍想通过爱情关系来获得拯救的话，那你将会一次又一次地遭受挫折。<span style="font-family:.PingFangSC-Regular;">但是如果你能承认爱情关系不是为了让你更幸福，</span><span style="color:#94110ff;">而是让你更有意识</span><span style="font-family:.PingFangSC-Regular;">，那么这种关系反而会为你提供拯救机会。你也会与更高的意识相结合，而这更高的意识正是想通过你而来到这个世界。</span>

明智，即意识，只有通过你才能来到这个世界上。<span style="color:#432ffff;">你的开悟不需要去等待这个世界变得明智，或别人变得有意识。</span><span style="font-family:.PingFangSC-Regular;">不要相互指责对方的无意识行为。从你开始争论的那一刻起，你就开始认同你的心理观点，并且你不仅在为这种心理观点辩护，而且还在为你的自我感辩护。这时，小我就占了主导地位。</span>

你可以指正伴侣的行为，可以在这样做的同时，不受小我的干扰——<span style="color:#432ffff;">不带责备、控诉或好胜心</span><span style="font-family:.PingFangSC-Regular;">。</span>

<span style="font-family:.PingFangSC-Regular;color:#432ffff;">去除破坏爱情关系的两个因素，</span><span style="font-family:.PingFangSC-Regular;">痛苦之身被改变，你也不再认同你的思维和心理立场：</span>
1. 学会在不责备对方的情况下表达你的感受，学会用一种开放的、非防御性的方式倾听你伴侣说话。请保持临在。
2. 给别人和你自己一些空间

判断是否开悟的另外一个方法是，<span style="color:#432ffff;">看一个人在困难或充满挑战的环境中或当事情出错时如何行事。</span>

<span style="font-family:.PingFangSC-Regular;color:#424242ff;">当两方都被自己痛苦之身所控制时，一种无意识的情感暴力、野蛮攻击和反击就出现了。</span>

<span style="font-family:.PingFangSC-Regular;color:#424242ff;">为什么女人更容易开悟？女人更容易去感受，去进入她的内在身体，所以她会自然地比男人更为贴近存在，更接近开悟状态。</span>

女性的痛苦之身通常有集体和个人两个层面。

<span style="font-family:.PingFangSC-Regular;"><u>放弃和你自己的关系</u></span>

你需要与你自己建立一个关系吗？你为什么不能成为你自己？当你与自己建立关系时，你就将你自己一分为二：​“我”和“我自己”​，主体和客体。这种由思维创造的二元性是你生活中所有问题和冲突的根源。在开悟状态中，你就是你自己，你与你自己合二为一。你不会批判你自己，你不会为你自己感到遗憾，你不会为你自己感到骄傲，你不会爱你自己，你也不会恨你自己。<span style="color:#432ffff;">开悟，不会再有一个需要你去保护、防卫和喂养的自己了。</span><span style="font-family:.PingFangSC-Regular;">你不会再有一种关系：就是你与你自己的关系。一旦你放弃了这种关系，你所有的其他关系都将会是爱的关系。</span>
