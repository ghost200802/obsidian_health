# 《你要如何衡量你的人生》

它的作者叫克里斯坦森。（创新者的窘境 作者） 得癌症后写下这本书。

他把人生分成了三块，如果你希望让自己的人生变得成功的话，有三块很重要。第一块就是你要有一个成功的事业；第二个，你要有很好的家庭和朋友的系统，就你需要有家人和友谊的支撑。第三块就是你要保持正直。

你的人生有没有战略？战略是由两样东西构成的，第一个是你的目标。战略不仅仅是目标，但是必须得有目标。第二个就是你将如何去实现这个目标。把这两个东西结合起来，这才叫作战略。

那么首先我们来解决目标的问题，你希不希望自己能够拥有一个能够给你带来真正激发良好动力的工作？

所以比激励理论更重要的是动因理论。那么哪些东西会给你带来发自内心的愿望呢？这里边总结了几点，一个叫作有挑战性。你需要做一些能够带来挑战性的事，这个会给你带来动力。第二个叫作获得认可，你在工作当中，在学习当中能够获得社会给你的正向反馈，让你觉得好棒。第三个叫责任感，就你觉得你应该做这样的事，做这样的事是对的。第四个就是在做这个事的时候，你能够得到个人的成长，你的个人能够不断地得到成长。

第二个要解决的问题是“在生活当中虽然有了目标，但你经常会遇到更多的机会。”

那么你要不要判断你该不该做一件事，判断你要不要岔开去抓住新的机会，还是放弃那个机会，咱们不能笼统地说放弃还是去做。

那到底哪些机会该抓住，哪些机会不该抓住？这是不是一个很困难的问题？所以在这个地方你需要学会一个理论，叫作<b><u>发现驱动计划理论</u></b><span style="font-family:.PingFangSC-Regular;">。什么叫发现驱动计划理论？我们在这个五十页我做了一个摘抄。就是接受一项任务之前，你应该考虑什么问题？</span>
1. <span style="font-family:.PingFangSC-Semibold;"><b>能帮你做这些关键事情的人，是不是需要牺牲自己的利益来帮你呢？</b></span><span style="font-family:.PingFangSC-Regular;">换句话说是不是因为要帮助你完成这项工作？先要假设这些人都是无私的。</span>
2. <span style="font-family:.PingFangSC-Regular;">同样重要的是问问你自己，</span>**哪些假设条件需要被验证是真实的，你才会对选择感到满意**？
3. <span style="font-family:.PingFangSC-Semibold;"><b>你做决策是根据外在的还是内在的动因？为什么你认为这会是你自己喜欢的，你有什么证据来证明吗？</b></span>
这一段话其实很重要在哪儿呢？就是在你做一个决策之前，你要问一下自己这段话。就是这个事能不能成功，它需要满足哪些前提？这些前提都能够存在，这件事才有成功的可能性。而如果这个前提里边有一项是虚假的，这件事是成功不了。

所以今后你人生当中当你遇到很多诱惑，你觉得这事我要不要去做的时候，你要问自己什么？这需要有哪些前提满足才能够实现这件事，这些前提通过我的努力能不能实现，还是说它就只是一个虚假的前提？用这种方法来决策我们到底该不该去抓住这些机会，面对这种突发的机遇，这是我们做人生战略的第二个大问题。

人生战略的第三个问题就是，你的战略需要和你的执行相配合。

你在战略层面说你要去做掌上型的，你在执行层面根本就没有给到他相应的支持。所以这就是我们说叫作，这个名词叫作错位激励，你所激励的事和你想要的东西根本达不到。

三个人生的战略

第一个就是找到一个能够让自己动心的，能够给自己带来内在激励的目标，这是动因理论。
第二个是面对取舍的时候，你要用发现驱动计划来问自己这个事的前提存不存在，来决定要不要抓住这个机会。
第三个，一旦你确定了该做这个事，一定要分配给它足够的时间。

有钱的公司甚至不考虑怎么样是对，有钱的公司只考虑这事是不是按计划在做，这就是大公司病。
