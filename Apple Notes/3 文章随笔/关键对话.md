# 关键对话

关键对话的三种情况：
1. 升职、加薪、面试、去女朋友家、第一次相亲
2. 对话双方的情绪很激烈，对话双方生气，两个人都在大声地嚷嚷
3. 双方的意见存在很大的分歧，你希望你的孩子不要玩游戏，而你的孩子特别喜欢玩游戏

最好的原则：你要对对方保持百分之百的尊重，同时做到百分之百的坦诚。

生气是为了找到一个借口和理由说这是怪他，如果不是他的话，我就不会这样。

<span style="font-family:.PingFangSC-Regular;color:#ff260ff;">极怒时莫与人书，极喜时莫与人物</span>

ABC原理认知模型。
A是事件发生，
B是你对这个事的看法，
C是你的压力状况。
<span style="font-family:.PingFangSC-Regular;color:#ff260ff;">很多人穷其一生都在试图通过改变</span><span style="color:#ff260ff;">A的方式来改变C。</span>

<span style="font-family:.PingFangSC-Semibold;color:#ff260ff;"><b>我们要改变自己对这个世界的看法，来解决事，而不是责怪他人。责怪他人，并没有帮助你把这个事做得更好。</b></span>

怎么进行关键对话？
1. <span style="font-family:.PingFangSC-Regular;">做准备，调整好心态，把自己的心管好（</span><span style="color:#ff260ff;">我们所有的苦恼都不是由他人引起的。</span>当你认为你所有苦恼都是由他人引起的时候，你会发现你特别容易生气。）
每次对话前问自己：
1⃣️ 这次对话我的目标是什么？我希望自己达成的目标是什么？

2⃣️ 这次对话之后，我希望为对方达成的目标是什么？

3⃣️ 这次对话之后，我希望为我们俩的关系达成一个什么样的目标？

4⃣️ 为了达成这些目标，我应该怎么做？


<span style="font-family:.PingFangSC-Regular;color:#ff260ff;">当你被对手激怒的时候，你发现你的智商就是零。</span>

<span style="font-family:.PingFangSC-Regular;color:#ff260ff;">双核对话：</span><span style="font-family:.PingFangSC-Regular;">一个核负责处理谈话的内容，一个核负责处理谈话的氛围（更重要）</span>
