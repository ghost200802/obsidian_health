# 群公告

欢迎参加善举日记打卡第9期活动！

善举日记是以“善良的行动”为核心，通过记录我们在生活中发送善意、接收善意以及看见善意发生的瞬间，为自己打造一个稳定的、生生不息的高能量系统。

<span style="font-family:.PingFangUITextSC-Regular;">我们将在日常生活中捕捉四类可被记录的“善意行为”</span>—— 1️⃣ <span style="font-family:.PingFangUITextSC-Regular;">善待自己（能量自生）</span> 2️⃣ <span style="font-family:.PingFangUITextSC-Regular;">善待他人（能量外放）</span> 3️⃣ <span style="font-family:.PingFangUITextSC-Regular;">被他人善待（能量接收）</span> 4️⃣ <span style="font-family:.PingFangUITextSC-Regular;">看见他人善待他人</span>（能量扩散）

这里没有压力、没有比较、没有评价。只有主动分享和记录生活中的善良与美好。群内随时以文字/图片形式分享即可。

❗️加入此群后，即代表同意：
1️⃣ 不在群里发布糟糕消息、不做冒犯他人的表达
2️⃣ 尊重他人，鼓励他人，不对他人的分享做评价
3️⃣ 连续4天未分享，会被抱出群
