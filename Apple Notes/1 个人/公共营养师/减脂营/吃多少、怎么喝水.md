# 吃多少、怎么喝水

“一掌半蛋白、一拳碳水、两拳蔬菜、一拇脂肪。”

1掌不含手指

视觉分盘法

| 🥬 蔬菜 | 50% | <span style="font-family:.PingFangUITextSC-Regular;">清炒西葫芦</span>+胡萝卜丝 |
| -- | -- | -- |



| 🍗 蛋白质 | 25% | <span style="font-family:.PingFangUITextSC-Regular;">鸡胸肉</span>/鱼/鸡蛋 |
| -- | -- | -- |



| 🍠 碳水 | 20% | <span style="font-family:.PingFangUITextSC-Regular;">山药</span>/红薯/小米/藜麦 |
| -- | -- | -- |



| 🥑 脂肪 | 5% | <span style="font-family:.PingFangUITextSC-Regular;">橄榄油</span>/亚麻籽/牛油果 |
| -- | -- | -- |



运动后或经期，碳水可以增加至30%或1拳半


✅ 如果她早上起床**有精神、体温正常、手脚温暖、运动后能恢复、情绪稳定** → 碳水比例是合适的。
⚠️ 如果她出现：
- 体温偏低（36℃以下）
- 一下午就疲惫
- 运动后恢复慢
- 晚上睡不着、易饿、情绪波动 👉 就说明碳水太低，需要**适当上调到45%左右**。
