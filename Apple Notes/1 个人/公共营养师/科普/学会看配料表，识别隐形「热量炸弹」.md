# 学会看配料表，识别隐形「热量炸弹」

提问：减脂期的我，下午3点饿了，根据以下食物的【配料表】，选出最适合减脂期的食物

A. 巧克力
B. 黑全麦小面包
C. 无糖山药沙琪玛
D. 酱牛肉

选出答案，并说明你为什么选这个哦～

————————————

 我来公布答案啦！正确答案是【D. 酱牛肉】

原因是：

<span style="font-family:.PingFangUITextSC-Regular;">配料表越短越好，</span>简单干净的食材，才更利于减脂和健康。

其它3个选项为何不可呢？

【牛奶巧克力】
1️⃣ 配料表**看前三位**<span style="font-family:.PingFangUITextSC-Regular;">，前三里有糖或油的名词，基本都是高糖高油，减脂直接避开！</span>

【黑全麦小面包】
1️⃣ 配料表**看前三位**。
宣称是全麦，但配料表第一位是小麦，它是假全麦面包！
2️⃣ 配料表前三里有起酥油
是高油食物，热量爆表，减脂直接避开！

【无糖山药沙琪玛】
1️⃣ <span style="font-family:.PingFangUITextSC-Regular;">无糖</span>≠ 没有糖
排第一、第三分别是麦芽糖醇和植物油是热量炸弹！吃了容易长胖，影响代谢！
2️⃣ 配料表也很长！如果有一堆看不懂的化学名词，直接不选！

## **No.1 先看前 3 位**

<span style="font-family:.PingFangUITextSC-Regular;">配料表</span>**越靠前，含量越高**。
<span style="font-family:.PingFangUITextSC-Regular;">如果前</span> 3 位出现：
<span style="font-family:.PingFangUITextSC-Regular;">白砂糖、蔗糖、麦芽糖浆、植物油、棕榈油、起酥油</span>……
<span style="font-family:.PingFangUITextSC-Regular;">基本就是</span>**高糖高油**，减脂直接避开！

<span style="font-family:.PingFangUITextSC-Regular;">🍞 全麦面包</span>→第一位必须是"全麦粉"
⚠️ 小麦粉排第一 = 假全麦

🥛酸奶→ 第一位必须是"生牛乳
⚠️ 水或糖排第一 = 饮料

## **No.2 配料表越短越好**

<span style="font-family:.PingFangUITextSC-Regular;">配料表越短越好，</span>简单干净的食材，才更利于减脂和健康。
配料表越长，代表**加工越多、添加剂越杂**，越不利减脂和健康。

## No.3 警惕「伪健康」陷阱

<span style="font-family:.PingFangUITextSC-Regular;">零糖</span> ≠ <span style="font-family:.PingFangUITextSC-Regular;">无糖。</span>
可能用果糖、麦芽糖、代糖，依然会影响体重和代谢。

<span style="font-family:.PingFangUITextSC-Regular;">非油炸</span> ≠ <span style="font-family:.PingFangUITextSC-Regular;">低脂。</span>
不油炸不代表没放油，一定要看**营养成分表**的脂肪含量。

<span style="font-family:.PingFangUITextSC-Regular;">零添加</span> ≠ <span style="font-family:.PingFangUITextSC-Regular;">无任何添加剂。</span>多是营销话术，通常只代表 “没加某一种”。

关键要看配料表，如果有一堆看不懂的化学名词，直接不选！
