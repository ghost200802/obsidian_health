# 主题：食欲和减肥调控的关系

我们常说减肥难，其实难的不是运动，也不是少吃几口食物，而是——食欲。

<span style="font-family:.PingFangUITextSC-Regular;">很多人都会有这种感受：明明不饿，</span>也知道身体不需要吃东西，但就是觉得嘴巴寂寞，想给它投喂点吃的。

我们以为是自己自制力差，其实不完全是。

食欲不是一个念头，而是一整个系统在推动你吃东西。

今天我带大家弄懂：食欲到底怎么来的？我们怎么做能让食欲稳定一点？

🌟 一、关于“食欲”的三个大误解

1<span style="font-family:.PingFangUITextSC-Regular;">、</span>一饿就是低血糖
<span style="font-family:.PingFangUITextSC-Regular;">很多人一有“想吃”的感觉，就以为是低血糖。但在临床上，真正发生低血糖的情况其实并不多。我们的身体有很强的保护机制，</span>对于没有相关疾病的健康人来说，很难在正常一餐的间隔内，把血糖降到真正危险的水平。

**大多数我们感受到的“饿”，更多来自激素波动（如血糖下降）、饮食习惯（到饭点了）、情绪或大脑信号，而不是真正的血糖过低。**

2<span style="font-family:.PingFangUITextSC-Regular;">、</span>热量缺口越大，食欲一定越强

热量缺口确实会影响食欲，但它并不是唯一决定因素。很多人即使吃够了，依然会“嘴巴寂寞”，这往往已经不是能量问题，而是食物奖赏、习惯或情绪在起作用。

3<span style="font-family:.PingFangUITextSC-Regular;">、</span>馋某种食物，是身体缺某种营养

比如“想吃炸鸡是缺钙”“想吃巧克力是缺镁”，这些说法听起来很有安慰效果，也符合大众习惯性思维——“因为A所以B”。但该说法**并没有可靠的科学证据支持**。这些说法真的挺离谱的。

总的来说，食欲从来不是单一原因造成的，而是多种信号叠加后的结果。从我们进食的那一刻起，各种饱腹感信号就开始工作了，直到停止进食；随着空腹时间不断增加，饥饿感信号开始工作，直到开始进食。这样的循环每天都在我们身上反复发生。

**🌟 二、我们为什么会“想吃”？**

1、食物奖赏：不是我馋，是它真好吃啊

导致人们过量进食的最大诱因：食品做得越来越好吃了。（注：加工后的食物叫食品，纯天然的原食材叫食物）

有些食物就是特别容易让人停不下来，这并不是你的问题。只要它：又香、又油、又甜或咸、口感层次丰富，或者某些特定口感（哽啾啾、糯叽叽、软fufu等），大脑里的奖赏系统就会被激活，让你产生明显的愉悦感。就是这种愉悦感，不断强化“吃它”的行为。

这也是为什么零食、甜品、外卖往往不容易真的吃饱，却很容易吃多。

好消息是：这种食物奖赏机制是可以被替换的。我们的大脑能通过奶茶/炸鸡/披萨产生愉悦感，它也能通过干净健康的家常菜产生愉悦感和满足感。

🔔 <span style="font-family:.PingFangUITextSC-Regular;">这里科普一个营养学和行为心理学的重要小概念：【</span>感觉特异性饱足感】

<span style="font-family:.PingFangUITextSC-Regular;">很多实验发现：</span>
<span style="font-family:.PingFangUITextSC-Regular;">当食物种类越丰富、味道越多样时，人往往会吃得更多；而当</span>持续吃同一种味道时，对它的愉悦感会下降，更容易产生“吃腻了”的感觉，从而自然停下来。

<span style="font-family:.PingFangUITextSC-Regular;">这也是</span>我反复强调进食顺序的原因。它不是为了限制你，而是帮你把被环境“驯坏”的口味，慢慢养回原始状态。市面上很少有减肥教练关注到这一点，但它被行为心理学反复验证，对减少进食量、恢复原始口味是有效的。甚至，换个更小的餐盘，也会有助于减少进食量。

2<span style="font-family:.PingFangUITextSC-Regular;">、</span>来自胃肠道的信号：身体在告诉你“饱了没有”

我们的肠胃道会不断向大脑发送“饿了该吃、饱了该停”的信号。这些信号很多，我们今天主要说几个重要的信号：

（1）胃扩张
当胃被撑起来时，大脑会更容易收到“已经吃够了”的信号。

所以减肥期间，多吃【大体积、低热量】的食物。

（2）胃促生长素（科普中听到的“饥饿素”就是它）
它是人体内唯一一个“饿的时候分泌多，饱的时候分泌少”的生长激素。

当我们睡眠不足、压力大、作息混乱时，它就容易升高，我们更容易感到饥饿，也更想吃高热量食物。

研究发现：很多肥胖者因为长期不当的饮食习惯，会导致饥饿素分泌异常，比如长时间不进食但感受不到饥饿（饥饿素分泌得不够多），进食后需要更长时间产生饱腹感（饥饿素下降得较慢）。

所以，我们保持睡眠充足和适度饮食，对好的减肥效果和健康的身体都大有裨益。

<span style="font-family:.PingFangUITextSC-Regular;">（</span>3<span style="font-family:.PingFangUITextSC-Regular;">）</span>CCK（缩胆囊素）
当我们摄入蛋白质+脂肪时，CCK会被分泌出来，让我们吃得更慢、更有饱足感。

这里的脂肪应当是来自坚果、牛油果、海鱼等富含omega-3的食物。

<span style="font-family:.PingFangUITextSC-Regular;">所以，若是早餐和午餐、或午餐和晚餐的间隔时间长（</span>5-6<span style="font-family:.PingFangUITextSC-Regular;">个小时），中途也不方便加餐，是很适合吃坚果</span>+<span style="font-family:.PingFangUITextSC-Regular;">鸡蛋</span>+<span style="font-family:.PingFangUITextSC-Regular;">牛奶做早餐，或中午吃牛油果鸡肉沙拉之类的食物。</span>当然，具体搭配仍需要根据个人肠胃、食欲水平和减脂阶段灵活调整。

<span style="font-family:.PingFangUITextSC-Regular;">（</span>4<span style="font-family:.PingFangUITextSC-Regular;">）</span>胰高血糖素样肽 （GLP-1，很多减肥药利用的机制）

它是体内一种抑制食欲的激素。它正常分泌的情况下，可以让我们更早感受到饱腹感、稳定食欲、降低进食欲望。

这个点是单纯为了给大家科普。因为当前市面上最火的减肥神药“司美格鲁肽”就是基于它研发出来的GLP-1受体激动剂。核心机制是通过抑制食欲，减少进食量，进而达到减肥的效果。但司美必须搭配一系列健康的减肥行为才能真正发挥作用，否则就是短期效果，一旦停药很快又会吃回去，和节食减肥后反弹的原理相似。

研究也发现，很多超重或肥胖人群体内的GLP-1确实存在分泌不足的情况。

但饮食方面依然有方法能帮助我们重新唤醒体内的“减肥剂”：

1、多吃全谷物的食物，如糙米/豆类，燕麦是最方便的。燕麦里麸皮对GLP-1的刺激作用表现很优秀。

4、按照方案里的进食顺序吃饭
5、多吃【能看见原食材的食物】

3<span style="font-family:.PingFangUITextSC-Regular;">、</span>代谢、激素和长期状态

**（1）瘦素**

它是一种饱腹感激素。脂肪越多，瘦素水平往往越高，但在肥胖状态下，大脑对瘦素的信号敏感性可能会下降，这就是常说的“瘦素抵抗”。

就会出现：**明明吃够足量的食物，大脑却还是觉得“不够”。**

**（2）胰岛素**

长期胰岛素水平偏高，一方面会促进脂肪储存，另一方面也常常和血糖波动、情绪变化一起出现，让人主观上更容易觉得“老是想吃点什么”。

（3）女性生理周期也会影响食欲

很多女性在经期前或经期更容易想吃甜的、热乎的或碳水类食物，也更容易出现情绪性进食。

这主要是和孕激素、雌激素的周期性波动有关。

一般情况下，经期结束后的一周内，雌激素逐渐升高，食欲通常比较稳定，精神状态也很好，可以通过增加一点热量缺口来达到更好的减肥效果。经期前孕激素上升，食欲会上升，建议适当增加一点主食，有助于稳定食欲。个体表现也略有差异。

这是一种常见也合理的生理现象，反映的是激素变化对食欲的影响，而不是意志力失控。

**🌟** 四、那我们能做些什么，让食欲慢慢稳定下来？

大道至简，所有的方法总结为一句话就是：用身体喜欢的方式去对待它。

1、与其靠自制力少吃，不如让自己真正吃饱

多用大体积、低热量、富含纤维的食物填饱肚子。比如冬瓜、黄瓜、西红柿、柚子、草莓等蔬果，能明显提高饱腹感，又不会有热量负担。

2<span style="font-family:.PingFangUITextSC-Regular;">、</span>按方案里的进食顺序吃

根据你们不同的食欲水平、饮食习惯，方案里设计的进食顺序略有不同，但都是贴合你们实际情况、对当下减肥最有帮助的顺序。

3、减少加工食物，多吃天然食物

很多人都发现，食物吃得越原始，食欲就越稳定。多吃能看见原食材的食物，参与的原材料越少越好，比如青椒炒牛肉，原材料只有生牛肉、青椒、盐、蒜末。

这个部分后面和学习看食物配料表一起详细说，教大家怎么选既有助于减肥又不会难吃的食物。（挖坑1）

4<span style="font-family:.PingFangUITextSC-Regular;">、</span>把睡眠当成减肥的重要环节

长期睡不好、睡不够的人，往往更容易觉得“不满足”。不仅在食欲方面不满足，在情绪方面也更容易波动，身体更容易处在应激状态，脂肪也更难分解。

5、学会识别触发“食欲”的诱因

我们每天生活节奏都很快，时刻都在接收大量来自手机/身边的信息。你某一刻产生“食欲”的原因可能非常多：单纯口渴了、别人提起甜品名称、别人吃的外卖香味、手机刷到的美食视频、一直没完成的KPI、收到高难度的工作任务、千辛万苦完成工作任务的痛快、感情进展不顺利，等等等。

想吃的时候，用我们的“10秒暂停法”，问问自己：我现在是身体饿，还是其他原因？

很多冲动在你思考的这10秒里，就会慢慢平息。
