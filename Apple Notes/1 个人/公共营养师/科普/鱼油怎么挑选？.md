# 鱼油怎么挑选？

一、先判断要不要补、每日吃多少

**基准：每周深海鱼 2～3 次足量**

1. ✅健康人、吃鱼达标、压力小：不用补
2. ✅健康人、吃鱼达标但长期高压：EPA+DHA 500～1000mg / 天
3. ✅健康人、吃不到足量深海鱼：
- ✅作息好压力小：250～500mg / 天
- ✅炎症高作息差：1000～2000mg / 天
1. ✅高血脂甘油三酯偏高、慢性炎症：2000～4000mg / 天，＞1000mg 建议遵医嘱

> <span style="font-family:.PingFangUITextSC-Regular;">备注：</span><span style="color:#ff260ff;">高胆固醇不能靠鱼油下降</span><span style="font-family:.PingFangUITextSC-Regular;">；植物亚麻酸转化率极低，没法替代鱼油。</span>

二、挑选产品 4 个硬性标准

1. **看有效量：只算 DHA+EPA 合计，不看胶囊净重**
2. **分子形态优先排序：rTG＞TG＞EE；EE 务必随餐吃提升吸收**
3. **新鲜度：TOTOX＜20，优选 10 以内，哈喇异味直接弃用**
4. **质控：带 IFOS 认证，可批号官网溯源单批次检测报告**

三、按需选 EPA/DHA 配比

1. 降脂、抗炎、护血管：EPA 偏高（EPA:DHA≈2:1～3:1）
2. 孕期 / 哺乳期 / 婴幼儿：高 DHA 配方
3. 无通用黄金配比，商家黄金比例多营销噱头

四、鱼油 / 藻油 / 磷虾油怎么选

1. 普通成人：**优选 rTG+IFOS 鱼油**，研究足、性价比高
2. 素食人群：只能选藻油（大多 DHA 高、EPA 少）
3. 磷虾油：价贵、研究少，非刚需不入手

五、服用关键提醒

1. 随正餐油脂同吃，吸收更好；
2. 营养素非药物，见效周期**3～6 个月**，多数改善体现在体检指标，体感变化微弱；
3. 不能靠鱼油减肥，仅通过抗炎间接辅助减脂。
