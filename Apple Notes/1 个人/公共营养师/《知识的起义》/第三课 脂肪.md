# 第三课 脂肪

我们的脂肪就是甘油三酯

细胞核

细胞质

脂肪：本质是脂肪酸，以甘油三酯的形式存在，城市中的油罐车，里面都是液态的甘油三酯，
形态是装着脂肪的小囊，减肥就是把每个小囊里的脂肪减少，小囊数量不会变，但容量会变大变小。

减肥的本质是：脂肪细胞的大小，瘦了就是脂肪细胞小了，胖了就是甘油三酯被利用的少了。

脂肪细胞的数量多少：成年人的数量一般是固定不变的，有2种情况会变： 1️⃣ 特别胖的话，前脂肪细胞会分化成熟的脂肪细胞；2️⃣ 抽脂是可以抽掉一部分脂肪细胞，不是自然健康的方法，属于外科手术范围，有风险。
孩童早期的脂肪细胞数量奠定了成年脂肪细胞的基础。
脂肪细胞的多少确实有一定程度的影响胖瘦，不仅仅是热量计算那么简单。

假设假设平时是个非常健康的人，一顿饭吃了500g碳水（大概4斤米饭），几乎不会变成脂肪，就算有，也很少。因为我们身体有办法把过多碳水缓冲出去，比如变成肌糖原、肝糖原，肌肉和肝脏都能超量储存，储存后大部分都烧掉了。 这样吃两三天都不会胖，但连续吃四五天是有可能会胖。

以上是不利于减肥的原因，不是直接原因。
激素正常的话，多吃一点也没事，激素不正常，稍微多吃就胖了。

2、储存

身体最喜欢了，脂肪进去脂肪储存，非常节能，只需要花3%就行。这是正常 情况下肥肉的主要来源。
我们吃进去的脂肪基本都储存起来了。

身体不正常的情况下，合成脂肪会变成主要来源。

肌内脂肪：属于异位脂肪储存，跟代谢疾病相关，肌内脂肪高，代谢疾病、胰岛素抵抗发病高。（不适用于运动员）

只要是高脂饮食，肌内脂肪必然高，而且是浮动的，和运动消耗有关。

我们需要的是消耗能力强。低脂饮食也可以。

典型的现代饮食脂肪含量都比较高，普遍都是高脂饮食。

> <span style="font-family:.PingFangUITextSC-Bold;"><b>脂肪是怎么消耗（代谢）掉的？</b></span>

细胞给细胞供能步骤：
1. 脂肪分解，分解成3个脂肪酸
2. 脂肪运输，进入血液
3. 进入细胞的线粒体，需要左旋肉碱带进去，但正常人肉碱足够用了，不需要额外补
4. 脂肪的氧化，烧了就消失了
5. 分解的脂肪酸比需要的多，没被利用的脂肪酸会在细胞里储存成脂肪，如果细胞里满了，它又返回到肝脏处理，肝脏又会把它放回血液里。（**所以能促进脂肪分解不等于脂肪减少**）

 **思考题：给肌肉塞肌糖原/肌内脂肪都会更大，健美备赛前怎么让肌肉更大**

> <span style="font-family:.PingFangUITextSC-Bold;"><b>能量代谢（热量消耗）</b></span>

食物热量
需要对食物热量了如指掌， 必须大致掌握，
甜的食物不一定热量高，健康的食物不一定热量低，**热量主要看水份，水分高热量低，脂肪高热量也高**<span style="font-family:.PingFangUITextSC-Regular;">。</span>

能量是怎么消耗出去的
1. 基础代谢：所有的计算都是估算的，每天都是浮动的
2. 食物热效应：身体消化吸收代谢储存转化，所消耗的热量，与减肥关系不大，
3. 活动热消耗：无意识和有意识的运动。体重公斤数 X 代谢当量 X 运动小时 约等于运动消耗，手表估算不准确。每日热量消耗 = 基础代谢X活动系数
