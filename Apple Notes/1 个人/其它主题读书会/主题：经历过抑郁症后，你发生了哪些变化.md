# <span style="font-family:.PingFangUIDisplaySC-Bold;"><b>主题：</b></span><span style="font-family:.AppleSystemUIFont;font-size:13pt;">经历过抑郁症后，你发生了哪些变化</span>

如果你曾经患有抑郁症，现在已经康复，不知道你是不是也偶尔思考过，**抑郁症的经历在多大程度上改变了你？在你的生命中留下了哪些「痕迹」？**

有人变得更敏感、也更柔软；
有人学会了照顾自己；
有人更能珍惜平凡生活的美好；
有人开始把生命的优先级重新排序；
有人变成了那个“自己曾经最需要的人”；
有人找到人生的意义；
…………

这次讨论会不是心理咨询、不是互相诉苦、不是比较痛苦的大小。
它是一个安全空间——我们只聊【走出来以后发生了哪些变化？】
如果你愿意，还可以分享你是如何做到改变的？
 每一句真实的感受，都可能成为别人黑暗时期的光 活着，并且愿意继续变好，本身就是一件值得被承认的事

如果你正在恢复、已经走出来、或正在路上，都欢迎你加入。

❗️报名前提示❗️
<span style="font-family:.PingFangUITextSC-Regular;">活动以积极正能量分享为主。</span>为了保护彼此，一旦报名，即代表同意：
 1）不提供建议，不分析、不评判、不对比 2）不诊断、不讨论药物，不替代专业治疗 3）内容保密，不在活动外提及他人分享的细节

活动流程：
1. 自我介绍 + 为什么来参加本次活动
2. 轮流分享：经历抑郁后，我发生了哪些变化？可以讲具体的行为变化、思维变化或人生观变化，等等
3. 听完其他人的分享，你听到了什么？、触动是什么？（不需要提建议）
4. 对未来的期许
