# 瘦了15斤的经验：早上奖励自己高热量食物

大家好，我是丹丹，10年体重管理经验，是你的私人减脂营养师。平时喜欢分享一些亲测有效、也适合大家操作的减脂方法。

<span style="font-family:.PingFangUITextSC-Regular;">我有个学员小</span>C<span style="font-family:.PingFangUITextSC-Regular;">，年纪轻轻，就已是“二高”人士了（高</span>/<span style="font-family:.PingFangUITextSC-Regular;">血</span>z/<span style="font-family:.PingFangUITextSC-Regular;">血</span>s<span style="font-family:.PingFangUITextSC-Regular;">）。来找我指导之前，小</span>C<span style="font-family:.PingFangUITextSC-Regular;">用</span>16+8<span style="font-family:.PingFangUITextSC-Regular;">减肥法，</span>每次刚掉两三斤又反弹，反反复复减肥好多次也不成功。

仔细一看她的饮食记录，我发现了关键问题——她非常爱吃高热量的东西，比如榴莲披萨、炸鸡、黄油面包。而且总是“越饿越晚吃”。

于是我建议她：你可以吃你喜欢的——但换个时间：**早上吃。**

2<span style="font-family:.PingFangUITextSC-Regular;">个月之后，她惊喜地告诉我：我居然瘦了</span>15<span style="font-family:.PingFangUITextSC-Regular;">斤！</span>白天精神好很多，晚上也不饿了。

/ / / 为什么早上吃高热量食物反而瘦了呢？

她之前早上不吃饭，中午吃很急，晚上又报复性暴食。就会陷入恶性循环。把高热量食物安排在早餐，是因为白天我们的代谢快、能量消耗大，吃进去的能量在白天马上被“用掉”。如果早上吃到“想吃的”，我们的满足感更高，晚上也不容易暴食。

/ / / 具体三餐要怎么安排呢？

✅ 早餐：可以放一点“快乐食物”
榴莲、披萨、面包、炸鸡都可以少量吃，选1种特别想吃的。 记得搭配一份蛋白质（如鸡蛋、牛奶）+ 6个小番茄。

✅ 午餐：均衡正常吃 一拳头粗粮饭+一手掌肉类+两手掌蔬菜，保证营养平衡。

✅ 晚餐：少油少盐、7点前吃完 蔬菜+蛋白质（豆腐、鱼虾、鸡蛋）+半个馒头。

小C现在体重降了15斤，裤子都大了一圈。 她开心地说：“原来我不用忌口，只要换个时间吃就行。”

## **💬 最后我想说**
减肥不一定要“忍”，更聪明的做法是顺应身体规律——**早上吃能量，晚上吃轻盈。**
这个操作方法很简单，但确实有用。如果你也有嘴馋暴食的困扰，不妨试试这个方法，或许有惊喜哦！
