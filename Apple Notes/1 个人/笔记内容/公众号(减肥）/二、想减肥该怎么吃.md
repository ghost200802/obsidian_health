# 二、想减肥该怎么吃

1、尽可能做到规律进食，保持血糖稳定

你如果饿得太久会导致血糖下降过低，容易头晕、暴食、情绪暴躁；但一次吃太多（特别是高GI食物）又会导致血糖快速飙升，胰岛素大量分泌，血糖又下降过快，导致饥饿感很快重来，还可能加速脂肪堆积。

把血糖想象成“电压”，电压忽高忽低，灯泡就会忽明忽暗，时间长了，电器也容易坏。我们规律进食就是为了让“电压稳定”，保证身体运作顺畅。

<span style="font-family:.PingFangUITextSC-Regular;">很多人</span>会把规律进食误解成“必须按时三餐”，实际上不必过于教条主义。只要做到以下3点就可以：
1、有固定的进食节奏：一日三餐/少食多餐，都可以；
2、避免长时间空腹：不要等到饿昏了头才吃，这样容易血糖下降+暴食过量；早上起床尽量1小时内吃早餐；
3<span style="font-family:.PingFangUITextSC-Regular;">、避免短时间内暴食过量：一次吃太多，</span>会造成血糖骤升骤降 → 容易困倦、饿得快，还促进脂肪堆积。

平稳的血糖不仅对减肥和健康有帮助，还会让你皮肤更好、工作学习效率更高。

2、控制总量，合理分配食物营养素

<span style="font-family:.PingFangUITextSC-Regular;">减重过程中在控制总能量摄入的基础上，不仅要注意食物选择的多样化与合理化，同时也要合理分配食物营养素。分配比例建议如下：</span>

- 碳水 **50–60%**（保留优质碳水，避免极端低碳）
- 蛋白质 **15–20%**（保护肌肉，增加饱腹感）
- 脂肪 **20–30%**（主要来自坚果、鱼类、橄榄油等）
👉 有运动 → 碳水比例高一些，给肌肉补能。
👉 无运动 → 蛋白质比例高一些，防止掉肌肉。
👉 脂肪 → 不需要额外增加，保持适量即可。
👉 碳水不能过低，否则会影响内分泌和生殖功能。
👉 **减脂备孕人群也适用。**
***如果本身有某些疾病，需要咨询专业人士，结合自身情况进行调整。**

3、 多喝水

水是 “0热量燃脂催化剂”。

补充充足的水分不仅能让身体加速脂肪代谢，还能增强饱腹感、促进消化防止便秘。

建议每日饮水量1500毫升-1700毫升，如果运动出汗较多，则需要额外补充水分。

分享几个减肥效率更高的喝水方法：
**晨起空腹喝 300~500ml 温水，唤醒身体进入燃脂模式**
**饭前 30 分钟喝 500ml 水，可直接减少正餐摄入量**
<span style="font-family:.PingFangUITextSC-Bold;"><b>运动后喝温水</b></span> **+** <span style="font-family:.PingFangUITextSC-Bold;"><b>蛋白质，</b></span>帮助肌肉修复
避免一次性吨吨吨，保持少量多次，每次100～200ml，可持续保持饱腹感。
感觉饿了先喝水，防止被假性饥饿欺骗，摄入多余能量
