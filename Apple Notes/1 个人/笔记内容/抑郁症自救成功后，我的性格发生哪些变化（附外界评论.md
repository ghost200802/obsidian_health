# 抑郁症自救成功后，我的性格发生哪些变化（附外界评论

上一篇我分享了我快速走出抑郁状态的7个小tips。感受到彻底自愈是抑郁症后的第18个月（也就是3个月前）。我对自愈的定义是：<b><u>「发现自己大部分时候能量满满；面对情绪波动，自我调节的速度比以前更快；面对重复发生的事情，却有了新的心态和新的回应。」</u></b>

以下是我发现的性格变化，但我并没有评价这些变化的好坏，因为凡事都有两面性，况且谁又能定义我的好坏呢。

这些变化主要是：

1⃣️ <span style="color:#432ffff;"><b>更爱自己，优先照顾好自己</b></span>
<span style="font-family:.PingFangSC-Semibold;"><b>以前的我：</b></span><span style="font-family:.PingFangSC-Regular;">基本不考虑自我感受，只考虑他人感受和需求，优先满足他人，忘记问自我感受。</span>
<span style="font-family:.PingFangSC-Semibold;"><b>现在的我：</b></span><span style="font-family:.PingFangSC-Regular;">会先照顾好自己，再询问他人的需求，再向他人提供帮助。</span>
<span style="font-family:.PingFangSC-Semibold;"><b>外界评论：</b></span><span style="font-family:.PingFangSC-Regular;">太自私</span>/<span style="font-family:.PingFangSC-Regular;">太自我</span>
<span style="font-family:.PingFangSC-Semibold;"><b>我的回应：</b></span><span style="font-family:.PingFangSC-Regular;">我连自己都照顾不好，我怎么能照顾好别人的需求？</span>

2⃣️ <span style="color:#432ffff;"><b>不纠结不内耗，超级直球</b></span>
<span style="font-family:.PingFangSC-Semibold;"><b>以前的我：</b></span><span style="font-family:.PingFangSC-Regular;">瞻前顾后，纠结内耗，迟迟做不出行动，错失解决问题的最佳时机。</span>
<span style="font-family:.PingFangSC-Semibold;"><b>现在的我：</b></span><span style="font-family:.PingFangSC-Regular;">不想猜，反正我也猜不对。直接向对方提问：你希望我为你做什么？然后考虑自己是否愿意做，再给出回应。</span>
<span style="font-family:.PingFangSC-Semibold;"><b>外界评论：</b></span><span style="font-family:.PingFangSC-Regular;">如果你重视我，你会主动为我做这些的，而不是问我。</span>
<span style="font-family:.PingFangSC-Semibold;"><b>我的回应：</b></span><span style="font-family:.PingFangSC-Regular;">我不是你，我不能代替你决定你此刻最想要的是什么。但是只要你告诉我，我就信；只要我愿意做，我毫无怨言去做到。</span>（语气平缓，眼神坚定）

3⃣️ <span style="font-family:.PingFangSC-Semibold;color:#432ffff;"><b>有意识</b></span><span style="color:#432ffff;"><b>拒绝不合理需求；勇敢提出自己的合理需求</b></span>
<span style="font-family:.PingFangSC-Semibold;color:#424242ff;"><b>以前的我：</b></span><span style="font-family:.PingFangSC-Regular;">面对别人提出的需求，即便觉得不合理，心里不高兴，也会尽量去满足；自己想要的事物，不敢说出口</span>
<span style="font-family:.PingFangSC-Semibold;"><b>现在的我：</b></span>能意识到自己要拒绝不合理的需求，拒绝+原因+表达我能提供的帮助，（但我需要稍作思考，才能完整表达出来，还做不到自然地脱口而出）；明确自己想要的东西，会直接向对方提出请求：你可以为我做XXXX吗？
<span style="font-family:.PingFangSC-Semibold;"><b>外界评论：</b></span>你太冷漠/你太没有人情味儿了
<span style="font-family:.PingFangSC-Semibold;"><b>我的回应：</b></span>我理解你为什么会这样想，但我接受自己不被你理解。（语气平缓，眼神坚定）

4⃣️ <span style="color:#432ffff;"><b>我终于拥有成年人心智，为自己所有的选择全权负责。</b></span>
<span style="font-family:.PingFangSC-Semibold;color:#424242ff;"><b>以前的我：</b></span><span style="color:#424242ff;">都是因为外界的人和事，才让我现在过得不好！我讨厌这个世界！</span>
<span style="font-family:.PingFangSC-Semibold;"><b>现在的我：</b></span>所有的结果都是我自己选择的，我愿意接受结果。这种感觉真的很自由！
<span style="font-family:.PingFangSC-Semibold;"><b>外界评论：</b></span>你的<span style="font-family:.PingFangSC-Regular;">选择也会对别人造成影响，这个后果你承担得起吗？</span>
<span style="font-family:.PingFangSC-Semibold;"><b>我的回应：</b></span>大家都是成年人，都应当对自己的行为负全责。这份指责我不接受。（语气平缓，眼神坚定）

5⃣️ <span style="color:#432ffff;"><b>面对纷扰的世界，我设置了我的「共情力开关」</b></span>
<span style="font-family:.PingFangSC-Semibold;color:#424242ff;"><b>以前的我：</b></span><span style="color:#424242ff;">共情能力超强，面对他人的困境，仿佛自己也身处其中，忍不住为他人付出更多行动，如果什么都不能做，甚至会感到无力。</span>
<span style="font-family:.PingFangSC-Semibold;"><b>现在的我：</b></span>给自己设置了<b><u>「共情力开关」</u></b><span style="font-family:.PingFangSC-Regular;">。面对他人的困境，先思考“我想为对方做什么？”想好后，就去做，其它都不再多想了。核心要点是：只专注在「做的事情」上，「不做的事情」就不想，想了也没用，以此来减轻自身负担。</span>
<span style="font-family:.PingFangSC-Semibold;"><b>外界评论：</b></span>你好冷漠
<span style="font-family:.PingFangSC-Semibold;"><b>我的回应：</b></span> 如果我这样的共情程度让你觉得我冷漠，那未来我会把这份共情留起来给愿意珍惜的人。（语气平缓，眼神坚定）

6⃣️ <span style="color:#432ffff;"><b>勇敢面对真实的不完美的自己</b></span>
<span style="font-family:.PingFangSC-Semibold;color:#424242ff;"><b>以前的我：明明怯懦但却很要强。总用高标准来要求自己和身边的人；听到别人对我的否定就如临大敌</b></span>
<span style="font-family:.PingFangSC-Semibold;"><b>现在的我：</b></span>先思考我是否认可对方的评价。如果我认可，那就坦然承认：确实是这样。如果不认可，就询问对方为什么会这么认为，有误会就澄清。
<span style="font-family:.PingFangSC-Semibold;"><b>外界评论：</b></span>你这些毛病怎么就改不掉呢？
<span style="font-family:.PingFangSC-Semibold;"><b>我的回应：</b></span>其实我也知道自己有这些毛病，已经在改的过程中，一时也急不来。（语气平缓，眼神坚定）

⬆️<b><u>写下这些是为了告诉正在经历抑郁症的朋友们，你们只是正处在自救的过程中，最终你们一定也会达成自救的，给自己一点时间，相信你一定能照顾好自己的。</u></b>

最最最重要的是‼️ 我此生得到最有力量的爱，是来自我自己。我再也不害怕了，因为即使全世界都不爱我不理我，我还有自己爱我自己❤️
