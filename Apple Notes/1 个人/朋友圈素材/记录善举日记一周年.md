# 记录善举日记一周年

距离我第一次记录善举日记，已经过去1年。

<span style="font-family:.PingFangUITextSC-Regular;">对我而言，这不只是我的能量来源，更像是在「重新训练大脑」。</span>

<span style="font-family:.PingFangUITextSC-Regular;">从神经科学角度来说，我们的大脑有「负面偏好」</span>——更容易注意风险、问题和不好的体验。这是生存本能。但在日常生活中，这些很容易让人陷入焦虑、内耗和冲突。

而每天去记录和创造善意，其实是在有意识地引导大脑，把注意力从“出现的问题”转向“拥有的资源”，反复强化那些温暖、稳定、有连接感的体验。

<span style="font-family:.PingFangUITextSC-Regular;">当这些记录不断重复时，大脑里的神经通路也在发生改变</span>——那些与安全感、愉悦感、连接感相关的积极回路被反复激活，变得更容易被调用；久而久之，大脑会自动扫描「美好的一面」。

而原本容易触发焦虑和恐惧的路径，逐渐不再那么「自动化」。

<span style="font-family:.PingFangUITextSC-Regular;">我这一年的变化，其实就是这种「慢慢被重塑」的结果：情绪恢复更快，内在的力量更坚定，也更容易看见他人和世界的善意，从世间万物中都能汲取能量。</span>

<span style="font-family:.PingFangUITextSC-Regular;">「世界没有变，但我眼里的世界变了」</span>

<span style="font-family:.PingFangUITextSC-Regular;">所以，善举日记对我来说，不只是记录生活，而是在一点点地改变我与自己、与他人、与世界、与灵性的关系。</span>

<span style="font-family:.PingFangUITextSC-Regular;">如果你最近有些心累，感到麻木，今晚可以试试记录一条小小的「还不错的瞬间」。或许，对你有帮助。</span>
