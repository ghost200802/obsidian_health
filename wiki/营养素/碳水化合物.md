# 碳水化合物

碳水化合物是人体最主要的能量来源，由碳、氢、氧三种元素组成。碳水化合物与[[蛋白质]]、[[脂肪]]共同构成三大宏量营养素。

## 定义与分类

### 碳水化合物的分类
- **简单碳水化合物**：单糖（葡萄糖、果糖）、双糖（蔗糖、乳糖）
- **复杂碳水化合物**：寡糖、多糖（淀粉、糖原、[[膳食纤维]]）

### 膳食碳水化合物的主要形式
- **淀粉**：谷物、薯类中的主要碳水形式
- **膳食纤维**：不能被消化吸收的多糖，是碳水化合物的重要组分
- **添加糖**：加工食品中额外添加的糖类

### 碳水化合物分类详表（教材）

| 分类 | 亚组 | 组成 |
|------|------|------|
| 糖（1-2个单糖） | 单糖 | 葡萄糖、果糖 |
| | 双糖 | 蔗糖、乳糖、麦芽糖 |
| | 糖醇 | 山梨醇、甘露醇 |
| 寡糖（3-9个单糖） | 异麦芽寡糖 | 麦芽糊精 |
| | 其他寡糖 | 棉子糖、水苏糖、寡糖 |
| 多糖（>=10个单糖） | 淀粉 | 直链淀粉、支链淀粉、变性淀粉 |
| | 非淀粉多糖 | 纤维素、半纤维素、果胶 |

### 抗性淀粉
抗性淀粉又称抗酶解淀粉或难消化淀粉，在小肠内无法消化吸收，但能被大肠内的微生物发酵利用。直链淀粉易老化形成难消化的抗性淀粉。

### 糖原
糖原也称动物多糖，由葡萄糖结合而成的支链多糖。包括肝糖原和肌糖原两种：
- 肝糖原：维持血糖水平的恒定
- 肌糖原：主要为肌肉供能，提供身体运动时所需能量

## 生理功能

1. **提供和贮存能量**：每克葡萄糖在体内进行生物氧化可产生4kcal的能量。大脑仅利用葡萄糖作为能量来源，大脑在活动时约消耗2/3的血糖
2. **构成组织及生理活性物质**：糖与蛋白质结合形成糖蛋白（保护胃黏膜的黏液、硫酸软骨素），参与抗体、酶、激素及核酸的组成；糖和脂肪形成糖脂是细胞膜的重要成分
3. **节约蛋白质**：碳水化合物供应充足时，可以预防体内膳食蛋白质消耗供能
4. **抗生酮作用**：葡萄糖氧化生成草酰乙酸，促进脂肪酸进入三羧酸循环彻底氧化，防止酮症酸中毒
5. **解毒作用**：葡萄糖醛酸在肝脏与细菌毒素、乙醇、砷等有毒物质发生反应
6. **增强肠道功能**：非淀粉多糖促进肠道蠕动，增加结肠发酵，促进有益菌生长，产生短链脂肪酸

## 食物来源

- 粮谷类含碳水化合物60%-80%
- 薯类含15%-29%
- 豆类含40%-60%
- 单糖和双糖主要来源于白糖、糖果、甜食、糕点、水果、含糖饮料和蜂蜜等

## 消化吸收机制

碳水化合物在消化道的消化吸收过程：

1. **口腔**：唾液淀粉酶开始分解淀粉为糊精和麦芽糖
2. **胃**：酸性环境抑制淀粉酶活性，消化暂时停止
3. **小肠**：胰淀粉酶将淀粉分解为麦芽糖、麦芽三糖和糊精；肠黏膜刷状缘酶（麦芽糖酶、蔗糖酶、乳糖酶等）将双糖分解为单糖
4. **吸收**：葡萄糖和半乳糖通过主动转运（SGLT1）吸收，果糖通过易化扩散（GLUT5）吸收

## 血糖调控

碳水化合物消化吸收后以葡萄糖形式进入血液，血糖调控涉及：

- **胰岛素**：促进葡萄糖摄取和利用，降低血糖
- **胰高血糖素**：促进糖原分解和糖异生，升高血糖
- **[[血糖生成指数]]**（GI）：衡量食物引起血糖升高能力的指标
- 低 GI 食物（全谷物、豆类）有利于血糖稳定
- 高 GI 食物（精制糖、白米面）导致血糖快速波动

## 寡糖（功能性低聚糖）

- 低聚果糖、异麦芽低聚糖、海藻糖、低聚木糖及大豆低聚糖等
- 不能被胃酸破坏，也不能被消化酶分解
- 可被肠道细菌发酵利用，产生短链脂肪酸以及乳酸
- 能有效促进肠道内双歧杆菌生长繁殖，称为"双歧因子"

## 膳食纤维

膳食纤维是碳水化合物中的重要组分，成人 AI 为 25~30 g/d（2023版），参见 [[膳食纤维]]。

## 与健康的关系

- 适量碳水化合物是健康膳食的基础
- 过量摄入添加糖与[[肥胖]]、[[代谢综合征]]、2型糖尿病风险增加相关
- 碳水化合物质量（全谷物 vs 精制碳水）比总量更重要
