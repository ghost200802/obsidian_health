# 维生素B9（叶酸）

维生素B9（叶酸，Folic Acid），即维生素B9，是一种水溶性[[维生素]]，参与一碳单位代谢和DNA合成，在细胞分裂和增殖过程中发挥关键作用。

## 定义

- **化学本质**：蝶酰谷氨酸
- **特性**：不耐热，食物加工过程中损失较大
- **膳食叶酸当量（DFE）**：考虑到合成叶酸与食物叶酸的生物利用度差异，DRIs 采用 DFE 作为单位（1 μg DFE = 1 μg 食物叶酸 = 0.6 μg 合成叶酸）

## 功能

- 参与一碳单位代谢（一碳单位的供体和受体）
- 参与 DNA 和 RNA 的合成
- 参与氨基酸代谢（如同型半胱氨酸转化为蛋氨酸）
- 参与红细胞形成

## DRIs 推荐摄入量（2023版）

| 人群 | RNI（μg DFE/d） |
|------|-----------------|
| 成人 | 400 |
| 孕妇 | 600 |
| 乳母 | 550 |
| [[可耐受最高摄入量|可耐受最高摄入量]] | 1000 |

### 2023版变化

- 成人RNI维持400 μg DFE/d
- PI-NCD：目前数据不足以制定叶酸PI-NCD
- 孕妇RNI维持600 μg DFE/d，备孕期建议每日补充400 μg合成叶酸

## 孕期补充的重要性

- **备孕及孕早期**补充叶酸可预防胎儿神经管缺陷（NTDs）
- 建议从孕前 3 个月开始，每日补充 400 μg 合成叶酸
- 参见 [[妊娠期营养]]、[[妊娠期铁推荐摄入量]]

## 富含叶酸的食物

| 食物类别 | 代表食物 | 叶酸含量 |
|---------|---------|---------|
| 深绿叶蔬菜 | 菠菜、西兰花、芦笋 | 高 |
| 豆类 | 黄豆、扁豆、鹰嘴豆 | 高 |
| 动物肝脏 | 猪肝、鸡肝 | 高 |
| 坚果 | 花生、葵花籽 | 中 |

## 叶酸缺乏与健康问题

- **巨幼细胞贫血**：影响红细胞分裂和成熟
- **胎儿神经管缺陷**：孕早期叶酸缺乏是最重要的可控风险因素
- **高同型半胱氨酸血症**：与心血管疾病风险增加相关

## 常见食物叶酸含量数据

根据中国食物成分表数据，常见食物的叶酸含量（每100g可食部）：

| 食物 | 叶酸（μg） | 备注 |
|------|-----------|------|
| 猪肝 | 425 | 叶酸含量最高的动物性食物 |
| 鸡肝 | 1172 | 叶酸极高 |
| 黄豆 | 130 | 植物性优质来源 |
| 菠菜 | 61 | 深绿叶蔬菜代表 |
| 西兰花 | 51 | 十字花科蔬菜 |
| 芦笋 | 113 | 高叶酸蔬菜 |
| 鸡蛋 | 32 | 每天一个鸡蛋可贡献部分叶酸 |
| 花生 | 107 | 坚果中较高 |
| 猕猴桃 | 25 | 水果中较高 |
| 橘子 | 13 | 常见水果 |
| 牛奶 | 5 | 含量较低 |

> 食物叶酸不耐热，烹饪后损失可达50%~90%，实际摄入量需考虑烹饪损失（参见[[烹饪营养]]）。

## 营养与食品卫生学（第8版）补充

> 数据来源：《营养与食品卫生学》第8版（孙长颢主编），基于2013版DRIs。

- 2013版RNI：400ug DFE/d，UL 1000ug/d，孕妇RNI 600ug DFE/d
- DFE计算：DFE = 膳食叶酸(ug) + 1.7 x 补充剂叶酸(ug)
- 功能：参与一碳单位转移，DNA/RNA合成，与VB12协同参与同型半胱氨酸代谢
- 孕早期缺乏：胎儿神经管缺陷（NTDs）风险显著增加
- 叶酸缺乏致巨幼红细胞性贫血
- 食物来源：深绿色蔬菜（菠菜、芦笋）、豆类、动物肝脏
