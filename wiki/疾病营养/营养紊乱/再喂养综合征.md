# 再喂养综合征

## 定义

再喂养综合征（refeeding syndrome, RFS）由Burger于1945年首次报道。RFS指长期饥饿或严重[[营养不良]]者，在重新摄入营养物质特别是高碳水化合物（糖）营养物质后的早期阶段，出现以**低磷血症**为特征的电解质代谢紊乱及由此产生的一系列症状及多器官系统损害。

## 流行病学

| 人群 | RFS发生率 |
|------|-----------|
| 住院成年患者 | 0.8% |
| 恶性肿瘤患者 | 24.5% |
| 接受TPN治疗患者 | 42% |
| ICU重症患者（喂食后2天低磷） | 34% |

**病死率可高达48%**。

## 病理生理

### "四低二高"特征

饥饿→胰岛素分泌减少→分解代谢→磷、钾、镁消耗。重新喂养（尤其高糖）→血糖升高→胰岛素恢复→合成代谢→电解质转入细胞内→

| 异常 | 机制 |
|------|------|
| **低磷血症**（主要特征） | 磷参与核苷酸、磷脂合成；缺乏→溶血性贫血、心肌/横纹肌溶解、氧解离曲线左移 |
| **低钾血症**（死亡重要原因） | Na+-K+-ATP泵增强→细胞外钾降低→细胞超极化→肌无力、心律失常、肠麻痹 |
| **低镁血症**（进展标志） | 多种酶活化受损→抽搐、癫痫、心律失常、低血压 |
| **低维生素B1** | 丙酮酸→乳酸堆积→乳酸性酸中毒；胆碱酯酶抑制→韦尼克脑病 |
| **高血糖**（始动因素） | 大量碳水化合物→血糖迅速升高→胰岛素分泌增加 |
| **高容量**（水钠潴留） | 胰岛素抗利钠作用→肾脏排泄钠水减少→肺水肿、心力衰竭 |

## 临床表现

- 低磷血症、低钾血症、低镁血症
- 维生素B1缺乏
- 充血性心力衰竭、心律失常
- 外周性水肿
- 肌肉疼痛、横纹肌溶解
- 肌无力、癫痫发作、谵妄、韦尼克脑病
- 溶血
- 呼吸肌无力、呼吸衰竭
- 急性肾小管坏死（继发于横纹肌溶解）

> [!warning]
> 电解质紊乱和心血管并发症通常在再喂养开始1周内出现（2-4天内），神经系统症状通常在这些变化之后出现。

## 诊断

### ASPEN诊断标准

再喂养启动5天内，血清磷、钾、镁减少：

| 程度 | 标准 |
|------|------|
| 轻度 | 电解质低于正常低限的10%-20% |
| 中度 | 电解质低于正常低限的20%-30% |
| 重度 | >30%和/或器官功能障碍 |

具体标准：
- 低钾血症（K+ < 2.5 mmol/L）
- 低磷血症（磷 < 0.32 mmol/L）
- 低镁血症（Mg2+ < 0.5 mmol/L）
- 出现循环/呼吸/神经系统症状
- 外周性水肿，体重急剧增加

### NICE风险筛查（三层级）

| 筛查标准一 | 筛查标准二 | 筛查标准三 |
|-----------|-----------|-----------|
| BMI < 18.5 | BMI < 16 | BMI < 14 |
| 3-6个月体重丢失 > 10% | 3-6个月体重丢失 > 10% | 体重丢失 > 20% |
| 没有或少量营养摄入 > 5天 | 没有或少量营养摄入 > 10天 | 没有或少量营养摄入 > 15天 |
| 既往酗酒/药物滥用史 | - | - |
| - | 再喂养前低磷、低钾、低镁血症 | - |

> [!important]
> 具备标准一中的**两项**，或**标准二 + 标准三中的一项**，均评估为RFS高风险。

## 预防

### 营养治疗前

- 检查血尿电解质、心电图
- 发现并纠正低磷、低钾、低镁血症
- 已有明显水电解质紊乱者**延迟营养治疗12-24小时**

### 启动营养治疗后前10天

#### 电解质经验性补充

| 电解质 | 推荐量 |
|--------|--------|
| 磷 | 0.5-0.8 mmol/(kg·d) |
| 钾 | 1-3 mmol/(kg·d) |
| 镁 | 0.3-0.4 mmol/(kg·d) |

#### 维生素B1

- 营养治疗前至少30分钟：静脉或肌内注射**200-300 mg**
- 此后每日经口或经静脉补充**200-300 mg**
- 复合维生素制剂每日按2倍参考剂量补充

#### 能量启动方案

| 阶段 | 能量供给 |
|------|----------|
| 第一个24小时 | 100-150g葡萄糖 或 10-20 kcal/kg |
| 递增 | 每1-2天增加目标能量的33% |
| 中高危电解质紊乱者 | 减慢或暂停能量增加，直至纠正 |

#### 监测

- 治疗开始后4-6小时监测电解质，以后每天1次

## 确诊后治疗

1. 下调能量供给
2. 提高脂肪供能比
3. 适量增加补充矿物质
4. 强化补充维生素B1或多种水溶性维生素
