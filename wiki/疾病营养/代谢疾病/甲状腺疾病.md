# 甲状腺疾病

## 定义

甲状腺通过滤泡上皮细胞合成和分泌T4和T3，在调节机体代谢和生长发育中起重要作用。碘是合成甲状腺激素的必需原料，人体80%-90%的碘来源于食物。碘缺乏→地方性甲状腺肿、克汀病、甲状腺癌；碘过量→散发性甲状腺肿、碘甲亢、自身免疫性甲状腺病。

## 一、甲状腺肿

### 代谢特征
- 多数患者甲状腺功能正常，无代谢异常
- 地方性甲状腺肿流行区：尿碘降低、甲状腺摄碘率增高、T4下降、反T3升高

### 尿碘评估

| MUI (μg/L) | 碘营养状态 |
|------------|-----------|
| <100 | 碘缺乏 |
| 100-200 | 最适当 |
| 200-299 | 碘超足量 |
| >300 | 碘过量 |

### 营养治疗
- 纠正碘缺乏：食盐碘化（10-15mg/kg盐）是最有效的预防措施
- WHO建议成年人碘适宜供给量150μg/d；妊娠和哺乳期250μg/d
- 供给充足蛋白质和能量改善甲状腺功能
- 补充硒可能降低TPO-Ab水平
- 纠正维生素A不足
- 避免过多食用含致甲状腺肿物质的食物（十字花科蔬菜中硫氰酸盐，豆类/橘子/苹果中类黄酮）

## 二、甲状腺功能亢进症（甲亢）

### 代谢特征
- **高代谢**：怕热、多汗、体重下降、大便次数增加、情绪激动、低热
- 糖代谢：促进肠道葡萄糖吸收、肝糖原分解、糖异生→糖代谢异常
- 脂代谢：胆固醇降解>合成→血TC和LDL-C偏低；TG和FFA升高
- 蛋白质：蛋白质分解→负氮平衡→肌肉消耗
- 低钾血症（尤其东方青年男性）
- B族维生素、维生素C、维生素A消耗增多
- 镁、锰、锌等微量元素明显降低

### 营养治疗

**原则：高能量、高蛋白质、高维生素、限制碘摄入**

| 营养素 | 推荐量 |
|--------|--------|
| 能量 | 较正常人增加50%-70%，3000-3500 kcal/d |
| 碳水化合物 | 占能60%-70% |
| 蛋白质 | 1.5-2.0 g/(kg·d)，保证优质蛋白质 |
| 水分 | 足量饮水（出汗多） |
| 维生素 | B族维生素和维生素C重点补充，充足维生素D |
| 碘 | **忌用**富含碘食物（海带、紫菜、牡蛎）和含碘药物（胺碘酮）；提倡无碘盐 |

## 三、甲状腺功能减退症（甲减）

### 代谢特征
- **低代谢**：乏力、食欲下降、怕冷、水肿、便秘、体重增加
- 脂代谢紊乱：胆固醇代谢缓慢→高TG和高TC血症
- 蛋白质代谢：白蛋白降低
- 糖代谢：可能出现低血糖
- 碘代谢：缺碘可致甲减；碘过量也可引起甲减
- 容易贫血

### 营养治疗

| 措施 | 具体方案 |
|------|---------|
| 补碘 | 食用加碘食盐最有效；缺碘者选用海带、紫菜 |
| 蛋白质 | 足量补充；动物蛋白（蛋类、乳类、鱼、肉）+植物蛋白（黄豆及豆制品） |
| 限制脂肪 | 限制富含胆固醇食物（蛋黄、奶油、动物脑和内脏）；减少高脂肪食物 |
| 纠正贫血 | 补充富含铁和B12的膳食（动物肝脏）；补充叶酸（蔬菜水果） |
| 用药注意 | 左甲状腺素钠片空腹早餐前1小时服用；不与钙片和牛乳同服 |
