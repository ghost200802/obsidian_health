# 维生素异常

## 概述

维生素是维持人体正常生理功能的必需营养物质，需要量虽然很少，但缺乏会引起特殊的临床表现，过量摄入（尤其脂溶性维生素）可引起急性和慢性中毒。

## 维生素缺乏临床表现总览

| 维生素 | 缺乏的临床表现 |
|--------|---------------|
| 维生素A | 夜盲症、干眼症、角膜干燥 |
| 维生素B1 | 脚气病、韦尼克脑病 |
| 维生素B12 | 巨幼细胞贫血、周围神经变性、口腔溃疡、口干症 |
| 叶酸 | 巨幼细胞贫血、腹泻、疲劳、抑郁、意识模糊 |
| 维生素C | 坏血病、毛囊过度角化、伤口愈合迟缓、牙龈出血 |
| 维生素D | 儿童佝偻病、成年人骨软化症 |
| 维生素E | 肌病、贫血和神经性疾病 |
| 维生素K | 凝血障碍 |

## 维生素A

### 缺乏

**原发性缺乏**：自身吸收、代谢、利用异常或遗传性
**继发性缺乏**：膳食摄入少、长期饮酒、胆汁/胰液不足、慢性肝病

**临床表现**：
- **眼部**：干眼症（泪液分泌不足、杯状细胞减少、毕脱斑、角膜软化）、夜盲症
- **上皮功能异常**：毛囊角质化、皮肤干燥、"蟾蜍变"
- **免疫功能受损**：淋巴细胞和NK细胞减少、抗体反应减弱
- **感染性疾病患病率和死亡率升高**

### 过量

- 急性中毒：达RNI 100倍
- 慢性中毒：长期达RNI 10倍
- 妊娠早期过量可致胚胎畸形
- β-胡萝卜素毒性较低，过量可致皮肤黄染

## 维生素B1（硫胺素）

### 缺乏

**高危人群**：酗酒者、长期利尿剂使用者、胃旁路术后患者

**缺乏疾病**：

#### 韦尼克脑病（WE）

维生素B1缺乏导致的严重神经系统综合征。人体仅储备30-50mg，持续3-4周不摄入即可完全耗尽。

**"三联征"**：眼球运动障碍、小脑性共济失调、精神意识障碍
**"四联征"**：三联征 + 多发性神经病

**发病机制**：硫胺素二磷酸（TDP）是丙酮酸脱氢酶、alpha-酮戊二酸脱氢酶、磷酸戊糖途径的辅助因子。缺乏时ATP合成减少、氧化损伤和细胞死亡，大脑等高代谢组织首先受累。

**治疗**：一旦怀疑应**立即给予维生素B1**，无须等待检查结果。应在摄入任何碳水化合物之前给予。静脉或肌内注射100-500 mg/d，连续3天。**维生素B1补充同时需补镁治疗**。

#### 脚气病

| 类型 | 特点 |
|------|------|
| 湿性脚气病 | 心脏问题和水肿，多见于婴儿 |
| 干性脚气病 | 肌肉乏力、感觉异常、步态不稳，多见于成年人 |
| 混合型 | 同时出现心血管和神经系统症状 |

### 过量

食用天然食物不会引起过量。服用补充剂达RNI 100倍以上时可能存在潜在毒性。

## 维生素C

### 缺乏（坏血病）

- 成人缺乏45-80天后出现明显体征
- 表现：牙龈肿胀出血、牙齿脱落、嗜睡、疲劳、皮肤损害
- 细胞内缺乏可导致氧化应激，缺血性心脏病风险增加
- 缺乏加重[[缺铁性贫血]]（影响铁还原和转运）

### 过量

- 儿童长期超量影响骨质钙磷吸收
- 可致尿路草酸钙结石
- 大量使用>60天停药后可出现反跳性缺乏
- 破坏维生素B12结构，引起[[巨幼细胞贫血|巨幼细胞贫血]]
