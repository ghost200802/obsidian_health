# NAFLD营养

## 定义

非酒精性脂肪性肝病（NAFLD）是一种与胰岛素抵抗（IR）和遗传易感性密切相关的代谢应激性肝损伤，是肝细胞内脂质蓄积过多的病理状态。疾病谱包括：非酒精性肝脂肪变→非酒精性脂肪性肝炎（NASH）→肝硬化→肝细胞癌。

> 最新国际专家共识建议以**代谢相关脂肪性肝病（MAFLD）**代替NAFLD，更好地反映疾病本质。

### 流行病学
- 全球患病率25.2%，其中约59.1%为NASH
- 我国患病率高达29.2%，且逐年上升
- 我国第一大慢性肝病
- 与T2DM、CVD、MetS甚至结直肠肿瘤高发密切相关

### 诊断标准
1. 肝活检病理学检查发现5%以上大泡性肝脂肪变或影像学显示弥漫性脂肪肝
2. 无饮酒史或饮酒折合酒精量男性<30g/d、女性<20g/d
3. 排除基因3型丙肝、药物性肝病、自身免疫性肝病、TPN、肝豆状核变性、IBD、甲状腺功能减退症等

## 代谢特征

### 脂类代谢
- 高TG血症多见
- 肝细胞内TG异常堆积原因：脂肪酸输入增多 + β氧化减少 + VLDL分泌减少
- BCAA水平升高与IR密切相关，促进肝脏脂肪酸从头合成

### 糖代谢
- 与MetS互为因果
- T2DM患者49%-62%伴发NAFLD，进展为NASH风险增加2-3倍

### 肠道菌群失调
- 长期过多能量摄入→肠道菌群失调→肠道屏障受损→细菌及代谢产物经门静脉到达肝脏
- 肠源性内毒素引起炎症应激反应

## 营养治疗

### 核心原则

通过改变生活方式、平衡膳食、科学运动、保持正常体重

### 具体措施

| 措施 | 具体方案 |
|------|---------|
| 控制体重及腰围 | 1年内减重3%-5%可改善MetS组分和逆转NAFLD；三餐规律，七八成饱 |
| 蛋白质 | 占能10%-20%，1.0-1.5g/(kg·d)，动物:植物=1:1；营养不良者1.5-2.0g/(kg·d)；儿童青少年2.0-4.0g/(kg·d) |
| 脂肪 | 占能20%，禁用动物油，选用橄榄油、茶油（MUFA），不吃肥肉和动物内脏 |
| 碳水化合物 | 适当降低供能比；多吃粗粮（红薯、玉米、荞麦、燕麦）；不吃高糖糕点 |
| "隐形肥胖" | BMI正常但体脂率或腰围超标：25-30 kcal/(kg·d) |
| 维生素和膳食纤维 | 低糖水果200g/d，新鲜蔬菜300-500g/d；B族维生素和维生素E参与脂肪肝代谢 |
| 饮水 | 2500ml/d左右；少量多次白开水、矿泉水、淡茶水 |
| 运动 | 中等强度有氧运动150min/周（5天/周，30min/天） |

### NAFLD伴肥胖儿童
- 改善生活方式是最重要和最基本治疗
- 低碳水化合物及高蛋白质膳食对减肥效果更好
- 限制零食、含糖饮料、油炸食品

### 多学科会诊
- NAFLD多伴有心血管、内分泌系统疾病、肥胖
- 需要多学科团队诊治
- 力争将血压、血脂、血糖、体重及腰围控制在理想范围
