# 隐性饥饿

## 定义

隐性饥饿（hidden hunger）是指在总体食物摄入充足的前提下，由于某些因素导致特定营养素摄入不足或利用受阻，从而诱发的营养失衡状态。也称微量营养素缺乏（Micronutrient Deficiencies, MND）。

## 流行病学

全球至少有**20亿人**受到一种或数种微量营养素缺乏的影响。由于临床表现隐匿，人们可能对此毫无察觉，但其对健康、工作效率及心理健康的负面影响甚至可能持续终身。

## 发病原因（5类）

1. **食物选择不当**：长期单一食物，某些营养素摄入严重不足
2. **消化吸收障碍**：消化/吸收功能障碍，不能有效利用食物中的营养素
3. **生理需求增加**：生长发育期、妊娠期、哺乳期
4. **疾病因素**：透析患者等疾病增加营养素需求或流失
5. **药物或其他外部因素**：影响营养素的摄入、消化、吸收或利用

## 病理生理（6类机制）

1. **营养素缺乏**：关键营养素低于需求，细胞/组织/器官功能受损
2. **细胞能量代谢障碍**：维生素和矿物质缺乏影响能量底物供给和关键酶活性
3. **酶活性受损**：营养素是酶的辅酶或辅基，缺乏时酶活性受影响
4. **免疫功能下降**：免疫反应减弱，感染风险增加
5. **组织和器官损伤**：长期营养不足导致结构和功能受损
6. **神经和内分泌功能失调**：认知障碍、激素平衡异常

## 营养代谢影响

1. **代谢途径改变**：B族维生素缺乏干扰糖代谢和脂肪酸氧化
2. **蛋白质合成障碍**：锌、铁缺乏影响蛋白质合成和肌肉合成
3. **神经递质合成和功能受损**：B6、B12、叶酸缺乏→情绪障碍和神经功能下降
4. **氧化应激和炎症反应**：维C、维E、硒缺乏→自由基积聚
5. **激素合成和调节障碍**：碘缺乏→甲状腺功能异常

## 临床表现（8类）

| 类别 | 典型表现 | 可能缺乏营养素 |
|------|----------|---------------|
| 全身性症状 | 疲劳、乏力、频繁感染 | 铁、B族维生素 |
| 皮肤与毛发 | 干燥苍白、反复皮肤问题、脱发 | 锌、维生素A |
| 神经系统 | 麻木、刺痛、肌肉无力 | 维生素B12 |
| 口腔 | 口角炎、牙龈出血 | 维生素C |
| 视听觉 | 夜盲、视力下降、耳鸣 | 维生素A |
| 消化系统 | 腹泻、便秘 | 锌 |
| 肌骨关节 | 骨折、骨痛、关节疼痛 | 钙、维生素D、维生素K2 |
| 心理和情绪 | 注意力不集中、记忆力下降、抑郁焦虑 | 维生素B6、B12 |

## 诊断

### 综合评估

1. **膳食史**：饮食习惯、食物选择、摄入量
2. **症状史**：上述临床表现
3. **既往史**：消化吸收障碍、长期用药
4. **体格检查**：体型、皮肤、头发、眼睛、口腔
5. **实验室检查**：血常规、肝肾功能、电解质、血清维生素D、B12、叶酸、锌、钙、磷
6. **膳食评估**：专业营养师进行膳食回顾或食物频率调查
7. **特殊检查**：双能X线评估骨骼健康、肌肉功能测试
8. **鉴别诊断**：排除甲状腺功能减退、慢性疾病等

## 治疗

### 营养代谢治疗

1. **个性化膳食建议**：确保摄入均衡充足的营养，鼓励多样化膳食
2. **营养补充**：根据缺乏情况提供相应补充剂
3. **健康教育**：科普隐性饥饿相关知识
4. **营养随访**：定期评估和调整治疗方案

### 药物治疗

- 特定营养缺乏（如[[缺铁性贫血]]）需药物治疗
- 疾病导致的营养不良需治疗原发病
