# 痛风中医辨证分型

## 定义

痛风中医辨证分型是根据中医理论对[[高尿酸血症]]与[[痛风]]人群进行的证候分类，不同证型对应不同的食养方案，是中医"辨证辨体，因人施膳"的核心依据。

## 病因病机

中医学认为[[痛风]]是一种**本虚标实**的疾病：
- 或先天禀赋不足，或后天失养所致
- 加之过食肥甘厚味，日久产生痰浊、湿热、瘀血
- 痹阻于筋骨关节，导致疾病发生

**核心病机**：先天脾胃虚弱，或后天饮食失养，损伤脾胃是痛风发病的关键因素；脾胃失调、湿浊内生是痛风反复不愈的症结所在。

## 四种辨证分型

### 湿浊证

| 项目 | 内容 |
|------|------|
| 常见分期 | 无症状[[高尿酸血症]]期、[[痛风]]间歇期 |
| 体质特点 | 痰湿体质多见 |
| 主要表现 | 肢体困乏沉重，形体肥胖，嗜食肥甘，口中粘腻不渴，大便黏 |
| 舌脉特征 | 舌淡胖，或有齿痕，苔白腻，脉滑 |
| 食养原则 | 避免寒冷诱发痛风急性发作；严格限制嘌呤和蛋白质总摄入量；低盐膳食；避免甜、油腻、酸、涩食物 |
| 推荐食物 | 薏苡仁、橘皮、茯苓代茶饮 |
| 推荐食药物质 | 橘皮、生姜、赤小豆、郁李仁、茯苓、葛根、菊苣、百合、砂仁、木瓜、黄芪 |

### 湿热证

| 项目 | 内容 |
|------|------|
| 常见分期 | 急性痛风性关节炎期 |
| 体质特点 | 湿热体质多见 |
| 主要表现 | 关节红肿灼热疼痛、疼痛剧烈，发作频繁，或伴发热，烦躁不安，口苦、口臭，大便黏滞或臭秽，或大便干 |
| 舌脉特征 | 舌质红，苔黄腻或黄厚，脉弦滑或滑数 |
| 食养原则 | 除严格限制嘌呤、蛋白质摄入之外，尤其应该限盐和烟酒；避免外感风寒、风热诱发急性发作 |
| 推荐食物 | 山竹、西瓜、荸荠等；赤小豆、木瓜、薏苡仁等煮汤饮用 |
| 推荐食药物质 | 薏苡仁、赤小豆、金银花、茯苓、桑叶、葛根、山药、马齿苋、菊苣、荷叶、栀子、决明子、白扁豆、余甘子、百合、鲜白茅根、菊花、五指毛桃、玉米须 |

### 痰瘀证

| 项目 | 内容 |
|------|------|
| 常见分期 | 痛风间歇期、慢性痛风性关节炎期 |
| 体质特点 | 血瘀和痰湿体质多见 |
| 主要表现 | 关节肿痛反复发作，关节局部有硬结或皮色暗红，或关节刺痛，屈伸不灵活、关节变形 |
| 舌脉特征 | 舌质紫暗，苔白腻，脉弦或弦滑 |
| 食养原则 | 多饮水；严格限制高脂、高胆固醇食物和高盐膳食 |
| 推荐食物 | 木耳、山楂、桃仁等 |
| 推荐食药物质 | 橘皮、桔梗、昆布、生姜、佛手、木耳、山楂、桃仁、姜黄、当归、山药、薏苡仁、茯苓、白扁豆、赤小豆、荷叶、葛根、菊苣、莱菔子、紫苏子、莲子、罗汉果、鱼腥草、重瓣玫瑰 |

### 脾肾亏虚证

| 项目 | 内容 |
|------|------|
| 常见分期 | 慢性痛风性关节炎期 |
| 体质特点 | 阳气亏虚体质多见 |
| 主要表现 | 关节疼痛反复发作、活动不灵活、僵硬或变形，腰膝酸软，乏力明显，肢体困倦沉重，腹胀，大便黏滞或溏稀 |
| 舌脉特征 | 舌淡胖，或有齿痕，舌苔白腻，脉沉缓或沉细 |
| 食养原则 | 宜食温热性食物 |
| 推荐食物 | 生姜、黄芪、茯苓、核桃、荔枝等 |
| 推荐食药物质 | 山药、芡实、莲子、茯苓、白扁豆、赤小豆、薏苡仁、葛根、菊苣、砂仁、大枣、龙眼肉、五指毛桃、人参（人工种植≤5年）、党参、黄芪 |

## 证型与西医分期的对应关系

| 西医分期 | 常见中医证型 |
|---------|------------|
| 无症状高尿酸血症期 | 湿浊证 |
| 急性痛风性关节炎期 | 湿热证 |
| 痛风间歇期 | 痰瘀证 |
| 慢性痛风性关节炎期 | 脾肾亏虚证 |

## 食养关键原则

- 痛风食养关键在于**调理脾胃**
- 推荐食用：白扁豆、玉米须、麦芽、山药、芡实、大枣、橘皮、山楂、五指毛桃、茯苓等
- 脾主肌肉，选择和缓、少量、持续的运动方式，使筋骨舒展，脾胃得健
