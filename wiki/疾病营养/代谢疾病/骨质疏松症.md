# 骨质疏松症

## 定义

骨质疏松症（Osteoporosis, OP）是一种以骨量减少、骨组织微结构破坏，导致骨脆性增加，易发生骨折为特征的全身性代谢性骨病。可发生于任何年龄，但多见于绝经后女性和老年人。人体骨量约在30-40岁达峰值骨量，主要由遗传因素决定，与性发育、营养和生活方式也密切相关。

## 分型

| 分型 | 特点 |
|------|------|
| 原发性I型（绝经后骨质疏松） | 与雌激素下降有关，多发于绝经后女性 |
| 原发性II型（老年性骨质疏松） | 与年龄相关的骨丢失，男女均可发生 |
| 继发性骨质疏松 | 继发于其他疾病（如甲亢、库欣综合征）或药物（如糖皮质激素） |
| 特发性骨质疏松 | 青少年和年轻成人中原因不明 |

## 诊断标准

基于DXA测定骨密度（BMD）：

| 分类 | T值 |
|------|------|
| 正常 | T >= -1.0 |
| 骨量减少 | -2.5 < T < -1.0 |
| 骨质疏松 | T <= -2.5 |
| 严重骨质疏松 | T <= -2.5 + 脆性骨折 |

## 代谢特征

骨重建失衡：骨吸收增加和/或骨形成减少，处于负平衡。

## 营养代谢紊乱特点

### 钙
- 成年人全身钙量约1100-1200g，99%存在于骨骼和牙齿
- 每天400-1000mg钙摄入可维持钙平衡
- 长期钙摄入不足+吸收不良+排泄增多→骨矿物质丢失→血钙降低→继发性甲旁亢→骨吸收增强→OP

### 磷
- 成年人全身磷约500-800g
- 高磷摄入→抑制1,25(OH)2D3生成→刺激PTH分泌→骨吸收增强
- 钙磷比1:1-2:1时钙吸收最充分

### 维生素D
- 血清25-羟维生素D是反映维生素D营养状态的最佳指标
- OP患者应长期维持25-羟维生素D>=30ng/ml（75nmol/L）
- >150ng/ml可能出现高钙血症
- 缺乏或不足者：每日口服D3 1000-2000IU，2-3个月后检测

### 维生素A
- 参与骨胶原和糖胺聚糖合成
- 过低（<500μgRE/d）或过高（>15000μgRE/d）均损害骨骼健康

### 维生素K
- 骨钙素分子中谷氨酸残基需在维生素K依赖性羧化酶作用下羧化为γ-羧化谷氨酸
- γ-羧化谷氨酸与骨羟磷灰石中钙离子结合
- 维生素K缺乏→与羟磷灰石结合力低下→骨骼矿化异常

### 蛋白质
- 双重性：足够蛋白质促进骨基质合成+刺激骨骼肌蛋白合成；过多→尿钙增加
- 蛋白质>75g/d且钙<600mg/d→负钙平衡

## 营养治疗

### 保持合理体重
- 避免体重过低（女性BMI>=23，男性BMI>=24）
- 防止骨骼肌量不足

### 保证充足钙摄入
- 首选牛乳及乳制品
- 成年人每日摄入钙元素（膳食+补充）800mg/d
- 50岁以上1000mg/d

### 补充维生素D
- 充足阳光照射（每天约15分钟，避免涂防晒霜）
- 富含维生素D食物：三文鱼、金枪鱼、牡蛎、明虾
- 缺乏者：D3 1000-2000IU/d
- OP患者：血清25-羟维生素D长期维持>=30ng/ml

### 适量磷摄入
- 适宜供给量700mg/d，可耐受最高摄入量3000mg/d

### 避免过多钠盐
- 成年人每日食盐<=5g

### 适量蛋白质
- 成年人1.0-1.2g/(kg·d)，动物:植物=1:1
- 老年人比正常成年人每日增加10%

### 补充维生素K
- 多吃发酵大豆（纳豆）、海藻、绿叶蔬菜、茶叶

### 科学烹调
- 含草酸高的蔬菜先用热水焯烫
- 面粉玉米粉加入发酵剂发酵使植酸水解

### 纠正不良生活方式
- 戒烟限酒（每日酒精<=15g）
- 避免过量咖啡、碳酸及含糖饮料

## 2026版食养与风险评估补充

### 关键流行病学与适用范围

- 2018 年中国研究数据显示：50 岁以上人群[[骨质疏松症]]患病率为 **19.2%**，其中女性 **32.1%**、男性 **6.9%**
- 65 岁以上患病率升至 **32.0%**，其中女性 **51.6%**、男性 **10.7%**
- 2026 版食养指南主要面向**成人原发性骨质疏松症**人群，强调食养用于预防和辅助改善，不替代药物治疗

### 食养重点

| 项目 | 建议 |
|------|------|
| 钙 | 50 岁以上 1000-1200 mg/d，尽量通过膳食达到 1000 mg 以上 |
| 维生素D | 18-64 岁 10 μg/d；65 岁及以上 15 μg/d；治疗期 25(OH)D 宜维持 >30 ng/mL |
| 蛋白质 | 1.2-1.5 g/(kg·d)，优质蛋白比例不低于 50% |
| 奶类 | ≥300 mL/d 或相当量奶制品，发酵乳制品有额外优势 |
| 全谷物 | 不低于主食量的 30% |
| 鱼虾 | 每周至少 2 次，推荐深海鱼 |
| 蔬菜水果 | 18-64 岁蔬菜 300-500 g/d、水果 200-350 g/d；65 岁及以上蔬菜 300-450 g/d、水果 200-300 g/d |
| 运动 | 每周不少于 2 次，每次 30-60 分钟，负重/抗阻/平衡训练结合 |

### 风险筛查工具

- **IOF 1 分钟测试题**：19 项危险因素中任一项阳性，即提示存在骨质疏松风险
- **OSTA**：`[体重(kg) - 年龄(岁)] x 0.2`
  - `>-1` 低风险
  - `-4 至 -1` 中风险
  - `<-4` 高风险
- 如需更具体的营养干预和执行建议，可联读[[骨质疏松营养]]与[[骨代谢标志物]]
