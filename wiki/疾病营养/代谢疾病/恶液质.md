# 恶液质

## 定义

恶液质（cachexia）是一种以持续性骨骼肌丢失（伴或不伴脂肪丢失）为特征的多因素综合征，不能通过常规营养支持完全逆转，导致进行性功能障碍。其本质是疾病相关性炎症介导的[[厌食]]条件下的静息能量消耗（REE）增加。

## "1-2-3-4-5"框架

| 要素 | 内容 |
|------|------|
| 1个本质 | 疾病相关炎症介导的厌食条件下REE增加 |
| 2大核心 | 营养摄入减少 + 体重持续丢失 |
| 3个阶段 | 恶液质前期→恶液质期→难治期 |
| 4大诊断要素 | 现患疾病、厌食、体重进行性丢失、炎症负荷升高 |
| 5项治疗原则 | 多模式、多靶点、多方面、全方位整合治疗 |

## 诊断标准演变

### 2008年国际专家共识

- 慢性疾病
- 12个月内体重下降>=5% 或 BMI<20 kg/m2
- 以下至少3项：肌力下降、疲劳、厌食、FFMI低、生化异常（CRP>5.0mg/L, IL-6>4.0pg/ml, Hb<120g/L, ALB<32g/L）

### 2011年肿瘤恶液质国际共识

- 无节食条件下6个月内体重下降>5%
- 或BMI<20 kg/m2患者体重下降>2%
- 或ASMI符合[[肌少症]]标准患者体重下降>2%

### 2023年亚洲恶液质工作小组

**主要标准**：
1. 现患潜在疾病
2. 3-6个月内体重非自主性丢失>2% 或 BMI<21 kg/m2

**辅助标准**（符合1项即可）：
3. 厌食（主观症状）
4. 握力下降（男性<28kg，女性<18kg）
5. CRP升高（>5mg/L）

## 分期

| 分期 | 诊断标准 |
|------|----------|
| 恶液质前期 | 体重减轻<=5%，厌食和代谢改变 |
| 恶液质期 | 体重减轻>5% 或 BMI<18.5且体重减轻>2% 或 肌肉减少且体重减轻>2%，常有食物摄入减少/系统性炎症 |
| 恶液质难治期 | 分解代谢增强，对治疗无反应，低体能评分，预期生存期<3个月 |

### CSS分期评分表

于世英教授团队发明的快速诊断分期评分表（cachexia staging score, CSS），临床区分能力更强、预后预测更准、操作更简便。

## 预后预测指标

低AGR（白蛋白/球蛋白比值）、低Hb、低NRI及低握力是肿瘤恶液质的独立预后指标。

## 营养治疗

### 营养配方推荐

| 参数 | 推荐量 |
|------|--------|
| 能量密度 | >1.0 kcal/ml |
| 蛋白质 | >1.2 g/(kg·d) |
| EPA | 2.0 g/d |
| BCAA | >35%（特别是亮氨酸） |
| omega-3 PUFA | 高含量 |

> [!important]
> 推荐以FSMP在餐间补充，不推荐以FSMP代替日常食物。单纯增加能量摄入不能逆转恶液质进程。

### 营养代谢调节

| 药物/营养素 | 作用 | 剂量 |
|-------------|------|------|
| HMB（亮氨酸代谢产物） | 修饰NF-kB预防恶液质，促进骨骼肌合成 | 3-6 g/d |
| 左旋肉碱 | 改善疲劳、肌肉无力、生活质量 | 4 g/d |
| DCA（二氯乙酸盐） | 激活丙酮酸脱氢酶，促进氧化磷酸化 | - |
| 维生素B1（大剂量） | 抗增殖效应，促进肿瘤凋亡 | >推荐剂量75倍 |
| 谷氨酰胺 | 抑制体重丢失 | 14 g/d |
| 联合方案 | 增加瘦体质 | 谷氨酰胺14g + HMB 3-6g + 精氨酸3g |

### 药物治疗

| 药物类别 | 代表药物 | 备注 |
|----------|----------|------|
| 孕酮类 | 醋酸甲地孕酮 | 逆转体重下降，但难以逆转肌肉丢失 |
| NSAIDs | 塞来昔布200mg b.i.d | 降低REE，增加摄食量 |
| 沙利度胺 | 抑制TNF-α | 改善瘦体组织 |
| omega-3脂肪酸 | EPA | 抗炎 |
| 食欲刺激剂 | 阿那莫林 | 选择性胃饥饿素受体激动剂，57%患者3周后摄入增加 |
| 糖皮质激素 | 强的松20-40mg/d | 短期使用 |
| 5-HT3拮抗剂 | 米氮平/奥氮平 | 改善食欲 |

> [!note]
> 联合用药比单独用药效果好，提倡多药联合使用。
