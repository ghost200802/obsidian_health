# 衰弱综合征

## 定义

衰弱（frailty）是一种多个生理系统功能和储备下降致压力源易感性增加的状态，其特征是生理储备下降导致机体易损性增加、抗应激能力减退。表现为疲乏、行动迟缓、肌力减弱、体力活动下降和无意识的体重减轻。

## 病因与危险因素

### 四类危险因素

1. **社会环境因素**：高龄、女性、少数民族、低收入、低教育水平、独居
2. **临床因素**：多病（慢性疾病、[[营养不良]]、[[肥胖]]、抑郁、认知受损）、多药治疗
3. **行为和生活方式**：缺乏体力活动、低蛋白质摄入、吸烟、酒精摄入增加
4. **生物学因素**：炎症（细胞因子/CRP升高）、内分泌因子异常、微量营养素缺乏（类胡萝卜素、维生素B6、维生素D、维生素E）

### 发病机制

慢性低度炎症（炎性衰老）是关键病理生理过程，通过中间生理系统（肌肉骨骼、内分泌、血液系统）引起各脏器生理储备功能减退。

## 诊断

### Fried衰弱表型（5条标准）

| 标准 | 判断标准 |
|------|----------|
| 低体力活动 | 只参与日常生活走路和低体力活动 |
| 疲乏 | CES-D>=10分 |
| 低握力 | 男性<26.5kg，女性<17kg |
| 步速减慢 | 男性<0.85m/s，女性<0.75m/s |
| 体重减轻 | 过去1年体重减少>5kg 或 BMI<18.5 |

| 符合项数 | 诊断 |
|----------|------|
| 0项 | 非衰弱 |
| 1-2项 | 衰弱前期（pre-frailty） |
| >=3项 | 衰弱 |

### FRAIL量表

| 条目 | 评分 |
|------|------|
| 疲惫（过去1个月总是/大部分时间感觉疲惫） | 1分 |
| 耐力下降（独立上下10级台阶受限） | 1分 |
| 行走能力下降（独立行走100米受限） | 1分 |
| 多种疾病共存（>=5种慢性疾病） | 1分 |
| 体重减轻（最近1年体重下降>5%） | 1分 |

| 总分 | 诊断 |
|------|------|
| 0分 | 健康 |
| 1-2分 | 衰弱前期 |
| >=3分 | 衰弱 |

### 衰弱生物标志物

联合检测血肌酐和胱抑素C：
- **公式**：切点值 = 55.56 x 胱抑素C(mg/L) - 34.03 x 肌酐(mg/dl) + 56.95
- **切点值<77**：低风险
- **77-119**：中风险
- **>119**：高风险

## 治疗

### 运动治疗（最有效方式之一）

- **抗阻运动**：增加肌肉量和力量
- **有氧运动**：促进心肺健康，减少氧化损伤
- **平衡训练**：防止跌倒
- **柔韧性运动**

### 营养治疗

- **蛋白质补充**：可增加肌肉量，改善肌力
- **地中海膳食**：降低衰弱综合征发生率，改善肠道微生物菌群
- **营养治疗+运动锻炼**有协同作用

### 间充质干细胞（MSC）疗法

- 骨髓间充质干细胞移植已完成I期和II期试验
- 可改善免疫功能、6分钟步行距离、1秒用力呼气量
- 降低血TNF-α水平
- 暂未发现不良反应
