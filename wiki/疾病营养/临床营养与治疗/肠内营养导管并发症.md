# 肠内营养导管并发症

## 概述

[[肠内营养]]的导管相关并发症是指营养导管在放置和长期应用过程中出现的各种并发症，主要与胃肠营养管的质地、管径、置管部位、置管方法和操作技术、导管护理等因素有关。

## 鼻、咽及食管损伤

### 原因
- 营养管局部长时间压迫或反复滑动
- 固定胶带对皮肤的刺激

### 表现
- 反复出现的鼻翼部皮肤糜烂
- 鼻、咽、食管黏膜充血、糜烂或坏死
- 咽喉部或食管溃疡、出血，甚至形成狭窄
- 鼻窦炎、中耳炎等

### 预防措施
1. **规范置管**：置管前以医用润滑剂充分润滑管前端，轻柔推送，遇阻力即调整
2. **妥善固定**：避免扭曲、受压，定期更换粘贴部位
3. **注意鼻咽部情况**：保持无管腔侧鼻孔通畅，做好口腔和鼻腔清洁
4. **更换置管方式**：长期管饲（>4周）建议经胃造口或空肠造口置管

## 营养管堵塞

### 常见原因
1. 使用细管径营养管而护理不周
2. 输注前后未冲管或冲洗不够
3. 营养液黏度较高和/或大分子聚合物较多
4. 经营养管给药时药片研磨不充分、不易溶解或与营养液混合后生成沉淀
5. 营养管侧孔吸入胃肠道黏膜

### 处理方法
1. 注入温水，浸泡管腔后用5ml注射器反复抽吸疏通
2. 无效：改用碳酸氢钠溶液或碳酸饮料冲管
3. 仍无效：注入胰酶溶液充满管腔，夹闭5-10分钟后再反复抽吸
4. 以上均无效：及时更换营养管

### 预防措施
1. **冲管**：每次输注营养液或药物前后均应冲管，用小号注射器脉冲式方法
2. **给药注意**：避免固体药物；必须用时完全溶解或研磨成细粉；暂停营养液供给；避免配伍禁忌
3. **管径选择**：高黏度配方（如匀浆膳）宜使用14Fr及以上直径的营养管

## 营养管移位和脱出

### 常见原因
- 患者自行拔出
- 营养管固定不牢或松脱
- 患者翻身不慎所致牵拉
- 咳嗽、打喷嚏、拍背时腹内压力升高

### 预防措施
1. 选择患者耐受性更好的细径、软质营养管
2. 意识障碍或躁动不安者使用约束带、腹带
3. 妥善固定、定期检查管道位置
4. 确认经皮胃肠造口管的外露标记位置

## 营养管渗漏或折断

### 常见原因
- 未及时更换营养管
- 冲管不及时、不充分
- 暴力疏通营养管
- 长时间使用、反复输注油脂等物质致材质变性

### 预防措施
- 按期更换营养管
- 按要求冲管
- 避免输注损伤营养管的物质
- 加强导管维护指导
