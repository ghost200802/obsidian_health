# 临床营养治疗

## 定义

临床营养治疗是通过营养干预手段预防、治疗疾病，促进患者康复的综合方法。它涵盖营养筛查、营养诊断、营养干预和营养监测的完整流程，是现代临床治疗不可或缺的组成部分。

![[raw/中国营养学会/临床营养学 (石汉平，陈伟) (z-library.sk, 1lib.sk, z-lib.sk).pdf#page=97]]

## 五阶梯营养治疗

营养治疗按照干预强度递进分为五个阶梯，优先选择较低阶梯的治疗方式：

```
阶梯1：饮食指导 + 教育
   ↓（效果不佳时升级）
阶梯2：饮食调整 + 食物强化
   ↓
阶梯3：全营养补充（口服营养补充，ONS）
   ↓
阶梯4：肠内营养（EN）— 管饲喂养
   ↓
阶梯5：肠外营养（PN）— 静脉营养
```

| 阶梯 | 方式 | 适用场景 |
|------|------|---------|
| 第1阶梯 | 饮食指导+营养教育 | 轻度营养风险，可经口进食 |
| 第2阶梯 | 饮食调整+食物强化 | 经口进食但需要调整膳食结构 |
| 第3阶梯 | 口服营养补充（ONS） | 经口进食不足，需要额外营养补充 |
| 第4阶梯 | [[肠内营养]]（EN） | 胃肠功能存在但无法充分经口进食 |
| 第5阶梯 | [[肠外营养]]（PN） | 胃肠功能严重障碍，无法使用EN |

> [!important]
> 营养治疗应遵循"只要胃肠道有功能，就首选肠内营养"的原则。肠外营养仅在胃肠道功能完全丧失或无法满足营养需求时使用。

## 营养三级诊断

### 一级诊断：营养筛查

- **目的**：快速识别存在营养风险的患者
- **时机**：入院24小时内完成
- **工具**：
  - **NRS 2002**（Nutritional Risk Screening 2002）：住院患者首选筛查工具，评分>=3分表示有营养风险
  - **MNA**（Mini Nutritional Assessment）：适用于老年患者
  - **MUST**（Malnutrition Universal Screening Tool）：英国通用筛查工具

### 二级诊断：营养评估

- **目的**：对筛查阳性的患者进行详细评估
- **内容**：膳食调查、体格检查、人体测量、生化指标
- **方法**：
  - PG-SGA（患者主观整体评估）
  - SGA（主观整体评估）
  - 膳食回顾/记录

### 三级诊断：综合评定

- **目的**：确定营养不良的类型、程度和原因
- **内容**：结合前两级结果，进行综合判断
- **输出**：营养诊断、营养治疗计划

## 重要筛查工具详解

### NRS 2002

适用于一般住院患者的营养风险筛查：
- **营养状况受损评分**（0-3分）：BMI、体重变化、膳食摄入
- **疾病严重程度评分**（0-3分）：根据疾病对营养的影响
- **年龄调整**：>=70岁加1分
- **总分>=3分**：存在营养风险，需要制定营养治疗计划

### MNA

专为老年患者设计的营养评估工具：
- 筛查部分（6项）：快速判断营养状况
- 全面评估部分（12项）：详细评估营养风险
- 总分<17分：营养不良；17-23.5分：有营养不良风险；>=24分：正常

### MUST

适用于社区和住院患者的通用筛查工具：
- BMI评分
- 体重下降评分
- 急性疾病影响评分（禁食5天以上）
- 分为低风险、中等风险、高风险

## 营养治疗一线化

推动营养治疗从"辅助治疗"走向"一线治疗"：

- 营养不良本身就是一种疾病，需要独立诊断和治疗
- 营养治疗应与药物、手术同等对待
- 早期营养干预可改善临床结局、缩短住院时间、降低医疗费用
- "营养是一线治疗"的理念正在被广泛接受

## 精准营养方向

未来临床营养治疗的发展趋势：

- **基因组营养学**：基于基因多态性制定个性化营养方案
- **代谢组学**：通过代谢物谱分析评估营养状态
- **肠道微生态**：利用益生菌、益生元调节肠道菌群
- **人工智能**：辅助营养诊断和方案制定
- **个体化营养制剂**：根据患者代谢特征定制营养配方

## 营养治疗的适应证

| 适应证类别 | 具体疾病/情况 |
|-----------|-------------|
| 营养不良 | 重度营养不良、恶液质 |
| 消化系统疾病 | 短肠综合征、炎性肠病、胰腺炎 |
| 危重症 | 创伤、感染、大手术后 |
| 肿瘤 | 癌症相关营养不良、放化疗期间 |
| 慢性病 | CKD、COPD、心力衰竭 |
| 神经系统 | 卒中后吞咽障碍、痴呆 |
| 烧伤 | 大面积烧伤后高代谢状态 |

## 临床营养相关治疗（第十二章补充）

> 数据来源：临床营养学第十二章

[[临床营养相关治疗]]是指根据疾病种类、病情和患者营养状况，通过营养素调整和营养治疗改善患者疾病状况、促进康复和预防并发症的治疗方法。主要包括：

| 治疗方式 | 说明 |
|---------|------|
| [[营养代谢调节治疗]] | 通过膳食、营养手段调整机体代谢过程（能量、碳水化合物、蛋白质、脂肪、微量营养素代谢调节） |
| [[消化液回输]] | 将收集的消化液经处理后重新输入消化道（适用于高流量肠瘘等） |
| [[粪菌移植]] | 移植健康人功能菌群重建肠道微生态（适用于艰难拟梭菌感染、炎性肠病等） |
| [[免疫调节治疗]] | 补充免疫营养物质增强免疫调节能力（omega-3脂肪酸、精氨酸、谷氨酰胺等） |
| [[减重与增重]] | 通过营养干预实现体重管理 |

## 营养治疗流程（第八章补充）

> 来源：[[source-临床营养学]] 第八章：临床营养治疗通则

### 营养管理完整流程

```
1. 营养筛查（入院24h内）
   → 筛查工具：NRS-2002、MNA、MUST等
   → 筛查阳性 → 进入营养评估

2. 营养评估与诊断
   → 膳食调查 + 人体测量 + 生化检查
   → SGA/PG-SGA评估
   → GLIM标准确诊营养不良
   → 确定营养不良分型和分级

3. 营养干预
   → 确定营养需求（能量、蛋白质等）
   → 选择治疗阶梯和方式
   → 制定营养处方

4. 营养监测
   → 监测营养治疗效果
   → 调整营养方案
   → 评估不良反应
```

### 营养治疗的核心原则

| 原则 | 说明 |
|------|------|
| 个体化 | 根据患者的疾病状态、代谢特点、营养需求制定方案 |
| 优先EN | 只要胃肠道有功能，首选肠内营养 |
| 循序渐进 | 从低剂量开始，逐渐增加至目标量 |
| 动态评估 | 定期评估效果，及时调整方案 |
| 整体性 | 营养治疗与药物治疗、手术治疗协调配合 |

### 能量与蛋白质需求

| 参数 | 一般患者 | 危重症/高代谢 |
|------|---------|-------------|
| 能量 | 25-30 kcal/(kg*d) | 25-35 kcal/(kg*d) |
| 蛋白质 | 1.0-1.2 g/(kg*d) | 1.5-2.0 g/(kg*d) |
| 热氮比 | 150:1 | 100:1 |

### 营养治疗的监测指标

| 监测项目 | 频率 | 目标 |
|---------|------|------|
| 体重 | 每周2-3次 | 体重稳定或上升 |
| 前白蛋白 | 每周1-2次 | 趋于正常范围 |
| 氮平衡 | 每1-2周 | 转为正平衡 |
| 血糖 | 每日（PN时） | 7.8-10.0 mmol/L |
| 电解质 | 每周1-2次 | 维持正常范围 |
| 肝肾功能 | 每周1次 | 监测器官功能 |
