# 器官移植营养

## 定义

器官移植（Organ Transplantation）患者是特殊高危营养不良人群，术后长期使用免疫抑制剂，可致免疫力下降、感染风险增加、代谢紊乱。营养治疗对移植患者至关重要。

## 代谢特征

| 类型 | 特点 |
|------|------|
| 能量代谢 | 免疫抑制+感染风险+应激 → 能量需求增加 |
| 糖代谢 | 移植肝缺血影响+促分解激素增加+糖耐量下降+抗排斥药抑制胰岛素释放 → IR |
| 蛋白质代谢 | 合成减少、分解增强 → 负氮平衡 |
| 脂代谢 | 糖皮质激素增加VLDL分泌+环孢素减少清除 → 14%-43%高脂血症 |
| 维生素/微量元素 | 免疫抑制剂致维生素D、B12、叶酸、铁、钙、镁缺乏 |

## 不同器官移植特点

### 肝移植
- 术前：肝病终末期，糖脂蛋白代谢失调，低蛋白血症、腹水、血氨升高
- 术后：无肝期肠道缺血再灌注 → 肠通透性增加；强烈应激 → 分解代谢增强

### 肾移植
- 术前：低蛋白质膳食+透析 → 营养不良
- 术后：免疫抑制剂影响糖、蛋白、脂、尿酸、电解质 → 高血糖、高血脂、骨质疏松

### 肺移植
- 术前营养不良发生率20%-70%，合并呼吸衰竭者高达60%
- 机械通气患者>70%营养不良
- 术后营养不良 → 呼吸肌萎缩、呼吸机依赖率增高

### 小肠移植
- 术前长期PN → 营养不良比例极高
- 术后经历缺血再灌注、去神经、淋巴回流中断等漫长恢复

## BMI相对禁忌证（表18-6）

| 器官 | BMI相对禁忌证 |
|------|-------------|
| 心 | <18.5 或 >35 kg/m² |
| 肺 | <16 或 >35 kg/m² |
| 肝 | >40 kg/m² |
| 肾 | 不应仅因肥胖排除 |

## 营养治疗方案

### 途径选择

- 日常膳食首选，少量多餐
- ONS补充不足
- 不能经口 → 管饲EN
- EN不足或不能实施 → PN
- 肠道有部分功能就应用EN，完全无功能才TPN

### 营养素需要量（表18-7）

| 营养素 | 推荐量 |
|--------|--------|
| 能量 | BEE×(1.2-1.5) 或 15-35 kcal/kg（理想体重） |
| 蛋白质 | 1.2-2.5 g/(kg·d)（理想体重） |
| 糖类 | 占总能量40%-50% |
| 脂肪 | 占总能量30% |
| 液体 | 1ml/kcal，根据出入量调整 |

### 恢复进食时间

- 一般术后3-5日恢复膳食
- 肾移植后第1日即可进食
- 小肠移植需1周以上

### 器官特异性营养制剂

| 器官 | 营养制剂特点 |
|------|------------|
| 肝功能不全 | 含BCAA多、芳香氨基酸少 |
| 肾功能不全 | 足够能量、必需氨基酸、组氨酸、少量脂肪及电解质 |
| 肺移植 | 低碳水高脂肪营养品（减少CO2） |
