# 医院膳食

## 定义

医院膳食（Hospital Diet）是指在医疗机构中为住院患者提供的各类膳食，包括基本膳食和治疗膳食两大类。医院膳食是[[临床营养治疗]]的重要组成部分，根据患者的疾病特点、营养需求和消化功能进行个体化设计。

## 基本膳食

基本膳食适用于一般住院患者，是医院膳食的基础形式：

| 膳食类型 | 特点 | 适用人群 |
|---------|------|---------|
| 普食 | 与健康人日常膳食基本相同，营养均衡 | 消化功能正常、无特殊营养需求的患者 |
| 软食 | 质地柔软、易咀嚼、易消化，减少膳食纤维和粗糙食物 | 低热患者、消化功能轻度减弱者、老年患者 |
| 半流质 | 外观半流体，含少量细软纤维，易消化吸收 | 发热、消化功能中度减弱、口腔疾病患者 |
| 流质 | 完全流体状态，无渣，易吞咽，热量和营养密度较低 | 高热、消化道术后、急性感染、吞咽困难患者 |

### 普食

- 每日能量约1800-2200 kcal
- 蛋白质约60-75 g/d
- 三餐分配：早餐25%-30%，午餐40%，晚餐30%-35%

### 软食

- 在普食基础上，将食物切碎、煮烂
- 减少粗纤维和刺激性食物
- 每日3-5餐

### 半流质

- 每日能量约1500-1800 kcal
- 蛋白质约50-60 g/d
- 少食多餐，每日5-6餐
- 限制粗纤维、浓汤、刺激性调味品

### 流质

- 每日能量约800-1200 kcal（营养密度低，不宜长期使用）
- 蛋白质约30-40 g/d
- 每日6-7次，每次200-250ml
- 分为清流质（无脂无蛋白）和浓流质

## 治疗膳食

治疗膳食是在基本膳食基础上，根据特定疾病需要进行的营养调整。参见[[医学营养治疗膳食]]了解详细分类。

### 常见治疗膳食分类

| 治疗膳食 | 调整内容 | 适用疾病 |
|---------|---------|---------|
| 低盐膳食 | 每日钠<2g | 高血压、心力衰竭、肾病水肿 |
| 低脂膳食 | 脂肪<40g/d或<20%能量 | 胆囊疾病、高脂血症、胰腺炎 |
| 低蛋白膳食 | 蛋白质0.5-0.8 g/(kg·d) | 慢性肾脏病、肝性脑病 |
| 高蛋白膳食 | 蛋白质1.2-1.5 g/(kg·d) | 营养不良、术后恢复、烧伤 |
| 低嘌呤膳食 | 嘌呤<150mg/d | [[痛风]]急性期 |
| 糖尿病膳食 | 控制碳水总量和GI | [[糖尿病]] |
| 低渣膳食 | 减少膳食纤维 | 肠道手术前后、消化道出血 |
| 高纤维膳食 | 膳食纤维25-35g/d | 便秘、憩室病 |
| 低碘膳食 | 碘<50μg/d | 甲亢、放射性碘治疗前 |
| 管饲膳食 | 经管饲途径提供 | 吞咽障碍、意识障碍 |
