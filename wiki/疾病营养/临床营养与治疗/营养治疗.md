# 营养治疗

## 定义

营养治疗（Nutrition Therapy）是通过营养干预手段预防、治疗疾病，促进患者康复的方法总称。它涵盖了从饮食指导到肠外营养的完整干预谱系。

## 营养治疗的范畴

营养治疗是[[临床营养]]的核心实践内容，包括：

- **营养筛查与诊断**：识别营养风险和营养问题，参见[[营养诊断]]
- **膳食治疗**：通过调整膳食结构治疗疾病，参见[[医学营养治疗膳食]]
- **营养支持**：通过[[肠内营养]]或[[肠外营养]]为无法正常进食者提供营养
- **营养教育**：提高患者和公众的营养知识水平

## 营养治疗的目标

1. **预防疾病**：通过合理营养预防营养缺乏病和慢性病
2. **辅助治疗**：作为疾病治疗的重要手段，改善临床结局
3. **促进康复**：加速术后/病后恢复
4. **提高生活质量**：改善患者的营养状态和功能状态
5. **降低医疗费用**：通过有效的营养干预降低总体医疗支出

## 营养治疗的基本流程

```
患者入院/就诊
    → 营养筛查（NRS2002/MNA/MUST）
    → 筛查阳性？
        → 是 → 营养评估（膳食/体格/生化/人体组成）
        → 营养诊断（PES格式）
        → 制定营养治疗方案
        → 实施（参见[[五阶梯营养治疗]]）
        → 监测与随访
        → 方案调整
```

## 营养治疗的重要性

- 营养不良住院患者并发症发生率和死亡率显著升高
- 有效的营养治疗可缩短住院时间、降低再入院率
- 营养治疗已被纳入多部临床指南的推荐

## 营养治疗在现代综合治疗中的地位

营养治疗在现代综合治疗中与医疗和护理**同等重要**，是临床治疗不可或缺的三大支柱之一。随着循证医学的发展，营养治疗已被越来越多的临床指南纳入核心推荐。

> **中医经典引用**：*"毒药攻邪，五谷为养，五果为助，五畜为益，五菜为充"* ——《黄帝内经》，体现了中国古代医学对营养治疗的深刻认识。

## 营养治疗六大目的详解

根据《实用营养师手册》第6章，营养治疗具有以下六大治疗目的：

### 1. 调整营养素摄入

根据疾病类型和病理生理特点，针对性地调整[[蛋白质]]、[[脂类]]、[[碳水化合物]]等[[宏量营养素]]的摄入量和比例。例如，肾病综合征患者需增加优质蛋白摄入，慢性肾脏病患者则需限制蛋白质摄入。

### 2. 减轻脏器负荷

通过限制某些营养素的摄入量，减轻患病器官的代谢负担，为组织修复创造有利条件。例如：
- 低蛋白饮食减轻肾脏负荷
- 低脂饮食减轻肝胆负荷
- 限钠饮食减轻心脏负荷

### 3. 控制营养素摄入

对特定营养素进行严格的量化控制，使其维持在治疗所需的范围内。例如：
- 糖尿病患者控制碳水化合物摄入量及血糖生成指数
- 高尿酸血症患者控制嘌呤摄入量
- 高血压患者控制钠盐摄入量

### 4. 有利于消化吸收

通过调整食物的质地、烹调方式和餐次安排，适应患者消化功能的变化，确保营养物质能够被有效消化和吸收。参见[[医院基本饮食]]中的软食、半流质和流质饮食。

### 5. 补充热能及营养素消耗

补偿因疾病导致的额外能量和营养素消耗，纠正或预防营养不良。例如：
- 烧伤患者的高代谢状态需要大幅增加热能和蛋白质供给
- 手术后患者需要额外营养支持促进伤口愈合
- 恶性肿瘤患者需对抗肿瘤消耗导致的营养不良

### 6. 辅助临床诊断

通过特定的饮食安排配合临床检查，提高诊断的准确性。例如：
- 隐血试验饮食：排除干扰隐血检测的食物
- 葡萄糖耐量试验前的饮食准备
- 代谢平衡试验饮食
