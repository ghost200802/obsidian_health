# 治疗饮食

## 定义

治疗饮食（Therapeutic Diet），又称**疗效饮食**，是指为治疗疾病或代谢机能紊乱而专门设计的饮食。它在热能、[[宏量营养素]]种类和数量、食物连贯性（质地）、烹调方法等方面进行针对性调整，以配合临床治疗、促进康复。

治疗饮食是在[[医院基本饮食]]（普食、软食、半流质、流质）的基础上发展而来的，一切治疗饮食、试验饮食和代谢饮食均以四种基本饮食为基础形式。

> **中医经典引用**：*"毒药攻邪，五谷为养，五果为助，五畜为益，五菜为充"* ——《黄帝内经·素问》，强调药物治疗与饮食调养的配合关系，饮食是疾病治疗中不可或缺的组成部分。

## 疗效饮食的基本原则

1. **成分调整不应违反营养原则**：在调整营养素种类和数量时，须保证足量营养素的供给，避免因过度限制导致新的营养问题
2. **个体化原则**：根据患者的疾病类型、严重程度、代谢状态和营养状况制定个性化方案
3. **动态调整**：随病情变化及时调整饮食方案
4. **兼顾可接受性**：在满足治疗目的的同时兼顾患者的饮食习惯和接受度

## 常见需要疗效饮食的疾病

| 疾病/代谢紊乱 | 对应治疗饮食 | 调整要点 |
|-------------|------------|---------|
| [[糖尿病]] | 糖尿病饮食/低GI饮食 | 控制碳水化合物总量及GI值，参见[[糖尿病营养]] |
| 苯丙酮尿症 | 低苯丙氨酸饮食 | 严格限制苯丙氨酸摄入 |
| 肾脏疾病 | 低蛋白饮食/限磷限钾饮食 | 控制蛋白质0.5-0.8 g/(kg·d)，限制磷钾摄入 |
| 半乳糖血症 | 无半乳糖饮食 | 完全排除含半乳糖的食物 |
| [[痛风]]/高尿酸血症 | 低嘌呤饮食 | 嘌呤摄入<150mg/d，参见[[痛风营养]] |
| 高血压 | 低盐饮食 | 每日钠<2g |
| 高脂血症 | 低脂饮食 | 脂肪<40g/d或<20%能量 |
| 消化道疾病 | 低渣/无渣饮食 | 减少膳食纤维摄入 |
| 肝性脑病 | 低蛋白饮食 | 限制蛋白质以减少氨的产生 |

## 治疗饮食与基本饮食的关系

```
基本饮食（普食/软食/半流质/流质）
    └── 在此基础上调整 → 治疗饮食
        ├── 调整热能总量（高能/低能饮食）
        ├── 调整营养素比例（高蛋白/低蛋白/低脂/低糖饮食）
        ├── 调整特定成分（低盐/低嘌呤/低碘/限磷限钾饮食）
        └── 调整食物质地（低渣/无渣/管饲饮食）
```
