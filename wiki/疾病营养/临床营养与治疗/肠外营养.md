# 肠外营养

## 定义

肠外营养（Parenteral Nutrition, PN）是指人体所需的营养素不经胃肠道而直接进入血液循环，以满足维持和修复机体组织需要的一种营养治疗方式。

### 分类
- **部分肠外营养（PPN/SPN）**：补充性肠外营养，肠内营养不足时部分能量和营养素由肠外营养补充
- **全肠外营养（TPN）**：全部能量和营养素均通过静脉补充

## 历史发展

| 年份 | 里程碑 |
|------|--------|
| 1616 | Harvey W 发现血液循环，为静脉输液奠定基础 |
| 1831 | Latta T 用煮沸后食盐水经静脉给霍乱患者补液 |
| 1945 | Brunschwing A 等首次提出周围静脉营养概念 |
| 1961 | Wretlind A 研发大豆油脂肪乳剂并成功应用 |
| 1967 | Dudrick SJ 和 Wilmore DW 首次应用全静脉营养成功救治短肠综合征婴儿 |
| 1968 | 证实TPN可使动物和人生长发育并维持正氮平衡 |
| 1969 | Shils ME 开展第1例家庭肠外营养（HPN） |
| 1986 | 复旦大学附属中山医院为全小肠切除术患者实施HPN，存活30年并在HPN 4年后正常怀孕生育 |
| 1990 | 中华医学会外科学分会营养治疗学组成立 |
| 2004 | 中华医学会肠外肠内营养学分会成立 |

## 适应证

凡需要营养治疗，但又不能通过[[肠内营养]]提供能量及营养素者或肠内营养无法满足能量与营养素目标需要量者：

1. **不能经口进食或经口进食不能满足需求**：口腔和头颈部疾病、脑卒中、颅脑损伤等致吞咽功能障碍；严重创伤或烧伤、难治性呕吐和腹泻、重症胰腺炎早期或严重腹腔感染、器官移植术后、恶性肿瘤、厌食等
2. **胃肠道消化与吸收功能障碍或衰竭**：肠缺血、炎性肠病、短肠综合征、高流量肠瘘、严重放射性肠炎、手术后消化道麻痹、完全性肠梗阻等
3. **EN不足时补充**：EN实施3-7天后，EN提供的能量及蛋白质仍达不到目标需要量60%的患者，应考虑SPN

## 禁忌证

1. 肠道功能正常，能获得足量营养者
2. 预计需要PN少于5天者
3. 心血管功能紊乱或严重代谢紊乱尚未控制或纠正者
4. 预计发生PN并发症的风险大于其可能带来的益处者
5. 临终者

## PN实施要点

- **循序渐进**：最初7-10天提供80%能量需求和100%蛋白质供应，之后增加至目标值
- **术前营养不良患者**：PN应从术前7-14天开始，术后继续维持
- **肠功能恢复后**：立即给予EN，随着EN耐受性提高逐渐减少PN量
- **停用标准**：EN可满足人体 > 60%能量及蛋白质目标需要量时，停用PN

## PN常用通路

### 通路选择原则
| 情况 | 推荐通路 |
|------|---------|
| 渗透压 >= 900mOsm/L 和/或 PN > 14天 | CVAD |
| 渗透压 < 900mOsm/L、短期PN | PVC（建议不超过14天） |

### PVC（外周静脉导管）
- **首选**：上肢静脉
- **限制**：渗透压 < 900mOsm/L、pH不能 < 5和 > 9、氨基酸浓度 < 5%、葡萄糖浓度 < 10%、滴速 < 50-60滴/min、连续输注 < 10-14天
- **优点**：快速建立静脉通道、操作简单、费用低、避免CVC相关并发症

### CVAD（中心静脉血管通路装置）

#### CVC（中心静脉导管）
- **首选**：锁骨下静脉
- **穿刺部位**：锁骨下静脉、锁骨上静脉、颈内静脉、颈外静脉、股静脉
- **留置时间**：2-4周
- **注意**：置管后常规行影像学检查确定导管尖端位置并排除气胸

#### PICC（外周中心静脉导管）
- **首选**：贵要静脉
- **留置时间**：最长可达12个月
- **优势**：并发症更少、成功率更高、导管留置时间更长

#### PORT（静脉输液港）
- **适用**：预计静脉输液时间超过6个月
- **常用**：胸壁港（首选）、手臂港
- **优点**：减少反复穿刺痛苦、增加日常自由度、隐蔽不影响形象
- **并发症**：气胸、血胸、空气栓塞、导管夹闭综合征、心律失常

### 通路比较

| 类别 | PVC | CVC | PICC | PORT |
|------|-----|-----|------|------|
| 首选途径 | 上肢静脉 | 锁骨下静脉 | 贵要静脉 | 胸壁港 |
| 输注液渗透压 | <900mOsm/L | 可>=900 | 可>=900 | 可>=900 |
| 留置时间 | 72-96h | 2-4周 | <1年 | >半年 |

## PN处方

### 全营养混合液（TNA/AIO）
将六大营养素按比例混合所得的液体：水、碳水化合物、氨基酸、脂肪、矿物质及维生素。

### 能量供给

#### 葡萄糖
- 成年人每日用量100-250g
- 供能4kcal/g
- 非蛋白能量占TNA供能的40%-60%
- 输注速度2-2.5mg/(kg*min)
- 葡萄糖和胰岛素10:1开始，血糖控制在 < 8mmol/L
- 配制后葡萄糖总浓度 <= 23%

#### 脂肪乳剂
- 供能9kcal/g
- 一般用量1.0-1.5g/(kg*d)
- 占非蛋白能量的30%-50%
- 危重症、呼吸功能不全、肿瘤：可占40%-60%
- **注意**：甘油三酯 > 3.0mmol/L（成人）或 > 1.7mmol/L（儿童）时应减慢输注或停用

#### 氨基酸
- 一般需要量1.0-1.5g/(kg*d)，热氮比150:1
- 严重分解代谢/重度营养不良：可达1.5-2.5g/(kg*d)，热氮比100:1
- TNA液中氨基酸浓度 >= 2.5%
- 肝功能不全：选用富含BCAA的专用制剂
- 慢性肾功能不全：选用以必需氨基酸为主的专用制剂

#### 电解质（每日给予量）
- Na+: 80-100mmol
- K+: 60-150mmol
- Ca2+: 2.5-5mmol
- Mg2+: 8-10mmol

#### 关键比例
- **热氮比**：150:1（一般）/ 100:1（高分解代谢）
- **糖脂比**：（1-2）:1
- **葡萄糖与胰岛素**：（4-10）g : 1IU

## PN处方配制原则

TNA混合液应在PIVAS（静脉用药配制中心）集中调配，配制顺序：
1. 将磷酸盐加入氨基酸或高浓度葡萄糖注射液中
2. 将其他电解质、微量元素加入葡萄糖或氨基酸注射液中（**注意**：不能与磷酸盐加入同一稀释液；钙离子和镁离子不能加入同一稀释液）
3. 用脂溶性维生素溶解水溶性维生素后加入脂肪乳剂
4. 加入一次性静脉营养输液袋顺序：氨基酸/含磷酸盐氨基酸注射液 → 其他液体 → 脂肪乳注射液

## 并发症总览

### 导管相关并发症
详见[[肠外营养导管并发症]]
- 置管穿刺处红肿外渗、周围静脉炎
- 导管堵塞、静脉血栓
- 导管相关感染（CRBSI）

### 代谢相关并发症
详见[[肠外营养代谢并发症]]和[[肠外营养相关性肝病]]
- 糖代谢紊乱（高血糖、低血糖）
- 氨基酸代谢紊乱（高氨血症、氮质血症）
- 脂肪超载综合征
- 微量营养素缺乏和过剩
- 再喂养综合征
- 肝功能损害（PNALD）
- 胆囊内胆泥和结石形成
- 肾结石、代谢性骨病
