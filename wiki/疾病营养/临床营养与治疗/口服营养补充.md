# 口服营养补充（ONS）

## 定义

口服营养补充（Oral Nutritional Supplements, ONS）是指除日常食物外，经口摄入的含有多种宏量营养素和微量营养素的特殊医学用途配方食品（FSMP），用于补充日常饮食中能量和营养素的不足。ONS是[[五阶梯营养治疗]]的**第3阶梯**，也是[[肠内营养]]的重要形式之一。

## ONS的特征

| 特征 | 说明 |
|------|------|
| 途径 | 经口摄入 |
| 形式 | 液体、粉剂、布丁、饼干等 |
| 成分 | 含蛋白质、碳水化合物、脂肪、维生素、矿物质等 |
| 能量密度 | 通常1.0-1.5 kcal/mL |
| 位置 | 在五阶梯治疗中处于第3阶梯 |

> [!important]
> ONS与普通食物的区别：ONS是经过科学配比的特殊医学用途配方食品，不属于普通食品，也不属于药品。

## 适应证

### 1. 营养风险筛查阳性患者
- [[NRS-2002|营养风险筛查#NRS-2002]] 评分 >= 3分的住院患者
- [[MNA]] 评分 < 24分的老年患者

### 2. 营养摄入不足
- 日常食物摄入量 < 目标需要量的60%
- 经口进食量持续减少
- 食欲下降导致摄入不足

### 3. 疾病相关营养不良或消耗
- 恶性肿瘤
- 慢性阻塞性肺疾病（COPD）
- 慢性肾功能不全
- 短肠综合征恢复期
- 术后恢复期

### 4. 高代谢状态
- 烧伤
- 严重创伤
- 重症感染

## ONS的使用方法

### 用量标准

| 情况 | 推荐用量 | 能量补充 |
|------|---------|---------|
| 标准补充 | 200-400 mL/次，2次/d | 400-600 kcal/d |
| 高需求 | 200-400 mL/次，3次/d | 600-900 kcal/d |
| 维持治疗 | 200 mL/次，1-2次/d | 200-400 kcal/d |

### 使用时机

- **餐间服用**：在两餐之间服用，不影响正常进食
- **分次小口服用**：避免一次性大量饮用
- **温度适中**：室温或微温，避免过冷

### 注意事项

1. ONS应**在两餐之间**饮用，不能替代正常饮食
2. 服用后应**保持口腔清洁**，避免残留
3. 应**循序渐进**，从少量开始逐渐增加
4. 长期使用需要定期监测营养指标

## ONS与管饲EN的区别

| 项目 | ONS | 管饲EN |
|------|-----|--------|
| 给予方式 | 经口自行饮用 | 经管道输注 |
| 胃肠道功能要求 | 基本正常 | 可有一定程度障碍 |
| 依从性 | 需要患者配合 | 依赖管路 |
| 适用阶梯 | 第3阶梯 | 第4阶梯 |
| 灵活性 | 患者可自主决定用量和时间 | 医护人员控制 |

## ONS的临床证据

### 循证医学证据

| 疾病/场景 | ONS效果 |
|----------|---------|
| 老年营养不良 | 降低并发症率、缩短住院时间 |
| 术后患者 | 促进伤口愈合、降低感染率 |
| 肿瘤患者 | 改善营养状况、提高治疗耐受性 |
| COPD | 增加体重、改善呼吸功能 |
| 骨质疏松 | 配合钙和维生素D，改善骨密度 |
| 骨骼肌减少症 | 增加肌肉量和肌力 |

## ONS的类型

| 类型 | 特点 | 适用人群 |
|------|------|---------|
| **标准型** | 均衡营养配方 | 一般营养不良患者 |
| **高蛋白型** | 蛋白质含量增加 | 肌肉减少症、术后恢复 |
| **高能量型** | 能量密度高（1.5 kcal/mL） | 需要限液的患者 |
| **纤维型** | 添加膳食纤维 | 便秘、肠道功能调节 |
| **疾病专用型** | 针对特定疾病调整配方 | 糖尿病、肾病、肺病等 |
| **免疫调节型** | 添加免疫营养素（如omega-3、核苷酸） | 围术期、危重症、肿瘤 |
