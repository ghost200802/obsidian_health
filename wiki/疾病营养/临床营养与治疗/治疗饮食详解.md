# 治疗饮食详解

## 页面定位 / 适用范围

本页是治疗饮食的细分页与方案库，展开低蛋白、低盐、低脂、糖尿病、低嘌呤等具体饮食方案的适应证、标准和食物选择。
- 若只需要治疗饮食的概念总览和分类入口，见[[治疗饮食]]
- 若需要回到医院基础膳食体系，见[[医院基本饮食]]
- 若需要查看临床营养主干治疗流程，见[[临床营养治疗]]

## 定义

治疗饮食（Therapeutic Diet）是在[[医院基本饮食]]（普食、软食、半流质、流质）基础上，根据疾病治疗需要，对能量、营养素种类和数量进行针对性调整的膳食方案。治疗饮食是[[临床营养治疗]]的重要手段，也是[[五阶梯营养治疗]]第1-2阶梯的具体实施形式。

> [!important]
> 治疗饮食的成分调整不应违反营养原则，须保证足量营养素的供给，避免因过度限制导致新的营养问题。

## 治疗饮食分类总览

| 分类方法 | 类别 | 具体饮食 |
|---------|------|---------|
| **按能量调整** | 高能量饮食、低能量饮食 | 增减总热能摄入 |
| **按蛋白质调整** | 高蛋白饮食、低蛋白饮食 | 调整蛋白质摄入量 |
| **按脂肪调整** | 低脂肪饮食、低胆固醇饮食 | 限制脂肪和胆固醇 |
| **按碳水化合物调整** | 糖尿病饮食、低碳水饮食 | 控制碳水摄入量和GI |
| **按矿物质调整** | 低盐饮食、限钾饮食、限磷饮食、低钙饮食 | 调整矿物质摄入 |
| **按特殊成分调整** | 低嘌呤饮食、低碘饮食、无麸质饮食 | 限制特殊成分 |
| **按食物质地调整** | 低渣饮食、无渣饮食、高纤维饮食、糊状饮食 | 调整食物质地 |

## 低蛋白饮食

### 适应证
- 急性/慢性肾功能不全（非透析）
- 肝功能衰竭/肝性脑病
- 尿毒症

### 营养标准

| 项目 | 标准 |
|------|------|
| 蛋白质摄入 | 0.5-0.8 g/(kg*d) |
| 优质蛋白比例 | >= 50%（动物蛋白和大豆蛋白） |
| 能量 | 30-35 kcal/(kg*d)，保证充足能量以减少蛋白质分解 |
| CKD 3-4期 | 0.6 g/(kg*d) + alpha-酮酸 |
| CKD 5期（非透析） | 0.5-0.6 g/(kg*d) + alpha-酮酸 |

### 食物选择

| 宜用 | 限制/禁用 |
|------|----------|
| 麦淀粉（代替部分主食） | 豆类（大量） |
| 薯类、藕粉 | 坚果 |
| 低蛋白米面 | 瘦肉（限量） |
| 蔬菜、水果 | 蛋类（限量） |
| 适量牛奶、鸡蛋（优质蛋白来源） | 大豆制品（限量） |

### 注意事项
- 必须保证充足能量摄入，否则蛋白质会被分解供能
- 配合alpha-酮酸类似物使用（CKD患者）
- 定期监测肾功能、血尿素氮、血清白蛋白
- 限制磷和钾的摄入

## 低盐饮食

### 适应证
- 高血压
- 心力衰竭
- 肾脏疾病伴水肿
- 肝硬化腹水
- 妊娠高血压综合征

### 营养标准

| 级别 | 每日钠摄入 | 每日食盐量 | 适用情况 |
|------|-----------|-----------|---------|
| 低盐 | < 2000 mg | < 5g | 一般高血压、轻度水肿 |
| 无盐 | < 500 mg | 烹调不加食盐 | 严重高血压、明显水肿 |
| 低钠 | < 500 mg | < 1g | 严重心力衰竭、急性肾炎 |

### 高钠食物（需避免）

| 类别 | 食物 |
|------|------|
| 调味品 | 酱油（每15ml含盐约3g）、盐渍/腌制食品 |
| 加工食品 | 火腿、腊肉、咸鱼、罐头、方便面 |
| 其他 | 味精（含钠）、小苏打（含钠） |

### 低盐饮食技巧
- 使用醋、柠檬汁、香料等替代食盐调味
- 避免加工食品和腌制食品
- 注意"隐形盐"（酱油、味精、加工食品中的钠）

## 低脂肪饮食

### 适应证
- 胆囊疾病（胆囊炎、胆石症）
- 胰腺疾病
- 高脂血症
- 脂肪泻/吸收不良综合征

### 营养标准

| 级别 | 脂肪摄入 | 适用情况 |
|------|---------|---------|
| 轻度限脂 | 脂肪 < 50g/d（占总能量<25%） | 高脂血症、肥胖 |
| 中度限脂 | 脂肪 < 40g/d（占总能量<20%） | 胆囊疾病恢复期 |
| 重度限脂 | 脂肪 < 20g/d（占总能量<10%） | 急性胰腺炎、脂肪泻 |

### 低脂食物选择

| 宜用 | 限制/禁用 |
|------|----------|
| 脱脂/低脂奶 | 全脂奶、奶油、黄油 |
| 瘦肉（鸡胸、鱼肉） | 肥肉、动物内脏 |
| 蒸、煮、炖烹调 | 油炸、油煎食品 |
| 豆腐、豆制品 | 油炸豆腐 |
| 新鲜蔬菜水果 | 坚果、种子类（大量） |

## 糖尿病饮食

### 适应证
- 1型糖尿病
- 2型糖尿病
- 妊娠糖尿病
- 糖耐量异常（IGT）

### 营养标准

| 项目 | 标准 |
|------|------|
| 总能量 | 25-30 kcal/(kg*d)（理想体重） |
| 碳水化合物 | 占总能量45%-60%（130-180g/d） |
| 蛋白质 | 占总能量15%-20%（1.0 g/(kg*d)） |
| 脂肪 | 占总能量20%-30% |
| 膳食纤维 | 25-30 g/d |
| 钠 | < 5g/d（< 2000 mg/d） |

### 碳水化合物管理

| 指标 | 说明 |
|------|------|
| GI值 | 选择低GI食物（GI < 55） |
| 碳水总量 | 每餐均匀分配，避免集中大量摄入 |
| 进餐顺序 | 蔬菜 → 蛋白质 → 碳水化合物（有利于血糖控制） |

### 食物交换份法

| 食物类别 | 每份能量 | 每份重量 | 主要营养素 |
|---------|---------|---------|-----------|
| 谷薯类 | 90 kcal | 25g（生重） | 碳水化合物20g，蛋白质2g |
| 蔬菜类 | 90 kcal | 500g | 碳水化合物17g，蛋白质5g |
| 肉蛋类 | 90 kcal | 50g | 蛋白质9g，脂肪6g |
| 乳类 | 90 kcal | 160g（牛奶） | 蛋白质5g，脂肪5g，碳水6g |
| 水果类 | 90 kcal | 200g | 碳水化合物21g |
| 油脂坚果类 | 90 kcal | 10g（油）/15g（坚果） | 脂肪10g |

## 低嘌呤饮食

### 适应证
- 痛风急性期
- 高尿酸血症
- 尿酸性肾结石

### 营养标准

| 项目 | 标准 |
|------|------|
| 嘌呤摄入 | < 150 mg/d（急性期）；< 300 mg/d（缓解期） |
| 总能量 | 维持理想体重 |
| 蛋白质 | 0.8-1.0 g/(kg*d) |
| 液体摄入 | >= 2000 mL/d（促进尿酸排泄） |
| 酒精 | 禁止（尤其啤酒和白酒） |

### 食物嘌呤含量分类

| 类别 | 嘌呤含量（mg/100g） | 食物举例 |
|------|-------------------|---------|
| **高嘌呤**（禁用） | > 150 | 动物内脏、浓肉汤、凤尾鱼、沙丁鱼、鱼籽、酵母 |
| **中嘌呤**（限制） | 50-150 | 猪牛羊肉、禽肉、豆类、菠菜、蘑菇 |
| **低嘌呤**（宜用） | < 50 | 精白米面、蔬菜（多数）、水果、牛奶、鸡蛋 |

## 低渣饮食（少渣饮食）

### 适应证
- 消化道手术前后
- 消化道出血
- 肠炎、腹泻
- 结肠镜检查前
- 炎症性肠病急性期

### 营养标准
- 膳食纤维 < 10-15 g/d
- 选择低纤维、易消化食物
- 烹调以煮、蒸、炖为主

### 食物选择

| 宜用 | 禁用 |
|------|------|
| 精白米面 | 全谷物、糙米、燕麦 |
| 去皮瓜类蔬菜 | 粗纤维蔬菜（芹菜、韭菜） |
| 熟透的水果（去皮去籽） | 坚果、种子 |
| 瘦肉（切碎煮烂） | 豆类（整粒） |
| 过滤果汁 | 纤维素高的水果（如带皮苹果） |

## 高纤维饮食

### 适应证
- 便秘
- 憩室病
- 糖尿病（辅助控制血糖）
- 高脂血症（辅助降低胆固醇）
- 肥胖（增加饱腹感）

### 营养标准
- 膳食纤维 25-35 g/d
- 逐渐增加纤维摄入，避免突然大量增加引起腹胀

### 高纤维食物

| 食物类别 | 高纤维来源 | 纤维含量（g/100g） |
|---------|-----------|-------------------|
| 谷类 | 燕麦、糙米、全麦面包 | 8-15 |
| 豆类 | 黄豆、绿豆、芸豆 | 10-25 |
| 蔬菜 | 芹菜、西兰花、菠菜 | 2-5 |
| 水果 | 梨、苹果（带皮）、香蕉 | 2-4 |
| 坚果 | 杏仁、核桃、花生 | 6-10 |
