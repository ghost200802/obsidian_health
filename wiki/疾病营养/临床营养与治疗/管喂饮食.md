# 管喂饮食

## 定义

管喂饮食（Tube Feeding Diet）是通过管道将配制好的营养液或食物匀浆输送到患者胃肠道内的营养方式，是[[肠内营养]]（EN）的重要组成部分，也是[[医院膳食]]中的一种特殊供给方式。当患者无法安全经口进食但胃肠道功能尚存时，管喂饮食是首选的营养支持方式。

## 适应证

| 适应证类别 | 具体情况 |
|-----------|---------|
| 神经系统疾病 | 脑卒中后吞咽障碍、昏迷、痴呆 |
| 口腔/咽喉疾病 | 口腔手术、颌面部创伤、咽喉肿瘤 |
| 消化系统疾病 | 食管狭窄/梗阻、胃瘫（需空肠管） |
| 术后/创伤 | 大手术后、严重创伤 |
| 精神心理疾病 | 神经性厌食（严重） |
| 高代谢状态 | 烧伤、重症感染 |

## 管道类型与选择

| 管道类型 | 放置位置 | 适用时间 | 特点 |
|---------|---------|---------|------|
| **鼻胃管（NG管）** | 胃 | < 4周 | 操作简便，最常见的短期管饲方式 |
| **鼻肠管（NJ管）** | 十二指肠/空肠 | < 4周 | 适用于胃排空障碍或高误吸风险 |
| **胃造瘘管（PEG）** | 胃 | > 4周 | 经皮内镜下胃造瘘，适合长期管饲 |
| **空肠造瘘管（JEJ）** | 空肠 | > 4周 | 适用于胃功能障碍，术后放置 |
| **咽造口管** | 咽部 | 长期 | 适用于经鼻置管困难者 |

## 管喂饮食的配方类型

### 1. 匀浆膳（自制）

| 项目 | 说明 |
|------|------|
| 配方 | 将日常食物（米饭、肉类、蔬菜、水果等）加水捣碎、过滤制成 |
| 优点 | 食材天然、价格低廉、符合饮食习惯 |
| 缺点 | 营养成分不稳定、颗粒不均匀、易堵塞管道、易污染 |
| 营养密度 | 约1 kcal/mL |

### 2. 要素膳（要素饮食）

| 项目 | 说明 |
|------|------|
| 配方 | 含游离氨基酸或短肽、葡萄糖/寡糖、中链脂肪酸、维生素矿物质 |
| 特点 | 无需消化或仅需 minimal 消化即可吸收 |
| 优点 | 营养成分精确、吸收好、残渣极少 |
| 缺点 | 口感差（仅供管饲）、价格较高 |
| 适用 | 消化功能严重受损者、短肠综合征 |

### 3. 整蛋白配方

| 项目 | 说明 |
|------|------|
| 配方 | 含完整蛋白质、碳水化合物、脂肪、维生素矿物质 |
| 特点 | 需要正常消化功能 |
| 优点 | 口感较好、营养均衡、价格适中 |
| 适用 | 消化功能基本正常者 |

## 管喂饮食的实施要点

### 输注方式

| 方式 | 特点 | 适用情况 |
|------|------|---------|
| **一次性输注** | 每次注入200-300mL，每天4-6次 | 胃肠功能好、鼻胃管 |
| **间歇性输注** | 缓慢滴注，每次2-4小时 | 胃肠功能中等 |
| **连续性输注** | 24小时持续匀速输注 | 胃肠功能差、空肠管、危重症 |

### 输注管理"五个度"

| 项目 | 要求 |
|------|------|
| **速度** | 从低速开始，逐渐增加。初始30-50 mL/h，目标80-100 mL/h |
| **温度** | 室温至微温（37-40°C），避免过冷刺激 |
| **浓度** | 从低浓度开始，逐渐增加至目标浓度 |
| **总量** | 从少量开始（500-700 mL/d），3-5天内达到目标量（1500-2000 mL/d） |
| **坡度** | 输注时抬高床头30-45度，预防误吸 |

### 喂养监测

| 监测项目 | 频率 | 正常范围/标准 |
|---------|------|-------------|
| 胃残留量 | 每4-6h（鼻胃管） | < 200-250 mL |
| 腹围 | 每日 | 无明显增加 |
| 腹胀/腹泻 | 每次输注时 | 无明显不适 |
| 体重 | 每周2-3次 | 稳定或上升 |
| 血糖 | 每日（特殊配方时） | 7.8-10.0 mmol/L |

## 管喂饮食并发症

### 胃肠道并发症（最常见，约30%-50%）

| 并发症 | 原因 | 处理 |
|--------|------|------|
| 腹泻 | 渗透压过高、输注过快、污染、乳糖不耐受 | 减慢速度、稀释配方、更换配方 |
| 便秘 | 缺乏膳食纤维、水分不足 | 补充水分、使用含纤维配方 |
| 恶心/呕吐 | 输注过快、量过多、胃排空延迟 | 减慢速度、减少单次量、抬高床头 |
| 腹胀 | 配方产气、输注过快 | 减慢速度、使用低产气配方 |

### 机械性并发症

| 并发症 | 预防 |
|--------|------|
| 管道堵塞 | 定期冲管（每4h用20-30mL温水冲洗） |
| 管道移位 | 妥善固定、定期检查位置 |
| 鼻咽部损伤 | 选择合适管径、定期护理 |

### 代谢性并发症

| 并发症 | 原因 | 处理 |
|--------|------|------|
| 水电解质紊乱 | 配方不均衡、腹泻/呕吐 | 监测电解质、调整配方 |
| 高血糖 | 应激反应、高糖配方 | 监测血糖、必要时加胰岛素 |
| 再喂养综合征 | 长期营养不良后突然大量营养 | 从低剂量开始，监测磷镁钾 |
