# 慢性心力衰竭营养

## 定义

慢性心力衰竭（CHF）是多种原因导致心脏结构和/或功能异常改变，使心室收缩和/或舒张功能障碍的临床综合征。主要表现为呼吸困难、疲乏和液体潴留。营养不良发病率16%-62%，重度和急性失代偿者高达75%-90%，心力衰竭恶液质约15%。

## 代谢特点

1. **能量代谢**：心肌偏向糖酵解，ATP合成减少（[[辅酶Q10]]缺乏可加重能量代谢障碍）；胃肠道血流障碍→营养摄入不足→心室射血分数下降（恶性循环）
2. **糖代谢**：早期葡萄糖摄取率和糖酵解率增加；晚期血氧严重缺乏→IR→CHF后糖尿病
3. **脂类代谢**：脂肪酸氧化率降低，堆积增加心肌毒性
4. **微量营养素**：维生素B2缺乏27%，维生素D仅9%达最佳水平，铁缺乏37%-61%（不良结局独立预测因子）

## 营养治疗

### 分期营养方案

| 阶段 | 能量 | 蛋白质 |
|------|------|--------|
| 急性发作期 | 约600 kcal/d | 25-30 g/d |
| 缓解期 | 1000-1500 kcal/d | 40-50 g/d |
| 恢复期 | 按理想体重 | 1.0 g/(kg·d) |

### 核心原则

1. **限钠限水**：
   - 急性发作伴容量过重：钠2g/d
   - 轻度/稳定期：不主张严格限钠
   - 严重CHF：液体1.5-2.0L/d

2. **膳食宜软烂、清淡、易消化**：富含膳食纤维的粗杂粮、杂豆、蔬菜水果

3. **少食多餐**：每日5-6次（3次主餐+2-3次辅餐）

4. **水电解质平衡**：注意低钾血症和镁缺乏

5. **补充B族维生素和矿物质**：
   - 铁缺乏需补铁（鸭血、鸡血、猪血等）
   - 维生素B1缺乏风险
   - 钙与心肌收缩性密切相关

6. **适当运动**：瑜伽和太极拳可能改善运动耐力和生活质量

### 恢复期膳食方案

| 食物类别 | 推荐量 |
|---------|--------|
| 谷薯类 | 250-400g/d（全谷物+杂豆50-150g，薯类50-100g） |
| 蔬菜 | 300-500g/d（深色蔬菜占1/2） |
| 水果 | 200-350g/d |
| 脂肪 | <25%，SFA<7% |
| 牛乳 | 300g/d（较重者脱脂或酸乳） |
| 食盐 | <3g/d |

### FSMP应用
- 经口摄食不足 → 口服FSMP
- FSMP仍不足 → 管饲EN → 必要时SPN或TPN
