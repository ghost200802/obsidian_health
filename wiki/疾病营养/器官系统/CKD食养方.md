# CKD食养方

> 针对[[慢性肾脏病]]（CKD）7种常见症状的传统食养方，来源于国家卫生健康委《成人慢性肾脏病食养指南（2024年版）》附录4。需根据CKD分期和[[CKD中医辨证分型]]合理选用。

> 主要来源：[[source-成人慢性肾脏病食养指南]]

## 使用原则

- **CKD 1-2期**：无特殊情况可对应证型适当食用
- **CKD 3期以上**：需根据血钾、血磷、尿酸等指标在医生指导下选用
- 食养方不能替代药物治疗，仅作为膳食调理辅助手段

## 食养方一览

### 1. 黄芪鲤鱼汤 — 气虚湿盛证

**适应症状**：水肿

**辨证**：气虚证 + 湿证

**材料**：
- 鲤鱼 500g（1条）
- 生黄芪 30g
- 赤小豆 30g
- 生姜、葱、盐少许

**做法**：
1. 鲤鱼去鳞、鳃、内脏，洗净
2. 黄芪、赤小豆洗净
3. 所有材料放入锅中，加水适量
4. 大火煮沸后转小火炖1-2小时
5. 加少量盐调味

**功效**：补气利水消肿

### 2. 薏米绿豆汤 — 湿热下注证

**适应症状**：尿频尿急

**辨证**：湿证 + 热证（湿热）

**材料**：
- 薏苡仁 30g
- 绿豆 20g

**做法**：
1. 薏苡仁、绿豆洗净，浸泡2小时
2. 加水适量，大火煮沸
3. 转小火煮至薏苡仁和绿豆软烂
4. 不加糖或少加糖

**功效**：清热利湿

### 3. 桂圆红枣粥 — 气血亏虚证

**适应症状**：贫血

**辨证**：气虚证 + 血虚证

**材料**：
- 大米 50g
- 龙眼肉 12g
- 大枣 12g
- 莲子肉 12g

**做法**：
1. 大米洗净，龙眼肉、大枣去核
2. 所有材料加水煮粥
3. 煮至软烂即可

**功效**：补气养血

### 4. 扁豆山药粥 — 脾虚湿盛证

**适应症状**：腹泻、纳差（食欲不振）

**辨证**：气虚证 + 湿证（脾虚）

**材料**：
- 鲜山药 30g
- 大米 30g
- 白扁豆 15g

**做法**：
1. 山药去皮切块，白扁豆提前浸泡
2. 大米洗净
3. 所有材料加水煮粥
4. 煮至软烂即可

**功效**：健脾化湿，和中止泻

### 5. 三仁润肠饮 — 阴虚便秘

**适应症状**：便秘（阴虚型）

**辨证**：阴虚证

**材料**：
- 甜杏仁 10g
- 郁李仁 10g
- 火麻仁 10g
- 蜂蜜 10g

**做法**：
1. 三仁打碎，加水煎煮30分钟
2. 去渣取汁
3. 加蜂蜜调匀，温服

**功效**：滋阴润肠通便

### 6. 槐花茶 — 实热便秘

**适应症状**：便秘（实热型）

**辨证**：热证（实热）

**材料**：
- 槐花 10g
- 蜂蜜 6g
- 绿茶 6g

**做法**：
1. 槐花、绿茶放入茶杯
2. 沸水冲泡，加盖焖5-10分钟
3. 加蜂蜜调味，代茶饮

**功效**：清热通便

### 7. 清热利咽茶 — 阴虚火旺证

**适应症状**：咽干咽痛

**辨证**：阴虚证 + 热证（虚火）

**材料**：
- 金银花 12g
- 桔梗 10g
- 青果 10g
- 薄荷 5g

**做法**：
1. 所有材料放入茶壶
2. 沸水冲泡，加盖焖10-15分钟
3. 代茶饮

**功效**：清热解毒，利咽润喉

### 8. 猪腰二仁汤 — 肾阳亏虚证

**适应症状**：腰膝酸软、夜尿多

**辨证**：阳虚证

**材料**：
- 猪腰 100g
- 核桃仁 20g
- 益智仁 10g

**做法**：
1. 猪腰剖开去筋膜，切片，飞水去腥
2. 核桃仁捣碎，益智仁洗净
3. 所有材料加水适量，小火炖1小时
4. 少量盐调味

**功效**：温补肾阳，固精缩尿
