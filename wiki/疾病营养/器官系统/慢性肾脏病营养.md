# 慢性肾脏病营养管理

## 页面定位 / 适用范围

本页是 CKD 的营养管理页，承接分期营养参数、食养原则和食物选择，不再重复疾病定义与诊断标准。
- 若需要查看 CKD 的定义、诊断与分期，见[[慢性肾脏病]]
- 若需要查看传统食养方案，见[[CKD食养方]]

> [[慢性肾脏病]]患者膳食管理的核心是：能量充足、蛋白质适量（分期管理）、[[限磷限钾]]、主食优化，以延缓疾病进展、改善生活质量。

> 主要来源：[[source-成人慢性肾脏病食养指南]]

## CKD分期营养管理核心参数

| 营养素 | CKD 1-2期 | CKD 3-5期（非透析） | CKD 5期（透析） |
|--------|----------|-------------------|----------------|
| 能量 kcal/(kg IBW·d) | 30-35 | 30-35 | 30-35 |
| 蛋白质 g/(kg IBW·d) | 0.8，避免>1.3 | 0.6，50%高生物价；糖尿病0.6-0.8 | 1.0-1.2，50%高生物价 |
| 碳水化合物占比 | 50%-65%，<10%精制糖 | 50%-65% | 50%-65% |
| 脂肪占比 | 30%-35%，SFA<10% | 30%-35%，增加MCT和ω-3 | 30%-35%，增加MCT和ω-3 |
| 膳食纤维 | 14g/1000kcal | 14g/1000kcal | 14g/1000kcal |
| 液体 | 1.5-1.7L | 1.5-1.7L | 血透：0.5L+前日尿量；腹透：0.5L+前日尿量+透析脱水量 |
| 钠 | ≤2300mg | ≤2300mg | ≤2300mg |
| 钾 | 通常不限制 | ≤2000-3000mg | ≤2000-3000mg（腹透个体化） |
| 钙 | 800mg/d，≤2000mg/d | 800-1000mg/d | ≤1500mg/d |
| 磷 | 个体化 | ≤800-1000mg/d | ≤800-1000mg/d |
| 膳食模式 | 植物性食物为主 | 植物性食物为主，需调整 | 植物性食物为主，需调整 |

> **理想体重(IBW)计算**：身高(cm) - 105

## 10条食养原则

### 1. 食物多样，分期选配

- 每日≥12种食物，每周≥25种
- 少盐、少调味品、限酒或不饮酒
- 限制/禁食浓肉汤或老火汤
- CKD 1-2期：[[植物性食物]]为主，全谷物、杂豆、薯类及水生蔬菜为主食
- CKD 3-5期：[[低蛋白饮食]]（0.6g/kg IBW/d）；选择高淀粉低蛋白食物替代部分谷类
- CKD 5期透析：适当调整动物性食物、豆类、蔬菜水果摄入量

### 2. 能量充足，体重合理，谷物适宜，主食优化

**能量**：30-35 kcal/(kg IBW·d)，超重/肥胖者可减少500-750 kcal

**BMI控制**：18.5-23.9 kg/m²（65岁以上老年人20.0-26.9 kg/m²）

**CKD 1-2期**：谷薯类为主，1/3为粗杂粮

**CKD 3-5期主食优化** — 核心策略：减少米面（蛋白质高且非优质），选择低蛋白高能量替代物：

| 替代类别 | 具体食物 |
|---------|---------|
| 低蛋白高能量根茎类 | 红薯、土豆、木薯、山药、芋头、沙葛 |
| 淀粉制品 | 小麦淀粉、红薯粉、土豆粉、木薯粉、西米、绿豆粉丝、豌豆粉丝 |
| 水生蔬菜 | 莲藕、马蹄 |
| 专用产品 | 低蛋白大米、肾病专用能量补充剂 |

### 3. 蛋白适量，合理摄入鱼禽豆蛋奶肉

- 优质蛋白占蛋白质总量50%以上
- 动物性食物优先白肉类（鱼禽），红肉每周1-2次，每次≤50g
- 蛋类每天≤1个，奶类≤300mL
- [[植物性食物]]有利于减轻肾脏负担：优先选择大豆及其制品
- **蛋白质来源优先级**：鱼禽类 → 大豆类 → 蛋、奶、畜肉

### 4. 蔬菜充足，水果适量

- 蔬菜300-500g/d，水果200-350g/d（[[糖尿病]]肾病患者100-200g/d）
- 水肿/高钾血症时需谨慎选择，蔬菜推荐清水浸泡并飞水弃汤
- 绿叶蔬菜约含蛋白质2-4g/100g，瓜菜约含0-1g/100g
- 植物化学物（类胡萝卜素、类黄酮、花青素、[[有机硫化物|有机硫化合物]]）通过抗氧化、抗炎及改善肠道菌群防治CKD

### 5. 少盐控油，[[限磷限钾]]

- **盐**：≤5g/d（水肿者≤3g/d，严重水肿无盐或低钠膳食）
- **烹调油**：25-40g/d，脂肪占总能量≤35%
- CKD 3-5期可增加富含MCT或ω-3脂肪酸的油脂（亚麻籽油、紫苏籽油）
- **磷**：≤800-1000mg/d，植物性食物磷吸收率低，推荐以植物性食物为主
- **钾**：个体化调整，高钾血症时≤2000-3000mg/d

### 6. 适量饮水，量出为入

- 无水肿且尿量正常：1500-1700mL/d
- 水肿/少尿者：根据尿液排出量和透析脱水量计划

### 7. 合理选择食药物质，调补有道

**因人制宜**：
| 证型 | 推荐食药物质 |
|------|------------|
| 气虚 | 山药、茯苓 |
| 血虚 | 阿胶 |
| 湿热 | 赤小豆、金银花 |
| 血瘀 | 桃仁、山楂 |

**因时制宜**：
| 季节 | 推荐食药物质 |
|------|------------|
| 春季 | 大枣、龙眼肉、核桃仁（温补） |
| 夏季 | 赤小豆、冬瓜、绿豆、薏苡仁（清热利湿） |
| 秋季 | 雪梨、山药、蜂蜜（养肺润燥） |
| 冬季 | 当归、阿胶（温补） |

**因地制宜**：
| 区域 | 推荐食药物质 |
|------|------------|
| 东南 | 茯苓、橘皮、薏苡仁（甘淡渗湿） |
| 西北 | 百合、大枣、龙眼肉（温热滋润） |

### 8. 合理选择营养健康食品

- CKD患者易出现微量营养素缺乏（B族维生素、维生素D、钙、铁、锌等）
- 低蛋白饮食不当→体重下降、消瘦、肌肉减少
- 必要时给予：膳食营养补充剂、肾病型能量补充剂、[[特殊医学用途配方食品]]

### 9. 规律进餐，限制饮酒，适度运动

- 一日三餐及加餐时间相对固定，减少外卖和聚餐
- CKD 3-5期不应饮酒
- 运动：每周3-5次，每次30-60分钟中等强度，抗阻运动每周3次

### 10. 定期监测，强化自我管理

| 监测频率 | 监测项目 |
|---------|---------|
| 每日 | 血压、体重、尿量 |
| 每周 | 饮食记录 |
| 每月 | 营养风险筛查 |
| 每季度/年 | 人体成分分析、握力、上臂围、小腿围、生化指标 |

## 问答补充（2024年版）

> 补充内容来自[[source-成人慢性肾脏病食养指南（2024年版）问答]]

- 问答稿再次强调 CKD 食养的关键是**分期管理**，不能把 1-2 期、3-5 期和透析期混成一套模板
- 对超重或肥胖患者，可在现有方案基础上适当减少 **500-750 kcal/d**
- 蔬菜一般推荐 **300-500 g/d**，但高钾或水肿场景要回到分期和化验结果来调整
- 适合用来向患者解释“为什么主食优化、蛋白控制和量出为入必须一起做”

## 食物选择指导

### 推荐食物
- 优质低蛋白：蛋清、牛奶（≤300mL）、瘦肉（限量）、鱼禽类
- 低蛋白高能量主食：藕粉、粉丝、凉粉、小麦淀粉制品、低蛋白大米
- 低钾蔬菜：白菜、冬瓜、黄瓜、丝瓜、西葫芦、葫芦
- 大豆制品：豆腐、腐竹

### 限制/避免食物
- 高钾：香蕉、橙子、哈密瓜、牛油果、土豆、蘑菇、菠菜、紫菜
- 高磷：动物内脏、坚果、加工食品（含磷添加剂）、奶酪
- 高钠：腌制食品、酱菜、加工肉制品、浓肉汤、老火汤

### 烹调技巧
- 方法：清蒸、炖、焖、煮
- 降磷钾：先飞水去磷钾，后烹调，最后下调味品
- 用电子秤称量食物（包括油盐调味品）
