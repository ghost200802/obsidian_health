# 消化系统疾病营养

## 定义

消化系统疾病营养（Nutrition in Gastrointestinal Diseases）是指针对消化系统各类疾病（肝脏疾病、炎症性肠病、短肠综合征、胰腺炎等）进行的针对性营养管理。消化系统是营养物质消化吸收的主要场所，其疾病直接影响患者的营养状态，而营养干预又是治疗的重要组成部分。

## 肝病营养

### 肝硬化

肝硬化患者营养不良发生率高达65%-90%，是影响预后的重要因素：

| 营养问题 | 营养干预 |
|---------|---------|
| 能量不足 | 能量供给25-35 kcal/(kg·d)，避免禁食 |
| 蛋白质-能量营养不良 | 蛋白质1.2-1.5 g/(kg·d)，肝性脑病急性期可暂降至0.5 g/(kg·d)，病情好转后逐步增加 |
| 糖代谢异常 | 夜间加餐（200kcal碳水化合物），减少糖异生 |
| 微量营养素缺乏 | 注意补充维生素A、D、E、K及锌、硒 |
| 腹水/水肿 | 限钠（<2g/d），限液 |

> [!important]
> 肝硬化患者应避免长时间禁食（>6小时），推荐少量多餐和夜间加餐，以防止糖异生和肌肉分解。

### 脂肪肝（NAFLD）

- **限制能量**：减重5%-10%可改善肝脂肪变
- **限制果糖**：减少含糖饮料和高果糖食物
- **增加膳食纤维**：全谷物、蔬果
- **适量蛋白质**：优质蛋白为主
- **规律运动**：每周150分钟中等强度有氧运动

## 炎症性肠病（IBD）营养

### Crohn病和溃疡性结肠炎

- **缓解期**：均衡饮食，避免个体不耐受食物，无需常规限制
- **活动期**：低渣饮食，减少肠道机械刺激
- **营养支持**：Crohn病常累及小肠，吸收障碍明显，需要[[肠内营养]]支持
- **肠内营养诱导缓解**：儿童Crohn病首选肠内营养诱导缓解
- **[[肠外营养]]**：严重短肠、高位瘘、完全性肠梗阻时使用

## 短肠综合征营养

### 定义

因手术切除大量小肠（残留小肠 < 100cm 或 < 1g/kg 体重）导致消化吸收功能严重障碍。

### 营养管理策略

| 阶段 | 策略 |
|------|------|
| 急性期 | [[肠外营养]]为主，维持水电解质平衡，减少腹泻 |
| 适应期 | 逐步引入[[肠内营养]]，促进肠道适应 |
| 恢复期 | 尽可能经口进食，根据残留肠段功能调整饮食 |
| 终末期 | 长期家庭肠外营养（HPN）或考虑小肠移植 |

### 饮食调整要点

- 高能量、高蛋白
- 低脂肪（中链甘油三酯MCT优于长链脂肪）
- 补充维生素（尤其是脂溶性维生素A、D、E、K）
- 补充矿物质（钙、镁、锌、铁）
- 口服补液盐维持水和电解质平衡
