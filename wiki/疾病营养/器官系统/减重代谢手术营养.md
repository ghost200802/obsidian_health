# 减重代谢手术营养

## 定义

减重代谢手术（Bariatric-Metabolic Surgery, BMS）是治疗肥胖及其相关代谢病的有效途径，由于消化道重建导致代谢重塑，围手术期营养管理是决定术后效果的重要因素。

## 常见术式

| 术式 | 缩写 | 原理 | 特点 |
|------|------|------|------|
| 腹腔镜胃袖状切除术 | LSG | 缩小胃容积 | 操作简单、占国内87.5%；术前合并GERD不建议选择；术后蛋白质和维生素缺乏 |
| 腹腔镜Roux-en-Y胃旁路术 | LRYGB | 缩小胃+改变食物通道 | "金标准"术式；T2DM肥胖者首选旷置大胃囊；术后铁缺乏常见 |
| 腹腔镜胆胰转流与十二指肠转位术 | LBPD-DS | 持久减重+高代谢病缓解率 | 手术难度大，营养并发症明显 |

## 术前营养

- 加强宣教，帮助选择术式，增加依从性
- 严重营养不良是术后营养并发症高危因素
- 完善检查，充分营养评估，制订营养计划

## 术后早期

- 术后1-5天酌量给予清流质 → 低糖低脂无咖啡因流质 → 半流质 → 软食 → 固体食物
- 避免倾倒综合征和脂肪泻（LRYGB）
- 首选ONS，无法3-7天内胃肠喂养满足需求 → SPN
- SPN：能量10-20 kcal/(kg·d)，蛋白质60-80g/d

## 术后长期膳食

### 膳食模式

- 地中海膳食、健康餐盘膳食、DASH膳食
- 精细并富含蛋白质的食物
- 增加水果、蔬菜、全谷物、ω-3脂肪酸
- 避免浓缩糖

### 饮食建议

1. 少食多餐，三餐间1-2次加餐
2. 减慢进食并仔细咀嚼
3. 每餐/加餐份量<250ml
4. 进固体和液体食物间隔30分钟
5. 避免过于黏稠、夹生或坚硬食物
6. 补充矿物质和维生素
7. 每天体力活动
8. 每天记录摄食情况
9. 每月至少记录1次体重（每周最多1次）

### 营养素补充

| 营养素 | 推荐量 |
|--------|--------|
| 水分 | >2000ml/d |
| 蛋白质 | 60-80g/d 或 1.2-1.5g/(kg·d)（HPD膳食法）；BPD-DS者增加30% |
| 碳水化合物 | 50-130g/d |

### 微量营养素补充（表18-8）

| 营养素 | 预防性补充 | 治疗性补充 |
|--------|-----------|-----------|
| 维生素B1 | 口服50mg/d或复合B | 严重：静脉500mg 3-5天后口服250mg至症状消失 |
| 维生素B12 | 口服350-1000μg/d；肌注1000μg/月 | 口服1000μg/d至正常后恢复推荐量 |
| 叶酸 | 口服400-800μg/d；育龄800-1000μg/d | 口服1000μg/d至正常 |
| 维生素D | D3 3000IU/d至25-OH-D>30ng/ml | D3 3000-6000IU/d |
| 维生素A | RYGB/SG: 5000-10000IU/d；DS: 10000IU/d | 角膜改变者：50000-100000IU/d |
| 钙 | BPD/DS: 1800-2400mg/d；RYGB/SG: 1200-1500mg/d | 同预防量 |
| 铁 | 男18mg/d；女45-60mg/d | 口服150-300mg/d+维生素C |
| 锌 | BPD/DS: 16-22mg/d；RYGB: 8-22mg/d；SG: 8-11mg/d | 每补8-15mg锌同步补1mg铜 |
| 铜 | BPD/DS或RYGB: 2mg/d；SG: 1mg/d | 轻中度：口服3-8mg/d；严重：静脉2-4mg/d 6天 |

### 随访

- 术后1月开始有规律运动：有氧150-300min/周 + 2-3次/周强化肌肉
- BMI>35者术后1月超声检查预防胆结石
- 育龄女性术后12-18个月适当避孕
- 随访1年后除骨密度外均每年检查1次；骨密度每2年1次
