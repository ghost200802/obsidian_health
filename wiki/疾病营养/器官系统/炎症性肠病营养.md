# 炎症性肠病营养

## 定义

炎症性肠病（Inflammatory Bowel Disease, IBD）包括溃疡性结肠炎（UC）和克罗恩病（CD），是主要累及胃肠道的非特异性、慢性、复发性、炎症性疾病。营养治疗有助于CD的诱导缓解和维持治疗。

## 代谢特点

- 能量需求与健康人群相似（高代谢但活动减少）
- 炎症活动和糖皮质激素应用 → 负氮平衡
- 微量营养素缺乏常见（活动期和缓解期均如此）
- CD累及回肠 → 维生素B12和脂溶性维生素缺乏
- 铁缺乏：肠道出血、十二指肠吸收障碍、炎症抑制促红细胞生成素

## 营养代谢紊乱特点

- UC和CD均常伴营养不良
- CD营养不良更常见且更难纠正（可累及胃肠道任何部位）
- UC主要局限于结肠，较少出现消化吸收障碍

## 营养治疗

### 一般膳食
- 多吃水果蔬菜，少摄入蔗糖
- 富含ω-3 PUFA、低ω-6 PUFA膳食可降低发病风险
- 母乳喂养>6个月降低IBD风险
- 减少深度加工食品（含羧甲基纤维素等）

### EN治疗（CD的核心治疗手段）

| 要点 | 内容 |
|------|------|
| UC | EN主要纠正营养不良和降低营养风险 |
| CD | 除营养作用外，更重要的是通过多种机制**抑制炎症，诱导和维持缓解** |
| 儿童青少年CD | EEN是诱导缓解**第一选择**，疗效与皮质激素相当但无不良反应 |
| 成年人CD | EEN缓解率高达80%，药物治疗无效者EN是有效替代 |
| EEN vs PEN | EEN抗炎效果优于PEN |
| 疗程 | 儿童/青少年6-12周；成年人4-6周 |

### EN制剂选择
- 肠道病变严重或有严重消化吸收不良 → 氨基酸型EN制剂
- 肠道病变不严重 → 整蛋白型或短肽型（性价比更高）

### EN通路选择
- 优先口服EN制剂（模拟管饲法口服：每3-5分钟口服30-50ml）
- 上消化道狭窄/穿透性病变/吞咽障碍 → 经鼻或经皮置管管饲

### 围手术期营养管理
- 50%-70%的CD患者需要外科手术
- 严重营养不良患者先接受强化营养治疗7-14天
- 经口/EN摄取不足>7天 → 联合EN和PN
- 末段回肠切除>20cm → 及时补充维生素B12

### 缓解期
- 避免食用可诱发病情活动的食物
- 柳氮磺吡啶和甲氨蝶呤可致叶酸缺乏 → 预防性补充
- 骨质疏松监测（血清25-羟维生素D）

### 肠道菌群调节
- 单用益生菌对CD缓解无效
- UC无法耐受5-氨基水杨酸时，可考虑益生菌替代
- 多重益生菌制剂可预防结肠贮袋炎
