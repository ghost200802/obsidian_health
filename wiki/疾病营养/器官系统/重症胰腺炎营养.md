# 重症胰腺炎营养

## 定义

重症急性胰腺炎（Severe Acute Pancreatitis, SAP）是急性胰腺炎的最严重类型，伴有持续性器官功能衰竭。营养治疗在SAP综合治疗中具有重要地位。

> SAP综合治疗"PANCREAS"：灌注(Perfusion)、止痛(Analgesia)、营养(Nutrition)、临床影像评价(Clinical and radiological assessment)、内镜(Endoscopy)、抗生素(Antibiotics)、手术(Surgery)

## 代谢特征

| 代谢类型 | 特点 |
|---------|------|
| 糖代谢 | 高血糖（β细胞损伤+胰岛素抵抗+糖异生增加）；外源性葡萄糖部分抵消蛋白质降解，但过量→高血糖+高碳酸血症 |
| 蛋白质代谢 | 骨骼肌明显消耗；尿尿素氮、肌酐排出增多；80%出现负氮平衡，氮损失20-40g/d；低蛋白血症+CRP升高 |
| 脂肪代谢 | 常见高脂血症；脂肪动员加速；部分脂肪分解/氧化障碍；约10%的SAP本身存在高脂血症（发病诱因） |
| 其他 | 40%-50%低钙血症；低锌、维生素B和叶酸缺乏 |

## 营养治疗

### 核心转变

- **"胰腺休息"和"肠道休息"理论已经过时**
- EN不仅是营养治疗方式，更是治疗SAP的重要措施
- EN作为首选途径

### EN实施时机

- 入院后24-48小时内开始
- 只要胃肠动力及吸收能力能耐受
- 多采用鼻胃管或鼻空肠管
- 应用输液泵控制连续滴注：初始500ml，25-50ml/h，3-4天后逐渐过渡到全量
- 鼻胃管 vs 鼻空肠管：耐受性、并发症和病死率无显著差异

### 营养素需要量

| 营养素 | 推荐量 |
|--------|--------|
| 能量 | 25-30 kcal/(kg·d)（无代谢车时） |
| 碳水化合物 | 3-6 g/(kg·d) |
| 蛋白质 | 1.2-1.5 g/(kg·d) |
| 脂肪乳剂 | 可至2g/(kg·d)，监测TG |
| 血糖控制 | >10mmol/L给予胰岛素 |
| TG控制 | 血浆TG<3-4mmol/L |

### 特殊情况

- **高甘油三酯血症相关AP**：考虑限制口服摄入，加速TG清除；更严重者静脉注射胰岛素+液体复苏或血浆置换；恢复EN时使用贝特类药物；PN尽量降低脂肪含量
