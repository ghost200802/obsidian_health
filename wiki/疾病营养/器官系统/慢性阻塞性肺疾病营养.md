# 慢性阻塞性肺疾病营养

## 定义

慢性阻塞性肺疾病（COPD）以持续存在的气流受限及相应的呼吸困难、咳嗽、咳痰等呼吸道症状为主要特征。30%-60%的COPD患者出现与疾病有关的营养不良。严重的COPD患者常常伴有蛋白质-能量营养不良，体重进行性下降，临床称为"肺恶液质综合征"。

## 代谢特点

1. **能量消耗增加**：每日用于呼吸的耗能约430-720kcal，较正常人高出10倍
2. **摄入不足**：胃肠道淤血、消化吸收障碍、长期平喘药刺激胃黏膜
3. **肌肉减少**：呼吸肌力量和质量下降 → 限制性通气不足 → 肺顺应性降低 → 加重呼吸困难（恶性循环）
4. **缺氧和高碳酸血症**：骨骼肌血供不足 → 乳酸酸中毒和CO2潴留

## 营养不良三大表现

- 恶液质（约5%发生率，随肺功能恶化增加）
- 肌肉减少（营养不良-肌肉减少症综合征对病死率影响更大）
- 骨质疏松

## 营养治疗

### 能量

公式：**BEE x C x 1.1 x 活动系数**

| 参数 | 数值 |
|------|------|
| C（校正系数） | 男1.16，女1.19 |
| 1.1 | 校正COPD患者体重减轻增加的10% BEE |
| 活动系数 | 卧床1.2，轻度1.3，中度1.5，剧烈1.75 |

- 急性应激期：20-30 kcal/(kg·d)
- 稳定期：30-35 kcal/(kg·d)
- 分多次供给，避免食欲下降和高能量负荷所致通气量增加

### 宏量营养素

| 营养素 | 稳定期 | 应激状态 |
|--------|--------|---------|
| 蛋白质 | 1.0-1.5 g/(kg·d)，15%-20% | — |
| 脂肪 | 20%-30% | 40%-45%（减少CO2生成） |
| 碳水化合物 | 50%-60% | <40%（避免CO2潴留） |

- 补充BCAA有助于改善呼吸肌收缩力
- 减少SFA，增加ω-3 PUFA有利于抗炎
- 适当添加支链脂肪酸可提高脂肪代谢率

### 维生素和矿物质

- 维生素C和E降低氧化应激水平
- 维生素D：抗炎、抗感染、抗肿瘤、改善神经肌肉功能
- 补充维生素C和D可改善FEV1、降低急性加重风险
- 长期使用类固醇者钙需求增加
- 适量补充硒（抗氧化功能）
- 限制钠盐

### 液体
- 适当补液保持黏液排出
- 急性期或合并肺心病/肺动脉高压/液体潴留者限制水摄入
