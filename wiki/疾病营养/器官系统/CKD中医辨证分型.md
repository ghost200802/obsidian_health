# CKD中医辨证分型

> 中医将[[慢性肾脏病]]归为"肾衰""虚劳"等范畴，病位涉及脾、肾、心、肺、肝、三焦，病因病机以**虚实夹杂、本虚标实**为主。食养需因人、因时、因地制宜。

> 主要来源：[[source-成人慢性肾脏病食养指南]]

## 辨证总纲

**本虚**（根本虚损）+ **标实**（病理产物/外邪）= 虚实夹杂

```
本虚证（四型）              标实证（三型）
┌──────────┐              ┌──────────┐
│  气虚证   │              │   湿证    │
│  血虚证   │  ←虚实夹杂→  │   热证    │
│  阴虚证   │              │   瘀证    │
│  阳虚证   │              │          │
└──────────┘              └──────────┘
```

## 本虚证四型

### 1. 气虚证

**临床表现**：
- 气短懒言
- 乏力
- 自汗
- 纳差（食欲不振）

**食养推荐** → [[CKD食养方]]
- 山药、茯苓
- 黄芪鲤鱼汤（气虚湿盛证时）

### 2. 血虚证

**临床表现**：
- 面色无华
- 唇甲色淡
- 贫血表现（头晕心悸等）

**食养推荐** → [[CKD食养方]]
- 阿胶
- 桂圆红枣粥（气血亏虚证时）：大米50g、龙眼肉12g、大枣12g、莲子肉12g

### 3. 阴虚证

**临床表现**：
- 五心烦热
- 潮热盗汗
- 口干咽燥
- 便秘

**食养推荐** → [[CKD食养方]]
- 三仁润肠饮（阴虚便秘）：甜杏仁10g、郁李仁10g、火麻仁10g、蜂蜜10g
- 清热利咽茶（阴虚火旺）：金银花12g、桔梗10g、青果10g、薄荷5g

### 4. 阳虚证

**临床表现**：
- 怕冷
- 手脚凉
- 夜尿多
- 腰膝酸软

**食养推荐** → [[CKD食养方]]
- 猪腰二仁汤（肾阳亏虚证）：猪腰100g、核桃仁20g、益智仁10g

## 标实证三型

### 1. 湿证

**临床表现**：
- 颜面/肢体浮肿
- 纳差
- 口中黏腻

**食养推荐**：
- 茯苓、薏苡仁
- 扁豆山药粥（脾虚湿盛证）：鲜山药30g、大米30g、白扁豆15g

### 2. 热证

**临床表现**：
- **实热**：口苦口黏、尿频尿急
- **虚火**：心烦不眠、潮热盗汗

**食养推荐**：
- 湿热→赤小豆、金银花
- 薏米绿豆汤（湿热下注证）：薏苡仁30g、绿豆20g
- 槐花茶（实热便秘）：槐花10g、蜂蜜6g、绿茶6g

### 3. 瘀证

**临床表现**：
- 面色黧黑
- 肢体刺痛
- 舌瘀斑

**食养推荐**：
- 桃仁、山楂

## 食药物质选择三原则

### 因人制宜

| 证型 | 推荐食药物质 |
|------|------------|
| 气虚 | 山药、茯苓 |
| 血虚 | 阿胶 |
| 湿热 | 赤小豆、金银花 |
| 血瘀 | 桃仁、山楂 |

### 因时制宜

| 季节 | 推荐食药物质 | 功效 |
|------|------------|------|
| 春季 | 大枣、龙眼肉、核桃仁 | 温补 |
| 夏季 | 赤小豆、冬瓜、绿豆、薏苡仁 | 清热利湿 |
| 秋季 | 雪梨、山药、蜂蜜 | 养肺润燥 |
| 冬季 | 当归、阿胶 | 温补 |

### 因地制宜

| 区域 | 推荐食药物质 | 功效 |
|------|------------|------|
| 东南 | 茯苓、橘皮、薏苡仁 | 甘淡渗湿 |
| 西北 | 百合、大枣、龙眼肉 | 温热滋润 |

## 使用注意

> **CKD 1-2期**患者无特殊情况可对应证型适当食用食养方。
> **CKD 3期以上**需根据血钾、血磷、尿酸等指标在医生指导下选用食养方和食药物质。
