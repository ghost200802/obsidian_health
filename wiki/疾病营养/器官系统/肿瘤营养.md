# 肿瘤营养

## 定义

肿瘤营养疗法（Cancer Nutrition Therapy, CNT）是通过计划、实施、评价营养治疗，治疗肿瘤及其并发症或身体状况，从而改善肿瘤患者预后的过程，包括营养诊断、营养治疗、疗效评价3个阶段。CNT是与手术、放疗、化疗等肿瘤基本治疗方法并重的独立治疗方法，应贯穿于肿瘤治疗的全过程。

> 营养治疗应该成为肿瘤综合治疗中的基础措施和核心内容，应该成为肿瘤患者的基础治疗、一线治疗。

## 代谢特点

### 肿瘤代谢重编程（Metabolic Reprogramming）

1. **能量代谢**：
   - **Warburg效应**：氧供充足条件下，肿瘤细胞多达50%以上ATP来自低产能效率的有氧糖酵解
   - 肿瘤宿主REE平均高于正常人约10%
   - 肿瘤细胞能源可塑性：葡萄糖利用困难时可利用乳酸、脂肪酸、氨基酸、酮体和乙酸

2. **糖代谢**：
   - 肿瘤细胞糖酵解能力可达正常细胞的20-30倍
   - 戊糖磷酸途径（PPP）明显增强
   - 肿瘤宿主存在胰岛素抵抗和葡萄糖利用障碍
   - 肿瘤与宿主肝脏之间的**乳酸-葡萄糖循环**（类Cori循环）：每1次循环净消耗4个ATP，每天额外消耗约300kcal

3. **蛋白质/氨基酸代谢**：
   - 谷氨酰胺消耗量最大，是其他氨基酸的10倍
   - 骨骼肌不断降解、瘦体质下降
   - 急性期蛋白合成增加，白蛋白合成减少

4. **脂类代谢**：
   - 肿瘤细胞脂肪酸从头合成增强
   - 脂肪组织分解动员增强
   - 腹膜后储存脂肪早期即严重下降

### 营养不良特征（7大特征）

1. 营养不良发病率更高（中国三甲医院80%，恶液质37%）
2. REE消耗升高（平均升高10%）
3. 持续生理心理应激
4. 慢性低度不可逆炎症（高炎症负荷死亡风险增加33.1%）
5. 消耗性代谢紊乱（发生率96%）
6. 显著肌肉减少（18%，女性>男性）
7. 治疗难度大

## 营养治疗

### 治疗时机
- 越早越好，诊断为肿瘤的同时进行营养诊断
- 医院-社区-家庭（H-C-H）延续、分级营养治疗

### 治疗目标
4个目标：抗消耗、抗炎症、抗肿瘤、增强免疫

### 能量与蛋白质需求

| 参数 | 推荐量 |
|------|--------|
| 能量（卧床） | 20-25 kcal/(kg·d) |
| 能量（活动） | 25-30 kcal/(kg·d) |
| 蛋白质（一般） | 1.2-1.5 g/(kg·d) |
| 蛋白质（恶液质） | 1.8-2.0 g/(kg·d)，BCAA>=0.6g/(kg·d)，EAA>=1.2g/(kg·d) |
| 蛋白质（严重营养不良冲击治疗） | 2.0-2.5 g/(kg·d) |
| 蛋白质（轻中度营养不良长期补充） | 1.5 g/(kg·d) |

### 分类治疗

| 营养状况 | 治疗策略 |
|---------|---------|
| 无营养不良 | 不需要营养治疗，直接抗肿瘤治疗 |
| 轻度营养不良 | 营养教育 + 抗肿瘤治疗 |
| 中度营养不良 | 医学营养（EN、PN）+ 抗肿瘤治疗 |
| 重度营养不良 | 先医学营养1-2周，然后继续医学营养 + 抗肿瘤治疗 |

### 肿瘤特异性营养配方

三大营养素供能比（表19-1）：

| 状态 | 途径 | C:F:P |
|------|------|-------|
| 无瘤患者 | EN | 45-55:25-30:15 |
| 荷瘤患者 | EN | 30-50:40-25:15-30 |
| 无瘤患者 | PN | C:F=70:30 |
| 荷瘤患者 | PN | C:F=40-60:60-40 |

> 荷瘤患者推荐高脂肪、低碳水化合物配方，C:F可达1:1

### 制剂选择要点
- 脂肪制剂：中/长链脂肪酸更适合（尤其肝功能障碍）；海洋来源ω-3 PUFA；橄榄油来源ω-9 MUFA
- 蛋白质制剂：整蛋白型适用于绝大多数；乳清蛋白更好改善营养状况；推荐富含BCAA的氨基酸制剂
- 药理营养：EAA/亮氨酸/HMB、精氨酸、ω-3 PUFA、核苷酸、谷氨酰胺、锌联合使用

### 五阶梯治疗原则
参见 [[临床营养治疗]] 页面

### 体力活动
- 推荐每周至少5次中高强度体力活动，每次30-60分钟
- 每天>60分钟高强度运动反增加疲劳
- 晚饭后快步走：快+长（>30分钟）
