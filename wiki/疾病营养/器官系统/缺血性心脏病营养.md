# 缺血性心脏病营养

## 定义

缺血性心脏病（IHD）是指冠状动脉发生粥样硬化引起管腔狭窄或闭塞，导致心肌缺血缺氧或坏死而引起的心脏病，也称冠状动脉粥样硬化性心脏病（CHD）。多发于40岁以上成年人，近年发病呈年轻化趋势。

## 代谢特点

- 正常心脏能量50%-75%来自脂肪酸，20%-50%来自葡萄糖
- 中度缺血时：糖酵解加速，脂肪酸氧化增强，葡萄糖氧化磷酸化受抑制
- 脂肪酸供能较葡萄糖耗氧多，使氧利用率下降
- 中国IHD患者约80%合并糖代谢异常
- HDL-C降低，LDL-C升高；氧化型LDL+富含TG脂蛋白→泡沫细胞→纤维帽→病变破裂→血栓→心肌梗死

## 营养紊乱特点

1. 糖代谢异常比例高（约80%）
2. 脂代谢异常（LDL-C增高，HDL-C降低，常伴腹型肥胖）
3. 维生素D缺乏促进冠脉钙化

## 营养治疗

### 总体原则

| 营养素 | 推荐量 |
|--------|--------|
| 能量 | 25-30 kcal/(kg·d) |
| 蛋白质 | 15%-20%，动物:植物=1:1 |
| 脂肪 | 20%-25%；植物:动物油脂>=2:1；胆固醇<300mg/d（高脂血症<200mg/d） |
| 碳水化合物 | 55%-65%，粗细搭配 |
| 膳食纤维 | 25-30g/d |
| 食盐 | <5g/d |

### 补充维生素
- 叶酸、维生素B6、B12降低同型半胱氨酸
- 膳食摄入较高叶酸可使IHD风险下降16%

### 急性冠状动脉综合征（ACS）

1. **急性期1-3天**：低脂肪流质，500-800 kcal/d，少食多餐5-6次/d
2. **病情好转**：低脂肪半流质，1000-1500 kcal/d，鱼类、鸡蛋清、蔬菜水果、面条米粉
3. **限制脂类**：脂肪<40g/d
4. **维持电解质**：镁300-450mg/d（对缺血心肌有保护作用）
5. 禁止肠胀气和刺激性食物，避免过冷过热食物
