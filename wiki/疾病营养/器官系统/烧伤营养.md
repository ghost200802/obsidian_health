# 烧伤营养

## 定义

烧伤泛指由热力、电流、化学物质、激光、放射线等造成的组织损害。烧伤后机体处于高度应激反应状态，代谢率显著升高，营养需求极大。

## 病理分期

1. **体液渗出期（休克期）**：体液大量渗出，基础代谢大幅下降
2. **急性感染期**：较长时间的代谢增加
3. **创面修复期**：代谢率逐渐趋于稳定
4. **康复期**：功能恢复

## 代谢特征

### 蛋白质代谢
- 儿茶酚胺、皮质醇、TNF-α、IL-1、IL-6分泌增加 → 蛋白质分解
- 早期高氨基酸血症，恢复期逐渐正常
- 尿素氮排出增多 → 负氮平衡

### 糖代谢
- 糖异生增强
- 胰岛素抵抗 → 组织摄取利用葡萄糖能力下降
- 易出现高糖血症

### 脂肪代谢
- 儿茶酚胺及胰高血糖素增高 → 脂肪分解
- 血浆FFA、甘油及TG增加
- 肝脏氧化脂肪酸效率降低
- 肉碱自创面大量丢失 → 酮体生成被抑制

## 营养治疗

### 能量评估

| 公式 | 表达 |
|------|------|
| HB公式 | 比实际需要量约低10% |
| Curreri公式 | 25×体重(kg) + 40×烧伤面积(%) kcal/d（估算偏高） |
| 第三军医大学公式 | 1000×体表面积(m²) + 25×烧伤面积(%) kcal/d |

> 体表面积(m²) = 0.0061×身高(cm) + 0.0128×体重(kg) - 0.1529

### 营养途径

- 烧伤面积<20%：经口摄入高能量高蛋白质食物
- 烧伤面积较大：尽早开展EN
- 不能耐受EN/存在禁忌且时间<14天：周围静脉PN
- 胃肠道和周围静脉均不能利用或时间>14天：中心静脉PN

### 宏量营养素

| 营养素 | 推荐量 |
|--------|--------|
| 蛋白质 | 占总能量15%-20%；严重烧伤约2g/(kg·d) |
| 脂肪 | 占总能量20%-30%；一般2g/kg，重度可增至3-4g/kg |
| 碳水化合物 | 占总能量50%-65%；每日400-600g，不超过5mg/(kg·min) |

### 微量营养素

- 电解质：每天常规足量补充
- 水溶性维生素：补充到正常需求量的2-5倍
- 脂溶性维生素：适当控制（可引起毒性反应）
- 微量元素：酌情补充锌、铜、铁等
