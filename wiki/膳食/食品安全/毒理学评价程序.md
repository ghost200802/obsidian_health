# 毒理学评价程序

> 食品安全性毒理学评价是按一定程序对食品中化学物质进行毒性测试和安全性评价的系统方法，依据GB 15193.1-2014标准执行。

## 核心要点

### 试验阶段（4阶段）

| 阶段 | 试验内容 | 目的 |
|------|---------|------|
| 1 | 急性经口毒性试验 | 测定LD50，进行毒性分级 |
| 2 | 遗传毒性试验（组合试验） | 筛查致突变性和潜在致癌性 |
| 3 | 28天经口毒性试验 | 初步评价，获得NOAEL |
| 4 | 90天经口毒性试验 | 确定NOAEL，为慢性试验剂量选择提供依据 |
| 补充 | 致畸试验、生殖毒性试验、毒物动力学试验 | 评估特殊毒性 |
| 最终 | 慢性毒性试验/致癌试验 | 最终安全性评价 |

### 结果判定原则
- LD50 < 人的推荐摄入量的100倍 → **放弃**
- NOAEL < 人的推荐摄入量的100倍 → **放弃**
- 100倍 < NOAEL < 300倍 → 需进行慢性毒性试验
- NOAEL ≥ 300倍 → 不必进行慢性毒性试验
- 遗传毒性组合中2项及以上阳性 → **放弃**
- 1项阳性 → 补做2项试验，再有阳性则放弃

### 安全系数（不确定系数）
- 通常100倍：种间差异10倍 x 种内差异10倍
- 可根据情况调整：特殊毒性、儿童接触、数据库不完整等可加大
