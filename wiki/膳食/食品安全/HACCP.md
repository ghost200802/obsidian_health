# HACCP

> HACCP（Hazard Analysis and Critical Control Points，危害分析与关键控制点）是一种预防性食品安全监控系统，通过对食品加工过程中危害因素的识别和控制来保障食品安全。

## 核心要点

### 7个原理
| 原理 | 内容 |
|------|------|
| 1. 危害分析 | 确定生物/化学/物理危害 |
| 2. 确定关键控制点（CCP） | 能预防、消除或降低危害的环节 |
| 3. 确定关键限值 | 区分可接受/不可接受的标准值 |
| 4. 建立监控措施 | 定时检测CCP是否受控 |
| 5. 建立纠偏措施 | 偏离时及时纠正 |
| 6. 建立验证措施 | 确认系统有效运行 |
| 7. 建立记录保存措施 | 完整记录存档 |

### 建立步骤（12步）
组建工作组→描述产品→确定用途→绘制流程图→现场确认→危害分析→确定CCP→确定关键限值→建立监控→建立纠偏→建立验证→建立记录

### 与GMP的关系
- 有效实施HACCP的**前提**是已建立完善的[[GMP]]体系
- GMP是基础卫生规范，HACCP是在GMP基础上的针对性危害控制
- 我国标准：GB/T 19538《HACCP体系及其应用指南》
