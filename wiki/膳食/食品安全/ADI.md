# ADI

> ADI（Acceptable Daily Intake，每日允许摄入量）是指人类终生每日摄入某种化学物质而不产生可检测到的健康危害的量，单位为 mg/(kg bw)。

## 核心要点

### 确定方法
- **公式**：ADI = NOAEL / 安全系数（通常100倍）
- **安全系数100倍** = 种间差异（10倍）x 种内差异（10倍）
- 可根据情况调整：特殊毒性、儿童接触、数据库不完整等可加大系数

### ADI分类
- **ADI值**：适用于单一物质，有具体数值上限
- **类别ADI（Group ADI）**：适用于一组结构类似的物质
- **无ADI规定**：不需限定摄入量的添加剂（如谷氨酸钠）

### 关键ADI值参考

| 添加剂 | ADI [mg/(kg bw)] |
|--------|-----------------|
| 苯甲酸及其钠盐 | 0-5 |
| 山梨酸及其钾盐 | 0-25 |
| BHA（丁基羟基茴香醚） | 0-0.5 |
| BHT（二丁基羟基甲苯） | 0-0.3 |
| PG（没食子酸丙酯） | 0-1.4 |
| 糖精钠 | 0-5 |
| 甜蜜素 | 0-11 |
| 阿斯巴甜 | 0-40（苯丙酮尿症患者禁用） |
