# 食品添加剂

## 定义

食品添加剂是指为改善食品品质和防腐需要，加入食品中的化学合成或天然物质。食品添加剂在规定范围内使用时是安全的，但滥用或超标使用可能危害健康。

## 食品添加剂分类

### 按功能分类
| 类别 | 功能举例 | 常见添加剂 |
|------|---------|-----------|
| 防腐剂 | 抑制微生物生长 | 苯甲酸、山梨酸 |
| 抗氧化剂 | 防止食品氧化变质 | BHA、BHT、维生素C |
| 着色剂 | 改善食品色泽 | 胭脂红、柠檬黄 |
| 甜味剂 | 提供甜味 | 蔗糖素、阿斯巴甜 |
| 增味剂 | 增强或改善风味 | 谷氨酸钠（味精） |
| 乳化剂 | 改善食品质地 | 卵磷脂、单甘酯 |
| 增稠剂 | 增加食品黏稠度 | 黄原胶、果胶 |
| 酸度调节剂 | 调节食品pH | 柠檬酸、乳酸 |

## 食品添加剂的管理

- 我国实行食品添加剂许可制度
- 使用范围和用量需符合GB 2760《食品安全国家标准 食品添加剂使用标准》
- 原则：技术上确有必要、安全可靠、不掩盖食品变质

## 食品添加剂与营养的关系

- 合理使用不影响食品营养价值
- 部分添加剂（如维生素C、铁强化剂）可增加营养价值
- 过度依赖添加剂的食品往往营养价值不高
- 消费者应关注食品标签，了解添加剂信息

## 营养与食品卫生学（第8版）补充

> 数据来源：《营养与食品卫生学》第8版（孙长颢主编），第10章。

### 分类（GB 2760-2014）
按功能分为23大类，包括：酸度调节剂、抗结剂、消泡剂、[[抗氧化剂]]、漂白剂、膨松剂、着色剂、护色剂、乳化剂、酶制剂、增味剂、面粉处理剂、被膜剂、水分保持剂、营养强化剂、[[防腐剂]]、稳定和凝固剂、[[甜味剂]]、增稠剂、食品用香料、食品工业用加工助剂、其他。

### 使用原则
- 必须经安全性毒理学评价，证明使用限量内长期食用安全无害
- 不影响食品感官性状和营养成分，不掩盖腐败变质
- 不降低营养价值，在达到目的前提下尽可能降低用量
- 严格遵守GB 2760规定的使用范围和限量

### 安全性评价——ADI
- **ADI（每日允许摄入量）**：人类终生每日摄入某化学物质而不产生可检测健康危害的量，单位 mg/(kg bw)
- ADI = NOAEL / 安全系数（通常100倍 = 种间差异10 x 种内差异10）
- 类别ADI：适用于一组结构类似的物质
