# 重金属污染

> 重金属污染指铅、镉、汞、砷等重金属通过工业排放、废水灌溉等途径进入食物链，在食品中蓄积并对人体健康造成危害。

## 核心要点

### 四大重金属危害与限量（GB 2762-2017）

| 重金属 | 主要来源 | 毒性特征 | 关键限量 |
|--------|---------|---------|---------|
| **铅（Pb）** | 工业污染、食品容器包装、汽车尾气 | 损害神经系统、造血系统、肾脏；影响儿童智力发育 | 谷物 ≤ 0.2 mg/kg，蔬菜 ≤ 0.1 mg/kg |
| **镉（Cd）** | 工业污染、含镉废水灌溉 | 肾脏毒性（肾小管损伤），骨质疏松（痛痛病），致癌 | 大米 ≤ 0.2 mg/kg，蔬菜 ≤ 0.05 mg/kg |
| **汞（Hg）** | 工业废水、含汞农药 | 甲基汞毒性最强，损害中枢神经系统（水俣病） | 水产品总汞 ≤ 0.5 mg/kg，甲基汞 ≤ 1.0 mg/kg |
| **砷（As）** | 工业废水、含砷农药 | As3+毒性 > As5+；原浆毒，与巯基结合，损害消化和神经系统 | 大米无机砷 ≤ 0.2 mg/kg，水产 ≤ 0.5 mg/kg |

### 预防措施
- 控制工业污染源头，禁止含重金属废水灌溉
- 加强食品中重金属限量监测
- 避免使用不合格的食品容器和包装材料
