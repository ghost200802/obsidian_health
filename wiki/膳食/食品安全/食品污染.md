# 食品污染

## 定义

食品污染是指食品在生产、加工、运输、储存过程中被有害物质污染的现象。食品污染可降低食品的营养价值和安全性，严重时可导致[[食源性疾病]]。

## 污染类型

### 1. 生物性污染

| 类型 | 常见污染物 | 危害 |
|------|-----------|------|
| 细菌 | 沙门氏菌、金黄色葡萄球菌、大肠杆菌 | 食物中毒、感染 |
| 病毒 | 诺如病毒、甲型肝炎病毒 | 急性胃肠炎、肝炎 |
| 真菌 | 黄曲霉、赭曲霉 | 霉菌毒素（如黄曲霉毒素） |
| 寄生虫 | 绦虫、旋毛虫、肝吸虫 | 寄生虫病 |

### 2. 化学性污染

| 类型 | 常见污染物 | 来源 |
|------|-----------|------|
| 农药残留 | 有机磷、有机氯等 | 农业生产中使用 |
| 重金属 | 铅、汞、镉、砷 | 工业废水、土壤污染 |
| 环境污染物 | 二噁英、多氯联苯 | 工业排放 |
| 兽药残留 | 抗生素、激素 | 畜牧养殖 |

### 3. 物理性污染

- 异物混入：玻璃碎片、金属碎片、石头、塑料
- 放射性污染：放射性核素污染

## 防控措施

- 源头控制：减少环境污染
- 加工规范：严格执行GMP和HACCP
- 储运管理：控制温度、湿度、储存时间
- 消费者注意：充分加热、生熟分开、注意保质期

## 营养与食品卫生学（第8版）补充

> 数据来源：《营养与食品卫生学》第8版（孙长颢主编），第9章。

### 微生物污染指标
- **菌落总数**：反映食品清洁状态，食品被污染程度的标志
- **大肠菌群**：反映食品受粪便污染程度的指标，肠道致病菌污染的间接指标
- **致病菌**：沙门氏菌、志贺氏菌、金黄色葡萄球菌等直接引起食源性疾病

### 农药残留重点
- 有机磷：最常见，抑制胆碱酯酶活性；需遵守安全间隔期
- 有机氯（DDT、六六六）：高蓄积性，脂溶性强，已被禁用/限用
- 预防：严格遵守安全间隔期，加强残留监测

### 重金属限量标准（GB 2762-2017）
- 铅：谷物 ≤ 0.2 mg/kg，蔬菜 ≤ 0.1 mg/kg，鱼类 ≤ 0.5 mg/kg
- 镉：大米 ≤ 0.2 mg/kg，新鲜蔬菜 ≤ 0.05 mg/kg（可致痛痛病）
- 汞：水产品总汞 ≤ 0.5 mg/kg，甲基汞 ≤ 1.0 mg/kg（水俣病）
- 砷：大米无机砷 ≤ 0.2 mg/kg，水产动物 ≤ 0.5 mg/kg

### 其他重要化学污染物
- [[杂环胺]]：蛋白质食物 >200°C 高温烹饪产生，致突变致癌
- 氯丙醇（3-MCPD）：酱油等酸水解植物蛋白中检出，限量 ≤ 1 mg/kg
- 食品接触材料：注意塑化剂迁移、金属溶出、荧光增白剂等
