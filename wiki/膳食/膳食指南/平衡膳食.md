# 平衡膳食

## 页面定位 / 适用范围

本页聚焦“平衡膳食”这一原则框架，强调营养充足、比例合理以及如何判断膳食是否平衡。
- 若需要查看官方指南文档总览，见[[膳食指南]]
- 若需要查看 8 条准则的逐条建议，见[[膳食准则]]
- 若需要查看群体层面的现实膳食模式，见[[膳食结构]]

## 定义

平衡膳食是指各类食物数量和比例合理、能够满足人体能量和营养素需要的膳食模式。平衡膳食是维持健康、预防疾病的基石。

## 八条膳食准则

《中国居民膳食指南（2022）》提出平衡膳食的八条准则：

1. **食物多样，合理搭配**：每天摄入12种以上食物，每周25种以上
2. **吃动平衡，健康体重**：各年龄段人群都应天天进行身体活动，保持健康体重
3. **多吃蔬果、奶类、全谷、大豆**：保证充足的[[膳食纤维]]、[[维生素]]和[[矿物质]]摄入
4. **适量吃鱼、禽、蛋、瘦肉**：优先选择鱼禽肉，鸡蛋不弃蛋黄
5. **少盐少油，控糖限酒**：培养清淡饮食习惯，少吃高盐和油炸食品
6. **规律进餐，足量饮水**：合理安排一日三餐，适量饮用白水或茶水
7. **会烹会选，会看标签**：学习烹饪技能，学会阅读食品标签
8. **公筷分餐，杜绝浪费**：选择新鲜卫生的食物，采用分餐制

## 平衡膳食的核心原则

- **食物多样性**：多种食物搭配，营养互补
- **适量性**：各营养素摄入量既不过少也不多余
- **均衡性**：三大宏量营养素（[[蛋白质]]、[[脂肪]]、[[碳水化合物]]）比例适当
- **规律性**：三餐定时定量

## 8条准则量化目标汇总

| 准则 | 核心量化指标 |
|------|------------|
| 食物多样合理搭配 | 每日>=12种食物，每周>=25种 |
| 吃动平衡健康体重 | BMI 18.5-23.9；每周>=150min中等强度运动 |
| 多吃蔬果奶类全谷大豆 | 蔬菜300-500g，水果200-350g，奶300-500g，大豆25g |
| 适量吃鱼禽蛋瘦肉 | 鱼虾40-75g/d，蛋40-50g/d，畜禽肉40-75g/d |
| 少盐少油控糖限酒 | 盐<5g，油25-30g，添加糖<25g，酒精<15g |
| 规律进餐足量饮水 | 三餐定时；饮水1500-1700ml |
| 会烹会选会看标签 | 优选蒸煮焯拌；会读[[食物营养标签]] |
| 公筷分餐杜绝浪费 | 使用公筷公勺；N-1点餐法 |

## 如何判断膳食是否平衡

- 每日食物种类是否达到12种以上
- 谷薯类是否以全谷物和杂豆占1/3-1/2
- 蔬菜是否>=300g，深色蔬菜是否占一半
- 奶类是否>=300ml
- 烹调油是否控制在30g以内
- 盐摄入是否控制在5g以内
- 饮水量是否达到1500-1700ml
- 是否坚持每周至少150分钟中等强度运动
