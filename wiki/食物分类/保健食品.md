# 保健食品

## 定义

保健食品（health foods）声称并具有特定保健功能或者以补充[[维生素]]、[[矿物质]]为目的的食品。适用于特定人群食用，具有调节机体功能，不以治疗疾病为目的，并且对人体不产生任何急性、亚急性或慢性危害。

## 我国保健食品法规演变

| 法规/标准 | 实施日期 | 定义要点 |
|-----------|---------|---------|
| 《食品卫生法》 | 1995.10 | 表明具有特定保健功能的食品 |
| 《保健食品管理办法》 | 1996.6 | 适宜特定人群，调节机体功能，不以治疗为目的 |
| GB 16740-1997 | 1997.5 | 具有一般食品共性，调节人体功能 |
| 《注册管理办法》 | 2005.7 | 增加"不产生任何危害"限定 |
| 《食品安全法》 | 2018.12 | 应当具有科学依据 |
| 《食品安全法》 | 2021.4 | 保健功能应有科学依据 |

## 历史沿革

- 古代食疗与现代植物性保健食品同根同源
- 孙思邈："夫为医者，当须先洞晓病源，知其所犯，以食治之；食疗不愈，然后命药"
- 1984年：中国保健食品协会成立
- 1995年：《食品卫生法》将保健食品作为特殊食品管理

## 分类

### 按功能分类
1. **营养素补充剂**：以补充维生素和矿物质为目的
2. **特定保健功能食品**：着眼于特殊消费群体，辅助调节功能

### 按加工分类
- 天然食品
- 添加某种成分的食品
- 剔除某种成分的食品
- 改变一种或多种成分的食品
- 改变生物利用度的食品

### 按剂型分类
- 一般食品形式：面食类、乳品类、饮料类、酒类
- 制剂形式：片剂、粉剂、胶囊、胶丸、口服液、茶等

## 功效成分

| 类别 | 举例 | 作用 |
|------|------|------|
| 多糖类 | 膳食纤维、香菇多糖 | 调节免疫、抑制肿瘤、降血糖降血脂 |
| 功能性甜味料 | 低聚糖、多元糖醇 | 预防龋齿、促进双歧杆菌增殖 |
| 功能性油脂 | EPA、DHA、磷脂 | 降血脂、抗凝血、改善大脑功能 |
| 自由基清除剂 | SOD、谷胱甘肽过氧化酶 | 抗氧化、清除氧自由基 |
| 维生素类 | beta-胡萝卜素、维生素E | 提高抗氧化能力、延缓衰老 |
| 肽与蛋白质 | 谷胱甘肽、免疫球蛋白、大豆多肽 | 清除自由基、增强肌肉、降胆固醇 |
| 活性菌类 | 乳酸菌、双歧杆菌 | 调节肠道菌群、调节免疫 |
| 微量元素类 | 硒、锌 | 维持正常生理功能 |
| 其他 | 二十八烷醇、植物甾醇、皂苷 | 多种生理活性 |

## 应用注意

- 不能替代常规膳食
- 摄入过量可能导致营养素过量，干扰其他营养素吸收
- 妊娠期、哺乳期、婴幼儿食用应咨询医师
- 注意使用目的是调节机体功能，不是治疗疾病
