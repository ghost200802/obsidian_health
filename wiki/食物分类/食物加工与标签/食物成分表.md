# 食物成分表

## 页面定位 / 适用范围

本页是“食物成分表”这一主入口页，聚焦《中国食物成分表》本身的版本、结构、使用方法和适用边界。
- 若需要理解数据库形态、数据采集与更新机制，见[[食物成分数据库]]
- 若需要理解营养成分数据如何被查询、计算和使用，见[[营养成分数据]]
- 若需要理解底层字段与样品 schema，见[[食物编码]]、[[食物样品描述]]、[[可食部分]]

## 定义

食物成分表是记录各类食物营养成分的标准参考数据集，是营养学研究和实践的基础工具。它系统收录了各类食物的能量及营养素含量，为膳食调查、食谱编制、营养评价和科学研究提供权威数据支持。

### 每种食物收录的营养素

每种食物条目包含以下营养成分数据：

**宏量营养素：**
- 能量（kcal、kJ）
- 水分（g）
- 蛋白质（g）
- 脂肪（g）
- 碳水化合物（g）
- 膳食纤维（g，含总膳食纤维、可溶性/不可溶性）

**微量营养素 - 维生素：**
- 维生素A（μg RAE）
- 维生素B1/硫胺素（mg）
- 维生素B2/核黄素（mg）
- 烟酸/维生素PP（mg）
- 维生素C/抗坏血酸（mg）
- 维生素E/生育酚（mg）
- 其他B族维生素

**微量营养素 - 矿物质：**
- 钙（mg）
- 磷（mg）
- 钾（mg）
- 钠（mg）
- 镁（mg）
- 铁（mg）
- 锌（mg）
- 硒（μg）
- 其他矿物质

**其他成分：**
- 胆固醇（mg）
- 氨基酸组成（部分食物）
- 脂肪酸组成（部分食物）

## 第一册与第二册互补

| 项目 | 第一册 | 第二册 |
|------|--------|--------|
| 内容侧重 | 常见食物、基础原料 | 加工食品、地区特色食物 |
| 食物数量 | 较多基础食物 | 较多加工和特色食物 |
| 数据详略 | 基础营养素全面 | 部分食物增加氨基酸/脂肪酸数据 |
| 使用场景 | 日常膳食调查和食谱编制 | 特殊食物查询、深度营养分析 |

> [!tip]
> 实际工作中两册应配合使用，第一册覆盖日常常见食物，第二册补充加工食品和特色食物。

## 使用方法

### 基本查询流程

```
确定目标食物 → 查找食物条目 → 获取营养素含量 → 计算膳食摄入量
```

### 典型应用场景

1. **膳食调查分析**
   - 记录患者24小时膳食
   - 查询每种食物的营养素含量
   - 计算总能量和各营养素摄入量
   - 与[[膳食营养素参考摄入量]]对比评价

2. **食谱编制**
   - 根据营养目标确定各营养素需要量
   - 选择适当食物组合
   - 查询食物成分计算配比
   - 调整至满足营养需求

3. **营养咨询**
   - 患者描述日常饮食
   - 通过食物成分表分析营养摄入
   - 找出不足或过量的营养素
   - 给出针对性饮食建议

4. **科学研究**
   - 营养流行病学研究的食物数据来源
   - 膳食暴露评估
   - 营养干预实验的基线评估

## 使用注意事项

- **食物可食部**：数据均为可食部分含量，需注意不可食部分（如骨头、皮壳）
- **烹饪加工影响**：数据多为生食状态，烹饪后营养素含量可能变化
- **地区差异**：同一食物不同产地营养素含量可能有差异
- **季节差异**：部分食物（如蔬菜）因季节不同营养素含量有变化
- **版本更新**：建议使用最新版本，旧版数据可能有修正

## 第6版新增内容

相比前几版，第6版有以下重要更新：

- **新增植物化学物数据**：包括[[大豆异黄酮]]、[[植物甾醇]]、[[叶黄素]]等常见植物化学物的含量数据，详见[[植物化学物]]
- **新增碘含量数据**：系统收录了各类食物的碘含量，补充了[[碘]]等重要微量元素数据
- **新增胆碱数据**：首次将[[胆碱]]纳入常规营养指标，收录了常见食物的胆碱含量
- **数据量扩充**：食物条目数量和营养素指标项目均有所增加
- **数据来源说明**：数据来源于全国代表性食物样品的直接化学分析，辅以部分文献数据和计算数据，分析方法遵循国家标准和国际通行方法（AOAC）

### 第二册的定位和价值

第二册的价值在于填补了第一册在加工食品和地方特色食物方面的数据空白。第一册以基础食物原料为主，涵盖20余大类的常见天然食物；第二册则系统收录了地方特色食物、市售包装食品、加工食品和新型食品的营养数据。许多在现代膳食中占据重要地位的市售食品（如袋装零食、速冻食品、即食食品等），其营养成分数据仅见于第二册。两册配合使用，才能完整覆盖中国居民日常膳食中各类食物的营养数据需求。

### 两册配合使用的方法

在实际工作中，建议按以下策略使用两册数据：

1. **日常基础食物**查第一册：如谷类、蔬菜、水果、畜肉、禽肉、水产、蛋类、乳类等天然食物原料
2. **加工和市售食品**查第二册：如袋装零食、速冻食品、即食食品、调味品、地方小吃等
3. **同名食物比对**：若两册中均有收录，以具体条目的食物描述（产地、加工方式、品牌等）为准选择最匹配的条目
4. **数据缺失处理**：若某食物在两册中均未收录，可选用成分相近的食物数据作为替代估算
