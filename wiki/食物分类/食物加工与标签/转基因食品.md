# 转基因食品

转基因食品（genetically modified food, GM food）是指利用基因工程技术改变生物体遗传物质所生产的食品或食品原料。转基因食品的安全性评价和法规管理是全球关注的重要议题。

## 定义

转基因技术通过将外源基因导入受体生物，使其获得新的性状：

- **输入性状**：抗虫、抗病、抗除草剂等
- **输出性状**：改善营养价值、改变脂肪酸组成、延长保鲜期等

## 安全性评价

### 评价原则

- **实质等同性原则**：比较转基因食品与传统对应食品在营养成分、有毒有害物质等方面的差异
- **个案分析原则**：每种转基因食品需单独评价，不能一概而论
- **逐步评估原则**：从分子特征到营养成分、毒理学、致敏性逐步评估

### 评价内容

| 评价项目 | 内容 |
|---------|------|
| 分子特征 | 插入基因的稳定性、表达产物的鉴定 |
| 营养成分 | 宏量和微量营养素含量比较 |
| 毒理学 | 急性、亚慢性毒性试验 |
| 致敏性 | 评价转入蛋白是否引起过敏 |
| 非预期效应 | 基因插入的非预期代谢变化 |
| 环境安全 | 对生态系统的潜在影响 |

### 国际共识

- 世界卫生组织（WHO）、联合国粮农组织（FAO）认为经过严格评价的转基因食品是安全的
- 目前上市转基因食品未见对人体健康有不良影响的证据
- 长期健康效应仍在持续监测中

## 法规管理

### 中国管理框架

- **审批制度**：转基因食品需经农业农村部安全评价审批
- **标识制度**：含有转基因成分的食品需在标签上明确标识
- **分级管理**：根据安全等级实施不同管理措施
- 参见 [[食品添加剂]] 和 [[食品污染]] 的相关法规体系

### 国际比较

| 地区 | 管理方式 |
|------|---------|
| 中国 | 审批+强制标识 |
| 欧盟 | 严格审批+严格标识 |
| 美国 | 自愿标识（近年法规有所调整） |
| 日本 | 强制标识（指定品种） |

## 常见转基因作物

- 大豆（抗除草剂）、玉米（抗虫/抗除草剂）、棉花（抗虫）、油菜（抗除草剂）、木瓜（抗病毒）
