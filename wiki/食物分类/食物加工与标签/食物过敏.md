# 食物过敏

## 定义

食物过敏（Food Allergy）是指机体免疫系统对某种食物蛋白产生的异常免疫反应。食物过敏与食物不耐受不同，前者涉及免疫机制，后者通常为非免疫性的代谢反应。食物过敏是[[临床营养]]中重要的疾病管理领域。

## 分类

### IgE介导的食物过敏

- 发病机制：食物蛋白刺激机体产生特异性IgE抗体，与肥大细胞、嗜碱性粒细胞表面的IgE受体结合，释放组胺等炎症介质
- **发病速度**：摄入致敏食物后数分钟至2小时内发作
- **典型症状**：
  - 皮肤：荨麻疹、血管性水肿、瘙痒
  - 消化道：恶心、呕吐、腹痛、腹泻
  - 呼吸道：喉头水肿、支气管痉挛、哮喘
  - 全身：过敏性休克（最严重，可危及生命）

### 非IgE介导的食物过敏

- **发病机制**：T细胞介导的迟发型免疫反应
- **发病速度**：摄入后数小时至数天
- **常见类型**：
  - 食物蛋白诱导的小肠结肠炎综合征（FPIES）
  - 过敏性嗜酸性食管炎（EoE）
  - 乳糜泻（麸质过敏）
  - Heiner综合征（牛奶诱发性肺含铁血黄素沉着症）

### IgE和非IgE混合型

- 部分疾病同时涉及IgE和非IgE机制
- 如特应性皮炎（部分由食物过敏诱发）

## 常见致敏食物

国际公认的"八大"致敏食物：

| 致敏食物 | 常见过敏表现 | 备注 |
|---------|------------|------|
| 牛奶 | 皮疹、胃肠道反应 | 儿童最常见 |
| 鸡蛋 | 皮疹、胃肠道反应 | 儿童最常见 |
| 花生 | 呼吸道、皮肤反应 | 可致严重过敏反应 |
| 坚果（核桃、杏仁等） | 呼吸道、皮肤反应 | 终身过敏倾向 |
| 大豆 | 胃肠道反应 | 多见于儿童 |
| 小麦 | 乳糜泻、皮疹 | 与乳糜泻相关 |
| 鱼类 | 皮肤、呼吸道反应 | 成人常见 |
| 甲壳类（虾蟹等） | 皮肤、呼吸道反应 | 成人常见 |

## 诊断方法

- **详细病史和膳食日记**：记录食物摄入与症状的关联
- **皮肤点刺试验（SPT）**：快速筛查IgE介导的过敏
- **血清特异性IgE检测**：定量检测过敏原特异性抗体
- **口服食物激发试验（OFC）**：食物过敏诊断的金标准
- **排除饮食试验**：排除可疑食物后观察症状是否缓解

## 管理原则

1. **严格回避致敏食物**：最根本的管理措施
2. **制定应急计划**：备好肾上腺素自动注射器（适用于严重过敏者）
3. **替代营养方案**：避免致敏食物的同时保证营养均衡
4. **定期随访评估**：部分儿童过敏可随年龄增长自然缓解
5. **营养监测**：长期回避多种食物可能导致[[营养不良]]，需定期评估

## 临床营养学补充（第十六章第九节）

> 来源：[[source-临床营养学]] 第十六章第九节

### IgE介导过敏的发病机制（SYK级联反应）

1. 致敏阶段：食物蛋白→抗原呈递细胞→Th2细胞→B细胞→产生特异性IgE
2. IgE与肥大细胞/嗜碱性粒细胞表面的高亲和力受体（FcεR1）结合
3. 再次接触→变应原与IgE交联→FcεR1聚集→激活酪氨酸激酶SYK
4. SYK激活下游信号→钙离子内流→脱颗粒
5. 释放组胺、白三烯、PAF等炎症介质→过敏症状

### IgE介导过敏亚型

| 亚型 | 特点 |
|------|------|
| FDEIA（食物依赖运动诱发性过敏） | 运动+特定食物共同诱发 |
| CMPA（牛乳蛋白过敏） | 婴儿最常见，多数3岁内缓解 |
| PFAS（花粉食物过敏综合征） | 口腔过敏综合征，花粉交叉反应 |

### 非IgE介导过敏类型详述

| 类型 | 发病年龄 | 潜伏期 | 主要表现 |
|------|----------|--------|----------|
| FPIES（食物蛋白诱导的小肠结肠炎综合征） | 婴儿为主 | 2-4小时 | 呕吐、腹泻、脱水、嗜睡 |
| FPIP（食物蛋白诱导的直肠结肠炎） | 婴儿为主 | 数小时至数天 | 血便 |
| SNAS（食物蛋白诱导的小肠吸收不良综合征） | 婴儿 | 数周至数月 | 腹泻、脂肪泻、生长发育迟缓 |

### 实验室检测方法

| 方法 | 适用类型 | 特点 |
|------|----------|------|
| 血清sIgE检测 | IgE介导 | 定量，可监测变化 |
| 皮肤点刺试验（SPT） | IgE介导 | 快速，但受抗组胺药影响 |
| 斑贴试验 | 非IgE介导（迟发型） | 48-72小时判读 |
| 食物特异性IgG检测 | 不推荐常规使用 | 临床意义不明确 |

### 口服食物激发试验（OFC）类型

| 类型 | 盲法 | 临床用途 |
|------|------|----------|
| DBPCFC（双盲安慰剂对照） | 双盲 | 诊断金标准 |
| 单盲安慰剂对照 | 单盲 | 临床研究 |
| 开放式激发 | 无盲法 | 临床筛查 |

### 营养管理方案

**低敏配方选择**：
- 深度水解配方（eHF）：适用于大多数CMPA
- 氨基酸配方（AF）：适用于重度过敏或eHF不耐受者
- 大豆配方：适用于非IgE介导的CMPA（IgE型可能有交叉过敏）

**新兴治疗方法**：
- 单克隆抗体（奥马珠单抗）：抗IgE治疗
- 益生菌：调节肠道免疫耐受
- 食品加工技术：改变蛋白质抗原性（加热、酶解、发酵）
