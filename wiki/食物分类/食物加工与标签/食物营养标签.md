# 食物营养标签

## 定义

食物营养标签是指预包装食品上向消费者提供食品营养信息和特性的说明，包括营养成分表、营养声称和营养成分功能声称。学会阅读食物营养标签是《中国居民膳食指南（2022）》[[会烹会选会看标签|准则七]]的重要技能。

## 营养成分表

### 常见标签口径与新标准过渡

截至**2026年6月5日**，市面上多数预包装食品仍主要按旧版 `GB 28050-2011` 的“1+4”思路标示；但新版[[source-GB 28050-2025 食品安全国家标准 预包装食品营养标签通则]]已于**2025年3月16日**发布，并将于**2027年3月16日**实施。阅读标签时要注意自己看到的是旧版还是新版格式。

#### 旧版常见“1+4”

| 项目 | 说明 |
|--------|------|
| **能量** | 以千焦（kJ）和千卡（kcal）表示 |
| **蛋白质** | 以克（g）表示 |
| **脂肪** | 以克（g）表示 |
| **碳水化合物** | 以克（g）表示 |
| **钠** | 以毫克（mg）表示 |

#### 新版 GB 28050-2025 强制标示内容

| 营养素 | 说明 |
|--------|------|
| **能量** | 以千焦（kJ）和千卡（kcal）表示 |
| **蛋白质** | 以克（g）表示 |
| **脂肪** | 以克（g）表示 |
| **饱和脂肪（酸）** | 需强制标示 |
| **碳水化合物** | 以克（g）表示 |
| **糖** | 需强制标示 |
| **钠** | 以毫克（mg）表示 |

此外，当食品或配料使用了氢化或部分氢化油脂时，需标示**反式脂肪酸**；营养成分表下方还应增加提示语：**“儿童青少年应避免过量摄入盐油糖”**。

### NRV% 含义

- **NRV**（Nutrient Reference Values）= 营养素参考值
- **NRV%** 表示100g（或100ml）该食品中某营养素含量占成人每日需要量的百分比
- 例如：某食品钠NRV%=30%，表示吃100g该食品就摄入了全天钠需要量的30%
- 快速判断技巧：NRV%越高的营养素，该食品含量越丰富

## 配料表排序规则

- 配料表按加入量**从多到少**递减排列
- 加入量不超过2%的配料可以不按递减顺序排列
- 实操要点：
  - 配料表前几位是产品的主要成分
  - 如果白砂糖、油脂、[[钠]]含量高的调味料排在前三位，说明产品高糖/高油/高盐
  - 配料表越短、成分越简单，通常越健康

## 营养声称标准

营养声称是对食品营养特性的描述声明，分为含量声称和比较声称：

| 声称类型 | 标准定义 |
|---------|---------|
| **无糖** | 糖含量 <= 0.5g/100g（或100ml） |
| **低糖** | 糖含量 <= 5g/100g（或100ml） |
| **低脂** | 脂肪含量 <= 3g/100g（液体）或 <= 1.5g/100g（固体） |
| **脱脂** | 脂肪含量 <= 0.5g/100g（液体）或 <= 0.5g/100ml |
| **低钠/低盐** | [[钠]]含量 <= 120mg/100g（或100ml） |
| **高钙** | [[钙]]含量 >= 240mg/100g（或 >= 120mg/100ml） |
| **含有/富含** | 营养素含量达到参考值的特定比例 |

## 健康选择技巧

1. **看能量**：选择能量密度较低的产品
2. **看钠**：优先选择钠NRV%低的产品（建议<15%）
3. **看脂肪**：注意区分总脂肪和饱和脂肪，饱和脂肪越低越好
4. **看糖**：选择添加糖少或无糖的产品
5. **看蛋白质**：选择蛋白质NRV%较高的产品
6. **看配料表**：配料表越短越好，前几位避免出现大量糖和油脂

## GB 28050-2025 实务补充

- 新版标准把消费者更容易忽略的**饱和脂肪**和**糖**拉进了强制标示内容，做减脂和慢病管理时会更好用。
- `NRV%` 仍然是快速判断工具，但更适合用于同类食品横向比较，不代表“吃满100%越多越好”。
- 按“每份”标示时，要同时看清每份的质量或体积，避免被看似很低的数值误导。
- 在儿童和青少年选购场景里，新的提示语对高盐、高油、高糖零食筛查很有帮助。
