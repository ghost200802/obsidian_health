# 可食部分

可食部分（edible portion）是指食物中可以供人食用的部分，是[[食物成分表]]数据表达的基础。食物成分表中所有营养成分含量均以每100g可食部分来表示。

## 定义

可食部分 = 食物总重 - 不可食部分（废弃物）重量

不可食部分包括：骨头、皮壳、果核、籽、根须、鳞片、内脏中的非食用部分等。不同食物的不可食部分差异很大。

## 计算方法

### 可食部分比例

$$可食部分比例（\%）= \frac{可食部分重量}{食物总重量} \times 100$$

### 膳食调查中的应用

在[[膳食调查]]中，当记录的是带废弃物的食物重量时，需通过可食部分比例换算为可食部重量：

$$可食部摄入量 = 食物摄入量 \times 可食部分比例$$

然后再根据[[食物成分表]]中每100g可食部的营养素含量计算实际营养素摄入量。

## 常见食物的可食部分比例

| 食物 | 可食部分比例 | 废弃部分 |
|------|------------|---------|
| 猪肉（里脊） | 100% | 无 |
| 带骨牛腩 | ~72% | 骨头 |
| 鸡肉（带骨） | ~66% | 骨头、皮 |
| 鸡蛋（去壳） | ~88% | 蛋壳 |
| 带鱼 | ~70% | 头、内脏、骨 |
| 香蕉（去皮） | ~59% | 皮 |
| 苹果（去皮） | ~76% | 皮、核 |
| 菠菜 | ~89% | 根 |
| 橘子 | ~77% | 皮、籽、白络 |

## 废弃率

废弃率（waste factor）是可食部分比例的互补概念：

$$废弃率 = 1 - 可食部分比例$$

废弃率受食物品种、采购规格和加工精细度影响。[[食物成分表]]中提供了常见食物的可食部分比例数据，是膳食计算的重要参考依据。

## 注意事项

- 食物成分表中部分食物标注为"食部 100%"，表示数据已是可食部分含量
- 不同地区对同一食物的加工习惯不同，可食部分比例可能存在差异
- 在实际工作中应根据具体情况选择合适的可食部分比例进行换算
