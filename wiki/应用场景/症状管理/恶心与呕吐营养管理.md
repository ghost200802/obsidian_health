# 恶心与呕吐营养管理

## 定义

恶心（nausea）是一种上腹部不适、紧迫欲吐的主观感觉。呕吐（vomiting）是胃内容物经食管、口腔强力排出的病理生理反射。恶心与呕吐是肿瘤患者最常见的[[营养影响症状]]之一。

## 病因与发病机制

### 呕吐中枢机制

呕吐反射由两个中枢调控：
- **呕吐中枢（VC）**：位于延髓，直接控制呕吐运动
- **化学感受器触发区（CTZ）**：位于第四脑室底，接受血液中化学刺激信号

### 三类病因

1. **中枢性**：颅内压增高、药物（化疗药物）、代谢紊乱（尿毒症、酮症酸中毒）
2. **周围性**：胃肠道疾病（胃炎、肠梗阻）、腹腔脏器炎症
3. **精神心理性**：焦虑、恐惧、条件反射

## 临床分级

### 恶心分级

| 分级 | 表现 |
|------|------|
| I级 | 可耐受，不影响进食 |
| II级 | 影响进食，但可进食部分食物 |
| III级 | 严重影响进食，几乎无法进食 |

### 呕吐分级

| 分级 | 表现 |
|------|------|
| I级 | 1-2次/日 |
| II级 | 3-5次/日 |
| III级 | >=6次/日 |
| IV级 | 顽固性呕吐，需静脉补液 |

## 营养治疗

### 膳食管理

1. **I-II级恶心呕吐**：
   - 清流质饮食→半流质饮食→软食→普食逐步过渡
   - 少量多餐，每日6-8餐
   - 避免高脂、高糖、辛辣、油腻食物
   - 食物温度适宜，避免过冷过热
   - 进食时少量饮用清凉饮料

2. **III-IV级恶心呕吐**：
   - 禁食、静脉补液纠正水电解质紊乱
   - 严重者考虑[[鼻饲管|管饲喂养]]或[[肠外营养]]

3. **鼻饲管喂养**：
   - 经鼻胃管或鼻肠管给予营养支持
   - 适用于无法经口进食但胃肠功能尚可的患者

4. **肠外营养**：
   - 适用于严重呕吐伴肠梗阻或胃肠道功能衰竭者

### 促消化药物

联合使用促动力药物可改善胃排空：
- 甲氧氯普胺（胃复安）：10mg，促进胃排空
- 多潘立酮（吗丁啉）：促进胃蠕动

### 止吐药物（7大类）

1. **5-HT3受体拮抗剂**：昂丹司琼、格拉司琼
2. **NK1受体拮抗剂**：阿瑞匹坦
3. **糖皮质激素**：地塞米松
4. **多巴胺受体拮抗剂**：甲氧氯普胺
5. **苯二氮卓类**：劳拉西泮
6. **H1受体拮抗剂**：苯海拉明
7. **大麻酚类**：屈大麻酚
