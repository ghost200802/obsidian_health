# 便秘营养管理

## 定义

便秘（constipation）是指排便频率减少、粪便干硬、排便困难或排便不尽感。是[[肠内营养|EN]]支持中最常见的消化道并发症之一，发生率达13.1%-85%。

## 病因

### 肠内营养相关因素
1. 缺乏膳食纤维
2. 水分摄入不足
3. 长期卧床，活动减少
4. 药物因素（阿片类镇痛药、抗胆碱药）
5. EN配方成分（高蛋白、低残渣）

### EN输注条件不当

| 参数 | 推荐范围 |
|------|----------|
| 输注温度 | 37-40°C |
| 间歇输注 | 每次45-60分钟 |
| 初始速度 | 30-50 ml/h |
| 目标速度 | 逐步增加至80-100 ml/h |
| 循环输注 | 12-16小时/天 |

## 营养治疗

### 膳食管理

1. **增加膳食纤维**：
   - 可溶性纤维：燕麦、水果（苹果、梨）、果胶
   - 不溶性纤维：全谷物、蔬菜、坚果
   - 总纤维摄入量25-30 g/d

2. **充足水分**：
   - 每日饮水量>=1500-2000 ml
   - EN期间额外补充水分

3. **含纤维EN配方**：
   - 含大豆多糖、燕麦纤维等成分
   - 可显著降低EN相关便秘发生率
   - 注意监测腹胀等不良反应

### 益生菌

- 补充双歧杆菌、乳酸杆菌等益生菌
- 改善肠道微生态，促进肠蠕动
- 可联合益生元（低聚果糖）使用

### EN输注管理

1. **间歇输注**优于持续输注（模拟正常进餐节律）
2. 循环输注（12-16h/天），留出肠道休息时间
3. 从低速开始，逐步增加至目标速度
4. 注意输注温度控制（37-40°C）
