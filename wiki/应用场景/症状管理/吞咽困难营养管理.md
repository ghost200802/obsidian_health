# 吞咽困难营养管理

## 定义

吞咽困难（dysphagia）是指食物从口腔经食管到胃的过程中出现障碍，表现为吞咽费力、哽噎感或吞咽疼痛。是头颈部肿瘤、神经系统疾病患者常见的[[营养影响症状]]。

## 病因

1. **机械性梗阻**：食管肿瘤、狭窄
2. **运动功能障碍**：脑卒中、帕金森病、重症肌无力
3. **头颈部肿瘤**：放疗后纤维化、手术后结构改变
4. **神经肌肉疾病**：肌萎缩侧索硬化、多发性硬化

## 筛查与评估

### 改良洼田饮水试验

- 对意识清楚、可配合的患者，先后尝试 **1 mL、3 mL、5 mL、30 mL** 温水
- 观察饮水时间、是否呛咳、是否停顿、是否鼻腔返流
- **3级及以上**提示异常，应进一步做吞咽功能评估

| 级别 | 表现 |
|------|------|
| 1级 | 一次顺利咽下，无呛咳 |
| 2级 | 分两次喝完，无呛咳 |
| 3级 | 一次咽下，但有呛咳 |
| 4级 | 分两次以上喝完，并有呛咳 |
| 5级 | 频繁呛咳，不能全部咽下 |

### 改良容积-粘度吞咽测试（VVST）

- 适用于洼田试验异常后进一步判断安全性与有效性
- 按 **3 mL -> 5 mL -> 10 mL** 的容积顺序，依次测试**中稠 -> 低稠 -> 水 -> 高稠**
- 观察安全性指标：咳嗽、音质改变、血氧饱和度下降
- 观察有效性指标：唇闭合差、口腔残留、分次吞咽、咽部残留
- 临床目标是在**安全前提下，选择最低可接受稠度和最大可接受容积**

## 营养治疗

### 经口进食管理

#### 食品质地分级

| 类别 | 分级 | 特点 | 适用场景 |
|------|------|------|----------|
| 液体 | 3级高稠 | 需用勺送食，不会自行流出 | 重度吞咽障碍 |
| 液体 | 2级中稠 | 可用吸管或杯饮，流速较慢 | 开始治疗性经口进食 |
| 液体 | 1级低稠 | 可轻松吸取，接近普通饮品 | 轻度吞咽障碍 |
| 固体 | 4级细泥型 | 均一泥状，无需咀嚼 | 需要舌和上颚压碎食物者 |
| 固体 | 5级细馅型 | 细碎、湿润、易成团 | 舌运送能力尚可者 |
| 固体 | 6级软食型 | 松软小块，可轻松碾压 | 吞咽和咀嚼轻度下降者 |

#### 食物选择示例

- **重度吞咽障碍**：南瓜泥、土豆泥、胡萝卜泥、鸡肉泥、鱼泥、豆腐泥、米糊
- **中度吞咽障碍**：蒸熟后捣碎的南瓜或西兰花、嫩煮鱼/鸡肉、软面条、软米饭、浓稠蔬菜汤
- **轻度吞咽障碍**：软煮蔬菜、小块嫩豆腐、软米饭、软面、浓稠汤水
- 质地改良后仍应兼顾[[蛋白质]]和能量密度，必要时可加入蛋白粉或联用[[口服营养补充]]

### EN适应证

出现以下情况需启动[[肠内营养|EN]]支持：
1. 经口进食量<目标量的60%
2. 意识障碍
3. 洼田饮水试验>=III级且难以安全经口进食
4. 严重口腔/咽喉功能障碍

### 管饲途径选择

| 途径 | 适用条件 |
|------|----------|
| 鼻胃管 | 短期使用（<4周） |
| 鼻肠管 | 胃排空障碍或高反流风险 |
| PEG | 长期使用（>4周），头颈肿瘤术后 |

### 水分管理

- 每日水分需求：21-43 ml/(kg·d)
- 注意稠液体和固体食物中的水分
- 不足时通过管饲额外补水

## 护理要点

- 进食时取坐位或半卧位，头部略前倾，避免平躺进食
- 食物温度以 **38-40℃** 为宜，避免过冷或过热刺激咽部
- 进食速度宜慢，小口分次；餐后保持坐位或半卧位 **30 分钟**
- 保持安静、专注的进食环境，降低呛咳和误吸风险
