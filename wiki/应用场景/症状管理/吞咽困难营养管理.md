# 吞咽困难营养管理

## 定义

吞咽困难（dysphagia）是指食物从口腔经食管到胃的过程中出现障碍，表现为吞咽费力、哽噎感或吞咽疼痛。是头颈部肿瘤、神经系统疾病患者常见的[[营养影响症状]]。

## 病因

1. **机械性梗阻**：食管肿瘤、狭窄
2. **运动功能障碍**：脑卒中、帕金森病、重症肌无力
3. **头颈部肿瘤**：放疗后纤维化、手术后结构改变
4. **神经肌肉疾病**：肌萎缩侧索硬化、多发性硬化

## 评估方法

### 洼田饮水试验（5级）

| 级别 | 表现 |
|------|------|
| I级 | 5秒内一次喝完，无呛咳 |
| II级 | 5秒以上分两次以上喝完，无呛咳 |
| III级 | 一次喝完，有呛咳 |
| IV级 | 分两次以上喝完，有呛咳 |
| V级 | 频繁呛咳，全部喝完困难 |

> [!warning]
> III级及以上提示吞咽功能障碍，需考虑EN支持。

## 营养治疗

### 经口进食管理

#### 六级饮食

| 级别 | 膳食质地 | 适用人群 |
|------|----------|----------|
| 1级 | 稀流体（清流质） | 严重吞咽障碍 |
| 2级 | 稠流体（增稠液体） | 轻中度吞咽障碍 |
| 3级 | 泥状/糊状食物 | 口期吞咽障碍 |
| 4级 | 细泥状/细碎食物 | 咀嚼困难 |
| 5级 | 软质食物 | 轻度吞咽障碍 |
| 6级 | 普通食物 | 基本正常 |

### EN适应证

出现以下情况需启动[[肠内营养|EN]]支持：
1. 经口进食量<目标量的60%
2. 意识障碍
3. 洼田饮水试验>=III级
4. 严重口腔/咽喉功能障碍

### 管饲途径选择

| 途径 | 适用条件 |
|------|----------|
| 鼻胃管 | 短期使用（<4周） |
| 鼻肠管 | 胃排空障碍或高反流风险 |
| PEG | 长期使用（>4周），头颈肿瘤术后 |

### 水分管理

- 每日水分需求：21-43 ml/(kg·d)
- 注意稠液体和固体食物中的水分
- 不足时通过管饲额外补水
