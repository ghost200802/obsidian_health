# 口干营养管理

## 定义

口干（dry mouth, xerostomia）是多种疾病引起口腔干燥的一种主观感觉，是多因素作用于口腔的结果，目前尚无标准化定义，患者主诉是诊断的主要依据。

## 流行病学

- 人群中患病率：5.5%-46%
- >65岁老年人中：30%-40%，女性更常见
- ICU重症患者口干是排名前三的不适症状之一
- ICU重度口干发生率：**17.9%**

## 病因

### 六类病因

1. **药源性因素**（最常见）：100种常用处方药最常见不良反应即为口干
   - 呋塞米：每增加30mg，口干发生率增加1.4倍
   - 利血平：抑制交感神经致唾液腺分泌障碍
   - 抗胆碱药物：竞争性结合胆碱受体抑制涎腺分泌

2. **放射性口炎**：80%鼻咽癌放疗患者因唾液腺受损出现口干

3. **风湿免疫疾病**：干燥综合征（累及唾液腺和泪腺）

4. **围绝经期综合征**：雌激素分泌减少致唾液腺腺泡细胞凋亡，维生素D缺乏致腮腺刺激性唾液流率降低

5. **营养因素**：贫血、高钠血症、高钙血症、维生素B12缺乏、维生素D缺乏、蛋白质缺乏

6. **其他**：全身/局部感染、口腔局部问题、抑郁症、焦虑症

## 病理生理

人体三大唾液腺（腮腺、下颌下腺、舌下腺）每天分泌唾液约**1.5L**。当唾液产生和消耗呈负平衡时出现口干。长期顽固口干可导致：
- 牙周炎
- 味觉改变
- 真菌感染
- 影响进食行为，导致能量和营养素摄入减少

## 评估方法

### 主观量表

| 工具 | 特点 |
|------|------|
| NRS（数字评分表） | 0-10分，0=无口干，10=剧烈口干 |
| VAS（视觉模拟评分表） | 与NRS相关性高 |
| TDS（口渴痛苦量表） | 6条目，5级评分，总分6-30分 |

### 客观方法
- 咀嚼刺激唾液流率法
- 动态唾液分泌测量法
- 主观评估工具更贴近患者真实感受

## 营养治疗

### 足量饮水

| 性别 | 每日总水适宜摄入量 | 饮水量 | 食物中水 |
|------|-------------------|--------|----------|
| 男性 | 3000 ml | 1700 ml | 1300 ml |
| 女性 | 2700 ml | 1500 ml | 1200 ml |

- 主动饮水，少量多次，每次约200ml
- 不要等到口渴再喝水
- 身体活动时前后各饮水100-200ml

### 量化用盐

- 每日食盐摄入<=5g
- 注意隐形盐：10ml酱油约1.6-1.7g盐，10g豆瓣酱约1.5g盐

## 药物治疗

| 药物 | 适应证 | 剂量 | 备注 |
|------|--------|------|------|
| 毛果芸香碱 | 放疗/干燥综合征口干 | 5mg/次，3次/d，4周显效 | FDA批准 |
| 西维美林 | 干燥综合征口干 | 30mg/次，3次/d | FDA批准 |
| 人工唾液 | 缓解口干 | - | 羧甲纤维素钠、黏蛋白 |
| alpha-干扰素 | 增加唾液分泌 | 150 IU/次，3次/d，12周 | - |
| 氨磷汀 | 放射性损害保护 | - | 预防性使用 |
