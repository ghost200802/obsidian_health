# 特殊人群体重管理

## 定义

特殊人群体重管理是指针对具有特定生理状态、年龄阶段或社会角色的群体，制定的差异化体重管理策略。不同人群在能量需求、代谢特点、健康风险和管理目标方面存在显著差异，因此需要个体化的管理方案。

> 依据《[[source-体重管理指导原则（2024年版）]]》，重点管理6类特殊人群。

---

## 6类特殊人群管理要点

### 1. 学龄前儿童（0-6岁）

| 管理维度 | 具体措施 |
|---------|---------|
| **膳食管理** | 培养健康饮食行为，规律进餐（每天3正餐+2次健康零食）；避免过度喂养；限制含糖饮料和高能量密度零食；食物多样化，保证优质蛋白质和微量营养素供给 |
| **身体活动** | 每天**180分钟**身体活动，其中中等到较大强度活动至少60分钟；以游戏化运动为主 |
| **睡眠管理** | 3-4岁每天10-13小时，5-6岁每天10-12小时；规律作息 |
| **行为习惯** | 减少屏幕时间（≤1小时/天）；培养自主进食能力；家庭营造健康饮食环境 |

> 注意事项：学龄前儿童处于快速生长发育期，**不建议采用限制能量饮食**。体重管理的重点在于培养健康生活方式，而非主动减重。参见[[儿童青少年肥胖营养]]了解更多。

### 2. 学龄儿童青少年（6-18岁）

| 管理维度 | 具体措施 |
|---------|---------|
| **饮食调整** | 保证充足营养供给以支持生长发育；减少高糖、高脂、高能量密度食物摄入；规律早餐，控制校外食品购买 |
| **身体活动** | 每天至少**60分钟**中等到较大强度运动；鼓励参加体育课和课外体育活动 |
| **屏幕时间** | 电子产品使用时间**< 2小时/天**；减少久坐时间 |
| **心理关注** | 关注体重相关的心理问题（如自卑、焦虑）；避免体重歧视和身材焦虑；建立积极的身体形象 |

> 注意事项：青少年体重管理需兼顾青春期发育需求，避免极端节食。[[PCOS营养管理]]对合并多囊卵巢综合征的青春期女性尤为重要。参见[[儿童青少年肥胖营养]]了解更多。

### 3. 孕产期女性

| 管理维度 | 具体措施 |
|---------|---------|
| **孕前管理** | 孕前BMI维持在**18.5-23.9**的理想范围；超重/肥胖女性建议孕前适度减重 |
| **孕期管理** | 根据孕前BMI分阶段控制能量摄入；定期监测体重增长曲线；保证叶酸、铁、钙等关键营养素 |
| **产后管理** | 产后**1年内**逐步恢复至孕前体重；鼓励母乳喂养（有利于产后体重恢复）；避免产后极端节食 |

| 孕前BMI | 孕期体重增长推荐范围 |
|---------|---------------------|
| 体重过低（< 18.5） | 12.5 ~ 18.0 kg |
| 正常体重（18.5 ~ 23.9） | 11.5 ~ 16.0 kg |
| 超重（24.0 ~ 27.9） | 7.0 ~ 11.5 kg |
| 肥胖（≥ 28.0） | 5.0 ~ 9.0 kg |

> 注意事项：孕期不宜减重，体重管理重点是控制增长幅度在合理范围内。参见[[妊娠期营养]]了解更多。

### 4. 老年人群（≥ 65岁）

| 管理维度 | 具体措施 |
|---------|---------|
| **健康饮食** | 适量优质蛋白质摄入（1.0-1.2 g/kg/d），预防[[老年少肌性肥胖|肌少症]]；充足蔬果和全谷物；适当补充维生素D和钙 |
| **规律活动** | 适合年龄的运动方案；有氧运动+平衡训练+抗阻运动；平衡训练预防跌倒 |
| **睡眠管理** | 保证约**7小时**睡眠；关注睡眠呼吸暂停 |
| **心理支持** | 关注心理健康，预防老年抑郁；社交活动促进心理健康 |

> 注意事项：80岁以上高龄老人BMI适宜范围为**22.0-26.9**，不建议过度减重。老年人体重管理需特别注意肌肉量维持，避免肌少症。参见[[老年少肌性肥胖]]了解更多。

### 5. 职业人群

| 管理维度 | 具体措施 |
|---------|---------|
| **健康食堂** | 推广健康食堂，提供低能量、营养均衡的工作餐；标注食物能量和营养成分 |
| **工间操** | 推行工间操制度，每天至少10-15分钟；利用碎片时间增加身体活动 |
| **减少久坐** | 优化工作环境，设置站立式办公；每久坐1小时起身活动5-10分钟 |
| **健康促进** | 职场健康教育和体重管理项目；建立健康支持性环境 |

> 注意事项：职业人群的特点是久坐时间长、应酬多、不规律饮食。管理重点在于环境改造和行为习惯调整。

### 6. 体重过低人群（BMI < 18.5）

| 管理维度 | 具体措施 |
|---------|---------|
| **膳食策略** | **高能量、高蛋白**饮食；增加健康脂肪摄入（坚果、牛油果等）；少食多餐，每天5-6餐 |
| **运动策略** | 以**增肌抗阻运动**为主（2-3次/周）；促进肌肉量增长；避免过度有氧运动消耗能量 |
| **疾病排查** | 排除导致体重过低的疾病因素：甲亢、消化道疾病、慢性感染、恶性肿瘤、精神心理疾病等 |

> 注意事项：体重过低可能提示潜在疾病，需先进行医学评估排除继发性因素。参见[[营养不良]]和[[营养风险筛查]]了解更多。

## 各人群特殊注意事项汇总

| 人群 | 核心原则 | 禁忌/注意 |
|------|---------|----------|
| 学龄前儿童 | 培养健康习惯 | 不建议限制能量饮食 |
| 学龄儿童青少年 | 保证发育+控制体重 | 避免极端节食、体重歧视 |
| 孕产期女性 | 控制孕期增重+产后恢复 | 孕期不宜减重 |
| 老年人群 | 维持肌肉量+适度控制 | 避免过度减重导致肌少症 |
| 职业人群 | 环境改造+行为改变 | 注意应酬饮食管理 |
| 体重过低人群 | 增加能量+排查疾病 | 排除继发性因素后再干预 |
