# 膳食调查

## 页面定位 / 适用范围

本页聚焦膳食调查这一**具体方法层**，重点是食物摄入数据的采集、换算、评价与临床应用。

- 若需要查看涵盖膳食、体格、生化和临床检查的综合调查框架，见[[营养调查]]
- 若需要查看膳食调查结果如何参与综合营养判断，见[[营养状况评价]]与[[营养评价]]

## 定义

膳食调查（Dietary Survey / Dietary Assessment）是营养评价的基本方法之一，通过了解患者日常食物摄入的种类和数量，计算能量和营养素的摄入量，并与推荐标准进行比较，评估患者的营养摄入是否充足。

## 膳食调查方法

### 1. 24小时膳食回顾法（24-Hour Recall）

| 项目 | 说明 |
|------|------|
| 方法 | 回顾过去24小时内摄入的所有食物和饮料 |
| 优点 | 快速简便、费用低、依从性好 |
| 缺点 | 依赖记忆、代表性有限 |
| 适用 | 住院患者、快速筛查 |

> **注意**：单日24小时回顾代表性有限，通常需要连续3天的回顾（含1个周末日）才能较好反映日常摄入。

### 2. 膳食记录法（Diet Record）

| 项目 | 说明 |
|------|------|
| 方法 | 患者或家属记录一段时间内摄入的所有食物和饮料 |
| 时长 | 通常3-7天（含周末） |
| 优点 | 准确性较高、不需要依赖回忆 |
| 缺点 | 依从性要求高、可能改变正常饮食行为 |

### 3. 食物频率问卷（FFQ）

| 项目 | 说明 |
|------|------|
| 方法 | 询问特定时间段内各种食物的食用频率和摄入量 |
| 周期 | 通常覆盖过去1个月、3个月或1年 |
| 优点 | 能反映长期膳食模式、成本低 |
| 缺点 | 问卷设计复杂、定量精度有限 |

### 4. 食物称重法

| 项目 | 说明 |
|------|------|
| 方法 | 精确称量实际摄入的食物生重和熟重 |
| 优点 | 最准确 |
| 缺点 | 费时费力、患者负担重 |
| 适用 | 营养研究、代谢病房 |

### 5. 化学分析法

| 项目 | 说明 |
|------|------|
| 方法 | 通过化学分析测定食物和体液中的营养素含量 |
| 优点 | 最精确 |
| 缺点 | 成本高、耗时长、需要实验室条件 |
| 适用 | 科学研究 |

## 膳食调查流程

```
1. 收集膳食信息
   → 选择合适的调查方法
   → 收集食物摄入种类和数量
   → 记录特殊饮食或补充剂

2. 计算营养素摄入量
   → 查阅食物成分表
   → 计算能量和各营养素摄入量
   → 与DRIs/RNIs进行比较

3. 评价膳食质量
   → 能量摄入是否充足
   → 蛋白质摄入是否充足
   → 维生素和矿物质摄入是否达标
   → 膳食结构是否合理

4. 给出膳食建议
   → 针对不足的营养素提出补充建议
   → 调整膳食结构
   → 结合疾病特点进行饮食调整
```

## 膳食调查在临床中的应用

| 应用场景 | 目的 | 方法选择 |
|---------|------|---------|
| 入院营养评估 | 了解患者营养摄入状况 | 24小时回顾 |
| 营养治疗效果监测 | 评估营养支持是否充足 | 膳食记录（3天） |
| 疾病相关饮食评估 | 评估特殊饮食依从性 | 膳食记录 + 食物频率问卷 |
| 出院饮食指导 | 制定个体化膳食方案 | 24小时回顾 + 食物频率问卷 |

## 膳食调查在GLIM标准中的应用

膳食调查结果可用于满足GLIM病因标准中的"食物摄入减少"项：
- 摄入量低于需要量的50%超过1周
- 任意程度的摄入减少超过2周

## 评估注意事项

> [!warning]
> 1. 住院期间膳食可能不代表患者平时的饮食习惯
> 2. 患者可能高估或低估摄入量
> 3. 需要考虑疾病对饮食的影响
> 4. 食物成分表的选择应适合本地食物
> 5. 需要考虑烹调方式对营养素的影响
