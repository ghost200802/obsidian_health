# 皮褶厚度

## 定义

皮褶厚度（Skinfold Thickness）是通过测量皮下脂肪厚度来评估人体脂肪储备和营养状况的重要人体测量指标，是[[营养状况评价]]中体格测量的核心项目之一。

## 测量部位

### 常用测量部位

| 部位 | 位置描述 | 主要用途 |
|------|---------|---------|
| **肱三头肌（TSF）** | 上臂背侧肩峰与鹰嘴连线中点处，沿长轴方向捏起皮褶 | 评估全身皮下脂肪储备的最常用指标 |
| **肩胛下** | 肩胛骨下角下方约1-2cm处，与脊柱成45度角向外上方捏起皮褶 | 与TSF联合评估躯干脂肪分布 |
| **腹部** | 脐旁右侧约1-2cm处，纵向（与躯干长轴平行）捏起皮褶 | 评估腹部脂肪沉积，与代谢性疾病风险相关 |

> [!tip] 测量要点
> 通常测量右侧肢体。TSF是最基本的测量部位，肩胛下皮褶常与TSF联合测量用于计算体密度和体脂百分比。

## 测量方法

### 器具要求

- **皮褶厚度计（皮褶计）**：表头读数精确至0.1mm
- **标准压力**：10 g/cm²（即卡尺接触面的压强）
- **卡尺面积**：接触面面积需符合标准（通常为6mm x 15mm）

### 操作步骤

1. **定位标记**：准确标记测量点位置
2. **捏起皮褶**：用拇指和食指捏起皮肤及皮下脂肪层，形成皮褶
   - 注意：只捏起皮肤和皮下脂肪，不应包含肌肉层
   - 捏起后两指间距离约需保持一致（通常为1cm）
3. **放置卡尺**：将皮褶计卡尺置于捏起皮褶的根部（即手指捏起点的正下方约1cm处）
4. **固定读数**：卡尺固定3秒后读取数值
5. **重复测量**：同一部位通常测量2-3次，取平均值，两次测量误差不应超过5%

> [!warning] 注意事项
> - 测量应在室温下进行，避免因寒冷导致血管收缩影响读数
> - 测量前皮褶计需校准至零点
> - 捏起皮褶时手法应一致，力度适中
> - 被测者取坐位或立位，上臂自然下垂

## 评价标准

### 肱三头肌皮褶厚度（TSF）参考值

| 人群 | 正常值（mm） | 判断标准 |
|------|-------------|---------|
| 男性 | 12.5 | 正常范围 8.3-16.5mm |
| 女性 | 16.5 | 正常范围 10.5-22.5mm |

### TSF占正常值的百分比评价

| 占正常值百分比 | 营养状况判断 |
|---------------|-------------|
| > 90% | 正常 |
| 80%-90% | 轻度脂肪储备消耗 |
| 60%-80% | 中度脂肪储备消耗 |
| < 60% | 严重脂肪储备消耗 |

> [!note] 评价方法
> 将实测TSF值与同性别同年龄的正常参考值比较，计算百分比：
> $$TSF\% = \frac{实测值}{标准值} \times 100\%$$

## 体密度与体脂推算

### 体密度公式

利用皮褶厚度可以推算体密度（Body Density, D），进而计算体脂百分比。以下为不同性别年龄的体密度推算公式：

**男性：**

| 年龄 | 公式 |
|------|------|
| 17-19岁 | $D = 1.1620 - 0.0630 \times \log(Z)$ |
| 20-29岁 | $D = 1.1631 - 0.0632 \times \log(Z)$ |
| 30-39岁 | $D = 1.1422 - 0.0544 \times \log(Z)$ |
| 40-49岁 | $D = 1.1374 - 0.0523 \times \log(Z)$ |
| 50岁以上 | $D = 1.1330 - 0.0504 \times \log(Z)$ |

**女性：**

| 年龄 | 公式 |
|------|------|
| 17-19岁 | $D = 1.1549 - 0.0678 \times \log(Z)$ |
| 20-29岁 | $D = 1.1599 - 0.0717 \times \log(Z)$ |
| 30-39岁 | $D = 1.1423 - 0.0632 \times \log(Z)$ |
| 40-49岁 | $D = 1.1333 - 0.0612 \times \log(Z)$ |
| 50岁以上 | $D = 1.1339 - 0.0645 \times \log(Z)$ |

其中 $Z = TSF + SSF$（TSF为三头肌皮褶厚度，SSF为肩胛下皮褶厚度，单位mm）。

### 体脂百分比计算（Siri公式）

$$体脂\% = (4.95 / D - 4.50) \times 100\%$$

## 上臂肌围与上臂肌面积

### 上臂肌围（MAMC）

上臂肌围反映肌肉蛋白储备状况，是评价蛋白质-能量营养状况的重要指标。

$$MAMC(cm) = MAC - 3.14 \times TSF$$

**参数说明：**
- **MAC**：上臂围（cm），在肩峰与鹰嘴连线中点处测量
- **TSF**：三头肌皮褶厚度（cm）

**正常参考值：**
- 男性：24.8 cm
- 女性：21.0 cm

**评价标准：**

| MAMC占正常值百分比 | 营养状况判断 |
|-------------------|-------------|
| > 90% | 正常 |
| 80%-90% | 轻度肌肉消耗 |
| 60%-80% | 中度肌肉消耗 |
| < 60% | 严重肌肉消耗 |

### 上臂肌面积（AMA）

上臂肌面积进一步精确反映肌肉量：

$$AMA(cm^2) = \frac{MAMC^2}{4 \times 3.14}$$

或使用校正公式：

$$AMA(cm^2) = \frac{(MAC - 3.14 \times TSF)^2}{4 \times 3.14}$$

> [!note] 评价意义
> MAMC和AMA主要用于评价蛋白质储备和[[营养不良]]的肌肉消耗程度，在[[营养评价]]中与TSF互补——TSF评估脂肪储备，MAMC评估蛋白质/肌肉储备。

## Oeder指数

Oeder指数是三头肌皮褶厚度与肩胛下皮褶厚度之和，用于综合评估皮下脂肪总量：

$$Oeder指数(mm) = TSF + SSF$$

**参考意义：**
- 可用于推算体密度（见上文公式）
- 也可作为评估肥胖程度的辅助指标
- Oeder指数越大，提示皮下脂肪蓄积越多
