# 营养诊断

## 定义

营养诊断是临床营养管理的核心环节，是对营养筛查阳性患者进行深入评估和综合判断的过程。营养诊断采用**三级诊断体系**，由浅入深地确定患者是否存在营养不良及其类型、程度和原因。

## 三级诊断体系

### 一级诊断：营养筛查

- **目的**：快速识别存在营养风险的患者
- **时机**：入院**24小时内**完成
- **方法**：使用标准化筛查工具
- **工具选择**：
  - 一般住院患者：[[营养风险筛查#NRS-2002]]
  - 老年患者：[[MNA]]
  - 肿瘤患者：NRS-2002（筛查）+ [[PG-SGA]]（评估）
  - 儿童患者：SGNA

### 二级诊断：营养评估

- **目的**：对筛查阳性患者进行全面评估，确定营养状况
- **方法**：
  - [[SGA]]（主观整体评估）
  - [[PG-SGA]]（患者参与的主观整体评估）
  - [[MNA]]（老年患者）
  - 膳食调查
  - [[人体测量]]
  - 生化检查（详见[[实验营养诊断]]）

### 三级诊断：综合评定

- **目的**：确定营养不良的诊断、类型和严重程度
- **方法**：[[GLIM]]标准
- **输出**：完整的营养诊断报告

## 营养诊断的完整流程

```
入院 → 24小时内完成营养筛查（一级诊断）
    → 筛查阴性 → 每周重复筛查
    → 筛查阳性 → 营养评估（二级诊断）
        → 膳食调查
        → 人体测量
        → 生化检查
        → SGA/PG-SGA评估
        → 综合评定（三级诊断）
            → GLIM标准诊断
                → 确诊营养不良
                    → 分型：饥饿型/炎症型/混合型
                    → 分级：轻度/中度/重度
                → 制定营养治疗计划
```

## 营养不良的诊断（GLIM标准）

### 诊断条件

营养不良诊断需同时满足：
- **至少1项表型标准**（体重下降/BMI降低/肌肉量减少）
- **至少1项病因标准**（食物摄入减少/炎症）

详见 [[GLIM]]

### 营养不良分型

| 分型 | 病因 | 特征 | 典型疾病 |
|------|------|------|---------|
| 饥饿型 | 摄入减少，无炎症 | 以营养素缺乏为主 | 进食障碍、消化道梗阻 |
| 炎症型（恶液质） | 摄入减少 + 炎症 | 代谢紊乱、肌肉分解 | 肿瘤、慢性感染、慢性炎症 |
| 混合型 | 摄入减少 + 炎症 | 兼具两型 | 严重创伤、大手术后 |

### 营养不良严重程度

| 严重程度 | 体重下降（6个月内） | BMI | 肌肉量 |
|---------|-------------------|-----|--------|
| 轻度 | 5%-10% | 18.5-< 20 | 轻度减少 |
| 中度 | 5%-10% | < 18.5 | 中度减少 |
| 重度 | > 10% | < 18.5 | 重度减少 |

## 营养诊断的四个维度

### 1. 膳食评估
- 24小时膳食回顾
- 膳食记录（3-7天）
- 食物频率问卷
- 营养素摄入量计算与比较

### 2. 人体测量
- 体重、身高、BMI
- 皮褶厚度
- 上臂围、上臂肌围
- 小腿围
- 腰围、臀围
详见 [[人体测量]]

### 3. 生化检查
- 血清白蛋白、前白蛋白
- 转铁蛋白、视黄醇结合蛋白
- 免疫功能指标
- 微量元素
- 维生素水平
详见 [[实验营养诊断]]

### 4. 临床评估
- 病史采集
- 体格检查
- SGA/PG-SGA评估
- 功能状态评估

## 营养诊断报告内容

一份完整的营养诊断报告应包括：

1. **筛查结果**：筛查工具名称、评分、判定
2. **膳食评估结果**：能量和营养素摄入量与推荐量的比较
3. **人体测量结果**：BMI、体重变化、皮褶厚度等
4. **生化检查结果**：血清蛋白、免疫功能等
5. **综合评定**：GLIM标准诊断结果
6. **营养不良分型与分级**
7. **营养治疗建议**：包括治疗方式、目标量、注意事项
