# 骨代谢标志物

## 一、概述

骨代谢标志物是反映骨形成和骨吸收过程中细胞活动和代谢产物变化的生化指标。骨骼是人体最大的钙磷储存库，也是维持矿物质平衡的重要器官。骨代谢标志物在[[骨质疏松症]]的早期筛查、疗效监测和骨折风险评估中具有重要价值。从[[营养评价]]角度看，骨代谢标志物可反映[[钙]]、[[维生素D]]、蛋白质等关键营养素对骨骼代谢的影响，是骨骼健康管理的重要辅助指标。

## 二、骨形成标志物

骨形成标志物反映成骨细胞活性及新骨形成状态。

### 2.1 骨钙素（BGP / Osteocalcin）

| 项目 | 英文缩写 | 参考范围 | 单位 |
|------|----------|----------|------|
| 骨钙素 | BGP (OC) | 4.5-14.5 | ng/mL |

- 由成骨细胞合成并分泌，是骨形成最特异的标志物
- 半衰期约5分钟，反映近期骨形成活性
- 增高：骨转换加速（甲亢、甲状旁腺功能亢进、骨软化症、骨折愈合期、儿童生长期）
- 降低：骨形成减少（甲减、长期应用糖皮质激素、严重[[维生素D]]缺乏、[[骨质疏松症]]晚期）

### 2.2 骨性碱性磷酸酶（B-ALP）

| 项目 | 英文缩写 | 参考范围 | 单位 |
|------|----------|----------|------|
| 骨性碱性磷酸酶 | B-ALP | 15.0-41.5 | U/L |

- 成骨细胞表面表达的特异性酶
- 比总ALP更能特异性反映骨形成
- 增高：骨转换加速、骨软化症、甲状旁腺功能亢进、骨肉瘤、骨折愈合期
- 降低：骨形成减少

### 2.3 I型前胶原N端前肽（PINP）/ C端前肽（PICP）

| 项目 | 英文缩写 | 参考范围 | 单位 |
|------|----------|----------|------|
| I型前胶原N端前肽 | PINP | 20.0-76.0 | ug/L |
| I型前胶原C端前肽 | PICP | 50.0-170.0 | ug/L |

- I型胶原是骨基质的主要成分（占骨有机基质的90%以上）
- PINP和PICP是I型前胶原在转化为成熟胶原过程中释放的片段
- PINP被IOF/IFCC推荐为骨形成的首选参考标志物
- 增高：骨形成活跃（骨折愈合、骨病活动期）
- 降低：抗骨吸收治疗有效

## 三、骨吸收标志物

骨吸收标志物反映破骨细胞活性及骨基质降解状态。

### 3.1 抗酒石酸酸性磷酸酶（TRACP）

| 项目 | 英文缩写 | 参考范围 | 单位 |
|------|----------|----------|------|
| 抗酒石酸酸性磷酸酶 | TRACP-5b | 1.2-4.8 | U/L |

- TRACP-5b由破骨细胞分泌，是破骨细胞活性的特异性标志物
- 增高：骨吸收加速（骨质疏松症、骨转移瘤、甲状旁腺功能亢进、变形性骨炎）
- 降低：抗骨吸收治疗有效

### 3.2 I型胶原交联C端肽（CTX）

| 项目 | 英文缩写 | 参考范围 | 单位 |
|------|----------|----------|------|
| 血清CTX | s-CTX | 0.14-0.58 | ng/mL |
| 尿CTX | u-CTX | 50-300 | ug/mmol Cr |

- I型胶原降解时释放的交联肽片段
- 是评估骨吸收最常用的标志物
- IOF/IFCC推荐CTX为骨吸收的首选参考标志物
- 增高：骨吸收加速
- 晨起空腹检测可减少昼夜变异

### 3.3 脱氧吡啶啉（DPD）

| 项目 | 英文缩写 | 参考范围 | 单位 |
|------|----------|----------|------|
| 尿DPD | u-DPD | 2.3-5.4 | nmol/mmol Cr |

- I型胶原的交联降解产物，仅来源于骨胶原
- 较CTX更特异，不受饮食中胶原干扰
- 需收集空腹第二次尿标本检测

## 四、骨质疏松症综合解读

### 4.1 骨量变化规律

| 年龄段 | 骨量变化特征 |
|--------|-------------|
| 出生至30-35岁 | 骨量持续积累，30-35岁达骨峰值 |
| 35-40岁 | 骨量相对稳定 |
| 40岁以后 | 骨量缓慢丢失，每年约丢失0.3-0.5% |
| 绝经后（女性） | 骨量快速丢失期，前5-10年每年丢失1-3% |
| 65岁以后 | 骨量丢失再次加速 |

- 骨峰值决定一生的骨骼健康基础
- 青少年和青年期积累足够的骨量至关重要
- 骨峰值每增加10%，骨质疏松性骨折风险降低约50%

### 4.2 骨质疏松症定义

| 指标 | 诊断标准 |
|------|----------|
| T值（与青年成人骨峰值比较） | |
| 正常 | T值 ≥ -1.0 |
| 骨量减少（骨量低下） | -2.5 < T值 < -1.0 |
| 骨质疏松症 | T值 ≤ -2.5 |
| 严重骨质疏松症 | T值 ≤ -2.5 + 一次或多次脆性骨折 |

- 骨密度（BMD）检测方法：双能X线吸收法（DXA），测量腰椎和髋部
- DXA是骨质疏松症诊断的金标准

## 五、易患人群

### 5.1 骨质疏松症高危因素

**不可改变因素**：
- 绝经后女性（雌激素下降 → 骨吸收加速）
- 老年人（年龄增长 → 骨形成减少）
- 女性骨量低于男性
- 有骨质疏松性骨折家族史
- 低体重/小体型（BMI < 20）
- 亚洲人种

**可改变因素**：
- 缺乏阳光照射 → [[维生素D]]缺乏
- 缺乏运动（尤其负重运动）
- 酗酒
- 吸烟
- 高盐饮食（钠促进尿钙排泄）
- 偏食/营养不良（[[钙]]、蛋白质、维生素D摄入不足）
- 长期使用糖皮质激素（泼尼松 ≥ 5 mg/d 超过3个月）
- 过量咖啡因摄入

### 5.2 重点关注人群

| 人群 | 风险因素 | 建议筛查 |
|------|----------|----------|
| 绝经后女性 | 雌素下降，骨吸收加速 | DXA骨密度 + 骨标志物 |
| 65岁以上老年人 | 年龄相关骨量丢失 | DXA + CTX/PINP |
| 长期糖皮质激素使用者 | 药物性骨质疏松 | 基线DXA + 每6-12月监测 |
| 糖尿病患者 | 低骨质量、骨折风险增加 | DXA + 骨标志物 |
| 甲状腺功能亢进者 | 甲状腺激素促进骨吸收 | DXA + 骨标志物 |
| 吸烟/酗酒者 | 骨形成抑制，骨吸收增加 | 骨标志物筛查 |

## 六、骨标志物临床应用

### 6.1 骨密度 vs 骨标志物

| 指标 | 骨密度（DXA） | 骨标志物 |
|------|---------------|----------|
| 反映内容 | 骨量（结构性） | 骨转换速率（功能性） |
| 变化所需时间 | 1-2年 | 1-3个月 |
| 诊断价值 | 金标准（骨质疏松症诊断） | 不能诊断，辅助评估 |
| 疗效监测 | 长期疗效评估 | 早期疗效评估（3个月即可变化） |
| 骨折预测 | 中长期预测 | 独立于骨密度的骨折风险预测因子 |

### 6.2 骨标志物的临床价值

- **早期疗效评估**：抗骨质疏松治疗后3个月骨标志物即可出现显著变化（下降30-50%），而DXA需1-2年
- **治疗依从性评估**：骨标志物未下降提示患者依从性差或药物无效
- **骨折风险评估**：骨标志物升高是独立于骨密度的骨折风险因子
- **骨转换状态评估**：鉴别高转换型骨质疏松（适合抗骨吸收治疗）与低转换型

### 6.3 骨标志物检测注意事项

- 标本采集时间：建议晨起空腹采血/留尿（减少昼夜变异）
- 避免影响因素：近期骨折、制动、妊娠、严重肝肾功能不全可影响结果
- 同一患者应使用同一实验室、同一方法连续监测
- 治疗后3个月复查评估疗效

## 七、营养学重点

### 7.1 骨骼健康的六大营养要素

**钙（Ca）**
- 骨骼中含约99%的体钙，钙是骨矿化的核心元素
- 成人每日钙推荐摄入量（RNI）：800 mg/d
- 50岁以上及绝经后女性：1000 mg/d
- 主要食物来源：牛奶及奶制品、豆腐、深绿色蔬菜、小鱼虾
- 详见 [[钙]]

**维生素D（VitD）**
- 促进肠道钙磷吸收，调节骨代谢
- 70%以上的维生素D由皮肤经日光（UVB）照射合成（7-脱氢胆固醇 → VitD3）
- 仅30%以下来源于膳食（富含VitD的食物有限：三文鱼、蛋黄、肝脏）
- 缺乏日光照射 → VitD缺乏 → 钙吸收下降 → 骨量丢失
- 详见 [[维生素D]] 和 [[维生素]]

**蛋白质**
- 骨基质主要由I型胶原（蛋白质）构成
- 适量蛋白质摄入促进骨形成
- 蛋白质不足 → 骨基质合成减少 → 骨量下降
- 但过量蛋白质（尤其高动物蛋白）增加尿钙排泄，需适量

**运动**
- 负重运动（跑步、跳绳、力量训练）和肌肉收缩运动刺激成骨细胞活性
- 运动是维持骨密度、增加骨强度的重要非药物干预
- 建议每周至少150分钟中等强度运动 + 2次力量训练

**限盐**
- 高钠摄入促进尿钙排泄（每排泄1000 mg钠约丢失26 mg钙）
- 高盐饮食是骨质疏松的不利因素
- 成人每日食盐摄入应 < 5 g（WHO建议）

**戒烟限酒**
- 吸烟抑制成骨细胞活性，降低骨密度
- 酒精干扰钙代谢和维生素D代谢
- 建议戒烟，饮酒量男性 < 25 g/d，女性 < 15 g/d

### 7.2 骨骼健康公式

> **钙 + 维生素D + 蛋白质 + 运动 + 限盐 + 戒烟限酒 = 骨骼健康**

### 7.3 维生素D与日光

- 皮肤日光照射是维生素D的最主要来源（占70%以上）
- UVB照射（波长290-315 nm）使皮肤中7-脱氢胆固醇转化为维生素D3
- 影响因素：
  - 纬度（高纬度地区UVB不足）
  - 季节（冬季UVB减弱）
  - 时间（上午10点-下午3点UVB最强）
  - 肤色（深色皮肤需更长时间照射）
  - 防晒霜（SPF 30减少约97%的VitD合成）
  - 年龄（老年人皮肤合成VitD能力下降约75%）
- 建议每日户外活动20-30分钟（充分暴露前臂和面部），不涂防晒霜
- 日照不足者需补充维生素D制剂

### 7.4 各年龄段的骨骼营养建议

| 年龄段 | 营养重点 |
|--------|----------|
| 儿童青少年 | 充足钙（1000-1200 mg/d）、VitD、蛋白质，加强运动，积累骨峰值 |
| 成年人（20-50岁） | 维持钙800-1000 mg/d、VitD，保持运动习惯，维持骨量 |
| 绝经后女性 | 钙1000 mg/d、VitD 800-1000 IU/d，蛋白质1.0-1.2 g/kg/d，负重运动 |
| 老年人 | 钙1000-1200 mg/d、VitD 800-1000 IU/d（必要时更高），蛋白质1.0-1.2 g/kg/d，防跌倒 |
