# 氮平衡

## 定义

氮平衡（Nitrogen Balance）是评估机体蛋白质代谢状态的核心指标，通过比较每日氮摄入量和氮排出量来反映体内蛋白质的合成与分解代谢状况。

## 计算方法

$$氮平衡(g/d) = 摄入氮量 - 排出氮量$$

### 摄入氮量计算

$$摄入氮量 = 蛋白质摄入量(g/d) \div 6.25$$

> 蛋白质中氮的平均含量约为16%，故除以6.25。

### 排出氮量计算

$$排出氮量 = 24小时尿尿素氮(g) + 4g（非尿素氮丢失）$$

其中非尿素氮丢失（约4g）包括：
- 粪便氮丢失：约1-1.5g
- 皮肤氮丢失：约0.5g
- 其他（汗液、毛发、呼吸等）：约0.5-2g

> [!important]
> 在大量烧伤、腹泻等情况下，非尿素氮丢失可能远超4g，需要根据实际情况调整。

## 判定标准

| 氮平衡状态 | 判定 | 临床意义 |
|-----------|------|---------|
| **正平衡** | 摄入氮 > 排出氮 | 合成代谢为主，蛋白质合成>分解，营养状况改善 |
| **零平衡** | 摄入氮 = 排出氮 | 维持状态，蛋白质合成=分解 |
| **负平衡** | 摄入氮 < 排出氮 | 分解代谢为主，蛋白质分解>合成，营养不良加重 |

## 氮平衡在临床中的应用

### 营养不良评估
- 持续负氮平衡提示营养不良存在或加重
- 创伤、感染、手术后机体处于应激状态，负氮平衡加重

### 营养支持效果评价
- 营养支持有效时，负氮平衡逐渐减轻
- 最终转为正氮平衡或零平衡
- 是评价蛋白质营养支持效果的**经典金标准**

### 营养需求确定
- 通过氮平衡试验确定蛋白质需要量
- 一般目标：氮平衡为+2~+4 g/d

## 影响氮平衡的因素

| 因素 | 影响 |
|------|------|
| 应激/创伤 | 增加蛋白质分解，加重负氮平衡 |
| 感染/发热 | 增加蛋白质消耗 |
| 糖皮质激素 | 促进蛋白质分解 |
| 高蛋白饮食 | 增加氮摄入 |
| 营养支持 | 改善氮平衡 |
| 肾功能不全 | 影响氮排泄 |

## 氮平衡的局限

- 需要准确收集24小时尿液
- 计算中的非尿素氮丢失为估计值
- 急性应激状态下难以达到正平衡
- 不能区分不同蛋白质的代谢情况
