# 人体测量

## 定义

人体测量（Anthropometric Measurement）是临床营养评价的基本方法之一，通过测量人体的各项形态学指标来评估营养状况和身体组成。主要包括身高、体重、BMI、皮褶厚度、上臂围、上臂肌围、小腿围等。

## 常用测量指标

### 1. 身高与体重

| 指标 | 测量方法 | 注意事项 |
|------|---------|---------|
| 身高 | 立位身高计，赤脚测量 | 晨起测量最准确；脊柱变形者测前臂长推算 |
| 体重 | 校准体重秤，晨起空腹、排空大小便后测量 | 需考虑水肿因素；卧床者用床秤 |

#### 身高推算（无法站立者）

- **前臂长推算身高**：身高 = 前臂长 x 1.44 + 69.6（女性）或 前臂长 x 1.44 + 72.4（男性）
- **膝高推算身高**：身高 = 膝高 x 2.02 + 56.7（女性）或 膝高 x 1.91 + 72.2（男性）

### 2. BMI（体质指数）

$$BMI = \frac{体重(kg)}{身高(m)^2}$$

#### 中国成人BMI标准

| BMI范围 | 判定 |
|---------|------|
| < 18.5 | 体重过低（消瘦） |
| 18.5-23.9 | 正常 |
| 24.0-27.9 | 超重 |
| >= 28.0 | 肥胖 |

#### WHO成人BMI标准

| BMI范围 | 判定 |
|---------|------|
| < 18.5 | 体重过低 |
| 18.5-24.9 | 正常 |
| 25.0-29.9 | 超重 |
| >= 30.0 | 肥胖 |
| >= 40.0 | 极度肥胖 |

#### 老年人BMI特殊标准

| BMI范围 | 判定（>= 70岁） |
|---------|----------------|
| < 22.0 | 营养不良风险 |
| 22.0-27.0 | 正常 |
| > 27.0 | 超重 |

### 3. 体重变化评估

#### 体重变化率计算

$$体重变化率(\%) = \frac{通常体重 - 当前体重}{通常体重} \times 100\%$$

#### 体重变化的意义

| 时间范围 | 变化程度 | 临床意义 |
|---------|---------|---------|
| 1周 | 体重下降1%-2% | 显著，需关注 |
| 1个月 | 体重下降5% | 严重，提示营养不良 |
| 3个月 | 体重下降7.5% | 非常严重 |
| 6个月 | 体重下降10% | 极度严重，预后不良 |

> [!important]
> **非自愿性体重下降**是营养不良的最重要指标之一。体重下降超过通常体重的10%即有临床意义。

### 4. 皮褶厚度

| 测量部位 | 方法 | 正常参考值 | 营养意义 |
|---------|------|-----------|---------|
| 三头肌皮褶厚度（TSF） | 肩峰与鹰嘴连线中点，捏起皮下脂肪测量 | 男性12.5mm，女性16.5mm | 反映体脂储备 |
| 肩胛下皮褶厚度 | 肩胛下角下方1-2cm处 | 男性10-20mm，女性12-25mm | 辅助评估体脂 |
| 髂前上棘皮褶厚度 | 髂前上棘处 | — | 辅助评估体脂 |

**TSF分级**：

| TSF（相当于正常值的百分比） | 判定 |
|--------------------------|------|
| > 90% | 正常 |
| 80%-90% | 轻度体脂减少 |
| 60%-80% | 中度体脂减少 |
| < 60% | 重度体脂减少 |

### 5. 上臂围（MAC）和上臂肌围（MAMC）

$$MAMC(cm) = MAC(cm) - 3.14 \times TSF(cm)$$

| 指标 | 测量位置 | 正常参考值 | 营养意义 |
|------|---------|-----------|---------|
| MAC | 上臂中点（肩峰至鹰嘴中点） | 男性25.3cm，女性23.2cm | 反映骨骼肌和脂肪储备 |
| MAMC | 计算值 | 男性25.3cm，女性23.2cm | 反映骨骼肌储备（排除脂肪因素） |

**MAMC分级**：

| MAMC（相当于正常值的百分比） | 判定 |
|----------------------------|------|
| > 90% | 正常 |
| 80%-90% | 轻度肌肉消耗 |
| 60%-80% | 中度肌肉消耗 |
| < 60% | 重度肌肉消耗 |

### 6. 小腿围（CC）

| 指标 | 正常参考值 | 营养意义 |
|------|-----------|---------|
| 小腿围 | >= 31cm | 老年人小腿围<31cm提示肌肉减少 |

### 7. 腰围与臀围

| 指标 | 男性参考值 | 女性参考值 | 营养意义 |
|------|-----------|-----------|---------|
| 腰围 | >= 85cm（中心性肥胖） | >= 80cm（中心性肥胖） | 评估中心性肥胖和代谢风险 |
| 腰臀比 | >= 0.9 | >= 0.85 | 评估脂肪分布 |

## 人体测量在营养评价中的应用

### SGA中的体格检查

SGA评估中的体格检查包括：
- 皮下脂肪评估（三头肌、肋下、髋部）
- 肌肉萎缩评估（颞肌、锁骨、肩胛带、肋间、大腿、小腿）
- 水肿/腹水评估

### GLIM标准中的表型指标

- 非自主性体重下降
- BMI降低
- 肌肉量减少（可通过人体测量评估）

## 人体测量的优势与局限

### 优势
- 简便、无创、可重复
- 设备要求低（仅需皮褶计和软尺）
- 成本低
- 可用于大规模筛查

### 局限
- 敏感性和特异性相对较低
- 受测量者操作影响
- 水肿可掩盖体重下降
- 不能反映急性营养变化
