# 营养缺乏病体征

## 定义

营养缺乏病体征是指在体格检查中发现的与特定营养素缺乏或不足相关的临床征象，是[[营养状况评价]]中临床检查的核心内容。通过系统观察和检查这些体征，可以辅助判断机体营养素缺乏的种类和程度。

## 营养缺乏的发展过程

营养缺乏病的发生是一个渐进的过程，从膳食摄入不足到出现临床症状通常经历以下阶段：

$$食物摄入不足 \rightarrow 营养素摄入不足 \rightarrow 组织中营养素缺乏 \rightarrow 生化代谢障碍 \rightarrow 机能变化 \rightarrow 病理改变（临床症状）$$

| 阶段 | 特征 | 可检出方法 |
|------|------|-----------|
| 1. 食物不足 | 膳食中某营养素供给量长期低于需要量 | 膳食调查 |
| 2. 营养不足 | 体内营养素储备开始下降 | [[生化检验]]（早期指标） |
| 3. 组织缺乏 | 组织中营养素含量降低，功能开始受影响 | 生化检验、功能试验 |
| 4. 生化障碍 | 酶活性改变、代谢异常 | 生化检验（特异性指标） |
| 5. 机能变化 | 生理功能出现可察觉的改变 | 功能检查 |
| 6. 病理改变 | 出现形态学改变和临床症状 | 临床体征检查 |

> [!important] 早期发现意义
> 营养缺乏在出现明显临床症状之前，往往已有生化改变和功能异常。通过系统的[[营养风险筛查]]和[[营养评价]]，可以在病理改变发生之前发现营养问题并及时干预。

## 各部位症状与营养素缺乏对应关系

以下为各身体部位常见营养缺乏病体征及其对应的营养素缺乏（来自《实用营养师手册》表16-5）：

### 头部

| 体征 | 可能缺乏的营养素 |
|------|----------------|
| 脱发、毛发稀疏、变脆 | 蛋白质-热能营养不良（PEM） |
| 色素减少、头发变灰白 | [[维生素B7（生物素）]]（Biotin）、铜 |
| 头发干燥、无光泽 | 蛋白质、必需脂肪酸 |

### 皮肤

| 体征 | 可能缺乏的营养素 |
|------|----------------|
| 皮肤干燥、脱屑、粗糙 | [[维生素A]]、必需氨基酸、必需脂肪酸 |
| 毛囊角化（毛囊性丘疹） | [[维生素A]] |
| 皮炎（暴露部位对称性皮炎） | [[维生素B3（烟酸）]]（烟酸，VitB3） |
| 皮肤瘀斑、出血点 | [[维生素C]]、[[维生素K]] |
| 脂溢性皮炎（鼻唇沟、耳后） | [[维生素B2（核黄素）]]（核黄素） |
| 伤口愈合延迟 | [[维生素C]]、蛋白质、锌 |

### 眼部

| 体征 | 可能缺乏的营养素 |
|------|----------------|
| 干眼病（结膜、角膜干燥） | [[维生素A]] |
| 夜盲、暗适应能力下降 | [[维生素A]] |
| Bitot斑（结膜角化斑） | [[维生素A]] |
| 眼睑炎、角膜周围充血 | [[维生素B2（核黄素）]] |
| 球结膜充血 | [[维生素B2（核黄素）]] |

### 口腔

| 体征 | 可能缺乏的营养素 |
|------|----------------|
| 口角炎（口角湿白、皲裂） | [[维生素B2（核黄素）]]、[[维生素B1（硫胺素）]]、[[铁]] |
| 口唇红肿、裂纹 | [[维生素B2（核黄素）]]、[[铁]] |
| 品红色舌（舌面紫红色） | [[维生素B2（核黄素）]] |
| 舌炎（舌乳头萎缩、舌面光滑） | [[铁]]、[[维生素B3（烟酸）]]、[[维生素B9（叶酸）]]、[[维生素B6（吡哆醇）]]、[[维生素B12（钴胺素）]] |
| 舌肿大（牛肉舌） | [[维生素B12（钴胺素）]]、[[维生素B9（叶酸）]]、铁 |
| 牙龈出血、肿胀 | [[维生素C]] |

### 指甲

| 体征 | 可能缺乏的营养素 |
|------|----------------|
| 反甲（匙状甲，指甲凹陷） | [[铁]] |
| 指甲变薄、脆裂 | 蛋白质、铁、钙 |
| 横沟（Beau线） | 蛋白质-热能营养不良 |

### 皮下组织

| 体征 | 可能缺乏的营养素 |
|------|----------------|
| 水肿（凹陷性水肿，以下肢为著） | 蛋白质-热能营养不良（PEM） |
| 水肿伴心脏扩大 | [[维生素B1（硫胺素）]]（脚气病） |
| 脂肪减少 | 热能摄入不足 |

### 肌肉骨骼系统

| 体征 | 可能缺乏的营养素 |
|------|----------------|
| 肌肉消瘦、萎缩、肌力下降 | 蛋白质-热能营养不良（PEM） |
| 弓形腿、O形腿或X形腿（儿童） | [[维生素D]]、[[钙]] |
| 骨痛、压痛 | [[维生素C]]、[[维生素D]] |
| 方颅、前囟闭合延迟（婴幼儿） | [[维生素D]]、钙 |
| 鸡胸、肋骨串珠（婴幼儿） | [[维生素D]] |
| 关节疼痛 | [[维生素C]] |

### 神经系统

| 体征 | 可能缺乏的营养素 |
|------|----------------|
| 多发性神经炎（手套/袜套样感觉异常） | [[维生素B1（硫胺素）]] |
| 腱反射减弱或消失 | [[维生素B1（硫胺素）]] |
| 意识模糊、共济失调 | [[维生素B1（硫胺素）]]（Wernicke脑病） |
| 精神症状（抑郁、记忆力减退） | [[维生素B12（钴胺素）]]、[[维生素B9（叶酸）]] |
| 味觉减退 | [[锌]] |

### 其他

| 体征 | 可能缺乏的营养素 |
|------|----------------|
| 甲状腺肿大 | [[碘]] |
| 味觉减退、食欲下降 | [[锌]] |
| 性功能减退 | [[锌]] |
| 生长发育迟缓 | 蛋白质、锌、铁、多种维生素 |
| 免疫功能低下 | 蛋白质、[[维生素A]]、[[维生素C]]、[[锌]]、[[铁]] |

## 临床检查顺序

### 坐位检查

按以下顺序依次检查：

1. **头发**：观察光泽度、稀疏程度、脱发情况
2. **面部**：面色（苍白-贫血）、鼻唇沟皮肤（脂溢性皮炎）、口角（口角炎）
3. **眼部**：结膜干燥、Bitot斑、角膜充血、暗适应询问
4. **口腔**：唇（红肿裂纹）、口角、黏膜、牙龈
5. **舌**：舌色（品红色舌）、舌乳头（萎缩/光滑/肿胀）
6. **指甲**：匙状甲、脆裂、横沟

### 卧位/全身检查

按以下顺序依次检查：

1. **全身皮肤**：干燥、毛囊角化、瘀点瘀斑、皮炎分布
2. **皮下组织**：水肿（按压胫骨前区检查凹陷性水肿）、脂肪厚度
3. **骨骼系统**：压痛、畸形（弓形腿、鸡胸等）
4. **肌肉系统**：肌肉萎缩、肌力检查
5. **神经系统**：感觉异常、腱反射、共济运动

> [!tip] 检查要点
> - 检查应在自然光线下进行，以准确观察皮肤和黏膜颜色
> - 检查前应询问病史和膳食史，有助于针对性检查
> - 体征的严重程度可按轻、中、重分级记录
> - 单一体征往往不能确诊某种营养素缺乏，需结合膳食调查和[[生化检验]]综合判断
