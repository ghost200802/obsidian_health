# 甲状腺功能检验

## 一、概述

甲状腺功能检验是通过检测血清中甲状腺激素、促甲状腺激素及相关抗体来评估甲状腺功能的检验项目。甲状腺激素参与人体的能量代谢、蛋白质合成、脂肪分解、碳水化合物代谢及生长发育调控，是维持基础代谢率的关键激素。甲状腺功能检验在[[营养评价]]中具有一定价值，[[碘]]是甲状腺激素合成的必需微量元素，甲状腺功能异常直接影响营养素代谢和能量消耗。

## 二、甲状腺激素

### 2.1 参考范围（按年龄分组）

| 项目 | 英文缩写 | 成年人参考范围 | 老年人参考范围 | 单位 |
|------|----------|----------------|----------------|------|
| 促甲状腺激素 | TSH | 0.27-4.20 | 0.50-6.00 | mIU/L |
| 总甲状腺素 | TT4 | 66.0-181.0 | 60.0-150.0 | nmol/L |
| 游离甲状腺素 | fT4 | 12.0-22.0 | 10.0-18.0 | pmol/L |
| 总三碘甲状腺原氨酸 | TT3 | 1.30-3.10 | 1.00-2.50 | nmol/L |
| 游离三碘甲状腺原氨酸 | fT3 | 3.10-6.80 | 2.50-5.50 | pmol/L |
| 反T3 | rT3 | 0.40-1.20 | 0.50-1.40 | nmol/L |

### 2.2 各项目说明

**促甲状腺激素（TSH）**
- 由垂体前叶分泌，促进甲状腺合成和分泌T3、T4
- 是评估甲状腺功能最灵敏的筛查指标
- TSH与fT4呈负反馈调节关系

**TT4 / fT4**
- TT4：总甲状腺素，包括与蛋白结合和游离两部分，受TBG浓度影响
- fT4：游离甲状腺素，不受TBG影响，是甲状腺激素的活性形式
- fT4比TT4更能准确反映甲状腺功能状态

**TT3 / fT3**
- T3是甲状腺激素的主要活性形式（活性约为T4的3-5倍）
- 约80%的T3由T4在外周组织脱碘转化而来
- fT3是评估甲状腺功能的直接指标

**rT3（反T3）**
- T4在外周组织经5-脱碘酶作用生成
- rT3无生物活性
- 增高：非甲状腺病态综合征（NTIS）、饥饿、重症疾病、低T3综合征
- 在严重[[营养不良]]时，T4向rT3转化增多 → T3降低、rT3升高，是机体节能的保护机制

## 三、甲状腺蛋白和抗体

### 3.1 参考范围

| 项目 | 英文缩写 | 参考范围 | 单位 |
|------|----------|----------|------|
| 甲状腺球蛋白 | Tg | < 55 | ng/mL |
| 甲状腺素结合球蛋白 | TBG | 12.0-28.0 | mg/L |
| 抗甲状腺球蛋白抗体 | ATg/TgAb | < 115 | IU/mL |
| 抗甲状腺过氧化物酶抗体 | TPOAb | < 34 | IU/mL |
| 促甲状腺激素受体抗体 | TRAb | < 1.75 | IU/L |

### 3.2 各项目临床意义

**甲状腺球蛋白（Tg）**
- 由甲状腺滤泡上皮细胞合成
- 甲状腺癌术后随访的肿瘤标志物
- 甲状腺全切术后Tg应接近零，若升高提示复发或转移

**甲状腺素结合球蛋白（TBG）**
- 运输T3、T4的结合蛋白
- TBG增高：妊娠、雌激素、口服避孕药 → TT3、TT4升高（假性增高）
- TBG降低：雄激素、糖皮质激素、肾病综合征、严重肝病

**抗甲状腺球蛋白抗体（ATg/TgAb）**
- 自身免疫性甲状腺疾病的标志物
- 桥本甲状腺炎阳性率约50-60%

**抗甲状腺过氧化物酶抗体（TPOAb）**
- 自身免疫性甲状腺疾病最灵敏的标志物
- 桥本甲状腺炎阳性率约90-95%
- 弥漫性毒性甲状腺肿（Graves病）阳性率约70-80%
- TPOAb阳性但甲状腺功能正常者，每年有2-4%进展为临床甲减

**促甲状腺激素受体抗体（TRAb）**
- Graves病的特征性抗体
- TRAb阳性 → 促甲状腺激素受体刺激型 → 甲亢
- 可用于Graves病诊断及疗效监测

## 四、临床意义

### 4.1 甲状腺功能异常分类

| 类型     | TSH   | fT4 | fT3 | 常见病因                        |
| ------ | ----- | --- | --- | --------------------------- |
| 临床甲亢   | 降低    | 增高  | 增高  | Graves病、毒性结节性甲状腺肿、亚急性甲状腺炎早期 |
| 亚临床甲亢  | 降低    | 正常  | 正常  | 早期甲亢、TSH瘤、外源性甲状腺激素过量        |
| 临床甲减   | 增高    | 降低  | 降低  | 桥本甲状腺炎、碘缺乏、甲状腺术后、¹³¹I治疗后    |
| 亚临床甲减  | 增高    | 正常  | 正常  | 早期桥本甲状腺炎、碘过量                |
| 低T3综合征 | 正常/降低 | 正常  | 降低  | 严重疾病、饥饿、[[营养不良]]（非甲状腺疾病）    |

### 4.2 桥本甲状腺炎

- 是自身免疫性甲状腺疾病最常见的类型
- TPOAb 90-95% 阳性，ATg 50-60% 阳性
- 早期可表现为一过性甲亢（甲状腺毒症期），最终进展为甲减
- 女性发病率远高于男性（约8:1）

## 五、营养学重点

### 5.1 碘——甲状腺激素合成的必需元素

- [[碘]]是合成甲状腺激素（T3、T4）的唯一必需微量元素
- 甲状腺激素分子中含有约65%的碘
- 成人每日碘推荐摄入量（RNI）：120 ug/d
- 安全摄入上限（UL）：600 ug/d

**碘缺乏后果**：
- 甲状腺激素合成减少 → TSH增高 → 甲状腺代偿性肿大（甲状腺肿/大脖子病）
- 严重碘缺乏 → 碘缺乏病：胎儿/新生儿脑发育障碍（克汀病）、成人甲状腺功能减退

**碘过量后果**：
- 碘过量也可诱发甲状腺功能异常（甲亢或甲减）
- Wolff-Chaikoff效应：大量碘可短暂抑制甲状腺激素合成

**食物碘来源**：
- 海产品（海带、紫菜、海鱼、贝类）是碘的最佳来源
- 加碘食盐（按标准20-30 mg/kg）
- 内陆地区、山区土壤含碘量低，容易缺碘

### 5.2 甲状腺激素与三大营养素代谢

**碳水化合物代谢**：
- 甲状腺激素促进糖原分解和糖异生 → 升高血糖
- 甲亢时空腹血糖升高，甲减时血糖偏低

**蛋白质代谢**：
- 生理剂量：促进蛋白质合成（正氮平衡）
- 过量（甲亢）：促进蛋白质分解（负氮平衡）→ 肌肉萎缩、体重下降
- 不足（甲减）：蛋白质合成减少 → 黏液性水肿

**脂肪代谢**：
- 甲状腺激素促进脂肪分解和脂肪酸氧化
- 甲亢 → 血胆固醇降低、体重下降
- 甲减 → 血胆固醇升高（LDL-C升高尤为明显）

### 5.3 甲亢时能量消耗增加与营养支持

- 甲亢时基础代谢率增高20-100%
- 每日能量消耗显著增加，蛋白质分解加速
- 出现消瘦、乏力、肌萎缩、骨质疏松等

**甲亢营养支持要点**：
- 高能量：每日能量供给 35-40 kcal/kg
- 高蛋白：每日蛋白质 1.2-1.5 g/kg，补偿蛋白质分解丢失
- 高维生素：补充B族维生素（参与能量代谢）及维生素C
- 适当补充钙和维生素D（甲亢加速骨吸收）
- 避免含碘丰富的食物（海带、紫菜、海鱼等）
- 避免刺激性食物（浓茶、咖啡、酒精）

### 5.4 甲减时的营养管理

- 基础代谢率降低，能量消耗减少
- 易发生体重增加、血脂异常
- 黏液性水肿 → 水钠潴留

**甲减营养管理要点**：
- 适当控制能量摄入，维持健康体重
- 限制脂肪和胆固醇摄入（甲减常伴高脂血症）
- 保证适量蛋白质摄入
- 补充膳食纤维预防便秘
- 适量碘摄入（桥本甲状腺炎患者酌情限碘）
- 适量运动促进代谢

## 六、体检项目选择建议

### 6.1 常规筛查

- **TSH**：甲状腺功能筛查的首选指标，灵敏度高、价格低
- 若TSH异常 → 加查fT4、fT3进一步明确

### 6.2 扩展/精准筛查

- TSH + fT4 + fT3 → 全面甲状腺功能评估
- TPOAb + ATg → 自身免疫性甲状腺疾病筛查
- TRAb → 疑似Graves病时加查

### 6.3 特殊人群推荐

- 女性（尤其育龄期/妊娠期）：TSH + fT4（妊娠期需特殊参考范围）
- 有甲状腺疾病家族史：TSH + fT4 + TPOAb
- 不明原因消瘦/体重增加：TSH + fT4 + fT3
- [[营养不良]]/重症患者：TSH + fT3 + rT3（评估低T3综合征）
- 甲状腺术后患者：TSH + fT4 + Tg
