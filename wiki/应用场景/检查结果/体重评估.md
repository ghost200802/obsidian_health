# 体重评估

## 定义

体重评估是对个体健康状况进行全面评估的过程，是制定个性化[[体重管理]]方案的基础。通过系统收集和分析个体在身体测量、人体成分、代谢指标、膳食习惯、身体活动及心理行为等方面的信息，全面了解体重相关健康风险，从而为后续干预提供科学依据。

> 依据《[[source-体重管理指导原则（2024年版）]]》，体重评估涵盖6个核心维度。

---

## 6个评估维度

### 维度一：个人史/既往史

| 评估项目 | 具体内容 | 临床意义 |
|---------|---------|---------|
| 家族史 | 肥胖、[[糖尿病]]、[[心血管疾病]]、[[高血压]]等代谢性疾病家族史 | 遗传易感性评估 |
| 减重史 | 过往减重尝试、减重方法、减重效果、体重反弹情况 | 评估减重难点和依从性 |
| 肥胖相关疾病史 | 2型糖尿病、高血压、[[血脂异常]]、睡眠呼吸暂停、[[多囊卵巢综合征|PCOS]]、[[痛风]]等 | 判断肥胖并发症严重程度 |
| 特殊用药史 | 糖皮质激素、抗精神病药、抗抑郁药等可能导致体重增加的药物 | 排除继发性体重增加因素 |

### 维度二：生活方式风险评估

| 评估领域 | 评估内容 | 评估工具/方法 |
|---------|---------|-------------|
| 饮食习惯 | 能量摄入水平、膳食结构、进餐频率、在外就餐情况、含糖饮料摄入 | [[膳食调查]]、24小时膳食回顾、食物频率问卷 |
| 运动情况 | 运动类型、频率、强度、时长、久坐时间 | 国际体力活动问卷（IPAQ） |
| 睡眠状况 | 睡眠时长、睡眠质量、是否存在睡眠呼吸暂停 | 匹兹堡睡眠质量指数（PSQI） |
| 心理状态 | 压力水平、情绪状态、是否存在情绪性进食 | 心理评估量表 |
| 吸烟饮酒 | 吸烟史、饮酒频率和量 | 生活方式问卷 |

### 维度三：体格检查/人体成分/能量消耗

| 指标 | 测量方法 | 参考标准 |
|------|---------|---------|
| BMI | 体重(kg) / 身高(m)² | 参见[[BMI与体重分类]]：正常18.5-23.9，超重24.0-27.9，肥胖≥28.0 |
| 腰围 | 脐水平周径，被测者直立、呼气末测量 | 男性 ≥ 90 cm，女性 ≥ 85 cm 为中心型肥胖 |
| 臀围 | 臀部最宽处周径 | 计算腰臀比（WHR），男性>0.9、女性>0.85为异常 |
| 体脂率 | [[人体成分分析]]（BIA法）或皮褶厚度测量 | 男性15%-20%，女性20%-25%为正常范围 |
| 内脏脂肪面积 | 生物电阻抗分析或CT/MRI | 内脏脂肪面积 ≥ 80 cm² 为异常 |
| 基础代谢率 | 间接测热法或公式（如Mifflin-St Jeor公式） | 用于计算每日能量需求 |

> 参见[[身体成分]]和[[人体成分分析]]了解更多测量原理与方法。

### 维度四：医学检查评估

| 检查类别 | 具体项目 | 临床意义 |
|---------|---------|---------|
| 血常规 | WBC、RBC、Hb、PLT | 排除贫血、感染等 |
| 尿常规 | 尿蛋白、尿糖、尿酮体 | 筛查肾脏损害、糖尿病 |
| 血糖相关 | 空腹血糖、餐后2h血糖、[[HbA1c]] | 评估糖代谢状态 |
| 血脂谱 | TC、TG、LDL-C、HDL-C | 评估[[血脂异常]]风险 |
| 肝功能 | ALT、AST、GGT | 筛查[[MAFLD营养管理|非酒精性脂肪性肝病]] |
| 肾功能 | Cr、BUN、eGFR | 评估肾脏功能 |
| 甲状腺功能 | TSH、FT3、FT4 | 排除甲状腺功能异常导致的体重变化 |
| 影像学检查 | 腹部B超 | 评估脂肪肝程度 |

> 参见[[检验报告]]、[[生化检验]]和[[营养相关检验]]了解各指标解读。

### 维度五：运动能力评估

| 评估维度 | 说明 | 常用方法 |
|---------|------|---------|
| 力量 | 肌肉力量 | 握力计测试 |
| 速度 | 有氧运动能力 | 固定自行车或跑步机测试 |
| 耐力 | 心肺耐力 | 6分钟步行试验、心肺运动试验（CPET） |
| 柔韧性 | 关节活动度 | 坐位体前屈测试 |
| 协调性 | 平衡能力 | 闭眼单脚站立测试 |
| 关节活动范围 | 运动安全性 | 关节ROM测量，排除运动禁忌 |

> 运动能力评估结果直接影响[[减重运动处方]]的制定。

### 维度六：特定人群评估要点

| 人群 | 重点评估内容 |
|------|------------|
| **儿童青少年** | 生长发育曲线、性发育状态、骨龄、心理健康 |
| **孕产妇** | 孕前体重、孕期增重曲线、胎儿发育状况、营养素水平 |
| **老年人** | 肌少症筛查（SARC-F量表）、跌倒风险、认知功能、营养素缺乏 |
| **疾病患者** | 原发病控制情况（如HbA1c、血压达标）、用药与体重的关系 |

## 评估注意事项

### 1. 综合评估

体重评估需要从多维度综合分析，不能仅依据BMI单一指标。例如，BMI正常的个体可能存在中心型肥胖或代谢异常（"正常体重肥胖"），需要结合腰围、体脂率、代谢指标等综合判断。

### 2. 个体化原则

不同年龄、性别、种族、健康状况的个体，其体重评估的侧重点和标准不同。例如：
- 老年人BMI适宜范围（22.0-26.9）高于一般成人
- 儿童青少年需使用年龄性别别BMI标准
- 孕妇有特定的体重增长标准

### 3. 定期随访

体重评估不是一次性事件，需要定期重复评估以：
- 监测体重变化趋势
- 评估干预效果
- 及时调整管理方案
- 发现新的健康风险
