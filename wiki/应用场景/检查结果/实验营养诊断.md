# 实验营养诊断

## 定义

实验营养诊断是利用生化检验和实验室指标对患者的营养状况进行客观评估的方法，是营养诊断体系中[[人体测量]]和膳食调查的重要补充。实验室指标可以早期发现营养缺乏或过量，为营养诊断和治疗效果评价提供客观依据。

## 实验诊断指标分类

### 一、蛋白质营养指标

#### 1. 血清白蛋白（Albumin, ALB）

| 项目 | 说明 |
|------|------|
| 正常范围 | 35-55 g/L |
| 半衰期 | 约20天 |
| 营养意义 | 反映体内蛋白质储备的慢性指标 |
| 临床意义 | < 30 g/L提示重度蛋白质营养不良 |

> [!warning]
> 白蛋白半衰期长（约20天），不是营养状况的敏感指标。低白蛋白更多反映炎症/疾病严重程度而非单纯的营养状态。

#### 2. 前白蛋白（Prealbumin, PA / Transthyretin）

| 项目 | 说明 |
|------|------|
| 正常范围 | 200-400 mg/L |
| 半衰期 | 约2天 |
| 营养意义 | 反映近期蛋白质营养变化的敏感指标 |
| 临床意义 | < 100 mg/L提示严重蛋白质缺乏 |

> 前白蛋白因半衰期短（约2天），是监测短期营养干预效果的优良指标。营养支持有效时，前白蛋白水平在1-2周内即可上升。

#### 3. 转铁蛋白（Transferrin, TFN）

| 项目 | 说明 |
|------|------|
| 正常范围 | 2.0-4.0 g/L |
| 半衰期 | 约8天 |
| 营养意义 | 反映蛋白质储备的中期指标 |
| 临床意义 | 受缺铁状态影响（缺铁时升高） |

#### 4. 视黄醇结合蛋白（Retinol-Binding Protein, RBP）

| 项目 | 说明 |
|------|------|
| 正常范围 | 25-70 mg/L |
| 半衰期 | 约12小时 |
| 营养意义 | 极短半衰期，反映极短期的营养变化 |
| 临床意义 | 受肾功能和维生素A状态影响 |

#### 血清蛋白半衰期比较

| 蛋白 | 半衰期 | 敏感性 | 适用场景 |
|------|--------|--------|---------|
| 白蛋白 | ~20天 | 低 | 长期营养监测 |
| 转铁蛋白 | ~8天 | 中 | 中期营养评估 |
| 前白蛋白 | ~2天 | 高 | 短期营养干预效果监测 |
| 视黄醇结合蛋白 | ~12小时 | 极高 | 极短期变化评估 |

### 二、免疫功能指标

| 指标 | 正常范围 | 营养意义 | 异常意义 |
|------|---------|---------|---------|
| 总淋巴细胞计数（TLC） | > 1.5 x 10^9/L | 反映细胞免疫功能 | < 1.0 x 10^9/L提示中度营养不良；< 0.8 x 10^9/L提示重度营养不良 |
| 皮肤迟发性超敏反应（DTH） | 硬结直径>5mm | 反映细胞介导免疫 | 无反应提示营养不良导致免疫功能受损 |

#### TLC营养评定

| TLC（x10^9/L） | 判定 |
|----------------|------|
| > 2.0 | 正常 |
| 1.2-2.0 | 轻度营养不良 |
| 0.8-1.2 | 中度营养不良 |
| < 0.8 | 重度营养不良 |

### 三、氮平衡

$$氮平衡(g/d) = 摄入氮量 - 排出氮量$$
$$排出氮量 = 24小时尿尿素氮 + 4g（非尿素氮丢失）$$

$$摄入氮量 = 蛋白质摄入量(g/d) \div 6.25$$

| 氮平衡 | 判定 | 临床意义 |
|--------|------|---------|
| 正平衡 | 摄入 > 排出 | 合成代谢为主，营养状况改善 |
| 零平衡 | 摄入 = 排出 | 维持状态 |
| 负平衡 | 摄入 < 排出 | 分解代谢为主，营养不良加重 |

> [!important]
> 氮平衡是评价蛋白质营养状况和营养支持效果的**经典指标**。负氮平衡持续存在提示营养不良加重或营养支持不足。

### 四、能量代谢指标

#### 基础代谢率（BMR）和静息代谢率（RER）

| 指标 | 定义 | 测量方法 |
|------|------|---------|
| BMR | 维持生命最低能量消耗 | 间接测热法（金标准） |
| RER | 安静静息状态下的能量消耗 | 间接测热法或公式估算 |

#### 常用能量估算公式

| 公式 | 适用人群 | 计算方法 |
|------|---------|---------|
| Harris-Benedict公式 | 一般成人 | BMR = 男性 66.5 + 13.75 x W + 5.003 x H - 6.775 x A；女性 655.1 + 9.563 x W + 1.850 x H - 4.676 x A |
| Harris-Benedict改良公式 | 肥胖患者 | 使用理想体重替代实际体重 |
| Mifflin-St Jeor公式 | 肥胖人群（更准确） | BMR = 男性 10 x W + 6.25 x H - 5 x A + 5；女性 10 x W + 6.25 x H - 5 x A - 161 |

> W = 体重(kg), H = 身高(cm), A = 年龄(岁)

#### 应激系数

| 应激程度 | 应激系数 | 适用疾病 |
|---------|---------|---------|
| 无应激 | 1.0 | 普通患者 |
| 轻度应激 | 1.1-1.2 | 普通手术 |
| 中度应激 | 1.2-1.4 | 感染、创伤 |
| 重度应激 | 1.4-1.6 | 大手术、多发性骨折 |
| 极重度应激 | 1.6-2.0 | 烧伤（体表面积>20%） |

$$TEE = BMR \times 应激系数 \times 活动系数$$

### 五、微量元素与维生素

#### 常用微量元素指标

| 微量元素 | 检测指标 | 正常范围 | 营养意义 |
|---------|---------|---------|---------|
| 铁 | 血清铁、铁蛋白、TIBC | 铁蛋白：男性15-200ug/L，女性12-150ug/L | 缺铁性贫血 |
| 锌 | 血清锌 | 11.5-22.5 umol/L | 味觉障碍、免疫功能下降 |
| 铜 | 血清铜 | 11.0-22.0 umol/L | 铜缺乏可致贫血、中性粒细胞减少 |
| 硒 | 血清硒 | 58-234 ug/L | 抗氧化、免疫功能 |
| 铬 | 血清铬 | 0.05-0.5 ug/mL | 糖耐量异常 |

#### 常用维生素指标

| 维生素 | 检测指标 | 正常范围 | 营养意义 |
|--------|---------|---------|---------|
| 维生素A | 血清视黄醇 | 0.3-0.7 umol/L | 夜盲症、免疫功能 |
| 维生素D | 25-(OH)D | >= 50 nmol/L（充足） | 骨代谢、免疫调节 |
| 维生素E | 血清alpha-生育酚 | 12-46 umol/L | 抗氧化 |
| 维生素B12 | 血清维生素B12 | 148-664 pmol/L | 巨幼细胞贫血 |
| 叶酸 | 血清叶酸 | > 6.8 nmol/L | 巨幼细胞贫血 |
| 维生素C | 血浆抗坏血酸 | 28.4-79.6 umol/L | 坏血病 |
| 维生素K | 凝血酶原时间 | 11-15秒 | 凝血功能 |

## 实验诊断指标的临床应用

### 营养不良诊断中的价值

| 指标类别 | 在GLIM诊断中的作用 | 局限 |
|---------|-------------------|------|
| 血清蛋白 | 辅助评估，不作为独立诊断标准 | 受炎症、肝肾功能影响 |
| 免疫指标 | 辅助评估营养相关免疫功能 | 受疾病、药物影响 |
| 氮平衡 | 评价蛋白质营养和治疗效果 | 需要准确收集24小时尿液 |
| 能量代谢 | 确定能量需求 | 公式估算准确性有限 |

### 营养支持效果监测

| 监测指标 | 监测频率 | 预期变化 |
|---------|---------|---------|
| 前白蛋白 | 每周1-2次 | 有效营养支持后1-2周内上升 |
| 白蛋白 | 每周1次 | 有效支持后2-3周开始上升 |
| TLC | 每周1次 | 营养改善后逐渐恢复正常 |
| 氮平衡 | 每1-2周1次 | 有效支持后转为正平衡 |
| 体重 | 每周2-3次 | 有效支持后体重趋于稳定或上升 |

## 实验诊断的局限性

> [!warning]
> 1. **炎症影响**：急性期反应蛋白（白蛋白、前白蛋白等）在炎症状态下下降，**不能单纯以低蛋白水平诊断营养不良**
> 2. **肝肾影响**：肝肾功能异常直接影响蛋白质代谢指标
> 3. **脱水/水肿**：影响血液浓缩程度
> 4. **药物影响**：皮质激素等药物影响蛋白质代谢
> 5. **单一指标不足**：不能仅凭单一实验室指标诊断营养不良
