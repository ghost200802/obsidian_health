# PG-SGA（患者参与的主观整体评估）

## 定义

PG-SGA（Patient-Generated Subjective Global Assessment）是Ottery于1994年在SGA基础上改良开发的营养评估工具，专门适用于**肿瘤患者**的营养评估。其最大特点是**患者参与**自我评估部分，配合医务人员评估部分共同完成。

## 评估结构

PG-SGA由两大部分组成：**患者自评部分**（Box 1-4）和**医务人员评估部分**（Box 5-7）。

---

### 患者自评部分（Box 1-4）

#### Box 1：体重

| 项目 | 评分 |
|------|------|
| 近1个月体重变化 | 记录具体数值 |
| 近2周体重变化 | 增加/无变化/减少 |
| 近6个月体重变化 | 记录具体数值 |

#### Box 2：膳食摄入

| 评估项目 | 评分要点 |
|---------|---------|
| 当前食物摄入量 | 与正常摄入量相比 |
| 持续时间 | 食物摄入变化持续的时间 |
| 进食类型 | 正常食物、较少固体食物、仅流质、仅营养补充、几乎不进食 |

#### Box 3：营养相关症状

患者自我评估以下症状（有症状为1分，严重影响可记为2-3分）：
- 食欲不振
- 口腔/咽喉疼痛
- 吞咽困难
- 恶心
- 呕吐
- 腹泻
- 便秘
- 口干
- 味觉改变
- 嗅觉改变
- 疼痛
- 其他症状

#### Box 4：活动和功能

| 功能水平 | 评分 | 描述 |
|---------|------|------|
| 无症状 | 0分 | 正常活动，无限制 |
| 症状轻微 | 1分 | 有症状但可正常活动 |
| 活动受限 | 2分 | 不卧床，但正常活动受限 |
| 卧床 | 3分 | 白天大部分时间卧床 |
| 完全卧床 | 4分 | 几乎完全卧床 |

---

### 医务人员评估部分（Box 5-7）

#### Box 5：代谢需求（疾病相关）

| 疾病分期 | 评分 |
|---------|------|
| 无应激 | 0分 |
| 轻度应激 | 1分（如择期手术） |
| 中度应激 | 2分（如化疗、放疗） |
| 重度应激 | 3分（如骨髓移植、重症感染） |

#### Box 6：体格检查

| 检查项目 | A（正常） | B（中度） | C（重度） |
|---------|----------|----------|----------|
| 皮下脂肪（三头肌、肋下） | 正常 | 轻度减少 | 明显减少 |
| 肌肉萎缩（颞肌、锁骨、肩胛、大腿） | 正常 | 轻度萎缩 | 明显萎缩 |
| 水肿/腹水 | 无 | 轻度 | 中-重度 |

#### Box 7：PG-SGA总评分与定性评定

##### 计分方法

将Box 1-6的评分相加，得到PG-SGA总分（0-35分）。

##### 评分与干预建议

| PG-SGA总分 | 评级 | 营养状况 | 干预建议 |
|-----------|------|---------|---------|
| 0-1分 | PG-SGA A | 营养良好 | 当前营养治疗无需调整 |
| 2-3分 | PG-SGA B | 中度营养不良/可疑营养不良 | 由营养师和家属进行营养指导 |
| 4-8分 | PG-SGA B | 中度营养不良 | 需要营养干预，结合症状治疗 |
| >= 9分 | PG-SGA C | 重度营养不良 | 急需营养治疗，可能需要药物辅助 |

## PG-SGA评分表

| 评分项 | 分值范围 | 说明 |
|-------|---------|------|
| Box 1 体重 | 0-4分 | 根据体重变化程度 |
| Box 2 膳食摄入 | 0-4分 | 根据摄入减少程度 |
| Box 3 症状 | 0-12分 | 每个症状0-3分 |
| Box 4 活动功能 | 0-3分 | 根据功能受限程度 |
| Box 5 代谢需求 | 0-3分 | 根据疾病应激程度 |
| Box 6 体格检查 | 0-3分 | 根据体检发现 |
| **总计** | **0-35分** | |

## PG-SGA的临床应用

### 使用场景
- 肿瘤患者入院营养评估
- 化疗/放疗期间营养监测
- 肿瘤患者围术期营养评估
- 肿瘤患者长期随访

### 评估频率
- 初次评估：入院时
- 随访评估：化疗/放疗周期内每2-4周一次
- 出院评估：出院前

### 优势
- 患者参与自评，提高了依从性和准确性
- 评分量化，便于追踪和比较
- 包含症状评估，可指导对症治疗
- 专门针对肿瘤患者的代谢特点设计
- 国际抗癌联盟（UICC）推荐

### 局限
- 需要患者配合完成自评部分
- 评估耗时较SGA更长
- 需要一定培训才能准确操作
