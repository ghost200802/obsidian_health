# 妊娠期高血压疾病营养

## 定义

妊娠期高血压疾病（Hypertensive Disorders of Pregnancy, HDP）是妊娠与血压升高并存的一组疾病，是孕产妇和围产儿死亡的主要原因之一。分为5类：
1. 妊娠期高血压（Gestational Hypertension）
2. 子痫前期（Preeclampsia）
3. 子痫（Eclampsia）
4. 慢性高血压并发子痫前期
5. 妊娠合并慢性高血压

## 代谢特征

### 脂肪代谢

- TG和LDL升高，HDL-C下降
- LDL沉积于血管壁 → 血管弹性降低 → 血压升高
- ω-3脂肪酸通过减少FFA可用性、增加磷脂合成、降低TG合成酶活性来调节氧化应激和炎症，对预防子痫前期有益

### 蛋白质

- 低蛋白质性营养不良是主要诱发因素
- 甲硫氨酸及牛磺酸有降血压作用：抑制RAAS系统、减少血管紧张素II生成、提高NO水平

### 碳水化合物

- 过量摄入 → 能量过剩 → 脂肪堆积 → 血压升高
- 膳食纤维降低血脂血压、抗炎

### 钙与维生素D

- 低钙摄入 → 刺激PTH和肾素释放 → 血管平滑肌细胞内钙浓度增加 → 血管收缩
- 钙减少子宫动脉和脐动脉血管收缩，影响子宫胎盘血流
- 维生素D缺乏增加HDP患病风险

### 钠

- 高钠 → 动脉壁对血管收缩物质致敏 → 血管收缩
- 管壁结合钠增加 → 吸收水分增加 → 管腔缩小
- 血管平滑肌细胞对钙离子通透性增加 → 收缩增强

## 营养治疗

### 能量

- 避免能量过剩，参照妊娠期高血糖能量推荐
- 孕前超重肥胖者适当减少，但一般不低于1600-1800 kcal/d

### 碳水化合物

- 占总能量50%-60%
- 膳食纤维25-30g/d
- 选择全谷类替代精白米面
- 避免过量蔗糖、果糖、代糖

### 蛋白质

- 适当增加蛋白质摄入（常出现蛋白尿致低蛋白血症）
- 每日最低70g
- 妊早中晚期分别增加0g、15g、30g
- 合并肾功能不全时适当限制
- 建议每周至少食用2次深海鱼

### 脂肪

- 占总能量25%以下
- SFA 6%-10%，MUFA和PUFA各8%-10%
- 胆固醇<300mg/d
- 烹调选用植物油（茶油、橄榄油、花生油）
- 避免动物油脂、浓肉汤、动物内脏、全脂乳制品

### 钠

| 高血压程度 | 每日食盐 |
|-----------|---------|
| 轻度高血压/无水肿 | 3-5g |
| 中度高血压 | 1-2g（折合酱油5-10ml） |
| 重度高血压 | 严格无盐膳食 |

- 少吃或避免腌渍食品（咸菜、咸鱼、咸肉、咸蛋、酱菜、腐乳等）
- 避免含钠高的调味剂加工食品

### 钙剂与维生素D

- 低钙摄入（<600mg/d）者建议补钙每日至少1.0g
- 主要来源：乳和乳制品、虾皮、鱼、海带、芝麻酱
- 维生素D推荐摄入量10μg/d，建议筛查维生素D水平
