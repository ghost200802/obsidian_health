# PEM（蛋白质-能量营养不良）

## 定义

蛋白质-能量营养不良（Protein-Energy Malnutrition, PEM）是由于蛋白质和能量摄入不足或吸收利用障碍引起的一种营养缺乏病，是儿童最常见的营养问题之一。

## 分型

| 类型 | 特征 |
|------|------|
| 消瘦型（marasmus） | 以能量严重不足为主，明显消瘦 |
| 恶性营养不良（kwashiorkor） | 以蛋白质严重缺乏为主，水肿突出 |
| 混合型（marasmic kwashiorkor） | 两者兼有 |
| 消瘦型肥胖 | 体重正常但体脂高、瘦体重低 |
| 营养不良性矮小 | 身高低于标准，体重可正常 |

## 营养治疗原则

### 能量
- 轻度：40-55 kcal/(kg·d)起始
- 逐步增至：120-170 kcal/(kg·d)
- 恢复期可超过同龄正常需要量

### 蛋白质
- 轻度：1.5 g/(kg·d)
- 中重度：2.0-3.0 g/(kg·d)
- 严重水肿：3.0-4.5 g/(kg·d)

### 补充原则
- 小量开始、逐步增加
- 首先纠正水、电解质紊乱
- 补充维生素和微量元素
- 食物从流质→半流质→软食→普食过渡
