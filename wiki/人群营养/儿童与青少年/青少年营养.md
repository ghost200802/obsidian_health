# 青少年营养

## 定义

青少年营养是指青春期（12-18岁）的特殊营养需求和管理。青春期是人生第二个生长高峰，体格快速发育、性成熟，对能量和营养素的需求达到一生中的最高峰之一。

## 青春期营养需求特点

### 生长发育高峰

- 身高快速增长，年均增长5-7cm（高峰期可达10-12cm/年）
- 体重显著增加
- [[蛋白质]]和能量需求大幅增加
- 骨量积累的关键时期（约积累一生中40%-60%的骨量）

### 营养素需求

| 营养素 | 特点 |
|--------|------|
| 能量 | 需求量接近或超过成人，男性约2400-2800 kcal/d，女性约2200-2400 kcal/d |
| [[蛋白质]] | 用于肌肉和骨骼生长，需求量1.0-1.2 g/(kg·d) |
| 钙 | 骨骼矿化关键期，需求量1000-1200 mg/d，达到一生中的最高峰 |
| [[铁]] | 女性因月经来潮需求量增加（20 mg/d），男性12-16 mg/d |
| 锌 | 生长发育和性成熟所需，男12-15 mg/d，女9-11.5 mg/d |
| [[维生素]] | 各类维生素需求增加，尤其[[维生素]]A、D、C、B族 |

### 钙铁需求高峰

- **钙**：青春期骨量积累速度最快，充足的钙和[[维生素]]D摄入对峰值骨量至关重要。奶及奶制品、深绿叶蔬菜、豆腐是优质钙源
- **[[铁]]**：男性因肌肉量快速增加需要更多铁；女性因月经来潮导致铁丢失，[[铁]]缺乏风险显著增加，需特别关注

## 饮食行为特点与问题

### 1. 饮食不规律
- 不吃早餐、暴饮暴食
- 外卖和快餐过多
- 正餐摄入不足，零食摄入过多

### 2. 零食饮料过多
- 含糖饮料摄入过量
- 高油高盐零食
- 替代正餐导致营养不均衡

### 3. 盲目节食
- 追求瘦身材而过度节食
- 导致能量-蛋白质营养不良
- 影响生长发育和月经来潮
- 需参照[[BMI与体重分类]]进行合理体重管理

### 4. 营养素缺乏
- [[铁]]缺乏（尤其女性青少年）
- 钙摄入不足
- [[维生素]]缺乏
