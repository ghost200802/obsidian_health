# 妊娠期高血糖营养

## 定义

妊娠期高血糖包括：
- **PGDM**（孕前糖尿病合并妊娠）：孕前糖尿病基础上合并妊娠
- **GDM**（妊娠期糖尿病）：孕前糖代谢正常，妊娠期发生的糖代谢异常

> 妊娠期高血糖妇女中90%以上为GDM。GDM患者糖代谢异常大多于产后恢复正常，但将来患T2DM机会增加。

## 代谢特征

- **妊娠早中期**：血糖随妊娠进展降低约10%（胎儿获取葡萄糖增加；肾排糖增加；雌激素和孕激素增加葡萄糖利用）
- **妊娠中晚期**：拮抗胰岛素样物质增加（TNF、瘦素、胎盘催乳素、雌激素、孕酮），胰岛素敏感性随孕周下降
- **胎盘催乳素**是引起GDM患者胰岛素抵抗增加的主要激素

## 营养治疗

### 治疗目标

- 血糖控制目标：
  - 餐前及FPG < 5.3 mmol/L
  - 餐后1h < 7.8 mmol/L 或餐后2h < 6.7 mmol/L
  - 避免夜间血糖 < 3.3 mmol/L
- HbA1c < 6%（无低血糖风险者）；有低血糖倾向可放宽至7%
- 医学营养治疗可使80%-90%的GDM达到理想血糖目标

### 能量

| 阶段 | 能量推荐 |
|------|---------|
| 妊娠早期 | 不低于1600 kcal/d，不增加食物量 |
| 妊娠中期 | 以往基础上平均每日增加250 kcal |
| 妊娠晚期 | 平均每日增加400 kcal，1800-2200 kcal/d为宜 |
| 多胎妊娠 | 单胎基础上每日增加200-300 kcal |
| 孕前超重肥胖 | 不低于1600-1800 kcal/d，不建议减重 |

> 过度限制能量（<1500 kcal/d）易出现酮症

### 碳水化合物

- 供能比50%-60%
- 每餐主食（生重）不低于50-75g
- 优先选择低GI食物（红薯、糯玉米等除外）
- 每日碳水摄入不低于175g（折合粮谷类约200g）
- 混合膳食使糖消化吸收减慢，有利于控制血糖
- 使用胰岛素治疗者保证碳水摄入避免低血糖

### 蛋白质

- 每日不低于70g，占总能量20%-25%
- 妊娠早期与孕前相同
- 妊娠中期增加15g
- 妊娠晚期增加30g
- 过多可引起蛋白尿，增加胎儿肾溶质负荷

### 脂肪

- 占总能量25%-30%
- SFA不超过7%，适当提高MUFA（橄榄油、山茶油，每天约20-25ml烹调油）
- 限制或不摄入反式脂肪酸
- 注意隐形油脂（坚果纳入总油脂）

### 膳食纤维

- 推荐每日25-30g
- 富含膳食纤维食物：新鲜蔬菜、水果、藻类、粗杂粮

### 维生素和矿物质

- 补充叶酸、镁、维生素D、锌有助于降低FPG和IR，降低GDM风险
- 孕期铁、叶酸、维生素D需要量较孕前增加1倍
- 钙、磷、维生素B1、B6增加33%-50%
- 食物摄入不足时选择营养素补充剂

### 餐次安排

- 少量多餐、定时定量
- 早中晚三餐：10%-15%、30%、30%
- 加餐：每次5%-10%
