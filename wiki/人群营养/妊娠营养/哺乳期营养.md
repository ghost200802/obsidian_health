# 哺乳期营养

## 定义

哺乳期营养是指乳母（lactating women）在哺乳期间为补偿妊娠分娩损耗、促进身体恢复、保证乳汁正常分泌和婴儿健康成长所需的合理营养供给。

## 泌乳生理

### 乳汁分泌量
- 产后6个月内平均每天约**750ml**
- 后6个月降至约**600ml**
- 乳汁含能量约**70kcal/100ml**

### 泌乳反射与射乳反射
- **泌乳反射**：催乳素→腺泡上皮细胞→合成分泌乳汁（受婴儿吸吮次数调节）
- **射乳反射**：催产素→肌上皮细胞→乳汁排出（受乳母心理因素影响）
- 乳母抑郁、紧张、疼痛→抑制射乳反射→乳汁排出减少
- 产后抑郁（PPD）及焦虑是泌乳的主要危险因素

### 影响乳汁分泌的因素
1. 内分泌因素（催乳素、催产素）
2. 乳母营养状况
3. 心理因素（自信心与母乳喂养程度及持续时间呈显著正相关）
4. 婴儿吸吮频率

## 乳母营养需求（RNI增加值）

### 能量
- 额外+500 kcal/d（其中最好100kcal来自蛋白质）
- 妊娠期贮存脂肪每天提供约165kcal
- 衡量标准：泌乳量使婴儿饱足 + 母亲体重逐步恢复

### 蛋白质
- 额外+25 g/d
- 乳汁蛋白质含量约12g/L
- 每天泌乳750ml→需蛋白质12.8-14.6g（转换效率70%）
- 建议优质蛋白占1/3以上

### 脂类
- 脂肪供能比20%-30%
- 母乳脂肪酸含量和组成与乳母膳食脂肪直接相关
- 鼓励增加n-3多不饱和脂肪酸（DHA）摄入
- 母乳中SFA:MUFA:PUFA约为4:3:2

### 矿物质

| 营养素 | RNI增加值 | 说明 |
|--------|----------|------|
| 钙 | +200 mg/d | AI 1000-1200mg/d；乳汁钙约30-34mg/100ml |
| 铁 | +4 mg/d | 乳汁铁含量较低(0.1-0.2mg/100ml) |
| 锌 | +4.5 mg/d | RNI 12mg/d |
| 碘 | +120 μg/d | RNI 240μg/d |
| 硒 | +15 μg/d | RNI 65μg/d |

### 维生素

| 营养素 | RNI | 说明 |
|--------|-----|------|
| 维生素A | 1300 μg RAE/d | 影响乳汁VA含量，过量不按比例增加 |
| 维生素D | 10 μg/d | 几乎不能通过乳腺→母乳VD含量低→婴儿需额外补充 |
| 维生素E | 14 mg α-TE/d | 可促进乳汁分泌 |
| 维生素B1 | 1.8 mg/d | 改善食欲、促进乳汁分泌；缺乏→婴儿脚气病 |
| 维生素B2 | 1.4 mg/d | 与膳食摄入直接相关 |
| 烟酸 | 18 mg/d | - |
| 维生素C | 150 mg/d | 乳汁中VC含量直接受膳食影响 |
| 叶酸 | 550 μg DFE/d | - |

## 乳母营养对乳汁的影响

### 泌乳量
- 营养不良→泌乳量明显减少（严重营养不良者泌乳量仅正常1/3-1/2）
- 营养改善后泌乳量可恢复
- 营养满足需要后，单纯增加营养素不能提高泌乳量

### 乳汁成分
- **蛋白质**：与乳母摄入量显著关联
- **脂肪**：种类和含量与膳食脂肪直接相关
- **钙**：乳汁钙含量相对恒定（母体代偿机制），但长期钙摄入不足影响母体骨骼健康
- **脂溶性维生素**：VA与膳食相关；VD、VK几乎不受影响
- **水溶性维生素**：VB1、VB2、VB6、烟酸、VC含量直接受膳食影响

## 乳母膳食要点

### 食物选择建议
- 每日至少摄入：粮谷类400-500g、牛奶400-500ml、蛋类100-150g、肉类100-200g、豆类及豆制品50-100g、蔬菜500g（绿叶菜占1/2以上）、水果200g
- 增加海产品摄入（补碘、补DHA）
- 多喝汤水（促进乳汁分泌）
- 避免辛辣刺激食物和含酒精饮料

### 促进泌乳的食物
- 传统推荐：鲫鱼汤、猪蹄汤、鸡汤
- 增加蛋白质和水分摄入

### 心理调适
- 保持积极乐观的心态
- 充分的睡眠和休息
- 家庭支持和社会支持
- 信心与母乳喂养程度呈正相关
