# 航海潜水人员营养

> 内容来源：《实用营养师手册》第45章 特殊条件人群营养——航海潜水人员

## 航行对营养的影响

### 消化机能变化

航海环境下，船员消化机能经历四个阶段的适应性变化：

| 阶段 | 时间 | 消化机能表现 |
|------|------|-------------|
| 不适应期 | 出航初期（约1-2周） | 食欲明显减退、恶心、胃肠不适 |
| 适应期 | 出航中期 | 消化功能逐步恢复适应 |
| 脱适应期 | 返航前或长期航行后期 | 再次出现食欲减退、消化功能下降 |
| 再适应期 | 返航后 | 逐渐恢复正常消化功能 |

### 常见消化问题

- **食欲减退**：因晕船、环境单调、情绪因素导致食欲下降
- **胃肠胀气**：船体晃动影响胃肠蠕动，加之气压变化，易产生腹胀
- **便秘**：活动空间有限、运动量减少、饮食结构变化、精神紧张等因素综合作用
- **胃酸分泌异常**：部分船员出现胃酸增多或减少

### 营养代谢改变

长期航海环境下，船员体内出现明显的代谢变化：

- **血糖升高**：可能与应激状态、活动量减少有关
- **血清蛋白增加**：反映体内代谢状态改变
- **血液pH降低**：可能与饮食结构、环境因素有关
- **钙磷代谢改变**：长期缺乏日照、[[维生素D]]合成减少，影响钙磷代谢

### 维生素缺乏

航海人员维生素缺乏问题尤为突出：

- **55%-66%的船员存在维生素缺乏**，以[[维生素C]]、[[维生素B1（硫胺素）]]、[[维生素B2（核黄素）]]缺乏最为常见
- **高温出汗**导致水溶性维生素大量流失（[[维生素C]]、[[维生素B1（硫胺素）]]、[[维生素B2（核黄素）]]等）
- 新鲜蔬菜水果供应不足是主要原因
- 潜水作业时高压环境加速维生素消耗

## 潜水作业对营养的特殊要求

### 高压环境影响

- 高压环境下氮气麻醉可影响神经系统功能
- 氧分压过高可导致氧中毒，需合理控制
- 体内气体溶解度增加，减压不当可导致减压病

### 营养需求特点

- 能量消耗增加（尤其在寒冷水域作业时）
- [[维生素B1（硫胺素）]]需求增加，参与高压环境下能量代谢
- [[维生素C]]需求增加，对抗高压环境下的氧化应激
- 蛋白质需求适当增加，支持组织修复
- 注意[[水盐代谢]]平衡，防止脱水

## 营养供给标准

### 航海人员营养供给建议

| 营养素 | 供给要点 |
|--------|----------|
| 能量 | 根据劳动强度确定，一般10.9-14.6MJ/d（2600-3500kcal/d） |
| 蛋白质 | 占总能量12%-15%，重视优质蛋白供应 |
| 脂肪 | 占总能量20%-30%，不宜过高 |
| 碳水化合物 | 占总能量55%-65% |
| [[维生素B1（硫胺素）]] | 注意补充，预防脚气病 |
| [[维生素B2（核黄素）]] | 注意补充，预防口角炎、舌炎 |
| [[维生素C]] | 适当增加供给，预防坏血病 |
| [[维生素D]] | 注意补充，因缺乏日照 |
| 钙 | 注意补充，预防骨质疏松 |

### 食物供应保障要点

- 保证新鲜蔬菜水果的定期供应和妥善储存
- 配备多种维生素制剂作为营养补充
- 注意饮水卫生和安全
- 合理安排膳食，改善食欲
- 食品储存需防潮、防霉、防变质
