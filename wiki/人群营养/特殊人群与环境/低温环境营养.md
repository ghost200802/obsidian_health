# 低温环境营养

## 定义

低温环境（low temperature environment）是人体所处的环境温度低于人体舒适温度范围、机体感觉寒冷的环境。目前以10度作为低温环境的温度界限。低温环境一般可分为三类：

- **地区低温**：北方、高原地区
- **季节低温**：冬季野外
- **职业低温**：水下作业和冷库作业

环境温度-20度以下时，体热大量散失，可引起能量不足相关的营养缺乏病、冷损伤及情绪心理损害，还可能诱发呼吸道疾病和心血管疾病。

## 生理功能变化

### 热平衡维持

- 深部温度（核心温度）一般35-39.5度
- 低温下深部温度分布区域缩小，集中在头部和内脏
- 冷诱导产热（CIT）：成年人主要来源是肌肉颤抖产热和非颤抖性产热
- 颤抖频率9-11次/min，代谢率增加4-5倍
- 非颤抖性产热以棕色脂肪组织产能量最多

### 基础代谢

- 基础代谢率增加10%-30%
- 细小动脉收缩，血液流动缓慢
- 血中儿茶酚胺浓度增高，心输出量增多、血压上升、心率加快

### 消化系统

- 胃酸分泌亢进，胃排空减慢
- 食物在胃内消化较充分
- 摄食和体重增加

### 神经系统

- 皮肤感觉异常、肌肉收缩力减弱
- 神经-肌肉协调性和灵活性降低

### 内分泌系统

- 下丘脑释放促甲状腺激素释放激素，促进甲状腺激素合成与释放
- 去甲肾上腺素和肾上腺素等冷应激激素分泌增强

## 营养代谢特点

### 能量和供能营养素

- 总能量需求增加5%-25%
- 我国北方男性和女性能量需要量比南方居民平均增加3%-5%和2%-4%
- 能量来源从以碳水化合物为主逐渐转变为以脂肪和蛋白质供能为主
- 适量健康脂肪有利于满足能量需求
- 机体对BCAA利用增强
- 摄入甲硫氨酸和酪氨酸可提高耐寒能力

### 维生素、矿物质和水

- 维生素消耗量明显增加（维生素C、维生素A、B族维生素）
- 维生素D合成减少，寒冷地区冬季缺乏率超过50%
- 血中锌、镁、钙和钠含量下降
- 肾脏泌尿作用增强，水需要量增加

## 能量和营养素供给量

| 能量和营养素 | 供给量 | 备注 |
|------------|--------|------|
| 能量 | 提高10%-15% | 多数低温从业者膳食能量不足 |
| 蛋白质 | 供能比13%-15% | 50%应为优质蛋白质 |
| 碳水化合物 | 供能比45%-50% | 每日至少400g以维持糖原储备 |
| 脂肪 | 供能比25%-35% | 能量57kcal/kg时高脂肪膳食是必然选择；7-14天高脂肪膳食不引起心血管损害 |
| 矿物质 | 最高补充食盐15-20g/d | 增加新鲜果蔬和乳制品，关注钙和钠 |
| 维生素 | 补充量高出30%-50% | 抗氧化维生素与脂肪有协同耐寒作用；B族维生素参与产热过程 |
| 水 | 1.5ml/kcal | -20度时24小时呼出水分高达1000ml，有脱水风险 |

## 特殊评估

1. **生理适应能力**：基础代谢率、棕色脂肪组织活性、心血管系统响应
2. **营养状况**：体重变化、身体脂肪含量、肌肉质量
3. **健康状况**：心血管疾病、呼吸道感染、免疫功能
4. **适应策略**：穿着、居住环境、锻炼习惯

## 营养建议

### 饮食保障

- 首要提供能量高、重量轻、体积小、易携带的耐寒膳食，最好可直接入口食用
- 尽量利用当地食物资源，拓展必需食物资源
- 充分利用强化食品（常见强化剂：[[维生素A]]、[[维生素C]]、[[钙]]、铁、钠）
- 餐次安排：早餐25%、上午加餐15%、午餐35%、晚餐25%
- 供应热食以保温取暖

### 饮水保障

- 每日摄入3-8L液体
- 含200mmol/L葡萄糖并强化矿物质和水溶性维生素的饮液
- 备有小型加热炉
- 膳食蛋白质不宜过多，避免多尿

### 提高耐寒能力的营养素和生物活性物质

- 甲硫氨酸、亮氨酸、共轭亚油酸、ω-3长链脂肪酸、[[维生素C]]、[[维生素E]]能诱导白色脂肪组织转化为棕色脂肪组织
- 辣椒素、儿茶素能增加产能量
- [[维生素C]]增加骨骼肌收缩力，促进颤抖产热，提升免疫力
