# 特殊职业环境营养

## 定义

特殊职业环境作业人员的健康损害及其预防属于职业卫生范畴。从营养学角度预防和降低健康损害只是众多预防措施中的一个方面。常见特殊职业环境包括航空、航天、航海以及接触化学毒物作业等。

## 一、航空人员

### 健康损害

- 处于高度精神紧张状态
- 受高空缺氧、低气压、加速度、噪声、振动、电离辐射等因素影响
- 消化系统异常：高空胃肠胀气、腹痛、溃疡病、慢性胃炎
- 心血管系统异常：高脂血症和高血压
- 空腹飞行易出现低血糖，餐后立即飞行升高血甘油三酯和胆固醇

### 营养代谢特点

- 能量代谢增高
- 急性缺氧时蛋白质分解增加，易发生负氮平衡

### 营养供给量

| 营养素 | 供能比/推荐量 |
|--------|-------------|
| 蛋白质 | 12%-15%（动物性蛋白质占30%-50%） |
| 脂肪 | 20%-30%（动物性脂肪不超过脂肪总量50%） |
| 碳水化合物 | 55%-65% |

### 膳食制度

- 禁止空腹或饱腹飞行
- 飞行日为四餐制，不飞行日为三餐制
- 禁食不易消化及含膳食纤维多的食物（甘薯、粗杂粮、干豆类、萝卜、黄豆芽、芹菜、卷心菜、黄豆）
- 禁饮含气饮料
- 禁止飞行前摄入含单糖或双糖丰富的膳食（蔗糖供能不超过总能量10%）
- 每日胆固醇<300mg
- [[维生素A]]至少1/3来自动物性食品
- **夜间飞行**：[[维生素A]]不少于3000μgRAE/d；烟酸、[[维生素B1（硫胺素）]]、[[维生素B2（核黄素）]]和[[维生素C]]参与夜视视网膜化学反应，应及时补充
- 夜餐以半流质为宜，蛋白质不宜过高
- 特殊飞行条件下补充[[维生素B6（吡哆醇）]]（与前庭器官敏感性密切相关）

## 二、航天员

### 健康损害

- 失重（微重力）环境严重影响生理功能和营养代谢
- 短期飞行：机体水分丢失、体液量及分布改变（液体向头部转移）、贫血（红细胞减少）
- 长期飞行：肌肉萎缩、骨质丢失、体脂消耗
- 胃肠道细菌数量增加，厌氧菌和双歧杆菌等屏障菌减少，胃肠微生态紊乱

### 营养代谢特点

- 肌肉损失最多的是抗重力肌肉（慢动肌纤维）
- 补充高水平亮氨酸能抑制蛋白质分解，促进蛋白质合成
- 钙和磷从尿及粪便排出增多，易形成肾结石
- 骨质丢失以下肢和脊柱等承重骨为主
- 补充膳食[[维生素K]]有助于对抗骨质丢失
- 高钙和[[维生素D]]膳食未能有效延缓骨质疏松进展

### 营养供给量

| 营养素 | 推荐量 |
|--------|--------|
| 蛋白质 | 供能比12%-15% |
| 脂肪 | 供能比30%以内，注意必需脂肪酸比例 |
| 碳水化合物 | 供能比52%-55%，单糖和双糖<10%，膳食纤维15-20g/d |
| 水 | 2500-3000ml/d（每消耗1kcal补充1ml水） |

### 膳食特点与要求

1. 航天膳食全部飞行前按计划配制，重量轻、体积小
2. 动物性食品加工成软、硬罐头类；果蔬类冷冻干燥或即食；脱水食品不超过50%
3. 食品包装方便在失重环境中应用，一口大小，无可流动汤汁
4. 按个人选择食谱和口味配餐，提供调味品
5. 食品安全可靠，能经受振动和加速度等冲击

### 三个营养阶段

1. **飞行前**：调整膳食摄入，减少对失重适应的过渡时间
2. **飞行中**：根据失重对机体生理影响修正营养素供给量
3. **飞行后**：改进膳食，加快损伤组织和营养储备恢复
4. **返回地面前**：补充含盐饮料，增加细胞外液和循环血量，缩短对地面重力环境的适应时间

## 三、航海人员

### 健康损害

- 船摇摆颠簸致晕船症状（恶心、食欲缺乏、冷汗、呕吐）
- 噪声影响情绪、睡眠和食欲
- 振动引起肌肉紧张，心率与呼吸加快
- 高温使水分、矿物质、维生素及含氮物质丢失增加
- 长期低剂量电离辐射影响蛋白质和维生素代谢

### 营养供给量

| 项目 | 推荐量 |
|------|--------|
| 能量 | 舰艇人员3300-3600kcal/d；潜艇船员3500-3700kcal/d |
| 蛋白质 | 供能比15%，约110g/d（动物性蛋白质占30%-50%） |
| 脂肪 | 供能比25%-30% |
| 碳水化合物 | 供能比55%-65% |

### 膳食特点

- 选营养价值高、体积小、耐储藏、易烹调、废弃量少的食物
- 食品原料经加工处理，不需洗涤，简单烹调即可
- 无新鲜果蔬时补充维生素制剂或强化食品
- 潜艇艇员额外补充[[维生素A]]和[[维生素D]]

### 晕船营养保障

1. 供给清淡菜肴，避免肥肉、鱼及腥味和膻味食物
2. 少量多餐，不宜过饱也不宜空腹；呕吐后适当进食
3. 维生素B6缓解呕吐症状，每天5-7.5mg
4. 适当选择生姜片、陈皮、山楂等开胃食品

### 维生素补充

- 维生素B6及[[维生素B1（硫胺素）]]有防晕船效果
- [[维生素B1（硫胺素）]]、[[维生素B2（核黄素）]]、[[维生素C]]和[[维生素B12（钴胺素）]]减少听力痉挛
- 大量维生素B6与琥珀酸盐或谷氨酸防护噪声损伤效果更好
- 长期航行维生素C水平大幅下降

## 四、接触化学毒物作业人员

### 接触铅作业人员

#### 健康损害

- 靶器官是骨骼，铅以磷酸铅形式在骨骼沉积
- 血钙低时骨铅进入血液，产生毒性
- 红细胞溶血、低血红蛋白性贫血、神经衰弱综合征、多发性周围神经炎、中毒性脑病、中毒性肝炎、肾衰竭
- 竞争性抑制锌、铁、钙吸收
- 导致维生素C缺乏和活性型维生素D3减少

#### 营养策略

| 营养素 | 推荐量/策略 |
|--------|-----------|
| 蛋白质 | 供能比15%，富含甲硫氨酸和胱氨酸的食物有排铅解毒作用 |
| 碳水化合物 | 供能比65%以上，葡萄糖醛酸参加铅解毒反应 |
| 脂肪 | 供能比低于20%（脂肪促进铅吸收） |
| 钙 | 800-1000mg/d（预防骨铅随骨钙溶出入血） |
| 维生素C | 150-200mg/d（排铅和维持巯基酶活性） |
| 铁、锌、铜 | 竞争性抑制铅吸收 |
| 维生素B12、叶酸 | 降低造血系统损害 |
| 维生素B1、B2、B6 | 预防神经系统铅损伤 |
| 膳食纤维 | 果胶和植酸降低肠道铅吸收 |

**排铅膳食策略**：
- 接触少量铅时：富含磷和硫的肉类与谷类食物，使骨铅形成可溶性磷酸氢铅入血排出
- 急性铅中毒时：富含钠、钾和钙的水果、蔬菜及乳类，使血铅沉积于骨骼
- 随后两类食物交替使用，促进铅逐步排出

### 接触苯作业人员

#### 健康损害

- 靶器官是神经系统和造血系统
- 急性中毒对中枢神经系统有麻醉作用
- 慢性中毒造成造血系统损害，严重者导致再生障碍性贫血或白血病
- 引起食欲降低、胃肠功能紊乱、蛋白质损失增加
- 对[[维生素C]]和铁影响突出

#### 营养策略

- 充足能量和碳水化合物
- 增加优质[[蛋白质]]（苯解毒转化所需谷胱甘肽与蛋白质营养状态和巯基酶活性有关）
- 限制膳食脂肪（脂肪促进苯吸收和蓄积）
- 增加维生素和矿物质，特别是[[维生素C]]
- 补充[[维生素B6（吡哆醇）]]、[[维生素B12（钴胺素）]]、叶酸、[[维生素A]]和[[维生素E]]
- 适当增加铁，预防苯中毒所致贫血
