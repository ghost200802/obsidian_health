# RNI — 推荐摄入量

**Recommended Nutrient Intake（RNI）** 是指某一特定性别、年龄及生理状态群体中对某一营养素需要量的估计值。RNI 是[[膳食营养素参考摄入量]]体系中最核心的指标之一，用于指导健康人群的营养素摄入目标。

## 定义

RNI 是在 [[平均需要量]]（平均需要量）的基础上，加上 2 个标准差（SD）后得到的摄入量水平。对于群体而言，RNI 可以满足某一特定性别、年龄及生理状态群体中 97%~98% 个体需要的摄入水平。

## 核心特征

- 健康个体摄入量达到 RNI 水平时，**几乎可以保证不发生该营养素缺乏**
- RNI 是个体营养素摄入量的**目标值**，而非上限
- RNI 与 [[可耐受最高摄入量]]（可耐受最高摄入量）共同构成营养素的安全摄入范围

## 常见营养素 RNI 示例（2023版）

| 营养素  | 人群       | RNI              |
| ---- | -------- | ---------------- |
| [[钙]]    | 成人       | 800 mg/d         |
| [[铁]]    | 成年男性     | 12 mg/d          |
| [[铁]]    | 18-49岁女性 | 18 mg/d          |
| [[铁]]    | 孕晚期      | +11 mg/d（在同龄基础上） |
| [[锌]]    | 成年男性     | 12.5 mg/d        |
| [[维生素A]] | 成年男性     | 770 μg RAE/d     |
| [[维生素C]] | 成人       | 100 mg/d         |
| [[维生素B9（叶酸）]]   | 孕妇       | 600 μg DFE/d     |
| [[蛋白质]]  | 成年男性     | 65 g/d           |
| [[蛋白质]]  | ≥65岁男性   | 72 g/d           |

> 关于孕期铁的详细推荐，参见 [[妊娠期铁推荐摄入量]]。更多数据参见 [[膳食营养素参考摄入量]]。

## 与其他 DRIs 指标的关系

- **RNI** > EAR（平均需要量）：RNI 在 EAR 基础上加 2 个 SD
- **RNI** < UL（[[可耐受最高摄入量]]）：超过 UL 有健康损害风险
- 当缺乏足够数据制定 RNI 时，使用 [[适宜摄入量]]（适宜摄入量）替代
