# 营养与氧化应激

氧化应激是指体内活性氧（ROS）和活性氮（RNS）等自由基的产生与抗氧化防御系统之间的失衡状态，导致生物大分子损伤。营养在维持氧化还原平衡中发挥关键作用。

## 自由基

### 定义与来源

自由基是含有一个或多个未配对电子的分子或离子，化学性质活泼：

- **内源性来源**：线粒体呼吸链电子漏（主要来源）、炎症反应中的呼吸爆发、细胞色素P450代谢
- **外源性来源**：紫外线辐射、电离辐射、环境污染、吸烟、某些药物

### 主要活性氧/氮种类

| 种类 | 特征 |
|------|------|
| 超氧阴离子（O₂⁻） | 线粒体产生的主要自由基 |
| 羟基自由基（OH·） | 氧化能力最强的自由基 |
| 过氧化氢（H₂O₂） | 非自由基的活性氧，可转化为OH· |
| 单线态氧（¹O₂） | 光化学反应产生 |
| 一氧化氮（NO） | 活性氮，参与信号传导和免疫防御 |
| 过氧亚硝酸盐（ONOO⁻） | NO与O₂⁻反应产物，强氧化剂 |

## 抗氧化营养素

### 酶性抗氧化系统

- **超氧化物歧化酶（SOD）**：催化O₂⁻转化为H₂O₂，需[[铜]]、[[锌]]、[[锰]]、[[硒]]作为辅因子
- **过氧化氢酶（CAT）**：催化H₂O₂分解为水和氧气
- **谷胱甘肽过氧化物酶（GPx）**：以[[硒]]为活性中心，还原脂质过氧化物

### 非酶性抗氧化营养素

| 抗氧化营养素 | 功能 | 来源 |
|-------------|------|------|
| [[维生素C]] | 水溶性抗氧化剂，再生维生素E，清除自由基 | 新鲜水果蔬菜 |
| [[维生素E]] | 脂溶性抗氧化剂，保护细胞膜磷脂 | 植物油、坚果 |
| β-胡萝卜素 | 淬灭单线态氧 | 胡萝卜、南瓜 |
| 硒（Se） | GPx活性中心，抗氧化酶必需成分 | 海鲜、巴西坚果 |
| 锌（Zn） | SOD辅因子，稳定细胞膜 | 海鲜、红肉 |
| 多酚类 | 清除自由基、螯合金属离子 | 茶、咖啡、蔬菜水果 |

## 氧化应激与慢性病

氧化应激是多种慢性病的重要病理机制：

- **心血管疾病**：LDL氧化是动脉粥样硬化的关键步骤
- **癌症**：DNA氧化损伤导致基因突变
- **[[糖尿病]]**：氧化应激损害胰岛β细胞、加重胰岛素抵抗
- **神经退行性疾病**：蛋白质氧化聚集（如阿尔茨海默病的β-淀粉样蛋白）
- **衰老**：自由基理论认为氧化损伤累积是衰老的原因之一
